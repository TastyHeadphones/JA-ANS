# 第33课（逐字人工精校）

## 1. 命令形・禁止形

| 题号 | 答案 | 解析 |
|---|---|---|
| 1-1 | 乗ります／乗れ | 礼貌形与命令形对应。 |
| 1-2 | 見ます／見るな | 禁止形：辞书形+な。 |
| 1-3 | 飲め／飲むな | 五段命令形+禁止形。 |
| 1-4 | 出ろ／出るな | 一段命令形「ろ」。 |
| 1-5 | 出します／出すな | 他动词禁止。 |
| 1-6 | 運動します／運動しろ | サ变命令形。 |
| 1-7 | 浴びろ／浴びるな | 一段动词。 |
| 1-8 | 行きます／行くな | 不规则禁止。 |
| 1-9 | 連れて来い／連れて来るな | 复合动词命令/禁止。 |
| 1-10 | 待ちます／待て | 五段命令形。 |

## 2. 助词填空

| 题号 | 答案 | 解析 |
|---|---|---|
| 2-1 | は／と | 主题+引用。 |
| 2-2 | に／と | 地点+引用内容。 |
| 2-3 | に／に | 方向+注意对象。 |
| 2-4 | は／を | 主题+宾语。 |
| 2-5 | に | 「会議に出席する」。 |

## 4. 表示语义（看板）

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 4-1 | 「無料」は〜という意味です。 | お金を払わなくてもいい | “免费”。 |
| 4-2 | 「禁煙」は〜という意味です。 | たばこを吸うな／たばこを吸ってはいけない | 禁止吸烟。 |
| 4-3 | 「使用中」は〜という意味です。 | 今使っている | 正在使用。 |
| 4-4 | 「立入禁止」は〜という意味です。 | 入るな／入ってはいけない | 禁止进入。 |

## 5. 传闻转述（〜と言っていました）

| 题号 | 答案 | 解析 |
|---|---|---|
| 5-1 | もうすぐ終わる | 句尾普通形后接「と言っていました」。 |
| 5-2 | 出席できない | 可能否定转述。 |
| 5-3 | あまりよくない | 形容词转述。 |
| 5-4 | 修理は無理だ | 名词句普通形「だ」。 |
| 5-5 | 駐車違反の罰金を15,000円払った | 过去事实转述。 |

## 6. 传话请求（〜と伝えていただけませんか）
