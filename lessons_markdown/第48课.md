# 第48课（逐字人工精校）

## 语法要点速览

| 语法 | 用法 |
|---|---|
| 使役形 | 让/使某人做某事（〜させる） |
| 使役授受 | 〜させていただく／〜させる |
| 使役+助词 | 人物多用「に/を」标记 |

## 1. 动词变形（普通形↔使役形）

| 题号 | 答案 | 解析 |
|---|---|---|
| 1-1 | 会わせます | 会います→会わせます。 |
| 1-2 | 働きます | 働かせます的原形。 |
| 1-3 | かけさせます | かけます→かけさせます。 |
| 1-4 | コピーさせます | サ变使役。 |
| 1-5 | 磨きます | 磨かせます的原形。 |
| 1-6 | 来させます | 来ます→来させます。 |
| 1-7 | 守ります | 守らせます的原形。 |
| 1-8 | 捨てさせます | 捨てます→捨てさせます。 |
| 1-9 | 飲ませます | 飲みます→飲ませます。 |
| 1-10 | 確かめます | 確かめさせます的原形。 |
| 1-11 | 下ろさせます | 下ろします→下ろさせます。 |

## 2. 助词与使役填空

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 2-1 | 部長は鈴木さん（　）アメリカ（　）（出張しました…）。 | を、へ、出張させました | 使役对象+目的地。 |
| 2-2 | わたしは毎日息子（　）皿（　）（洗います…）。 | に、を、洗わせます | 让儿子洗盘子。 |
| 2-3 | 子どものとき、父はわたし（　）剣道（　）（習いました…）。 | に、を、習わせました | 让某人学习。 |
| 2-4 | 電車の中では子ども（　）（立ちます…）います。 | を、立たせて | 使役て形连用。 |
| 2-5 | 子ども（　）歯（　）（磨きます…）ようにしています。 | に、を、磨かせる | 让孩子刷牙。 |
| 2-6 | 子ども（　）犬（　）世話を（します…）います。 | に、の、させて | 让孩子照顾狗。 |
| 2-7 | 子ども（　）お酒（　）（飲みます…）はいけません。 | に、を、飲ませて | 禁止让孩子喝酒。 |

## 3. 改写（使役）

| 题号 | 原句（核心） | 标准答案 | 解析 |
|---|---|---|---|
| 3-1 | わたしは日本へ留学しました。 | 父はわたしを日本へ留学させました。 | 由“父亲让我去留学”。 |
| 3-2 | 子どもは好きな仕事をやります。 | わたしは子どもに好きな仕事をやらせたいです。 | 使役+たい。 |
| 3-3 | 生徒は毎日日記を書いています。 | 先生は生徒に毎日日記を書かせています。 | 教师让学生做。 |
| 3-4 | 息子は毎朝自分の部屋を掃除します。 | わたしは息子に毎朝自分の部屋を掃除させます。 | 使役基本句。 |
| 3-5 | 鈴木さんは新しい製品について説明しました。 | 部長は鈴木さんに新しい製品について説明させました。 | 上级指示。 |
| 3-6 | 学生は授業中、英語を使いません。 | 先生は授業のとき、学生に絶対に英語を使わせません。 | 使役否定。 |
| 3-7 | 息子は1時間以上ゲームをしません。 | わたしは息子に1時間以上ゲームをさせません。 | “不让做”句型。 |
| 3-8 | 子どもは自由に外で遊びます。 | わたしは子どもを自由に外で遊ばせたいです。 | 使役+愿望。 |
| 3-9 | ハンス君は毎日家で日本語の本を読んでいます。 | ハンス君のお母さんは、ハンス君に毎日家で日本語の本を読ませています。 | 主语变换。 |

## 4. 会话填空（使役）

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 4-1 | もう遅いですから、息子（　）車で（送ります…）。 | に、送らせます | 让儿子开车送。 |
| 4-2 | あとで太郎（　）（電話します…）。 | に、電話させます | 让太郎回电。 |
| 4-3 | すぐ弟（　）車で（行きます…）から。 | を、行かせます | 让弟弟开车去。 |

## 5. 委托许可（〜させていただけませんか）

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 5-1 | 少し（考えます…）いただけませんか。 | 考えさせて | 让自己先想想。 |
| 5-2 | わたしもパーティーに（参加します…）いただけませんか。 | 参加させて | 请求被允许参加。 |
| 5-3 | 少し熱があるので、早く（帰ります…）いただけませんか。 | 帰らせて | 请求被允许早退。 |
| 5-4 | 1週間ほど（休みます…）いただけませんか。 | 休ませて | 请求被允许休假。 |
| 5-5 | 最近のデータを（コピーします…）いただけませんか。 | コピーさせて | 请求被允许复印。 |
