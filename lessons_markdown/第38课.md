# 第38课（逐字人工精校）

## 1. 助词填空

| 题号 | 答案 | 解析 |
|---|---|---|
| 1-1 | に | 参加对象。 |
| 1-2 | の／を | 名词修饰+宾语。 |
| 1-3 | を | 宾语标记。 |
| 1-4 | に／が | 时间/地点+主语。 |
| 1-5 | に／を | 位置+动作对象。 |

## 3. 自由作答题（答案示例）

| 题号 | 示例答案 | 解析 |
|---|---|---|
| 3-1 | 友達と旅行するのは（楽しいです）。 | 例句给出可成立搭配。 |
| 3-2 | たばこをたくさん吸うのは（体によくないです）。 | 名词化后接评价。 |
| 3-3 | 朝早く海岸を散歩するのは（気持ちがいいです）。 | 动作作主语。 |
| 3-4 | （外国語でスピーチをするのは）難しいです。 | 形容词评价。 |
| 3-5 | （ケータイを見ながら歩くのは）危ないです。 | 安全类评价。 |

| 4-5 | わたしは込んでいる電車に乗るのが嫌いです。 | 喜恶表达。 |
