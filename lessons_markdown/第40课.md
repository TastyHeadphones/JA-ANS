# 第40课（逐字人工精校）

## 1. 疑问从句（か／どうか）

| 题号 | 标准答案 | 解析 |
|---|---|---|
| 1-1 | ケーキが何個あるか、教えてください。 | 疑问词从句用「か」。 |
| 1-2 | シュミットさんはどんな料理が好きか、知りたいです。 | 从句后接愿望。 |
| 1-3 | ワットさんはどうして来なかったか、わかりますか。 | 过去原因问法。 |
| 1-4 | このカメラはいくらだったか、覚えていません。 | 过去事实不确定。 |
| 1-5 | この手紙は重さが25グラム以下かどうか、量ってください。 | 是非问句用「かどうか」。 |
| 1-6 | この答えは正しいかどうか、もう一度考えてください。 | 二选一判断。 |

## 3. 「〜てみる」

| 题号 | 标准答案 | 解析 |
|---|---|---|
| 3-1 | ちょうどいいかどうか、はいてみます。 | 试穿看看。 |
| 3-2 | サイズが合うかどうか、かぶってみます。 | 试戴看看。 |
| 3-3 | 足が痛くないかどうか、はいて歩いてみます。 | 试穿并行走确认。 |
| 3-4 | 来るかどうか、聞いてみてください。 | 先去问问。 |
| 3-5 | どんな店か、入ってみました。 | 试着进店体验。 |
