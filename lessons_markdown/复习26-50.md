# 复习26-50（逐字人工精校）

## 语法要点速览

| 模块 | 覆盖内容 |
|---|---|
| 助词综合 | `が/を/に/と/の/で/から` |
| 句型综合 | `〜そうだ`、`〜ようだ`、`〜はずだ`、`〜ところだ` |
| 敬语与授受 | 自谦、尊敬、`〜ていただく` |
| 条件表达 | `と／たら／ば／なら` 区别 |
| 句型变形 | `ます形`、普通形、名词化、使役/受身 |
