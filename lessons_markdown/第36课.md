# 第36课（逐字人工精校）

## 1. 助词与「ように」

| 题号 | 答案 | 解析 |
|---|---|---|
| 1-1 | に／×／を | 「ラッシュにあわないように」「7時ごろうちを出ます」。 |
| 1-2 | を／× | 「10時を過ぎたら」「できるだけ電話…」。 |
| 1-3 | に／× | 「仕事に慣れる」。 |
| 1-4 | ×／に | 「毎月、必ず母に…送る」。 |
