# 复习26-33（逐字人工精校）

## 语法要点速览

| 模块 | 内容 |
|---|---|
| 可能形与命令形 | 借りられる／借りろ／借りるな 等 |
| 助词综合 | `に/で/が/の/は/も` |
| 说明与请求 | `〜んです`、`〜ていただけませんか` |
| 意志形 | `〜よう`、`〜ましょう` 改写 |
| 常用副词 | `ずっと/たいてい/実は` 等 |

## 1. 可能/命令/禁止（1-8）

| 题号 | 答案 | 解析 |
|---|---|---|
| 1-1 | 借りられる、借りよう、借りるな | 三种形态对应：可能/意志/禁止。 |
| 1-2 | あきらめよう、あきらめろ | 意志形与命令形。 |
| 1-3 | 話せる、話せ、話すな | 可能/命令/禁止。 |
| 1-4 | 運転できる、運転しよう | 可能与意志。 |
| 1-5 | 来られる、来よう、来い | 不规则动词`来る`的变形。 |
| 1-6 | 置ける、置け、置くな | 一段/五段规则应用。 |
| 1-7 | 起きよう、起きるな | 意志与禁止。 |
| 1-8 | 立て、立つな | 命令与禁止。 |
