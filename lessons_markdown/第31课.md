|---|---|---|
| 1-1 | 続けよう | 五段动词意向形。 |
| 1-2 | 戻します | 按题目要求填入对应礼貌形。 |
| 1-3 | 休憩しよう | サ变动词意向形。 |
| 1-4 | 起きよう | 一段动词意向形。 |
| 1-5 | 置きます | 按表格要求填对应形。 |
| 1-6 | 持って来よう | 「来る」意向形「来よう」。 |
| 1-7 | 残ろう | 五段「る→ろう」。 |
| 1-8 | 降ります | 填对应礼貌形。 |
| 1-9 | 使おう | 五段「う→おう」。 |
| 1-10 | 見つけよう | 一段动词意向形。 |
| 1-11 | 並べます | 填对应礼貌形。 |
| 1-12 | 選ぼう | 五段「ぶ→ぼう」。 |
| 1-13 | 持とう | 五段「つ→とう」。 |

## 2. 把「〜ましょう」改成意向表达

| 题号 | 标准答案 | 解析 |
|---|---|---|
| 2-1 | 時間がないから、急ごう。 | 「急ぎましょう→急ごう」。 |
| 2-2 | おいしいワインをもらったから、いっしょに飲もう。 | 邀请语气改为简体意向。 |
| 2-3 | カリナさんがまだ来ていないから、もう少し待とう。 | 「待ちましょう→待とう」。 |
| 2-4 | 暑いから、エアコンをつけておこう。 | 先行准备「〜ておこう」。 |
| 2-5 | あしたは休みだから、東京スカイツリーに行かない？ 行こう。 | 会话里自然接续意向。 |
| 2-6 | あの喫茶店に入らない？ そうしよう。 | 回应提议用「〜しよう」。 |

## 3. 「〜と思っています」

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 3-1 | 会社をやめて、もう一度大学で＿＿＿と思っています。 | 勉強しよう | 意向形+と思っています。 |
| 3-2 | 子どもを動物園へ＿＿＿と思っています。 | 連れて行こう | 复合动词意向形。 |
| 3-3 | 犬を＿＿＿と思っています。 | 飼おう | 五段动词意向形。 |
| 3-4 | 自分でケーキを＿＿＿と思っています。 | 作ろう | 五段动词意向形。 |
| 3-5 | 先週見たマンションを＿＿＿と思っています。 | 借りよう | 一段动词意向形。 |

## 4. 「つもりです」

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 4-1 | 今の研究を続けますか。 | 続ける | 「V辞书形+つもりです」。 |
| 4-2 | 大学院の試験を受けますか。 | 受けない | 否定计划用「ない形+つもり」。 |
| 4-3 | 大阪まで新幹線で行きますか。 | 行く | 行动计划。 |
| 4-4 | 夏休みにアルバイトをしますか。 | しない | 「しないつもりです」。 |

## 5. 「予定です」

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 5-1 | 会議は何時に終わりますか。（4時に終わります） | 4時に終わる | 普通形修饰「予定」。 |
| 5-2 | 結婚式は何時までですか。（2時までです） | 2時までの | 名词化时常用「の予定」。 |
| 5-3 | 課長の午後の予定（支店へ行きます） | 支店へ行く | 动词普通形。 |
| 5-4 | 夏休みは何をしますか。（1週間北海道を旅行します） | 1週間北海道を旅行する | 动作计划。 |
| 5-5 | スキー旅行に行く人は何人ぐらいですか。（50人ぐらいです） | 50人ぐらいの | 数量+の予定。 |

## 6. 计划与未完成

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 6-1 | 松本さんはもう来ましたか。 | 来ていません／来る | 现状否定+将来计划。 |
| 6-2 | 結婚について両親に話しましたか。 | 話していません／話す | 「まだ〜ていません」+「〜つもり」。 |
| 6-3 | プレゼントはもう決めましたか。 | 決めていません／決めよう | 未决定+意向。 |
