## 第1課 まなぶ ―「なぞなぞ」

### 本课核心目标

通过孩子在小学学习英语、回家讲述课堂上的「なぞなぞ」（谜语）这件事，理解中级阶段如何描述「状态变化」「想做某事的愿望」「不希望发生的担心」「在没做某事的状态中」。

### 文法详细总结

#### 1. `〜ようになる / 〜なくなる`

- 用法：表示能力或习惯发生变化，「变得能……／变得不再……」。
- 接续：`V辞書形 + ようになる`、`Vない形（去い）+ くなる` / `Vないように + なる`。
- 例句：<ruby>友<rt>とも</rt></ruby>だちに<ruby>教え<rt>おしえ</rt></ruby>てもらって、<ruby>自転車<rt>じてんしゃ</rt></ruby>にのれるようになりました。
- 翻译：朋友教了我之后，我学会骑自行车了。

#### 2. `〜ように + する／しない／なる`

- 用法：努力（不）让某种状态实现／发生。
- 接续：`V辞書形 / Vない形 + ようにする`。
- 例句：<ruby>父<rt>ちち</rt></ruby>は<ruby>仕事<rt>しごと</rt></ruby>をやめてから、<ruby>運動<rt>うんどう</rt></ruby>するようになりました。
- 翻译：父亲退休后开始变得有锻炼的习惯。

#### 3. `〜たがる / 〜たがっている`

- 用法：客观叙述第三方"想做某事"的样子。
- 接续：`Vます形（去ます）+ たがる`。
- 例句：ランさんはご<ruby>両親<rt>りょうしん</rt></ruby>に<ruby>会い<rt>あい</rt></ruby>たがっているんだ。
- 翻译：兰小姐想见父母。

#### 4. `〜てはいけない / 〜といけない`

- 用法：「（如果……）就糟糕了／不行」，常用于说明「为了避免某种坏结果而……」的语境。
- 接续：`Vて + は + いけない`、`V辞書形 + と + いけない`。
- 例句：かぜを<ruby>引い<rt>ひい</rt></ruby>てはいけないから、あたたかい<ruby>服<rt>ふく</rt></ruby>を<ruby>着た<rt>きた</rt></ruby>ほうがいい。
- 翻译：为了不感冒，最好穿暖和点儿。

#### 5. `〜ないで／ずに + いる`

- 用法：处于"没做某事"的持续状态。
- 接续：`Vない形 + ないで／ずに + いる`。
- 例句：いそがしくて、<ruby>家族<rt>かぞく</rt></ruby>とゆっくり<ruby>話す<rt>はなす</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>持て<rt>もて</rt></ruby>ずにいます。
- 翻译：因为太忙，没能和家人好好说话。

### 文法详细总结

#### 1. `〜のに、〜てしまった`

- 用法：「明明……却（不情愿地／意外地）发生了Y」，常带轻微遗憾或意外感。
- 接续：`普通形 + のに、〜てしまった`。
- 例句：いつもはいっしょにかたづけるのに、<ruby>弟<rt>おとうと</rt></ruby>は<ruby>私<rt>わたし</rt></ruby>にかたづけさせてしまった。
- 翻译：明明平时是一起收拾的，弟弟却让我一个人收拾了。

#### 2. `〜うちに`

- 用法：在某种状态持续期间做（或发生了）某事。
- 接续：`V辞書形 + うちに`、`Vている + うちに`、`い形/な形/N + うちに`。
- 例句：<ruby>朝<rt>あさ</rt></ruby>のうちに、<ruby>宿題<rt>しゅくだい</rt></ruby>を<ruby>終わ<rt>おわ</rt></ruby>らせてしまおう。
- 翻译：趁早上把作业做完吧。

#### 3. `〜うちは`

- 用法：「在……期间是……（但变化后就不同了）」，常与对比的后句呼应。
- 接续：`V辞書形 + うちは`、`Vない形 + うちは`。
- 例句：<ruby>漢字<rt>かんじ</rt></ruby>が<ruby>読め<rt>よめ</rt></ruby>ないうちは<ruby>大変<rt>たいへん</rt></ruby>でしたが、<ruby>今<rt>いま</rt></ruby>は<ruby>好き<rt>すき</rt></ruby>な<ruby>本<rt>ほん</rt></ruby>が<ruby>読め<rt>よめ</rt></ruby>るようになりました。
- 翻译：不会读汉字时很辛苦，现在能读自己喜欢的书了。

#### 4. `〜(さ)せてあげる／やる`

- 用法：让／允许某人做某事（带恩惠语气）。
- 接续：`Vさせて + あげる／やる`。
- 例句：<ruby>将来<rt>しょうらい</rt></ruby>のために、<ruby>子供<rt>こども</rt></ruby>たちにいろいろなことを<ruby>習わ<rt>ならわ</rt></ruby>せてやりたい。
- 翻译：为了孩子的未来，想让他们学各种各样的东西。

### 文法详细总结

#### 1. `〜ことにする`

- 用法：说话人主动做出决定。
- 接续：`V辞書形 / Vない形 + ことにする`。
- 例句：あしたからはもっと<ruby>早く<rt>はやく</rt></ruby>おきることにします。
- 翻译：从明天起决定起更早。

#### 2. `〜ことにしている`

- 用法：持续保持的个人习惯、个人规则。
- 接续：`V辞書形 / Vない形 + ことにしている`。
- 例句：<ruby>寝る<rt>ねる</rt></ruby><ruby>前<rt>まえ</rt></ruby>にスマホを<ruby>見な<rt>みな</rt></ruby>いことにしている。
- 翻译：我决定（一直坚持）睡前不看手机。

#### 3. `〜(さ)せる`（再次复习——本课通过对比"过去 vs 现在"使用）

- 用法：使役，「让／使／命令／允许某人做某事」。
- 接续：`Vさせる`形。
- 例句：「<ruby>食事<rt>しょくじ</rt></ruby>を<ruby>楽しみ<rt>たのしみ</rt></ruby>にする<ruby>子供<rt>こども</rt></ruby>たちに<ruby>何を<rt>なにを</rt></ruby><ruby>食べ<rt>たべ</rt></ruby>させてやろうか」と<ruby>母親<rt>ははおや</rt></ruby>は<ruby>考え<rt>かんがえ</rt></ruby>た。
- 翻译：母亲想，"该给期待吃饭的孩子们吃些什么呢"。

#### 4. `〜としか〜ない`

- 用法：「只能说……，只能想成……」，表示其他可能性被排除。
- 接续：`普通形 + としか + V否定形`。
- 例句：<ruby>試験<rt>しけん</rt></ruby>は<ruby>受け<rt>うけ</rt></ruby>ましたが、<ruby>今<rt>いま</rt></ruby>はいっしょうけんめいやったとしか<ruby>言え<rt>いえ</rt></ruby>ません。
- 翻译：我考完了，现在只能说尽力了。

- 用法：「认定为……／被认定为……」。

- 用法：「随……不同而异」。
- 接续：`N + によって`。

## 综合复习清单（第1〜20课）

### 高频语法主题分组

- **状态变化**：〜ようになる／なくなる、〜につれて、〜ば〜ほど、〜とともに
- **意志与决定**：〜ことにする／ことにしている、〜(よ)うと、〜ことだ
- **使役・被动**：〜(さ)せてあげる／やる、〜(さ)せられる
- **不得不・必须**：〜ざるを<ruby>得な<rt>えな</rt></ruby>い、〜わけにはいかない、〜ないことには〜ない
- **请求／希望／反问**：〜ないでほしい、〜たがる、〜ないものか／ないものだろうか
- **常情／道理**：〜ものだ、〜ものではない、〜ことだ、〜べきだ／べきではない
- **推论**：〜はずだ、〜はずがない、〜わけだ、〜に<ruby>違い<rt>ちがい</rt></ruby>ない、〜のことだから、〜からといって〜わけではない
- **比较／程度**：〜ほど〜はない、〜ほど〜、〜まで、〜どころか、かえって〜、〜<ruby>割<rt>わり</rt></ruby>に（は）
- **比喻／引用／举例**：〜とか〜とか、〜として、〜なんて、〜をはじめ、〜に<ruby>限ら<rt>かぎら</rt></ruby>ず
- **意外発見・転換**：〜たところ、〜のに〜てしまった、〜といっても、〜。その<ruby>一方<rt>いっぽう</rt></ruby>（で）、〜
- **不可能・無法・否定推断**：〜ようがない、〜とは<ruby>思え<rt>おもえ</rt></ruby>ない、〜(よ)うともしない、〜としか〜ない、〜わけがない
- **怀旧／习惯／状态持续**：〜たものだ、〜うちに、〜うちは、〜まま、〜たきり、〜ようでは
- **强烈感觉**：〜てたまらない、〜たばかり、〜て／でならない、〜だけに
- **手段・原因・契机**：〜によって（手段／原因）、〜ことから、〜きっかけで／をきっかけに、〜たばかりに
- **立场・前提・基础**：〜にとって(は)、〜<ruby>上で<rt>うえで</rt></ruby>、〜た<ruby>上で<rt>うえで</rt></ruby>、〜にあたり／にあたって、〜次第（で）、〜はともかく
- **错觉・成见**：〜たつもり、〜<ruby>思い込む<rt>おもいこむ</rt></ruby>
- **极端／强调／专长**：〜さえ、〜さえ〜ば、〜にしろ〜にしろ、〜にもかかわらず、〜にかけては、さすが（に）
- **理由说明**：〜ものだから／もので、〜というのも、〜<ruby>通り<rt>とおり</rt></ruby>
- **让步**：〜としても／たとえ〜としても
- **时点／时机**：〜ところだ（要做／正在做／刚做完）、〜とともに
- **认定／客观叙述**：〜とする／〜とされる、〜なりに／なりの

### 整体写作・口语任务

1. 选取一项"自己的留学／异文化体验"，用 `〜たばかり`、`〜てたまらない`、`〜なんて`、`〜(よ)うともしない` 写一段约 200 字短文。
2. 选取一句日本谚语或一句自己国家的谚语，用 `〜ほど〜はない`、`〜とは思えない` 解释其含义。
3. 用 `〜にとって`、`〜にとっては`、`〜ようがない`、`〜たものだ` 写一段对童年色彩记忆／印象的反思。
4. 用 `〜はずだ`、`〜はずがない`、`〜ば〜ほど`、`〜として` 讨论"个人信息保护"的具体注意事项。
5. 用 `〜どころか、かえって〜`、`〜につれて`、`〜わけだ`、`〜からといって、〜わけではない` 描述一段"原以为如此，结果意外不同"的经历。
6. 用 `〜ざるを得ない`、`〜さえ`、`〜にしろ〜にしろ`、`〜た上で` 讨论"タテ社会 vs ヨコ社会"中你自己的立场。
7. 用 `〜たばかりに`、`〜て／でならない`、`〜としても`、`〜ないことには` 写一段关于故乡变化的反思（ふるさと<ruby>話題<rt>わだい</rt></ruby>）。
8. 用 `〜はともかく`、`〜さえ〜ば`、`〜ことから`、`〜ないものか` 谈一谈"什么是真正的丰饶（<ruby>豊か<rt>ゆたか</rt></ruby>さ）"。
9. 用 `〜割に（は）`、`〜にあたり／にあたって`、`〜に限らず`、`〜にかけては` 描述一种"手艺／专长（ものづくり・<ruby>職<rt>しょく</rt></ruby><ruby>人<rt>ひと</rt></ruby>のわざ）"的传承。
