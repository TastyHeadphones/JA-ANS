# 第10课 なれる ―「腕時計」

> 对应书内第10课（PDF 约第67–72页）。
> 这一课的主轴是「<ruby>時間<rt>じかん</rt></ruby>と<ruby>生活<rt>せいかつ</rt></ruby>、そして<ruby>喪失<rt>そうしつ</rt></ruby>からの<ruby>変化<rt>へんか</rt></ruby>」，重点学习"无论……都""随着……""非但没有……反而""难怪……""并不是说……就……"等表达。

## 本课主旨

作者的丈夫在一次海外出差途中，因原因不明的空难去世，至今快三年了。生前不爱说话的丈夫常说"退休后要搬到乡下，过与钟表无缘的生活"，这句话至今萦绕耳边。讨厌被时间追着跑的丈夫，临行前说"在机场买块新的"，把自己的手表留在了家——这成了那场什么都没能归来的事故里唯一留下的遗物。为了忘记事故，丈夫去世后的一段时间，作者出门、上班都不带表。起初觉得不便，但随着时间推移，非但不觉得不便，反而心里多了从容：不再边走边一次次看表担心"来得及吗"，也不再等车时焦躁。奇怪的是，在家也几乎不看钟了。但不看钟并不等于不在意时间——作者凭大致的感觉（<ruby>見当<rt>けんとう</rt></ruby>をつけて）生活，也不会差太多。如今她有时会对着丈夫的手表轻声说："我没搬去乡下，但已经习惯了没有手表的生活，不再被时间牵着走，过上了你想要的'与钟表无缘的生活'。"

## 导入问题

课前会先围绕下面三个问题展开：

- 你一天大概看几次时钟？
- 想知道时间的时候，除了钟表，你还会用什么东西？
- 你有没有觉得"时间不够、很忙"的时候？

## 文法要点

### 1. `〜（よ）うと（する／して）`（为了……／试图……）

- 用法：表示为了实现某个目的、抱着某种意图而去做后项的事，相当于 `〜（よ）うとして`。本课课文「<ruby>事故<rt>じこ</rt></ruby>を<ruby>忘れ<rt>わすれ</rt></ruby>ようと、<ruby>時計<rt>とけい</rt></ruby>を<ruby>持た<rt>もた</rt></ruby>なかった」即此用法。
- 接续：
  - `V意向形 + と + （後項：そのためにした行動）`
- 例句：
  - <ruby>少し<rt>すこし</rt></ruby>でも<ruby>運動<rt>うんどう</rt></ruby>をしようと、毎日30<ruby>分<rt>ふん</rt></ruby><ruby>歩く<rt>あるく</rt></ruby>ことにした。
  - <ruby>娘<rt>むすめ</rt></ruby>に<ruby>音楽<rt>おんがく</rt></ruby>の<ruby>勉強<rt>べんきょう</rt></ruby>をさせようと、<ruby>両親<rt>りょうしん</rt></ruby>はバイオリンを<ruby>習わ<rt>ならわ</rt></ruby>せた。
  - <ruby>事故<rt>じこ</rt></ruby>を<ruby>忘れ<rt>わすれ</rt></ruby>ようと、<ruby>仕事<rt>しごと</rt></ruby>に<ruby>集中<rt>しゅうちゅう</rt></ruby>した。
- 说明：
  - `〜（よ）うと` ＝ `〜（よ）うとして`，表示"想要……，（为此）做了……"，前项是意图/目的，后项是为此采取的行动。
  - 区别于 `〜（よ）うと（も）`（无论……都，让步）：本课是"为了/试图"，不是"无论"。
  - 也区别于 `〜ようとしない`（连试都不试）。

### 2. `〜につれて`

- 用法：表示随着某事物的变化，另一事物也相应地变化，"随着……"。
- 接续：
  - `V辞書形 + につれて`
  - `N + につれて`
- 例句：
  - <ruby>時間<rt>じかん</rt></ruby>が<ruby>経つ<rt>たつ</rt></ruby>につれて、<ruby>不便<rt>ふべん</rt></ruby>さよりも<ruby>心<rt>こころ</rt></ruby>の<ruby>余裕<rt>よゆう</rt></ruby>を<ruby>感じ<rt>かんじ</rt></ruby>るようになった。
  - <ruby>年<rt>ねん</rt></ruby>を<ruby>取る<rt>とる</rt></ruby>につれて、<ruby>時間<rt>じかん</rt></ruby>の<ruby>感じ方<rt>かんじかた</rt></ruby>が<ruby>変わ<rt>かわ</rt></ruby>ってきた。
  - <ruby>経験<rt>けいけん</rt></ruby>が<ruby>増え<rt>ふえ</rt></ruby>るにつれて、<ruby>考え方<rt>かんがえかた</rt></ruby>も<ruby>深く<rt>ふかく</rt></ruby>なる。
- 说明：
  - `〜につれて` 表示两件事同步变化，前件是基准变化，后件是随之发生的变化。
  - 与 `〜にしたがって` 意思相近，但 `〜にしたがって` 也有"按照……做"的意思，`〜につれて` 只用于变化。

### 3. `〜どころか、かえって〜`

- 用法：表示"非但不……，反而……"，结果与预期完全相反。
- 接续：
  - `V / い形 / な形 / N + どころか`
- 例句：
  - <ruby>薬<rt>くすり</rt></ruby>を<ruby>飲ん<rt>のん</rt></ruby>だどころか、かえって<ruby>具合<rt>ぐあい</rt></ruby>が<ruby>悪く<rt>わるく</rt></ruby>なった。
  - <ruby>慣れ<rt>なれ</rt></ruby>ようとしたどころか、かえってストレスが<ruby>増え<rt>ふえ</rt></ruby>た。
  - <ruby>時間<rt>じかん</rt></ruby>を<ruby>節約<rt>せつやく</rt></ruby>できるどころか、かえって<ruby>時間<rt>じかん</rt></ruby>がかかった。
- 说明：
  - `どころか` 表示"非但如此……连……都不是"，后续的 `かえって` 强调反向结果。
  - 注意：`〜どころか` 也可单独用于"不仅如此，甚至……"，需根据上下文判断。

### 4. `〜わけだ`

- 用法：表示"难怪……""这就说明……"，对某事给出合理解释或当然结论。
- 接续：
  - `普通形 + わけだ`
  - `な形容詞（な）+ わけだ`
  - `N（の）+ わけだ`
- 例句：
  - <ruby>毎日<rt>まいにち</rt></ruby><ruby>練習<rt>れんしゅう</rt></ruby>したのだから、<ruby>上手<rt>じょうず</rt></ruby>になったわけだ。
  - 5<ruby>年<rt>ねん</rt></ruby>も<ruby>日本<rt>にほん</rt></ruby>にいれば、<ruby>日本<rt>にほん</rt></ruby><ruby>語<rt>ご</rt></ruby>が<ruby>上手<rt>じょうず</rt></ruby>なわけだ。
  - いつも<ruby>早起き<rt>はやおき</rt></ruby>しているから、<ruby>健康<rt>けんこう</rt></ruby>なわけだ。
- 说明：
  - 说话人基于已知信息推导出理所当然的结论，有"怪不得……"的语感。
  - 与 `〜はずだ`（预期推断）区分：`わけだ` 更强调"这就解释了"的理解感。

### 5. `〜からといって〜わけではない`

- 用法：表示"虽然……但并不因此……"，否定从某条件直接得出的必然结论。
- 接续：
  - `普通形 + からといって + V / い形 / な形 + わけではない`
- 例句：
  - <ruby>日本<rt>にほん</rt></ruby>に<ruby>慣れ<rt>なれ</rt></ruby>たからといって、<ruby>自分<rt>じぶん</rt></ruby>の<ruby>文化<rt>ぶんか</rt></ruby>を<ruby>忘れ<rt>わすれ</rt></ruby>るわけではない。
  - <ruby>遅刻<rt>ちこく</rt></ruby>したからといって、<ruby>悪い<rt>わるい</rt></ruby><ruby>人<rt>ひと</rt></ruby>というわけではない。
  - お<ruby>金<rt>きん</rt></ruby>があるからといって、<ruby>幸せ<rt>しあわせ</rt></ruby>なわけではない。
- 说明：
  - 用于提醒"不能因为A就断定B"，有打破简单因果逻辑的效果。
  - 常与 `だからといって`（即便如此）搭配使用。

## 课文梳理

1. 作者的丈夫在海外出差途中，因原因不明的空难去世，至今快三年了。
2. 不爱说话的丈夫生前常说"退休后搬到乡下，过与钟表无缘的生活"，这句话至今萦绕在作者耳边。
3. 讨厌被时间追赶的丈夫临行前说"在机场买块新的"，把自己的手表留在了家；这成了那场什么都没能归来的事故里，唯一留下的遗物。
4. 为了忘记事故，丈夫去世后的一段时间，作者出门、上班都不带表。
5. 起初觉得不便，但随着时间推移，非但不觉得不便，反而心里多了从容——不再边走边反复看表担心"来得及吗"，等车时也不再焦躁。
6. 奇怪的是，在家也几乎不看钟了。但不看钟并不等于不在意时间——作者凭大致的感觉（見当）生活，也不会差太多。
7. 以前每次看表都被"还剩几分钟"追赶着；如今变成了从容着手做事的生活。
8. 她想：丈夫出发前买了怎样的表呢？或许正是在机内一次次看表时，事故发生了。
9. 如今她有时会对着丈夫的手表轻声说："我没搬去乡下，但已经习惯了没有手表的生活，不再被时间牵着走，过上了你想要的'与钟表无缘的生活'。"

## 练习题

### 一、词汇练习

1. `主人は、時間に（　　　　　）生活をきらっていた。`
   - 答案：`追われる`
   - 解析：`時間に追われる` = 被时间追赶。

2. `事故のあと、その腕時計が（　　　　　）残った物だった。`
   - 答案：`唯一`
   - 解析：`唯一` = 唯一。

3. `時計を持たなくなって、気持ちに（　　　　　）が持てるようになった。`
   - 答案：`余裕`
   - 解析：`余裕` = 从容、余地。

4. `今は、だいたい何時ごろか（　　　　　）をつけて生活している。`
   - 答案：`見当`
   - 解析：`見当をつける` = 估摸、大致判断。

### 二、语法练习

5. `事故を忘れ（　　　　　）と、時計を持たなかった。`
   - 答案：`よう`
   - 解析：`〜（よ）うと` = 想要……（意向形）；`忘れる` → `忘れよう`。

6. `時間が経つ（　　　　　）、気持ちが楽になった。`
   - 答案：`につれて`
   - 解析：`〜につれて` = 随着……。

7. `不便に思う（　　　　　）、かえって余裕が持てた。`
   - 答案：`どころか`
   - 解析：`〜どころか、かえって〜` = 非但……反而……。

8. `時計を見ない（　　　　　）、時間を意識しない（　　　　　）ではない。`
   - 答案：`からといって`、`わけ`
   - 解析：`〜からといって〜わけではない` = 虽然……但并不因此……。

### 三、短答题

9. `筆者は、時計を持たない生活がどう変わったと言っていますか。`
   - 参考答案：<ruby>不便<rt>ふべん</rt></ruby>どころか、かえって<ruby>気持ち<rt>きもち</rt></ruby>に<ruby>余裕<rt>よゆう</rt></ruby>が<ruby>持て<rt>もて</rt></ruby>、<ruby>時間<rt>じかん</rt></ruby>に<ruby>追わ<rt>おわ</rt></ruby>れずに<ruby>落ち着い<rt>おちつい</rt></ruby>て<ruby>物事<rt>ものごと</rt></ruby>に<ruby>取り<rt>とり</rt></ruby>かかれるようになった。（反而更从容）

10. `用 〜につれて 写一句话，描述某种随时间的变化。`
    - 参考答案：`時間が経つにつれて、悲しみが少しずつやわらいできた。`

## 本课复习要点

- 能用 `〜（よ）うと` 表达"为了……／试图……"（如 `忘れようと`）。
- 能用 `〜につれて` 表达"随着……也……"（两件事同步变化）。
- 能用 `〜どころか、かえって〜` 表达"非但没有……反而……"（结果与预期相反）。
- 能用 `〜わけだ` 表达"难怪……""这就说明……"（合理结论/理解）。
- 能用 `〜からといって〜わけではない` 表达"虽然……但并不因此……"（打破简单因果推断）。

### 課文：腕時計

**①** <ruby>海外出張<rt>かいがいしゅっちょう</rt></ruby><ruby>中に<rt>なかに</rt></ruby>、<ruby>原因不明<rt>げんいんふめい</rt></ruby>の<ruby>飛行機事故<rt>ひこうきじこ</rt></ruby>で<ruby>主人<rt>しゅじん</rt></ruby>が<ruby>亡く<rt>なく</rt></ruby>なり、もうすぐ3<ruby>年<rt>ねん</rt></ruby>になる。

- **译文**：丈夫在海外出差途中，因一场原因不明的空难去世，至今快三年了。
- **语法**：`〜中に`（在……期间）；`もうすぐ〜になる`（快要到……了）
- **单词**：`海外出張`（海外出差）；`原因不明`（原因不明）；`亡くなる`（去世）

---

**②** <ruby>口数<rt>くちかず</rt></ruby>のあまり<ruby>多く<rt>おおく</rt></ruby>なかった<ruby>主人<rt>しゅじん</rt></ruby>が、「<ruby>定年<rt>ていねん</rt></ruby>になったらいなかに<ruby>引っ込ん<rt>ひっこん</rt></ruby>で、<ruby>時計<rt>とけい</rt></ruby>とは<ruby>縁<rt>えん</rt></ruby>のない<ruby>生活<rt>せいかつ</rt></ruby>をする」とくり<ruby>返し<rt>かえし</rt></ruby><ruby>言って<rt>いって</rt></ruby>いたのが、<ruby>今<rt>いま</rt></ruby>も<ruby>耳<rt>みみ</rt></ruby>に<ruby>残っ<rt>のこっ</rt></ruby>ている。

- **译文**：不爱说话的丈夫生前反复说"退休后就搬到乡下，过与钟表无缘的生活"，这话至今还萦绕在我耳边。
- **语法**：`〜たら`（如果/到……时候）；`〜とは縁のない`（与……无缘的）；`〜くり返し〜ていた`（反复地……）
