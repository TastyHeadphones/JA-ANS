# 第20课 おぼえる ―「ものづくり」

> 对应书内第20课（PDF 约第119–124页）。
> 这一课的主轴是「<ruby>職人<rt>しょくにん</rt></ruby>のわざと<ruby>体<rt>からだ</rt></ruby>で<ruby>覚え<rt>おぼえ</rt></ruby>ること」，重点学习"相对于……（出乎意料地）""在……之际""不仅……""说到……方面"等表达。

## 本课主旨

国外有过这样一场风波——出于卫生考虑，规定寿司店必须戴薄手套，捏寿司的师傅们却抗议"戴手套就捏不出寿司的口感"。最后特别许可：可以光手作业。作者也是干了近40年的寿司师傅。年轻人来要求工作，他给两个月的"见習い（学徒）"期，只让磨菜刀。一段时间什么都不说，等熟悉后让对方摸他磨好的刀刃——能用手指辨认刀刃的细微落差的人，将来一定行。料理的颜色搭配、汤底火候、香味调和——一切都要"用眼睛、用手指、用舌头"去记，不可能用语言传达。不仅料理如此，染布、荞麦面、木工——所谓"摸摸看""舔舔看"，是只能用身体记忆的"<ruby>職人<rt>しょくにん</rt></ruby>のわざ"。

## 导入问题

课前会先围绕下面三个问题展开：

- 你有什么可以向别人自夸的拿手本领？
- 那是跟谁、怎么学的？
- 现在有什么想学的本事吗？

## 文法要点

### 1. `〜割に（は）`

- 用法：「相对于……，出乎意料地……」。前后两件事在程度上不成比例。
- 接续：`普通形 / な形〜な / N + の + 割に（は）`
- 例句：
  - <ruby>姉<rt>あね</rt></ruby>は<ruby>年齢<rt>ねんれい</rt></ruby>の<ruby>割<rt>わり</rt></ruby>に（は）、<ruby>若く<rt>わかく</rt></ruby><ruby>見え<rt>みえ</rt></ruby>ます。
  - このペンは<ruby>値段<rt>ねだん</rt></ruby>が<ruby>高い<rt>たかい</rt></ruby><ruby>割<rt>わり</rt></ruby>に（は）、うまく<ruby>書け<rt>かけ</rt></ruby>ません。
  - <ruby>弟<rt>おとうと</rt></ruby>はたくさん<ruby>食べ<rt>たべ</rt></ruby>る<ruby>割<rt>わり</rt></ruby>に（は）、ふとりません。
  - <ruby>騒ぎ<rt>さわぎ</rt></ruby>の<ruby>割<rt>わり</rt></ruby>にはすんなり<ruby>許可<rt>きょか</rt></ruby>が<ruby>出た<rt>でた</rt></ruby>ようだ。
- 说明：
  - "前句条件给一个预期，后句结果出乎预期"。
  - 与 `〜にしては` 接近，但 `〜割に（は）` 更口语。

### 2. `〜にあたり／〜にあたって（は）`

- 用法：「在……之际／面对……（重要场合）时」。
- 接续：`V辞書形 / N + にあたり／にあたって（は）`
- 例句：

- 说明：
  - 后续多接"<ruby>誰<rt>だれ</rt></ruby>にも<ruby>負け<rt>まけ</rt></ruby>ない／<ruby>引けを取らない<rt>ひけをとらない</rt></ruby>／<ruby>一番<rt>いちばん</rt></ruby>だ"。
  - 用于自夸或评价某人某地的强项。

## 课文梳理

1. 国外曾因为卫生考虑，规定寿司店必须使用料理用手套——经营寿司店的人们抗议："那样捏不出寿司了"，于是这事闹大。
2. 寿司独特的口感——既不太硬也不太软——是非素手不可的，他们如此辩护。结果出乎意料地顺利获得了特许。"花时间用身体记忆"这件事的重要性能被承认，是值得高兴的事。
3. 作者干寿司店将近40年。年轻人来求工作，他用"两个月作为见習い"的条件雇下，只让对方磨刀。
4. 一段时间什么都不说，等对方熟悉后让他摸自己磨好的刀刃。"哪里不太一样"——能确切感觉到的、用手指搓搓便能区分的人，将来一定能磨刀，也一定能成长。
5. 见习生中，可能有人觉得"是不是固执的师傅在为难我？"。但其实许多事言语描述不出来，唯有上手做才行。料理的色彩搭配、汤底加减、香味、口感——每一项都要"用眼、用手、用舌"去记，才能成为可以自信地端出料理的"一人前"。
6. 不仅料理，染物的色调、荞麦面、木匠的刨工……每行都有"在这门手艺上不输任何人"的工匠。他们不是不想教诀窍——是无法用语言传达，只能让对方"摸摸看""舔舔看"。这些"<ruby>言葉<rt>ことば</rt></ruby>にできない、<ruby>体<rt>からだ</rt></ruby>で<ruby>覚え<rt>おぼえ</rt></ruby>るよりほか<ruby>身に<rt>みに</rt></ruby>つかない<ruby>職人<rt>しょくにん</rt></ruby>のわざ"，必然存在于世界各地。

## 练习题

### 一、词汇练习

1. `40年近くまな板の前に立っている、すし屋を（　　　　　　　）人だ。`
   - 答案：`営む`
   - 解析：`営む` = 经营。

2. `毎日（　　　　　　　）を研ぐ。`
   - 答案：`包丁`
   - 解析：`包丁を研ぐ` = 磨刀。

3. `この技術にかけては、誰にも（　　　　　　　）を取らない。`
   - 答案：`引け`
   - 解析：`引けを取らない` = 不输。

### 二、语法练习

4. `姉は年齢の（　　　　　　　）、若く見えます。`
   - 答案：`割に（は）`
   - 解析：`〜割に（は）` = 相对于……出乎意料。

5. `定年退職する（　　　　　　　）、ごあいさつを申し上げます。`
   - 答案：`にあたり`
   - 解析：`〜にあたり` = 在……之际。

6. `女性（　　　　　　　）、男性も化粧品を使うようになった。`
   - 答案：`に限らず`
   - 解析：`〜に限らず` = 不仅……。

7. `料理（　　　　　　　）は、彼に勝てる人はいない。`
   - 答案：`にかけては`
   - 解析：`〜にかけては` = 在……方面。

### 三、短答题

8. `本文中、「触ってみろ」「なめてみろ」と伝えられているのは何か？`
   - 参考答案：是那些言语无法描述、只能用身体（手指、舌头、眼睛）去记的"<ruby>職人<rt>しょくにん</rt></ruby>のわざ"——只有亲自尝试、感受才能学到的工匠手艺。

9. `用 〜にかけては 写一句关于自己的句子。`
   - 参考答案：`漢字を書くスピードにかけては、クラスで一番だと思っている。`

## 本课复习要点

- 能用 `〜割に（は）` 表达"相对于……出乎意料地……"。
- 能用 `〜にあたり／〜にあたって` 在郑重场合表达"在……之际"。
- 能用 `〜に限らず` 强调"不仅……也……"。
- 能用 `〜にかけては` 表达"在……方面（无人能及）"。

### 課文：ものづくり

**①** <ruby>日本<rt>にほん</rt></ruby>の「ものづくり」の<ruby>精神<rt>せいしん</rt></ruby>を<ruby>理解<rt>りかい</rt></ruby>したくて、<ruby>東京<rt>とうきょう</rt></ruby>のある<ruby>寿司店<rt>すしてん</rt></ruby>を<ruby>訪ね<rt>たずね</rt></ruby>た。

- **语法**：`〜割には`（相对于……出乎意料地……）；`〜続ける`（持续……）；`〜とのことだ`（据说……）
