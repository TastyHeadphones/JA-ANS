# 第17课 なおす ―「命」

> 对应书内第17课（PDF 约第101–106页）。
> 这一课的主轴是「<ruby>医療<rt>いりょう</rt></ruby>と<ruby>命<rt>いのち</rt></ruby>のあり<ruby>方<rt>ほう</rt></ruby>」，重点学习"自从……（之后一直）""这样下去就……""一方面……另一方面""不能（不得不允许）……"等表达。

## 本课主旨

医疗的进步延长了人类的平均寿命，过去无药可救的疾病也变得可以治愈。但寿命延长的同时，社会高龄化也加剧，"活着到底是什么"成为不得不重新思考的严峻问题。医疗本应缓解病人的痛苦、让人恢复到原本的健康生活；然而现在所谓"植物状态"的病人，依靠药品和器械每天延长几分钟生命的治疗并不少见。作者反思：医疗真正应该追求的不是"活得多长"，而是"能健康地活的时间有多长"——这是必须认真重新审视的问题。

## 导入问题

课前会先围绕下面三个问题展开：

- 身边的人有没有得过大病？
- 长期住院会带来什么问题？
- 你认为"活着"到底意味着什么？

## 文法要点

### 1. `〜たきり`

- 用法：「自从……（以后再没……）」「自从……一直保持那个状态"」。
- 接续：`Vた形 + きり`
- 例句：
  - <ruby>友達<rt>ともだち</rt></ruby>にDVDを<ruby>借り<rt>かり</rt></ruby>たきり、まだ<ruby>返し<rt>かえし</rt></ruby>ていない。
  - <ruby>子供<rt>こども</rt></ruby>たちは<ruby>遊び<rt>あそび</rt></ruby>に<ruby>行った<rt>いった</rt></ruby>きり、<ruby>晩<rt>ばん</rt></ruby>ご<ruby>飯<rt>めし</rt></ruby>の<ruby>時間<rt>じかん</rt></ruby>になっても<ruby>戻っ<rt>もどっ</rt></ruby>てこない。
  - <ruby>弟<rt>おとうと</rt></ruby>は<ruby>自分<rt>じぶん</rt></ruby>の<ruby>部屋<rt>へや</rt></ruby>に<ruby>入っ<rt>はいっ</rt></ruby>た。それきり、<ruby>出て<rt>でて</rt></ruby>こない。
- 说明：
  - 强调"动作做完后，本应继续的下一步迟迟没来"。
  - 后续多为否定句。

### 2. `〜ようでは`

- 用法：「这样下去就……（不行）」。表示"如果以这种状态持续，就会……"的批评/担忧。
- 接续：`Vる / Vない + ようでは`
- 例句：
  - ひとりで<ruby>起き<rt>おき</rt></ruby>られないようでは、<ruby>会社<rt>かいしゃ</rt></ruby>で<ruby>働く<rt>はたらく</rt></ruby>なんて<ruby>無理<rt>むり</rt></ruby>だ。
  - <ruby>勉強<rt>べんきょう</rt></ruby>しないで<ruby>遊んで<rt>あそんで</rt></ruby>ばかりいるようでは、いい<ruby>大学<rt>だいがく</rt></ruby>に<ruby>入れ<rt>いれ</rt></ruby>ません。
  - <ruby>自分<rt>じぶん</rt></ruby>の<ruby>好き<rt>すき</rt></ruby>な<ruby>物<rt>もの</rt></ruby>しか<ruby>食べ<rt>たべ</rt></ruby>ないようでは、<ruby>大きく<rt>おおきく</rt></ruby>なりませんよ。
- 说明：
  - 后续多接消极结果（"不行""无理""だめだ" 等）。
  - 比 `〜ようだと` 语气更书面、批判性更强。

### 3. `〜。その一方（で）、〜`

- 用法：「一方面……另一方面……」，对比说明同时存在的两方面。
- 接续：句子 + `。その一方（で）、` + 另一句
- 例句：
  - <ruby>兄<rt>あに</rt></ruby>は<ruby>大学<rt>だいがく</rt></ruby>でコンピュータを<ruby>研究<rt>けんきゅう</rt></ruby>している。その<ruby>一方<rt>いっぽう</rt></ruby>で、コンピュータ<ruby>関係<rt>かんけい</rt></ruby>の<ruby>会社<rt>かいしゃ</rt></ruby>を<ruby>作り<rt>つくり</rt></ruby>、その<ruby>仕事<rt>しごと</rt></ruby>もしている。
  - <ruby>結婚<rt>けっこん</rt></ruby>もせず、<ruby>子供<rt>こども</rt></ruby>もいらないと<ruby>考え<rt>かんがえ</rt></ruby>る<ruby>女性<rt>じょせい</rt></ruby>が<ruby>増え<rt>ふえ</rt></ruby>ている。その<ruby>一方<rt>いっぽう</rt></ruby>で、<ruby>早く<rt>はやく</rt></ruby><ruby>結婚<rt>けっこん</rt></ruby>して<ruby>仕事<rt>しごと</rt></ruby>を<ruby>辞め<rt>やめ</rt></ruby>たい<ruby>女性<rt>じょせい</rt></ruby>も<ruby>少なく<rt>すくなく</rt></ruby>ないらしい。
  - <ruby>寝たきり<rt>ねたきり</rt></ruby>の<ruby>患者<rt>かんじゃ</rt></ruby>を<ruby>生か<rt>いか</rt></ruby>す<ruby>医療<rt>いりょう</rt></ruby>がある。その<ruby>一方<rt>いっぽう</rt></ruby>で、<ruby>家族<rt>かぞく</rt></ruby>も<ruby>犠牲を払う<rt>ぎせいをはらう</rt></ruby><ruby>現実<rt>げんじつ</rt></ruby>がある。
- 说明：
  - 用于客观陈述"同时存在的两个面"，强调对比。
  - 也可以用 `一方` 单独连接两个句子。

### 4. `〜わけにはいかない`

- 用法：「不能……」「不可以那样做」（不是物理不能，而是道义、社会等原因不允许）。
- 接续：
  - `V辞書形 + わけにはいかない`（不能做）
  - `Vない形 + ないわけにはいかない`（不得不做）
- 例句：
  - <ruby>今日<rt>きょう</rt></ruby>は<ruby>大切<rt>たいせつ</rt></ruby>な<ruby>仕事<rt>しごと</rt></ruby>があるから、<ruby>少し<rt>すこし</rt></ruby>ぐらい<ruby>熱<rt>ねつ</rt></ruby>があるからといって、<ruby>会社<rt>かいしゃ</rt></ruby>を<ruby>休む<rt>やすむ</rt></ruby>わけにはいかない。
  - <ruby>血<rt>ち</rt></ruby>を<ruby>分け<rt>わけ</rt></ruby>た<ruby>人<rt>ひと</rt></ruby><ruby>間<rt>かん</rt></ruby>からすると、そう<ruby>簡単<rt>かんたん</rt></ruby>に<ruby>割り切る<rt>わりきる</rt></ruby>わけにはいかない。
  - <ruby>約束<rt>やくそく</rt></ruby>したからには、<ruby>行か<rt>いか</rt></ruby>ないわけにはいかない。
- 说明：
  - 与 `〜ことはできない`（单纯不能）相比，`〜わけにはいかない` 强调"出于道义/事情/承诺等不能"。

## 课文梳理

1. 为了"哪怕多活一点"的愿望，医疗取得了显著进步。许多本来出生就会夭折的婴儿被救活，过去不能治的病变得可以治。
2. 然而平均寿命延长的同时，社会高龄化也在加深，必须重新思考"活着到底是什么"的严峻局面也由此产生。
3. 医疗本应缓解患者痛苦，让人恢复健康；但今天"植物状态"病人不在少数——明知无法回到健康，仍靠药与器械每天延长一点点的治疗，引发了诸多问题。
4. 作者认为：医疗应追求的不是"如何活得久"，而是"如何让能健康地活的时间延长"。有人主张："如果只能躺着，连身体都动不了，就不算真正活着"——但血脉相连的家属又怎能这么干脆地割舍？
5. 长期治疗不仅消耗病人，更累及家属：工作要调整，自己也可能因照护而病倒。若亲人能回到原本生活倒也罢，看不到终点的治疗让家庭付出沉重代价。"活着"到底意味着什么——这是当下必须严肃重新追问的问题。

   - 答案：`犠牲`
   - 解析：`犠牲を払う`。

### 二、语法练习

4. `友達に本を借りた（　　　　　　　）、まだ返していない。`
   - 答案：`きり`
   - 解析：`〜たきり` = 自从……以后没有……。

5. `ひとりで起きられない（　　　　　　　）、社会人として働けない。`
   - 答案：`ようでは`
   - 解析：`〜ようでは` = 这样下去就……。

6. `兄は会社員だ。（　　　　　　　）、自分の店も持っている。`
   - 答案：`その一方（で）`
   - 解析：`その一方（で）` = 另一方面。

7. `約束したから、行かない（　　　　　　　）。`
   - 答案：`わけにはいかない`
   - 解析：`〜わけにはいかない` = 不能不去。

### 三、短答题

8. `本文中、筆者が「医療が目指すべき」と言うのは何か？`
   - 参考答案：不是单纯延长寿命，而是延长"能健康地活着"的时间；医疗应缓解病人痛苦，让人尽可能恢复到原本的健康生活。

9. `用 〜わけにはいかない 写一句关于自己的句子。`
   - 参考答案：`明日大事な試験があるから、今日は早く寝ないわけにはいかない。`

## 本课复习要点

- 能用 `〜たきり` 表达"自从……以后再没……"。
- 能用 `〜ようでは` 表达"这样下去就（不行）"的批评/担忧。
- 能用 `〜。その一方（で）、〜` 对比说明两方面共存的现实。
- 能用 `〜わけにはいかない` 表达"出于道义不能（不去）"。
