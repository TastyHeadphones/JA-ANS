# 第7课 かざる ―「名刺」

> 对应书内第7课（PDF 约第49–54页）。
> 这一课的主轴是「<ruby>外見<rt>がいけん</rt></ruby>と<ruby>中身<rt>なかみ</rt></ruby>」，重点学习"甚至连……""根据/受……影响""难得……""应该……"等表达。

## 本课主旨

人们常说"<ruby>外見<rt>がいけん</rt></ruby>より<ruby>中身<rt>なかみ</rt></ruby>"（内在比外表重要），可现实里大多还是靠外表判断人，想被别人看好的人也不少。整容手术、热销的男性化妆品、宛如明星照的竞选海报，都说明大家都在意外表、容易受外表影响。学校设统一校服本是为了让学生不必在意外表、专心学习运动，但学生们却拿校服当话题、甚至以校服来决定升入哪所学校。名片也是如此：上面的公司名、头衔会影响接收者对你的判断，名气大的公司名会让名片显得气派。"<ruby>外見<rt>がいけん</rt></ruby>より<ruby>中身<rt>なかみ</rt></ruby>"当然重要，外表再光鲜，没有内在也没用；但在一个重视外表的社会里长大的人，很难突然无视外表。结论：既然都要装扮外表，那就同时不断磨练自己，努力成为"内在不输给外表"的人。

## 导入问题

课前会先围绕下面三个问题展开：

- 初次和人见面时，你会注意哪些方面？
- 自己的身体或脸上，喜欢的地方是哪里？讨厌的地方是哪里？
- 你觉得"很帅／很美／很有魅力"的，是怎样的人？

## 文法要点

### 1. `〜まで`（甚至连……都）

- 用法：举出一个极端、出人意料的例子，表示"甚至连……都（如此）"，强调程度之深。本课课文「<ruby>会社名<rt>かいしゃめい</rt></ruby>や<ruby>肩書き<rt>かたがき</rt></ruby>まで<ruby>見て<rt>みて</rt></ruby><ruby>判断する<rt>はんだんする</rt></ruby>」即此用法。
- 接续：
  - `N + まで`
  - `V（辞書形/ている等）+ + まで`
- 例句：
  - <ruby>日本<rt>にほん</rt></ruby>は<ruby>便利<rt>べんり</rt></ruby>だ。100<ruby>円<rt>えん</rt></ruby>で<ruby>時計<rt>とけい</rt></ruby>まで<ruby>買え<rt>かえ</rt></ruby>る。
  - <ruby>部屋<rt>へや</rt></ruby>が<ruby>静か<rt>しずか</rt></ruby>で、となりの<ruby>人<rt>ひと</rt></ruby>の<ruby>話<rt>はなし</rt></ruby>まで<ruby>聞こ<rt>きこ</rt></ruby>える。
  - <ruby>弟<rt>おとうと</rt></ruby>がそこまでまじめに<ruby>考え<rt>かんがえ</rt></ruby>ているとは<ruby>思わ<rt>おもわ</rt></ruby>なかった。
- 说明：
  - `〜まで` 在此不是"到……为止"，而是"甚至连……都"，列举极端例子以强调。
  - 与 `〜さえ`（连……都）意思相近；与表示终点的 `〜まで（に）`（到……为止/之前）用法不同，靠语境区分。

### 2. `〜によって`①（根据、视……而定）

- 用法：表示因条件、对象不同而结果不同，"根据……、因……而异"。
- 接续：
  - `N + によって / により`
- 例句：
  - <ruby>文化<rt>ぶんか</rt></ruby>によって、<ruby>礼儀<rt>れいぎ</rt></ruby>の<ruby>形<rt>かたち</rt></ruby>は<ruby>違う<rt>ちがう</rt></ruby>。
  - <ruby>人<rt>ひと</rt></ruby>によって、<ruby>名刺<rt>めいし</rt></ruby>に<ruby>対する<rt>たいする</rt></ruby><ruby>考え方<rt>かんがえかた</rt></ruby>は<ruby>様々<rt>さまざま</rt></ruby>だ。
  - <ruby>国<rt>くに</rt></ruby>によって、ビジネスマナーが<ruby>異なる<rt>ことなる</rt></ruby>。
- 说明：
  - 强调"以某个标准为依据"，前接的名词是判断、变化的依据。
  - 常见搭配：`〜によって違う / 異なる / 様々だ`。

### 3. `〜によって`②（通过……手段/由……造成）

- 用法：表示手段、方法，或动作的执行者（被动句中）。
- 接续：
  - `N + によって`
- 例句：
  - <ruby>礼儀<rt>れいぎ</rt></ruby>によって、<ruby>相手<rt>あいて</rt></ruby>への<ruby>尊重<rt>そんちょう</rt></ruby>が<ruby>伝わ<rt>つたわ</rt></ruby>る。
  - この<ruby>建物<rt>たてもの</rt></ruby>は<ruby>有名<rt>ゆうめい</rt></ruby>な<ruby>建築家<rt>けんちくか</rt></ruby>によって<ruby>設計<rt>せっけい</rt></ruby>された。
  - <ruby>言葉<rt>ことば</rt></ruby>によってではなく、<ruby>行動<rt>こうどう</rt></ruby>によって<ruby>示す<rt>しめす</rt></ruby>。
- 说明：
  - 与用法①区分：用法①表示"依据/因……而异"，用法②表示"通过某手段"或"由某人/某事造成"。
  - 在被动句中常表示行为的施动者，相当于 `〜に` 但语气更正式。

### 4. `せっかく〜のだから／せっかく〜のに`

- 用法：表示"难得……（所以应该好好利用）"或"难得……（却……，很可惜）"。
- 接续：
  - `せっかく + V / N + のだから / のに`
- 例句：
  - せっかく<ruby>日本<rt>にほん</rt></ruby>に<ruby>来た<rt>きた</rt></ruby>のだから、<ruby>日本<rt>にほん</rt></ruby>のマナーをしっかり<ruby>学び<rt>まなび</rt></ruby>たい。
  - せっかくきれいに<ruby>飾っ<rt>かざっ</rt></ruby>たのに、<ruby>誰も<rt>だれも</rt></ruby><ruby>気づ<rt>きづ</rt></ruby>いてくれなかった。
  - せっかくの<ruby>機会<rt>きかい</rt></ruby>だから、ぜひ<ruby>挑戦<rt>ちょうせん</rt></ruby>してみてください。
- 说明：
  - `せっかく〜のだから` 语气是"既然难得……，就要……"（正面利用）。
  - `せっかく〜のに` 语气是"难得……却……"（可惜、遗憾）。

### 5. `〜ことだ`

- 用法：对某人提出建议或劝告，表示"最重要的是……""应该……"。
- 接续：
  - `V辞書形 / Vない形 + ことだ`
- 例句：
  - <ruby>日本<rt>にほん</rt></ruby><ruby>語<rt>ご</rt></ruby>を<ruby>上手<rt>じょうず</rt></ruby>にしたければ、<ruby>毎日<rt>まいにち</rt></ruby><ruby>練習<rt>れんしゅう</rt></ruby>することだ。
  - <ruby>病気<rt>びょうき</rt></ruby>を<ruby>治し<rt>なおし</rt></ruby>たければ、<ruby>薬<rt>くすり</rt></ruby>を<ruby>飲ん<rt>のん</rt></ruby>でゆっくり<ruby>休む<rt>やすむ</rt></ruby>ことだ。
  - <ruby>相手<rt>あいて</rt></ruby>に<ruby>失礼<rt>しつれい</rt></ruby>にならないよう、まずマナーを<ruby>学ぶ<rt>まなぶ</rt></ruby>ことだ。
- 说明：
  - 用于给出建议或指出最关键的做法，带有一定的规劝语气。
  - 与 `〜ものだ`（一般常理）区分：`〜ことだ` 更针对具体对象提出具体建议。

## 课文梳理

1. 人们常说"<ruby>外見<rt>がいけん</rt></ruby>より<ruby>中身<rt>なかみ</rt></ruby>"，可现实中靠外表判断别人、或想被别人看好的情况并不少。
2. 为改善外表去整容、男性化妆品热销；竞选海报上甚至有宛如明星照的，让人疑惑到底想传达什么——这些都说明大家在意外表、容易受外表影响。
3. 学校设统一校服，本意是让学生不必在意外表、专心学习与运动；但学生却把校服当话题，甚至以校服来决定升入哪所学校。
4. 名片也一样：上面的公司名、头衔会影响接收方对你的判断，名气大的公司名会让名片显得气派。
5. "<ruby>外見<rt>がいけん</rt></ruby>より<ruby>中身<rt>なかみ</rt></ruby>"当然重要，外表再光鲜，没有内在也没用；但在重视外表的社会里长大的人，很难突然无视外表。
6. 结论：既然都要装扮外表，那就该同时不断磨练自己，努力成为"内在不输给外表"的人。

## 练习题

### 一、词汇练习

1. `よく「外見より（　　　　　）」と言われる。`
   - 答案：`中身`
   - 解析：`外見より中身` = 内在比外表重要。

2. `みんな外見を気にし、外見に（　　　　　）されやすい。`
   - 答案：`影響`
   - 解析：`影響される` = 受影响。

3. `名刺の会社名や（　　　　　）を見て、相手を判断する。`
   - 答案：`肩書き`
   - 解析：`肩書き` = 头衔、职衔。

4. `いくら外見を（　　　　　）も、中身がなければ何にもならない。`
   - 答案：`かざって`
   - 解析：`かざる`（<ruby>飾る<rt>かざる</rt></ruby>）= 装饰、装扮。

### 二、语法练习

5. `名刺を手にした側は、会社名や肩書き（　　　　　）見て判断する。`
   - 答案：`まで`
   - 解析：`〜まで` = 甚至连……（举出极端例子）。

6. `人（　　　　　）、名刺に対する考え方は違う。`
   - 答案：`によって`
   - 解析：`〜によって` = 因……而异。

7. `せっかく外見をかざる（　　　　　）、中身もみがきたい。`
   - 答案：`のだから`
   - 解析：`せっかく〜のだから` = 既然难得……就要……。

8. `中身をよくしたければ、努力を続ける（　　　　　）だ。`
   - 答案：`こと`
   - 解析：`〜ことだ` = 应该……（建议）。

### 三、短答题

9. `筆者は、外見と中身についてどうするのが大切だと言っていますか。`
   - 参考答案：既然都要装扮外表，就要同时磨练自己，努力成为内在不输给外表的人，并坚持下去。

10. `用 せっかく〜のだから 写一句话。`
    - 参考答案：`せっかく日本に来たのだから、いろいろな所を見て回りたい。`

## 本课复习要点

- 能用 `〜まで` 表达"甚至连……（都）"，并与表示截止时间的 `〜までに` 区分。
- 能区分 `〜によって`① "因……而异"和 `〜によって`② "通过……手段/由……造成"两种用法。
- 能用 `せっかく〜のだから` 表达"既然难得……就要……"，以及 `せっかく〜のに` 表达"难得……却……（可惜）"。
- 能用 `〜ことだ` 对他人提出具体的建议或劝告。

### 課文：名刺

**①** よく「<ruby>外見<rt>がいけん</rt></ruby>より<ruby>中身<rt>なかみ</rt></ruby>」と<ruby>言わ<rt>いわ</rt></ruby>れるが、じっさいには、<ruby>外見<rt>がいけん</rt></ruby>で<ruby>人<rt>ひと</rt></ruby>を<ruby>判断する<rt>はんだんする</rt></ruby>ことも、<ruby>相手<rt>あいて</rt></ruby>によく<ruby>見ら<rt>みら</rt></ruby>れようとすることも<ruby>少なく<rt>すくなく</rt></ruby>ない。

- **译文**：人们常说"外表不如内在"，但实际上，以外表判断别人、或想让自己在别人眼里更好看的情况，都不在少数。
- **语法**：`〜より〜`（比起……）；`〜ことも〜ことも少なくない`（……和……都不少）；`〜（よ）うとする`（想要……）
- **单词**：`外見`（外表）；`中身`（内在）；`判断する`（判断）

---

**②** <ruby>外見<rt>がいけん</rt></ruby>をよく<ruby>見せ<rt>みせ</rt></ruby>るために<ruby>手術<rt>しゅじゅつ</rt></ruby>をしたり、<ruby>男性<rt>だんせい</rt></ruby><ruby>化粧品<rt>けしょうひん</rt></ruby>も<ruby>売れ<rt>うれ</rt></ruby>ている。<ruby>選挙<rt>せんきょ</rt></ruby>のポスターなどにも、まるでスターの<ruby>写真<rt>しゃしん</rt></ruby>かと<ruby>思う<rt>おもう</rt></ruby>ような<ruby>物<rt>もの</rt></ruby>まであって、<ruby>見る<rt>みる</rt></ruby><ruby>者<rt>もの</rt></ruby>に<ruby>何を<rt>なにを</rt></ruby><ruby>伝え<rt>つたえ</rt></ruby>たいのかと<ruby>首<rt>くび</rt></ruby>をかしげたくなることがある。

- **译文**：有人为让外表更好看去做手术，男性化妆品也很畅销。就连竞选海报里，都有宛如明星写真的，让人忍不住歪头疑惑：到底想向观者传达什么？
- **语法**：`〜ために`（为了……）；`〜たり`（列举）；`まるで〜かと思うような`（简直像是……）；`〜まで`（甚至连……）
- **单词**：`手術`（手术）；`化粧品`（化妆品）；`選挙`（选举）；`首をかしげる`（歪头表示疑惑）

---

**③** みんなが<ruby>外見<rt>がいけん</rt></ruby>を<ruby>気に<rt>きに</rt></ruby>し、（<ruby>外見<rt>がいけん</rt></ruby>に）<ruby>影響<rt>えいきょう</rt></ruby>されやすいということを<ruby>表し<rt>あらわし</rt></ruby>ている<ruby>良い<rt>よい</rt></ruby><ruby>例<rt>れい</rt></ruby>だ。

- **译文**：这正是一个很好的例子，说明大家都在意外表、容易受外表影响。
- **语法**：`〜やすい`（容易……）；`〜ということを表す`（表明……这件事）
