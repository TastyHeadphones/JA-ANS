# 第5课 あきれる ―「満員電車」

> 对应书内第5课（PDF 约第37–42页）。
> 这一课的主轴是「<ruby>異文化<rt>いぶんか</rt></ruby>への<ruby>驚き<rt>おどろき</rt></ruby>とあきれ」，重点学习"刚……""……得受不了""连……的意思都没有""……什么的（感叹语气）"等表达。

## 本课主旨

叙述者来日本之前听说"日本人亲切、有礼貌"。来了之后发现日本的满员电车是了解社会风貌的最佳场所。某天，几个小学生上车后立即拿出游戏机大声喧哗，周围大人没人制止；老人上车时，沉迷游戏的孩子们不肯让座。叙述者一方面也经历过日本人的亲切，另一方面对这次所见感到愕然（あきれた）。结论：用自己的眼睛和耳朵观察，并思考为何如此，才是最重要的。

## 导入问题

课前会先围绕下面三个问题展开：

- 你去外国后，有没有感到惊讶的事？
- 来到日本后，你有没有觉得"和自己以前的方式不一样、这个更好"或"这个不太好"的事？
- 有没有怎么都适应不了、感到困扰的事？

## 文法要点

### 1. `〜たばかり`

- 用法：表示刚刚完成某动作，或某状态刚刚开始不久。
- 接续：
  - `Vた形 + ばかり`
  - `Vた形 + ばかりのN`
- 例句：
  - <ruby>日本<rt>にほん</rt></ruby>に<ruby>来た<rt>きた</rt></ruby>ばかりなので、まだ<ruby>携帯電話<rt>けいたいでんわ</rt></ruby>を<ruby>持っ<rt>もっ</rt></ruby>ていません。
  - この<ruby>漢字<rt>かんじ</rt></ruby>は<ruby>習っ<rt>ならっ</rt></ruby>たばかりなのに、もう<ruby>読み方<rt>よみかた</rt></ruby>を<ruby>忘れ<rt>わすれ</rt></ruby>てしまいました。
  - <ruby>車<rt>くるま</rt></ruby>を<ruby>買っ<rt>かっ</rt></ruby>たばかりのころは、よくいろいろな<ruby>所<rt>ところ</rt></ruby>へ<ruby>出か<rt>でか</rt></ruby>けました。
- 说明：
  - `〜たばかり` 强调动作刚完成，时间距离近。
  - `〜たところ` 也表示"刚……"，但更强调时间点上的刚刚完成，`〜たばかり` 则强调时间上"还不久"。

### 2. `〜て／でたまらない`

- 用法：表示感情或感觉强烈到"忍受不了"的程度。
- 接续：
  - `Vて + たまらない`
  - `い形容詞（くて）+ たまらない`
  - `な形容詞（で）+ たまらない`
- 例句：
  - きのうは<ruby>長い<rt>ながい</rt></ruby><ruby>時間<rt>じかん</rt></ruby>、<ruby>車<rt>くるま</rt></ruby>の<ruby>運転<rt>うんてん</rt></ruby>をしたので、こしが<ruby>痛く<rt>いたく</rt></ruby>てたまらなかった。
  - <ruby>満員<rt>まんいん</rt></ruby><ruby>電車<rt>でんしゃ</rt></ruby>の<ruby>中<rt>なか</rt></ruby>で、どうして<ruby>本<rt>ほん</rt></ruby>や<ruby>新聞<rt>しんぶん</rt></ruby>が<ruby>読め<rt>よめ</rt></ruby>るのかふしぎでたまりません。
  - やっとたばこをやめたが、ときどきすいたくてたまらない。
- 说明：
  - 表示"……得受不了"，程度极强。
  - 只能用于说话人自身的感受，不能用于第三方（第三方用 `〜てしかたがない/ようだ`）。

### 3. `〜(よ)うともしない`

- 用法：表示"连试也不试着做……、根本没有做……的意思"，含轻微批评或惊讶。
- 接续：
  - `V意向形 + ともしない`
  - `V意向形 + ともせずに`
- 例句：
  - <ruby>電車<rt>でんしゃ</rt></ruby>の<ruby>中<rt>なか</rt></ruby>で<ruby>私<rt>わたし</rt></ruby>の<ruby>足<rt>あし</rt></ruby>をふんだ<ruby>男<rt>おとこ</rt></ruby>の<ruby>人<rt>ひと</rt></ruby>はあやまろうともせずに、おりて<ruby>行って<rt>いって</rt></ruby>しまった。
  - <ruby>最近<rt>さいきん</rt></ruby>は<ruby>字<rt>じ</rt></ruby>を<ruby>見る<rt>みる</rt></ruby>のをいやがって、<ruby>新聞<rt>しんぶん</rt></ruby>や<ruby>本<rt>ほん</rt></ruby>を<ruby>読も<rt>よも</rt></ruby>うともしない<ruby>人<rt>ひと</rt></ruby>がふえてきた。
- 说明：
  - `〜ようとしない` 表示"不打算做"，强调主观意志。
  - `〜ようともしない` 更强烈，是"连想都不想做"。

### 4. `〜なんて`

- 用法：对某事感到惊讶、轻蔑或不可思议，带有强烈的感情色彩。
- 接续：
  - `普通形 / N + なんて`
- 例句：
  - お<ruby>金<rt>きん</rt></ruby>なんてなくてもいいと<ruby>考え<rt>かんがえ</rt></ruby>るのは、まだ<ruby>若い<rt>わかい</rt></ruby>からだ。
  - <ruby>結婚記念日<rt>けっこんきねんび</rt></ruby>をわすれるなんて、<ruby>信じ<rt>しんじ</rt></ruby>られない。
  - <ruby>会社員<rt>かいしゃいん</rt></ruby>なんて<ruby>人<rt>ひと</rt></ruby>に<ruby>使わ<rt>つかわ</rt></ruby>れるだけで、<ruby>面白く<rt>おもしろく</rt></ruby>ない。
- 说明：
  - `なんて` 相当于 `などというものは`，带强烈的主观情感。
  - 可表示惊讶（ええっ、〜なんて！）或轻蔑（〜なんてどうでもいい）。

## 课文梳理

1. 叙述者来日本前，曾听说"日本人亲切、有礼貌"。
2. 来日本后，认为了解日常社会风貌的最佳场所是每天坐的电车。
3. 某天，几个小学生上车后立即占座、拿出游戏机大声喧哗，大人们无人制止。
4. 途中一位带着大行李的老人上车，孩子们沉迷游戏，无人让座，叙述者感到愕然。
5. 但叙述者也经历过很多日本人亲切的帮助，以及在站台静静排队的风景。
6. 结论：重要的是用自己的眼睛和耳朵观察，思考"为什么会这样"，而不是直接评判。

## 练习题

### 一、词汇练習

1. `日本に来た（　　　　　）なので、まだ漢字があまり読めない。`
   - 答案：`ばかり`
   - 解析：`〜たばかり` = 刚……。

2. `小学生たちはゲームに（　　　　　）で、お年寄りに気づかなかった。`
   - 答案：`夢中`
   - 解析：`夢中になる` = 沉迷、入迷。

3. `その行動には本当に（　　　　　）てしまった。`
   - 答案：`あきれ`
   - 解析：`あきれる` = 愕然。

4. `電車の中で（　　　　　）で騒ぎ始めた。`
   - 答案：`大声`
   - 解析：`大声で` = 大声地。

### 二、语法练習

5. `きのうは運転しすぎて、こしが痛くて（　　　　　）なかった。`
   - 答案：`たまら`
   - 解析：`〜てたまらない`。

6. `男の人は謝ろう（　　　　　）もせずに、帰ってしまった。`
   - 答案：`と`
   - 解析：`〜ようともしない`。

7. `結婚記念日を忘れる（　　　　　）、信じられない。`
   - 答案：`なんて`
   - 解析：惊讶语气 `〜なんて`。

8. `この漢字は習った（　　　　　）なのに、もう忘れてしまった。`
   - 答案：`ばかり`
   - 解析：刚学不久，`〜たばかり`。

### 三、短答題

9. `文中で筆者が「あきれた」のはどんな場面でしたか？`
   - 参考答案：小学生在车厢里大声玩游戏，周围大人不制止；老人上车时，沉迷游戏的孩子们不让座。

10. `用 〜てたまらない 写一句话，描述自己的某种感受。`
    - 参考答案：`試験の前は緊張してたまりません。`

## 本课复习要点

- 能用 `〜たばかり` 表达"刚……"，并与 `〜たところ` 区分用法。
- 能用 `〜て/でたまらない` 描述强烈的感情或感觉。
- 能用 `〜(よ)うともしない` 表达"连做的意思都没有"。
- 能用 `〜なんて` 表达对事物的惊讶或轻蔑。
