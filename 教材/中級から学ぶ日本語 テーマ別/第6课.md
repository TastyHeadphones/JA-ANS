# 第6课 つたえる ―「<ruby>思いやり<rt>おもいやり</rt></ruby>」

> 对应书内第6课（PDF 约第43–48页）。
> 这一课的主轴是「<ruby>言葉<rt>ことば</rt></ruby>の<ruby>使い方<rt>つかいかた</rt></ruby>と<ruby>思いやり<rt>おもいやり</rt></ruby>」，重点学习"举例列举""虽说……但""理应如此""不该如此""自以为……"等表达。

## 本课主旨

作者在闷热或加班疲惫的日子会多付钱乘坐特急电车。车内反复播放的广播「<ruby>特急<rt>とっきゅう</rt></ruby>には<ruby>乗車券<rt>じょうしゃけん</rt></ruby>のほかに<ruby>特急<rt>とっきゅう</rt></ruby><ruby>券<rt>けん</rt></ruby>が<ruby>必要<rt>ひつよう</rt></ruby>です。お<ruby>間違い<rt>まちがい</rt></ruby>にならないでください」让作者在意——语法和敬语都没错，可反复听下来，却像在说"没有票就别上车"。作者还提到一位年轻母亲的投书：她对孩子说的话整天都是"别剩下，好好吃""电视要离远点看"之类的命令和禁止句，自己都觉得腻烦；但读到的人也许会从中感到对孩子的"<ruby>思いやり<rt>おもいやり</rt></ruby>"（关怀）。同样一句「〜なさい」，有时听起来像命令，有时却能传达温柔。结论：汉字、语法、词义固然重要，但更需要学习"考虑到对方会如何接收"的用词方式。

## 导入问题

课前会先围绕下面三个问题展开：

- 说话时，你会注意哪些方面？
- 听到对方说的话，有没有觉得"真讨厌"的时候？
- 至今为止别人对你说过的话里，有没有让你觉得"非常高兴"的话？

## 文法要点

### 1. `〜とか〜とか`

- 用法：从多个事物中举出几个代表性的例子，表示"……啊……啊之类的"。
- 接续：
  - `N / 普通形 + とか + N / 普通形 + とか`
- 例句：
  - <ruby>日本<rt>にほん</rt></ruby><ruby>語<rt>ご</rt></ruby>を<ruby>勉強<rt>べんきょう</rt></ruby>するには、<ruby>単語<rt>たんご</rt></ruby>とか<ruby>文法<rt>ぶんぽう</rt></ruby>とかをしっかり<ruby>覚え<rt>おぼえ</rt></ruby>る<ruby>必要<rt>ひつよう</rt></ruby>がある。
  - <ruby>週末<rt>しゅうまつ</rt></ruby>は<ruby>友達<rt>ともだち</rt></ruby>と<ruby>映画<rt>えいが</rt></ruby>を<ruby>見る<rt>みる</rt></ruby>とか、カラオケに<ruby>行く<rt>いく</rt></ruby>とかして<ruby>過ご<rt>すご</rt></ruby>します。
  - <ruby>彼女<rt>かのじょ</rt></ruby>は「<ruby>疲れ<rt>つかれ</rt></ruby>た」とか「<ruby>忙しい<rt>いそがしい</rt></ruby>」とか<ruby>言って<rt>いって</rt></ruby>、なかなか<ruby>会っ<rt>あっ</rt></ruby>てくれない。
- 说明：
  - `〜とか〜とか` 用于非穷举式列举，语气比 `〜や〜` 更口语。
  - 可以接名词，也可以接句子（普通形）。

### 2. `〜といっても`

- 用法：提出某个前提，随即对其加以限定或修正，表示"虽说……但（实际上没那么……）"。
- 接续：
  - `N / 普通形 + といっても`
- 例句：
  - <ruby>日本<rt>にほん</rt></ruby><ruby>語<rt>ご</rt></ruby>が<ruby>話せ<rt>はなせ</rt></ruby>るといっても、まだ<ruby>敬語<rt>けいご</rt></ruby>はあまりわかりません。
  - <ruby>旅行<rt>りょこう</rt></ruby>に<ruby>行く<rt>いく</rt></ruby>といっても、<ruby>一泊二日<rt>いっぱくふつか</rt></ruby>の<ruby>近い<rt>ちかい</rt></ruby><ruby>所<rt>ところ</rt></ruby>だ。
  - <ruby>家族<rt>かぞく</rt></ruby>といっても、もう<ruby>何年<rt>なんねん</rt></ruby>も<ruby>会っ<rt>あっ</rt></ruby>ていない。
- 说明：
  - 用来纠正对方可能产生的过高期待或误解。
  - 前半是承认的前提，后半是实际限制或补充说明。

### 3. `〜ものだ`

- 用法①：表示一般性的常识、理所当然的道理，带有"本来就是这样的"语感。
- 接续：
  - `V辞書形 / V ない形 + ものだ`
  - `い形容詞 / な形容詞 + ものだ`
- 例句：
  - <ruby>困っ<rt>こまっ</rt></ruby>たときには、<ruby>助け<rt>たすけ</rt></ruby><ruby>合う<rt>あう</rt></ruby>ものだ。
  - <ruby>人<rt>ひと</rt></ruby>の<ruby>話<rt>はなし</rt></ruby>を<ruby>聞く<rt>きく</rt></ruby>ときは、ちゃんと<ruby>顔<rt>かお</rt></ruby>を<ruby>向け<rt>むけ</rt></ruby>るものだ。
  - <ruby>子ども<rt>こども</rt></ruby>は<ruby>外<rt>そと</rt></ruby>で<ruby>元気<rt>げんき</rt></ruby>に<ruby>遊ぶ<rt>あそぶ</rt></ruby>ものだ。
- 说明：
  - 用于说明常理、习惯或社会规范，有时带有轻微的说教意味。
  - 不能用于自己当下正在做的事，而是讲一般性的道理。

### 4. `〜ものではない`

- 用法：表示"不应该做……""……是不对的"，是对行为的否定性评价。
- 接续：
  - `V辞書形 + ものではない`
- 例句：
  - <ruby>人<rt>ひと</rt></ruby>の<ruby>悪口<rt>わるくち</rt></ruby>を<ruby>言う<rt>いう</rt></ruby>ものではない。
  - <ruby>知ら<rt>しら</rt></ruby>ない<ruby>人<rt>ひと</rt></ruby>から<ruby>物<rt>もの</rt></ruby>をもらうものではないと、<ruby>親<rt>おや</rt></ruby>に<ruby>言わ<rt>いわ</rt></ruby>れた。
  - <ruby>人<rt>ひと</rt></ruby>の<ruby>前<rt>まえ</rt></ruby>で<ruby>大き<rt>おおき</rt></ruby>な<ruby>声<rt>こえ</rt></ruby>で<ruby>怒鳴る<rt>どなる</rt></ruby>ものではない。
- 说明：
  - 是 `〜ものだ` 的否定，表示"按道理不该这样做"。
  - 带有强烈的规范性，常用于忠告、教导或批评。

### 5. `〜たつもり`

- 用法：表示说话人"自以为做了某事"或"自认为处于某种状态"，但实际上未必如此。
- 接续：
  - `Vた形 + つもり`
  - `V辞書形 / Vない形 + つもり`（表意图时）
- 例句：
  - <ruby>気持ち<rt>きもち</rt></ruby>を<ruby>伝え<rt>つたえ</rt></ruby>たつもりだったが、<ruby>相手<rt>あいて</rt></ruby>には<ruby>伝わ<rt>つたわ</rt></ruby>っていなかった。
  - <ruby>日本<rt>にほん</rt></ruby><ruby>語<rt>ご</rt></ruby>の<ruby>敬語<rt>けいご</rt></ruby>はわかっているつもりだが、まだ<ruby>間違え<rt>まちがえ</rt></ruby>ることがある。
  - <ruby>十分<rt>じゅうぶん</rt></ruby><ruby>説明し<rt>せつめいし</rt></ruby>たつもりだったが、みんなに<ruby>理解<rt>りかい</rt></ruby>してもらえなかった。
- 说明：
  - `〜たつもり` 强调主观认为已经做了，实际结果可能不符合。
  - 与表示意图的 `〜つもりだ`（打算做）区分：`〜たつもり` 是回顾已做之事。

## 课文梳理

1. 作者在闷热难耐或加班疲惫的日子，会多花钱乘坐特急电车。
2. 车内反复播放「<ruby>特急<rt>とっきゅう</rt></ruby>には<ruby>乗車券<rt>じょうしゃけん</rt></ruby>のほかに<ruby>特急<rt>とっきゅう</rt></ruby><ruby>券<rt>けん</rt></ruby>が<ruby>必要<rt>ひつよう</rt></ruby>です。お<ruby>間違い<rt>まちがい</rt></ruby>にならないでください」的广播。语法、敬语都没错，但反复听下来，却像在被告知"没票就别上车"，让作者在意。
3. 作者曾读到一位年轻母亲的投书：她对孩子说的话整天都是"别剩饭，好好吃""电视尽量离远点看"之类的命令、禁止句，连自己都觉得腻烦。
4. 但读到这封投书的人，也许会从中读出对孩子的关怀（<ruby>思いやり<rt>おもいやり</rt></ruby>）——"多吃点才能长大""离太近看电视眼睛会坏"。
5. 即使顾及对方、想正确地传达，也未必如愿：同样一句「〜なさい」，有时听起来像命令，有时却能传达温柔。
6. 结论：汉字、语法、词义固然重要，但更需要学习"考虑到对方会如何接收"的用词方式。

## 练习题

### 一、词汇练习

1. `特急には、乗車券のほかに（　　　　　）が必要だ。`
   - 答案：`特急券`
   - 解析：乘坐特急需要乘车券之外的特急券。

2. `ちょっとした言葉の使い方が（　　　　　）ことがある。`
   - 答案：`気にかかる`
   - 解析：`気にかかる` = 在意、挂心。

3. `一日中、（　　　　　）や禁止の文ばかりで、うんざりだ。`
   - 答案：`命令`
   - 解析：`命令` = 命令；与 `禁止` 并列。

4. `お母さんの言葉から、子供への（　　　　　）が伝わる。`
   - 答案：`思いやり`
   - 解析：`思いやり` = 体贴、关怀。

### 二、语法练习

5. `「食べなさい」（　　　　　）「見なさい」（　　　　　）、命令の文ばかりだ。`
   - 答案：`とか`、`とか`
   - 解析：`〜とか〜とか` = ……啊……啊（举例列举）。

6. `命令の文（　　　　　）、子供への思いやりが伝わることもある。`
   - 答案：`といっても`
   - 解析：`〜といっても` = 虽说是……但……。

7. `人の気持ちを考えないで、自分のことばかり話す（　　　　　）ではない。`
   - 答案：`もの`
   - 解析：`〜ものではない` = 不应该……。

8. `正しく伝え（　　　　　）でも、相手には違って伝わることがある。`
   - 答案：`たつもり`
   - 解析：`〜たつもり` = 自以为做了……。

### 三、短答题

9. `筆者は、どうして特急電車のアナウンスが気にかかるのですか。`
   - 参考答案：<ruby>文法<rt>ぶんぽう</rt></ruby>も<ruby>敬語<rt>けいご</rt></ruby>も<ruby>間違っ<rt>まちがっ</rt></ruby>ていないのに、<ruby>何度<rt>なんど</rt></ruby>も<ruby>聞い<rt>きい</rt></ruby>ているうちに「<ruby>券<rt>けん</rt></ruby>がなければ<ruby>乗る<rt>のる</rt></ruby>な」と<ruby>言わ<rt>いわ</rt></ruby>れているように<ruby>感じ<rt>かんじ</rt></ruby>てしまうから。

10. `用 〜といっても 写一句话。`
    - 参考答案：`日本語が話せるといっても、敬語はまだ難しい。`

## 本课复习要点

- 能用 `〜とか〜とか` 进行非穷举式的举例列举。
- 能用 `〜といっても` 对前提做限定说明，表示"虽说……但实际上……"。
- 能区分 `〜ものだ`（理所当然）和 `〜ものではない`（不应该），理解其规范性用法。
- 能用 `〜たつもり` 表达"自以为做了某事"，并理解与现实可能存在落差。

- **语法**：`〜や〜など`（列举）；`〜には`（在……情况下）
- **单词**：`むし暑い`（い<ruby>形<rt>かたち</rt></ruby>，闷热）；`我慢する`（忍耐）；`残業`（加班）；`別料金`（额外费用）

- **语法**：`〜のほかに`（除……之外）；`お〜になる`（尊敬语）；`〜という+N`（叫做……的）

- **语法**：`〜のでも〜のでもない`（既不是……也不是……）
- **单词**：`選び方`（选择方式）；`敬語`（敬语）；`気に入る`（满意、中意）；`文法`（语法）
