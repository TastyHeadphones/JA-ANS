# 第12课 わける ―「ABO AB」

> 对应书内第12课（PDF 约第69–74页）。
> 这一课的主轴是「<ruby>血液型<rt>けつえきがた</rt></ruby>と<ruby>性格<rt>せいかく</rt></ruby>」，重点学习"按……（方式）""因为，其实是因为""被认定为……""根据……（因人而异）"等表达。

## 本课主旨

作者描述了日本人热衷于"血型性格论"的现象——A型重秩序、B型自由奔放、O型领导力强、AB型冷静客观，自己也是其中一员。虽然科学上没有依据，但很多人把它当作"了解对方类型后再安心相处"的小工具，是缓解人际压力的一种方式。课文反思：把人分成几种类型来理解，到底是简化沟通的智慧，还是带偏见的成见？

## 导入问题

课前会先围绕下面三个问题展开：

- 你相信血型与性格有关吗？
- 在你的国家，人们用什么方式给别人"分类"？
- 把人分类对人际交往有帮助吗？

## 文法要点

### 1. `〜通り（に/の）`

- 用法：表示"按……的样子/方式（去做）"。
- 接续：
  - `Vた形 / V辞書形 + 通り`
  - `N + の通り / Nどおり`
- 例句：
  - <ruby>地図<rt>ちず</rt></ruby>の<ruby>通り<rt>とおり</rt></ruby>に<ruby>歩い<rt>あるい</rt></ruby>たら、すぐに<ruby>分か<rt>わか</rt></ruby>りました。
  - <ruby>本<rt>ほん</rt></ruby>に<ruby>書い<rt>かい</rt></ruby>てある<ruby>通り<rt>とおり</rt></ruby>の<ruby>作り方<rt>つくりかた</rt></ruby>で<ruby>料理<rt>りょうり</rt></ruby>を<ruby>作っ<rt>つくっ</rt></ruby>てみた。
  - <ruby>友達<rt>ともだち</rt></ruby>が<ruby>言った<rt>いった</rt></ruby><ruby>通り<rt>とおり</rt></ruby>、<ruby>秋<rt>あき</rt></ruby>の<ruby>京都<rt>きょうと</rt></ruby>はとてもきれいでした。
- 说明：
  - 接 `〜に` 强调"按照（去做）"，接 `〜の` 接名词。
  - 后续可以是结果或评价。

### 2. `〜というのも`

- 用法：补充说明前句的原因，「这是因为……」。
- 接续：`〜。というのも、〜から（だ）。`
- 例句：
  - <ruby>友達<rt>ともだち</rt></ruby>に<ruby>何か<rt>なにか</rt></ruby>あったのではないかと<ruby>心配<rt>しんぱい</rt></ruby>だ。というのも、<ruby>最近<rt>さいきん</rt></ruby><ruby>何となく<rt>なんとなく</rt></ruby><ruby>楽しそう<rt>たのしそう</rt></ruby>に<ruby>見え<rt>みえ</rt></ruby>ないからだ。
  - <ruby>私<rt>わたし</rt></ruby>もこれにこっている。というのも、<ruby>周り<rt>まわり</rt></ruby>の<ruby>人<rt>ひと</rt></ruby>を<ruby>見る<rt>みる</rt></ruby>と<ruby>確かに<rt>たしかに</rt></ruby>よく<ruby>当た<rt>あた</rt></ruby>っていると<ruby>思う<rt>おもう</rt></ruby>からだ。
- 说明：
  - 用于书面或语义稍正式的口语，给前句"补一个原因解释"。
  - 后句常以 `〜から（だ）` 或 `〜のだ` 结尾。

### 3. `〜とする / 〜とされる`

- 用法：「认定为……、被认定为……」。表示主观判断或一般认可。
- 接续：`普通形 + とする / とされる（N + であるとされる）`
- 例句：
  - <ruby>警察<rt>けいさつ</rt></ruby>は<ruby>昨日<rt>きのう</rt></ruby>の<ruby>自転車<rt>じてんしゃ</rt></ruby><ruby>事故<rt>じこ</rt></ruby>の<ruby>原因<rt>げんいん</rt></ruby>を<ruby>信号無視<rt>しんごうむし</rt></ruby>であるとした。
  - <ruby>漢字<rt>かんじ</rt></ruby>が<ruby>日本<rt>にほん</rt></ruby>に<ruby>伝わ<rt>つたわ</rt></ruby>ったのは1600<ruby>年前<rt>ねんまえ</rt></ruby>のことだとされる。
  - AB<ruby>型<rt>かた</rt></ruby>は<ruby>物事<rt>ものごと</rt></ruby>を<ruby>客観的<rt>きゃっかんてき</rt></ruby>に<ruby>考え<rt>かんがえ</rt></ruby>るタイプが<ruby>多い<rt>おおい</rt></ruby>とされる。
- 说明：
  - `〜とする`：说话人/某主体作出的认定（主动）。
  - `〜とされる`：被普遍认为（被动；常用于客观描述）。

### 4. `〜によって`（因人/物而异）

- 用法：「随……不同而异」，强调因对象差异而产生不同结果。
- 接续：`N + によって`
- 例句：
  - <ruby>血液型<rt>けつえきがた</rt></ruby>についての<ruby>意見<rt>いけん</rt></ruby>は<ruby>人<rt>ひと</rt></ruby>によって<ruby>様々<rt>さまざま</rt></ruby>だ。

  - <ruby>国<rt>くに</rt></ruby>によって<ruby>食事<rt>しょくじ</rt></ruby>の<ruby>習慣<rt>しゅうかん</rt></ruby>が<ruby>異なる<rt>ことなる</rt></ruby>。
- 说明：
  - 与 `〜によって`（手段／原因）同形不同义，需要根据上下文判断。
  - 后接 `さまざまだ／違う／異なる` 等差异类词语。

## 课文梳理

1. 日本人初次见面常常问"你是什么血型？"——血液型话题极受欢迎，作者也是热衷者之一。
2. A型重秩序、人际、慎重准备；B型乐观、自由、声大有活力，画家音乐家多；O型有领导力、伙伴意识强，但被其他血型说成任性、不听人意见；AB型冷静客观。
3. 关于血型与性格的说法因人而异，被批"没有科学根据，是不靠谱的话"，但仍有许多人参考。
4. 作者认为：人们之所以喜欢把对方"先归类再相处"，是因为"先知道对方类型，安心地交往"——这是缓解人际压力的一种小工具。

## 练习题

### 一、词汇练习

1. `A型は何事も（　　　　　　　）に準備するタイプだ。`
   - 答案：`慎重`
   - 解析：`慎重` = 慎重、谨慎。

2. `血液型の話には科学的な（　　　　　　　）がない。`
   - 答案：`根拠`
   - 解析：`根拠` = 依据。

3. `物事を（　　　　　　　）に判断する人は信頼できる。`
   - 答案：`客観的`
   - 解析：`客観的` = 客观的。

### 二、语法练习

4. `本に書いてある（　　　　）に料理を作ってみた。`
   - 答案：`通り`
   - 解析：`〜通りに` = 按……。

5. `最近、彼の様子が気になる。（　　　　　　　）、声を聞かないからだ。`
   - 答案：`というのも`
   - 解析：补充原因。

6. `漢字が日本に伝わったのは1600年前のことだ（　　　　）。`
   - 答案：`とされる`
   - 解析：`〜とされる` = 一般被认为。

7. `何が大切かは、人（　　　　　　　）違う。`
   - 答案：`によって`
   - 解析：因人而异 `〜によって`。

### 三、短答题

8. `本文で、人を血液型に分けるのは何のためだと書かれているか？`
   - 参考答案：因为多少了解一下对方的类型，可以让自己安心地与对方交往，也是缓解人际压力的小手段。

9. `用 〜とされる 写一个关于自己国家的例句。`
   - 参考答案：`寿司は日本を代表する料理だとされる。`

## 本课复习要点

- 能用 `〜通り（に/の）` 表达"按照……（做）"。
- 能用 `〜というのも` 给前句补充一个原因。
- 能区分 `〜とする`（主体认定）与 `〜とされる`（被认定，客观）。
- 能用 `〜によって` 表达"因（人／对象）而异"。

### 課文：ABO AB
