## 整理原则

- **同类集中**：教材归教材，复习预习归一处，PDF 单独存放，根目录不再散落文件。
- **去重**：重复文件只保留最完整的一份（原 `lessons_markdown/` 与标准习题集重复，已合并删除）。
- **保链**：资料包内部保留原有 `./images/` 相对路径，图片链接不失效。
