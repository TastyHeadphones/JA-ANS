## 第36课

### 36课1 ～ようになります／なりました

#### I. 变形填空

1. 日本に来てから、勉強する時間が（長いです→　）なりました。  
答案：長く  
解析：い形容词「長い」变副词形「長く」后接「なりました」。

2. 日本に来てから、日本語が少し（上手です→　）なりました。  
答案：上手に  
解析：な形容词「上手」接续为「上手に」。

3. 日本に来てから、日本料理が（好きです→　）なりました。  
答案：好きに  
解析：名词/な形容词后用「に」表示状态变化。

4. 日本に来てから、漢字が（書けます→　）なりました。  
答案：書けるように  
解析：能力变化用「可能形 + ようになる」。

5. 日本に来てから、日本語で電話が（かけられます→　）なりました。  
答案：かけられるように  
解析：同上，可能形＋ようになる。

6. 日本に来てから、テレビのニュースが（わかります→　）なりました。  
答案：わかるように  
解析：動詞辞書形＋ようになる，表示“变得能懂”。

#### II. できるようになったこと（开放题）

题目：
- 「日本へ来たとき、できなかったこと」
- 「できるようになったこと」

示例答案：

1. 日本へ来たとき、ひらがなしか読めませんでした。今は新聞も読めるようになりました。  
解析：前后对比用「～ませんでした／～ようになりました」。

2. 日本へ来たとき、刺身が食べられませんでした。今は刺身が食べられるようになりました。  
解析：可能否定到可能肯定的变化。

3. 日本へ来たとき、日本語で電話がかけられませんでした。今は日本語で電話がかけられるようになりました。  
解析：「可能形＋ようになりました」固定表达能力提升。

题型：根据图写句子。

解析：年龄点＋能力变化。

2. （8）歳のとき、ピアノが弾けるようになりました。  
解析：同一语法套入不同动作。

3. （10）歳のとき、自転車に乗れるようになりました。  
解析：「乗れる」为「乗る」可能形。

### 36课3 「可能動詞」なくなりました

#### I. 日本へ来てから、できなくなったこと

1. 母の料理があまり食べられなくなりました。  
解析：「可能形＋なくなりました」表示“变得不能/不再能”。

2. 夜ふかしができなくなりました。  
解析：行为能力/许可变化也可用此结构。

3. 毎日ゲームができなくなりました。  
解析：同上，侧重“以前可以，现在不行”。

#### II-1. 絵を見て書きましょう

题目（原题）：例「今は川で泳げなくなりました。」の形で、絵を見て文を作る。  
配图：  
![36课3配图](./images/36-37/36-3_illustration.png)

题目例：今は川で泳げなくなりました。  
答案：今は川が汚くなったので、川で泳げなくなりました。  
解析：在基本句后补充原因，使表达完整。

#### II-2. あなたの国の「川・町・山」はどうですか

答案：わたしの町は車が増えて、前より静かに歩けなくなりました。  
解析：用「～なくなりました」描述环境变化。

### 36课4 ～ように／～ないように

#### I. 連線して文を作る

1. はっきり聞こえるように、**大きい声で言ってください**。  
解析：目标“听清楚”对应“请大声说”。

2. よく見えるように、**眼鏡をかけます**。  
解析：目的“看清楚”对应“戴眼镜”。

3. 病気にならないように、**野菜を食べています**。  
解析：避免结果用「ないように」。

4. 電話番号を忘れないように、**メモしてください**。  
解析：防止遗忘的手段是做记录。

5. すぐ連絡できるように、**携帯電話を持っています**。  
解析：为了“随时联系”而“带手机”。

#### II. 作文

1. 速く泳げるように、毎日30分練習しています。  
解析：目的＋持续行为。

2. 字をまちがえないように、書く前に必ず確認しています。  
解析：「ないように」强调防止错误。

### 36课5 ～ように／～ないようにしています

题目：どうすればお金持ちになれますか。  
词语：買い物、貯金、タクシー、服、新聞や雑誌、カード、食事

示例答案：

1. わたしは毎月、貯金するようにしています。  
解析：用「～ようにしています」表示习惯化行为。

2. 服はできるだけ安い店で買うようにしています。  
解析：结合「できるだけ」表达节约策略。

3. タクシーに乗らないようにしています。  
解析：避免高消费用「ないようにしています」。

4. カードで買い物しすぎないようにしています。  
解析：控制支出的习惯表达。

5. 新聞や雑誌は図書館で読むようにしています。  
解析：降低生活成本的自然表达。

题目：あなたの国では、どんなことに注意しなければなりませんか。

![36课6配图](./images/36-37/36-6_illustration.png)

示例答案：

1. 公共の場所では、列に並ぶようにしてください。  
解析：请求/提醒对方遵守规则。

2. 電車の中で大声で話さないようにしてください。  
解析：禁止或规劝用「ないようにしてください」。

3. 横断歩道を渡るようにしてください。  
解析：强调应做行为。

4. 夜おそく一人で危ない道を歩かないようにしてください。  
解析：安全提醒场景常用此句型。

### 36课7 文法チェックシート

1. 新聞が（読める／読む）ように、漢字を勉強しています。  
答案：読める  
解析：「ように」前接可能形。

2. テレビのニュースが（わかれる／わかる）ようになりました。  
答案：わかる  
解析：动词辞书形接「ようになる」。

3. 漢字を勉強しましたから、新聞が（読める／読む）ようになりました。  
答案：読める  
解析：能力变化用可能形。

4. 病気になりましたから、旅行に（行けるように／行けなく）なりました。  
答案：行けなく  
解析：因生病而“不能去”，用「行けなくなりました」。

5. 今すぐ事務所へ（来て／来るようにして）ください。  
答案：来て  
解析：直接请求当前动作时用「Vてください」。

6. 資料を1枚（ずつ／ぐらい）取ってください。  
答案：ずつ  
解析：「1枚ずつ」表示“每人/每次一张”。

7. 日本語が（かなり／必ず）話せるようになりました。  
答案：かなり  
解析：程度副词修饰能力。

8. 日本の習慣（に／を）慣れました。  
答案：に  
解析：「慣れる」接助词「に」。

9. このスーパーでは7時（に／を）過ぎたら、食べ物が安くなります。  
答案：を  
解析：「時刻＋を過ぎる」。

10. 来週テニスの試合（に／を）出る予定です。  
答案：に  
解析：「試合に出る」固定搭配。

## 第37课

### 37课1 受身の作り方

题目：（　）にひらがなを書きましょう。

1. く → （か）  
解析：I类动词受身把「う段」改为「あ段」+れる。

2. ぐ → （が）  
解析：同上。

3. す → （さ）  
解析：同上。

4. つ → （た）  
解析：同上。

5. ぶ → （ば）  
解析：同上。

6. む → （ま）  
解析：同上。

7. る（I类）→ （ら）  
解析：「とる→とられる」。

8. る（II类）→ （ら）  
解析：II类「去る＋られる」，如「たべる→たべられる」。

9. する → （さ）  
解析：「する→される」。

10. くる → （こ）  
解析：「くる→こられる」。

### 37课2 受身の練習

#### Iグループ

1. 使う → 使われる  
2. 行う → 行われる  
3. 聞く → 聞かれる  
4. 書く → 書かれる  
5. 話す → 話される  
6. 壊す → 壊される  
7. 汚す → 汚される  
8. 待つ → 待たれる  
9. 呼ぶ → 呼ばれる  
10. 選ぶ → 選ばれる  
11. 読む → 読まれる  
12. 踏む → 踏まれる  
13. 頼む → 頼まれる  
14. とる → とられる  
15. 作る → 作られる  
16. しかる → しかられる  
解析：I类统一规则「ない形词干＋れる」。

#### IIグループ

1. 褒める → 褒められる  
2. 調べる → 調べられる  
3. 建てる → 建てられる  
4. まちがえる → まちがえられる  
5. 捨てる → 捨てられる  
6. 見る → 見られる  
解析：II类「る」前词干后直接加「られる」。

#### IIIグループ

1. 発明する → 発明される  
2. 発見する → 発見される  
3. 輸入する → 輸入される  
4. 注意する → 注意される  
5. 来る → 来られる  
6. 連れて来る → 連れて来られる  
解析：III类不规则变化，单独记忆。

### 37课3 ～(ら)れます「人(A)は人(B)に～(ら)れます」

题目（原题）：絵の①〜④を見て、「わたしは〜に〜られました」の形で書く。  
配图：  
![37课3配图](./images/36-37/37-3_illustration.png)

（按图写句）

1. わたしは母にしかられました。  
解析：受害/被动对象作主语。

2. わたしは松本さんにさそわれました。  
解析：人物提示图的标准被动态写法（图示题可有多种自然答案）。

3. わたしは兄に足をふまれました。  
解析：「足を踏む」在受身中常作受害表达。

4. わたしは部長に呼ばれました。  
解析：被上级叫去，用受身最自然。

### 37课4 ～(ら)れます「人(A)は人(B)に[名詞]を～(ら)れます」

题目（原题）：絵の①〜③を見て、「わたしは〜に[名詞]を〜られました」の形で書く。  
配图：  
![37课4配图](./images/36-37/37-4_illustration.png)

1. わたしは松本さんに中村さんを紹介されました。

2. わたしは中村さんにお茶を入れられました。  
解析：表示“被要求/被迫做某事”的被动语感。

3. わたしは部長に仕事を頼まれました。  
解析：「頼む→頼まれる」，常见工作场景。

### 37课5 ～られます「人(A)は人(B)にAの物を～(ら)れます」

题目（原题）：I. 絵を見て受身文を書く。II. しかられたこと・とられたことなど自分の経験を書く。  
配图：  
![37课5配图](./images/36-37/37-5_illustration.png)

#### I. 看图写句（示例答案）

1. わたしは泥棒にかばんをとられました。  
解析：所有物受损失，典型受害被动。

2. わたしは弟にケーキを食べられました。  
解析：自己的东西被别人做掉。

3. わたしは雨にせんたくものをぬらされました。  
解析：自然现象也可作施事者。

4. わたしは妹に人形をこわされました。  
解析：受害对象作主语。

5. わたしは鳥に魚をとられました。  
解析：图中“鸟叼走鱼”的自然表达。

6. わたしは友達にノートを見られました。  
解析：隐私被看也常用受身。

#### II. 自分のことを書きましょう

示例答案：

1. わたしは子どものとき、父にしかられました。  
解析：与题目示例一致的经历表达。

2. わたしは中学生のとき、友達に自転車をとられました。  
解析：继续练习「AはBにAの物を～られました」。

### 37课7 ～年に／～(場所)で～が～られました

题目（原题）：地図の情報（年・場所・出来事）を使って受身文を完成し、IIで自分の国のことを書く。

![37课7配图](./images/36-37/37-7_illustration.png)

#### I. 日本の例を完成

解析：按同样句型扩展到本国事实。

### 37课8 ～られています

题目（原题）：図（輸入割合）を見て「～から輸入されています」「～で作られています」を書く。  
配图：  
![37课8配图](./images/36-37/37-8_illustration.png)

#### I. 図を見て書く（示例答案）

1. えびはインドネシアから輸入されています。  
解析：例句，表示现在持续状态。

2. 小麦はアメリカから輸入されています。  
解析：「～から輸入されています」是固定搭配。

3. 牛肉はアメリカやオーストラリアから輸入されています。  
解析：并列来源可用「や」。

4. 米は日本で作られています。  
解析：国内生产可用「～で作られています」。

#### II. 自分の国の食べ物（物）について

示例答案：

コーヒー豆はブラジルから輸入されています。  
解析：继续套用状态被动结构。

### 37课9 「受身」と「～てもらう」の区別

题目（原题）：I. 文脈に合うほう（受身 / ～てもらう）を選ぶ。II. 指定動詞で文を作る。  
配图：  
![37课9配图](./images/36-37/37-9_illustration.png)

#### I. 选出正确形式

1. 友達に自転車を（修理してもらいました）。  
解析：受益（别人帮我修）用「てもらう」。

2. 友達に自転車を（壊されました）。  
解析：受害（被弄坏）用受身。

3. 空港で知らない人に荷物を（持って行かれました）。  
解析：被拿走是受害被动。

4. 母に大切な本を（捨てられました）。  
解析：书被丢弃，主语受损失。

#### II. 「～られる」か「～てもらう」を使って書く

1. （買う）わたしは父にカメラを**買ってもらいました**。  
解析：父亲给我买，受益表达。

2. （とる）わたしは泥棒にカメラを**とられました**。  
解析：被偷，受害被动。

3. （見る）先生に日本語で書いたレポートを**見てもらいました**。  
解析：请老师看，是受益请求。

4. （見る）友達に試験を**見られました**。  
解析：考试内容被看见，受害语感。

### 37课10 文法チェックシート

1. わたしは毎朝、子どもを（起きます／起こします）。  
答案：起こします  
解析：对象是“子どもを”，要用他动词「起こす」。

2. ライト兄弟は飛行機を（発明しました／発見しました）。  
答案：発明しました  
解析：飞机是“发明”而不是“发现”。

3. 英語はいろいろな国で（話せています／話されています）。  
答案：話されています  
解析：一般事实被动态。

4. この絵はピカソ（について／によって）かかれました。  
答案：によって  
解析：施事者用「～によって」。

5. 事務所の人に住所を（聞かられました／聞かれました）。  
答案：聞かれました  
解析：受身正确形为「聞かれました」。

6. 兄はわたしの日記を読みました。（受身改写）  
答案1：わたしの日記は兄に読まれました。  
答案2：わたしは日記を兄に読まれました。  
解析：两种改写都正确，前者是中性被动，后者是受害被动。

7. 2000年にシドニー（で／に）オリンピックが開かれました。  
答案：で  
解析：活动举办地点用「で」。

8. ミラーさんの誕生日のパーティー（に／を）招待されました。  
答案：に  
解析：「～に招待される」。

9. ビールは麦（から／で）造られます。  
答案：から  
解析：原料形态变化大时常用「から」。

10. あの家は木（から／で）作られています。  
答案：で  
解析：材料形态基本可见时常用「で」。
