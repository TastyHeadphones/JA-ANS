# 第13课 おもいだす ―「夏のにおい」

> 对应书内第13课（PDF 约第75–80页）。
> 这一课的主轴是「家族との思い出」「におい」，重点学习"保持……的状态""应当（义务）""那个人嘛——肯定……""以……为契机"等表达。

## 本课主旨

作者回忆少年时与父亲在夏日傍晚一起干园艺活的场景：在父亲的指示下满头大汗地除草、运土，母亲喊"吃饭啦"，父亲让他们最后浇一次水再洗手吃饭——就在那一瞬间，干透的土地"咕咚咕咚"地把水喝进去，散发出独特的"傍晚泥土与草木复活的气味"，那是作者一生难忘的"夏のにおい"。课文借此引出：味道、气味这种非语言的记忆，是连接家人情感最深的纽带。

## 导入问题

课前会先围绕下面三个问题展开：

- 闻到什么味道时，你会想起家人？
- 你最怀念的童年场景是什么样的？
- 你的家乡有什么"季节的气味"？

## 文法要点

### 1. `〜まま`

- 用法：表示"在……的状态下保持着，不变地"。
- 接续：
  - `Vた形 + まま`
  - `Vない形 + ないまま`
  - `Nの + まま`、`い形 + まま`、`な形〜な + まま`
- 例句：
  - 疲れていて、電気をつけたまま寝てしまった。
  - 「ここはこんなふうに」と指示されるままに、汗まみれで働いた。
  - 靴を履いたまま部屋に入らないでください。
- 说明：
  - 强调"前一动作的状态没有改变就持续下去"。
  - `指示されるままに` 表示"任凭对方指挥"，常带被动语义。

### 2. `〜べき／〜べきだ／〜べきではない`

- 用法：表示"理应做（不该做）"，从义务、道德、当然的角度判断。
- 接续：
  - `V辞書形 + べき／べきだ`
  - `する → するべき／すべき`
- 例句：
  - 学生はできるだけ多くの時間を勉強のために使うべきです。
  - 正確には「夕方のにおい」と言うべきだろう。
  - 約束を破るべきではない。
- 说明：
  - 比 `〜なければならない` 多了"道义上应该"的语气。
  - 否定形：`〜べきではない`，表示"不应该……"。

### 3. `〜のことだから`

- 用法：「以那个人的性格／习惯来看，肯定……」。带有"我了解他"的语气。
- 接续：`N + のことだから`
- 例句：
  - 田中さんのことだから、もう来るよ。
  - 真面目な彼女のことだから、必ず時間通りに来るだろう。
  - 心配性な母のことだから、また電話してきた。
- 说明：
  - 用于已知对方性格/习惯，根据这一点做出推测。
  - 后句常是符合此人特征的判断。

### 4. `〜きっかけで / 〜をきっかけに（して）`

- 用法：「以……为契机／开端」，表示某事成为变化或决心的起点。
- 接续：
  - `N + がきっかけで`
  - `N + をきっかけに（して）`
  - `V普通形 + のがきっかけで`
- 例句：
  - 母の入院がきっかけで、医者になろうと決めた。
  - 手術をきっかけに、たばこをやめることにした。
  - 留学をきっかけに、日本文化に興味を持つようになった。
- 说明：
  - 强调"由某个事件开始，引发了变化"。
  - `〜がきっかけで` 较口语，`〜をきっかけに（して）` 偏书面。

## 课文梳理

1. 夏天的傍晚，父亲下班一回家就把作者和哥哥叫到庭院，让他们帮忙干园艺活。
2. 在父亲指示下，他们俩满头大汗地除草、运土，正觉得累的时候，母亲喊"开饭啦"。
3. 父亲说："最后浇一次水，再洗手吃饭"——就在水洒到地上的那一瞬间，独特的"夏天的气味"扑面而来。
4. 那是被烈日晒干的草木复活的气味，也是干透的土地咕咚咕咚地"喝水"的气味。
5. 那一瞬间凝结的气味、父亲的指示、母亲的呼唤，构成了作者对父亲与夏天最深的记忆。

## 练习题

### 一、词汇练习

1. `庭の草を（　　　　　　　）のは大変だ。`
   - 答案：`抜く`
   - 解析：`草を抜く` = 拔草。

2. `子供のころのにおいは今でも（　　　　　　　）。`
   - 答案：`懐かしい`
   - 解析：`懐かしい` = 怀念的。

3. `父の（　　　　　　　）に従って庭仕事をした。`
   - 答案：`指示`
   - 解析：`指示` = 指示。

### 二、语法练习

4. `テレビをつけた（　　　　　　　）寝てしまった。`
   - 答案：`まま`
   - 解析：`〜たまま` = 保持……状态。

5. `学生はもっと勉強する（　　　　　　　）だ。`
   - 答案：`べき`
   - 解析：`〜べきだ` = 应当。

6. `田中さん（　　　　　　　）、もう来ているはずだ。`
   - 答案：`のことだから`
   - 解析：`〜のことだから` = 以那个人的特点来看。

7. `この事故が（　　　　　　　）に、ヘルメットをかぶるようになった。`
   - 答案：`きっかけ`
   - 解析：`〜をきっかけに` = 以……为契机。

### 三、短答题

8. `「夏のにおい」とは何のにおいだと書かれているか？`
   - 参考答案：是被一整天的烈日烤干的草木在被浇水时"复活"的气味；同时也是干透的土地咕咚地吸收水时散发出的傍晚泥土的气味。

9. `用 〜まま 写一句关于自己的句子。`
   - 参考答案：`昨日は疲れていて、化粧を落とさないまま寝てしまった。`

## 本课复习要点

- 能用 `〜まま` 表达"保持……的状态不变"。
- 能区分 `〜べきだ`（道义应当）和 `〜なければならない`（义务）。
- 能用 `〜のことだから` 基于对方性格做出推测。
- 能用 `〜きっかけで／〜をきっかけに` 表达"以……为契机"。
