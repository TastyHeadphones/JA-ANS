# 第18课 はなれる ―「ふるさと」

> 对应书内第18课（PDF 约第107–112页）。
> 这一课的主轴是「<ruby>過疎化<rt>かそか</rt></ruby>とふるさと」，重点学习"就因为……所以……""非常……""即使……""不……就不能……"等表达。

## 本课主旨

作者作为过疎化现状调查员，到处走访渐渐凋零的渔村和农村。在一个小渔村，老人讲起年轻时鱼多到"船都要沉"、一天就能挣过去工厂一个月的钱，于是不顾未来拼命捕捞；如今"鱼没了，年轻人也没了，什么都没了"。农村也一样——松茸、竹笋满山可采的日子早已成历史，年轻人为找工作离开了村子，留下老人。地方政府推出"新故乡"计划吸引人回流，但很多人住不下来——不能融入当地的传统、生活、人际关系便又离开了。少子高龄化的浪潮无差别地涌向城乡，"如果现在不行动，将来连地图上都会消失"——这一念头始终萦绕在作者心头。

## 导入问题

课前会先围绕下面三个问题展开：

- 人口太多会带来什么问题？
- 人口太少又会带来什么问题？
- 你出生的地方人口是在增加还是减少？

## 文法要点

### 1. `〜たばかりに`

- 用法：「就因为……（这一点），结果……（不好的事）」。强调消极结果由某一不慎引起。
- 接续：`Vた形 / い形〜かった / な形〜だった + ばかりに`
- 例句：
  - あまり<ruby>考え<rt>かんがえ</rt></ruby>ずに<ruby>約束<rt>やくそく</rt></ruby>したばかりに、<ruby>後で<rt>あとで</rt></ruby><ruby>大変<rt>たいへん</rt></ruby>なことになってしまった。
  - <ruby>約束<rt>やくそく</rt></ruby>の<ruby>時間<rt>じかん</rt></ruby>を<ruby>間違え<rt>まちがえ</rt></ruby>たばかりに、わざわざ<ruby>遠く<rt>とおく</rt></ruby>から<ruby>来て<rt>きて</rt></ruby>くれた<ruby>友達<rt>ともだち</rt></ruby>に<ruby>会え<rt>あえ</rt></ruby>なかった。
  - <ruby>相手<rt>あいて</rt></ruby>の<ruby>電話番号<rt>でんわばんごう</rt></ruby>を<ruby>聞い<rt>きい</rt></ruby>ておかなかったばかりに、せっかくの<ruby>仕事<rt>しごと</rt></ruby>の<ruby>機会<rt>きかい</rt></ruby>を<ruby>失っ<rt>うしなっ</rt></ruby>た。
- 说明：
  - "ばかりに" 后接的多是说话人感到遗憾／后悔的结果。
  - 与 `〜せいで` 接近，但 `〜ばかりに` 更书面、更带"就那么一点小事就……"的口气。

### 2. `〜て／でならない`

- 用法：「……得不得了」「非常……」。形容感情或感觉极强、自己控制不住。
- 接续：
  - 形容词て形 + ならない
  - V使役被动 / Vてしまう<ruby>型<rt>かた</rt></ruby> + ならない
- 例句：
  - <ruby>試合<rt>しあい</rt></ruby>に<ruby>負け<rt>まけ</rt></ruby>たときは<ruby>残念<rt>ざんねん</rt></ruby>でならなかった。
  - <ruby>母<rt>はは</rt></ruby>が<ruby>亡く<rt>なく</rt></ruby>なってから、<ruby>父<rt>ちち</rt></ruby>はひとりで<ruby>食事<rt>しょくじ</rt></ruby>をするのがさびしくてならないようだ。
  - <ruby>子供<rt>こども</rt></ruby>は<ruby>友達<rt>ともだち</rt></ruby>のゲームがほしくてならないらしい。
- 说明：
  - 主语多是说话人或第三方明显表现出来的感情。
  - 与 `〜てたまらない` 同义；`〜てならない` 多用于感情、自然涌上的情感。

### 3. `〜としても／たとえ〜としても`

- 用法：「即使……也……」，假设条件下的让步。
- 接续：
  - `普通形 + としても`
  - 常与 `たとえ` 搭配
- 例句：
  - たとえ<ruby>会社<rt>かいしゃ</rt></ruby>を<ruby>辞め<rt>やめ</rt></ruby>させられたとしても、すぐに<ruby>別の<rt>べつの</rt></ruby><ruby>仕事<rt>しごと</rt></ruby>が<ruby>見つ<rt>みつ</rt></ruby>けられると<ruby>思う<rt>おもう</rt></ruby>。
  - いったんそこで<ruby>生活<rt>せいかつ</rt></ruby>を<ruby>始め<rt>はじめ</rt></ruby>たとしても、<ruby>多く<rt>おおく</rt></ruby>は<ruby>地域<rt>ちいき</rt></ruby>に<ruby>根<rt>ね</rt></ruby>づかない。
  - たとえ<ruby>雨<rt>あめ</rt></ruby>が<ruby>降っ<rt>ふっ</rt></ruby>たとしても、<ruby>試合<rt>しあい</rt></ruby>は<ruby>行い<rt>おこない</rt></ruby>ます。
- 说明：
  - 与 `〜ても` 接近，但 `〜としても` 更书面、强调"假设这样也……"。

### 4. `〜ないことには〜ない`

- 用法：「不……就不能……」「除非……否则就不会……」，强调前条件是后结果的必要条件。
- 接续：`Vない形 + ないことには`、`N + でないことには`
- 例句：
  - <ruby>親<rt>おや</rt></ruby>に<ruby>確認<rt>かくにん</rt></ruby>を<ruby>取ら<rt>とら</rt></ruby>ないことには、<ruby>出張<rt>しゅっちょう</rt></ruby>の<ruby>日取り<rt>ひどり</rt></ruby>を<ruby>決め<rt>きめ</rt></ruby>るわけにはいかない。
  - 今、<ruby>手<rt>て</rt></ruby>を<ruby>打た<rt>うた</rt></ruby>ないことには、<ruby>大変<rt>たいへん</rt></ruby>なことになる。
  - <ruby>自分<rt>じぶん</rt></ruby>で<ruby>見な<rt>みな</rt></ruby>いことには、<ruby>信じ<rt>しんじ</rt></ruby>られない。
- 说明：
  - 与 `〜なければ〜ない` 接近，但 `〜ないことには〜ない` 更强调"绝对必要的前提"。

## 课文梳理

1. 在一个小渔村，作者问老人"现在还去海上吗？"——老人短短一句"不去"。两人并坐着看夕阳染红的大海。
2. 过了一会儿，老人开口讲起年轻时的事："船都要沉似的，鱼那么多"——工厂一个月的工资，他们一天就能挣到。乘那股势头，根本没想未来就拼命捕捞。
3. 夕阳沉入海平面，老人淡淡地说："鱼没了、年轻人也没了、什么都没了。"
4. 农村也一样——过去"松茸、竹笋多到吃不完"，工作减少后年轻人去了城市，剩下的都是老人。祭典、近所小店早已消失。"再也回不到从前了……寂寞得难受。"
5. 政府推动地方自治体振兴过疎村；有的地方为吸引向往田园生活的年轻人或退休者，连住房、工作都准备好了。但人来了又留不住——融不进当地传统、生活和人际关系便又离开。
6. 少子高龄化的浪潮无差别冲向都市与过疎村；有的地区甚至可能从地图上消失。"现在不行动，就来不及"——这个念头一直萦绕在作者奔走调查的头脑里。

## 练习题

### 一、词汇练习

1. `若いころは船が（　　　　　　　）ほど魚がとれた。`
   - 答案：`沈む`
   - 解析：`沈む` = 沉。

2. `地方自治体が過疎化した村を（　　　　　　　）しようとしている。`
   - 答案：`活性化`
   - 解析：`活性化する` = 振兴。

3. `早く（　　　　　　　）を打たないと地図から消えてしまう。`
   - 答案：`手`
   - 解析：`手を打つ` = 采取措施。

### 二、语法练习

4. `あまり考えずに約束した（　　　　　　　）、大変なことになった。`
   - 答案：`ばかりに`
   - 解析：`〜たばかりに` = 就因为……结果……。

5. `母が亡くなって、父はさびしく（　　　　　　　）。`
   - 答案：`てならない`
   - 解析：`〜てならない` = ……得不得了。

6. `たとえ雨が降った（　　　　　　　）、試合は行います。`
   - 答案：`としても`
   - 解析：`〜としても` = 即使……。

7. `親に確認しない（　　　　　　　）、決められない。`
   - 答案：`ことには`
   - 解析：`〜ないことには〜ない` = 不……就不能……。

### 三、短答题

8. `本文中、なぜ「新しいふるさと作り」がうまくいかないのか？`
   - 参考答案：因为搬过去的人很多无法融入当地的传统、生活和人际关系，缺乏归属感，于是不久又离开了村子，无法真正在地区扎根。

9. `用 〜たばかりに 写一个自己的例句。`
   - 参考答案：`寝坊したばかりに、大事な試験に遅刻してしまった。`

## 本课复习要点

- 能用 `〜たばかりに` 强调"就因为一点小事，导致不好的结果"。
- 能用 `〜て／でならない` 表达"……得不得了"的强烈感情。
- 能用 `〜としても／たとえ〜としても` 表达"即使……也……"。
- 能用 `〜ないことには〜ない` 表达"不……就不能……"的必要条件。

### 課文：ふるさと

**①** <ruby>地方<rt>ちほう</rt></ruby><ruby>自治体<rt>じちたい</rt></ruby>から<ruby>過疎化<rt>かそか</rt></ruby>の<ruby>現状<rt>げんじょう</rt></ruby><ruby>調査<rt>ちょうさ</rt></ruby>を<ruby>依頼<rt>いらい</rt></ruby>され、<ruby>各地<rt>かくち</rt></ruby>の<ruby>漁村<rt>ぎょそん</rt></ruby>や<ruby>農村<rt>のうそん</rt></ruby>を<ruby>歩い<rt>あるい</rt></ruby>て<ruby>回っ<rt>まわっ</rt></ruby>ている。

- **译文**：受地方自治体委托调查过疏化现状，我走遍各地的渔村和农村进行调查。
- **语法**：`〜られる`（被动，受到……的委托）；`〜ている`（持续进行中）
- **单词**：`自治体`（名词，地方自治体）；`過疎化`（名词，人口过疏化）；`調査`（名词，调查）；`依頼する`（サ变，委托）

**②** ある<ruby>小さ<rt>ちいさ</rt></ruby>な<ruby>漁村<rt>ぎょそん</rt></ruby>で、<ruby>老<rt>ろう</rt></ruby><ruby>人<rt>ひと</rt></ruby>と<ruby>並ん<rt>ならん</rt></ruby>で<ruby>夕陽<rt>ゆうひ</rt></ruby>に<ruby>染まる<rt>そまる</rt></ruby><ruby>海<rt>うみ</rt></ruby>を<ruby>眺め<rt>ながめ</rt></ruby>ていると、<ruby>老<rt>ろう</rt></ruby><ruby>人<rt>ひと</rt></ruby>が「<ruby>今<rt>いま</rt></ruby>は<ruby>海<rt>うみ</rt></ruby>に<ruby>出な<rt>でな</rt></ruby>い」とぽつりと<ruby>言った<rt>いった</rt></ruby>。

- **译文**：在某个小渔村，我与一位老人并排眺望着被夕阳染红的大海，老人轻轻说了一句："现在不出海了。"
- **语法**：`〜ていると`（正在……时，引出下文变化）；`〜に染まる`（被染成……颜色）
- **单词**：`夕陽`（名词，夕阳）；`染まる`（自动词，染上）；`眺める`（他动词，眺望）；`ぽつりと`（副词，独自轻声地）

---

**③** 「<ruby>若い<rt>わかい</rt></ruby>ころは、<ruby>船<rt>ふね</rt></ruby>が<ruby>沈む<rt>しずむ</rt></ruby>ほど<ruby>魚<rt>さかな</rt></ruby>がとれた。<ruby>工場<rt>こうじょう</rt></ruby>の<ruby>一<rt>いち</rt></ruby>か<ruby>月分<rt>がつぶん</rt></ruby>の<ruby>給料<rt>きゅうりょう</rt></ruby>が、<ruby>一日<rt>いちにち</rt></ruby>でとれたもんだ」

- **译文**："年轻的时候，鱼多到船都快沉了。工厂一个月的工资，我们一天就能挣到。"
- **语法**：`〜ほど`（程度，到……程度）；`〜もんだ`（口语化的 `ものだ`，带感慨语气）
- **单词**：`沈む`（自动词，沉）；`給料`（名词，工资）；`とれる`（自动词，能捕获）

---

**④** その<ruby>勢い<rt>いきおい</rt></ruby>に<ruby>乗って<rt>のって</rt></ruby>、<ruby>先の<rt>さきの</rt></ruby>ことも<ruby>考え<rt>かんがえ</rt></ruby>ずに<ruby>夢中<rt>むちゅう</rt></ruby>で<ruby>魚<rt>さかな</rt></ruby>をとり<ruby>続け<rt>つづけ</rt></ruby>たばかりに、<ruby>今<rt>いま</rt></ruby>は<ruby>魚<rt>さかな</rt></ruby>も<ruby>若者<rt>わかもの</rt></ruby>も<ruby>何も<rt>なにも</rt></ruby>かも<ruby>姿<rt>すがた</rt></ruby>を<ruby>消し<rt>けし</rt></ruby>てしまった。

- **译文**：就因为乘着那股势头，什么都不想地拼命捕鱼，如今鱼没了、年轻人没了、一切都消失了。
- **语法**：`〜たばかりに`（就因为……，导致不好结果）；`〜ずに`（不……地）；`〜てしまう`（带遗憾感的完成）
- **单词**：`勢い`（名词，势头）；`夢中で`（沉迷地/拼命地）；`姿を消す`（惯用，消失）；`何もかも`（一切、全部）

---

**⑤** <ruby>農村<rt>のうそん</rt></ruby>の<ruby>事情<rt>じじょう</rt></ruby>も<ruby>変わ<rt>かわ</rt></ruby>らない。<ruby>若者<rt>わかもの</rt></ruby>は<ruby>仕事<rt>しごと</rt></ruby>を<ruby>求め<rt>もとめ</rt></ruby>て<ruby>都会<rt>とかい</rt></ruby>に<ruby>出て<rt>でて</rt></ruby><ruby>行き<rt>いき</rt></ruby>、<ruby>残さ<rt>のこさ</rt></ruby>れたのは<ruby>お年寄り<rt>おとしより</rt></ruby>ばかりで、さびしくてならない。

- **译文**：农村的情况也一样。年轻人离开去城市找工作，留下的都是老人，寂寞得不得了。
- **语法**：`〜てならない`（……得不得了，感情强烈自然涌现）；`〜ばかり`（只有……）
- **单词**：`農村`（名词，农村）；`求める`（他动词，求取）；`都会`（名词，城市）；`残される`（被留下）；`お年寄り`（名词，老人）

---

**⑥** たとえ<ruby>若者<rt>わかもの</rt></ruby>が<ruby>一度<rt>いちど</rt></ruby><ruby>帰っ<rt>かえっ</rt></ruby>てきたとしても、<ruby>地域<rt>ちいき</rt></ruby>の<ruby>伝統<rt>でんとう</rt></ruby>や<ruby>生活<rt>せいかつ</rt></ruby>になじめないことには、<ruby>根づく<rt>ねづく</rt></ruby>ことができない。

- **译文**：即使年轻人回来了，只要无法融入当地的传统和生活，就没办法扎根。
- **语法**：`たとえ〜としても`（即使……也……）；`〜ないことには〜ない`（不……就不能……）
- **单词**：`伝統`（名词，传统）；`なじむ`（自动词，融入/适应）；`根づく`（自动词，扎根）

**⑦** 今、<ruby>手<rt>て</rt></ruby>を<ruby>打た<rt>うた</rt></ruby>ないことには、やがてこの<ruby>村<rt>むら</rt></ruby>は<ruby>地図<rt>ちず</rt></ruby>から<ruby>消え<rt>きえ</rt></ruby>てしまう<ruby>恐れ<rt>おそれ</rt></ruby>がある。それを<ruby>思う<rt>おもう</rt></ruby>と、じっとしていられない。

- **译文**：如果现在不采取措施，这个村庄将来恐怕会从地图上消失。一想到这里，就坐立不安。
- **语法**：`〜ないことには〜ない`（不……就不能……）；`〜恐れがある`（有……的风险）；`〜ていられない`（无法静静地……）
