# 第3课 たべる ―「ごちそう」

> 对应书内第3课（PDF 约第25–30页）。
> 这一课的主轴是「食事と家族の関係」，重点学习"趁……（时机）做某事""只能说……""让某人做某事（表关怀）"等表达。

## 本课主旨

通过描写一个小学生独自对着电视吃晚饭、中学生姐姐加热母亲准备好的饭菜匆匆吃完再去学英语、父亲深夜才回家、母亲最后一个吃饭的当代家庭场景，反思"家人围坐一桌共享饭食"这一风景的逐渐消失。作者感慨，超市里食物充足、简单方便，但家人围坐的温暖渐成"怀念"。

## 导入问题

课前会先围绕下面三个问题展开：

- 你自己做饭吃吗？
- 你每天都能悠闲地吃饭吗？
- 你通常和谁一起吃饭？

## 文法要点

### 1. `〜うちに`

- 用法：在某种状态持续期间做某事，也用于不知不觉中发生了变化。
- 接续：
  - `V辞書形 + うちに`
  - `Vている + うちに`
  - `い形容詞 + うちに`
  - `な形容詞 + な + うちに`
  - `N + の + うちに`
- 例句：
  - 朝のうちに、宿題を終わらせてしまおう。
  - 本を読んでいるうちに、眠ってしまった。
  - 若いうちに、いろいろな経験をしておいたほうがいい。
- 说明：
  - 用于"趁着……还在，做某事"，或"在……期间，（不知不觉发生了Y）"。
  - `〜てしまった` 型的后续常表示"意外地、不知不觉地发生"。

### 2. `〜うちは`

- 用法：在某种状态持续期间，某个条件成立（有时含"一旦状态结束，就会改变"的对比语气）。
- 接续：
  - `V辞書形 + うちは`
  - `Vない形 + うちは`
  - `い形容詞 + うちは`
- 例句：
  - 朝のうちはすずしいが、昼になるとたいへん暑くなる。
  - 若いうちは分からないが、年を取るとわかる。
  - 漢字が読めないうちは大変だったが、今は好きな本が読める。
- 说明：
  - 与 `〜うちに` 的区别：`うちに` 侧重"趁……（期间）动作"，`うちは` 侧重"在……期间，某状态成立（后来会变化）"。

### 3. `〜(さ)せてあげる／やる`

- 用法：允许或让某人做某事（带有给予恩惠的语感）。
- 接续：
  - `V使役形（させて）+ あげる`（对外人或平辈）
  - `V使役形（させて）+ やる`（对晚辈、孩子、动物等）
- 例句：
  - 友達が辞書を持っていなかったので、使わせてあげました。
  - 将来のために、子供たちにいろいろなことを習わせてやりたいと思っています。
- 说明：
  - `あげる` 用于对等或往上；`やる` 用于对晚辈或亲子关系中的孩子。
  - 反过来请求对方让自己做，用 `〜(さ)せてもらえませんか／〜(さ)せていただけませんか`。

### 4. `〜としか〜ない`

- 用法：表示"除了……之外，没有别的话可说／别的方式可想"，强调唯一可能性。
- 接续：
  - `普通形（小句）+ としか + 言えない / 考えられない / 答えられない`
- 例句：
  - 試験は受けましたが、今はいっしょうけんめいやったとしか言えません。
  - ドアがあいているのはどうしてだろう。誰かが入ったとしか考えられない。
  - いくら「手伝ってほしい」と言われても、「時間があれば」としか答えられない。
- 说明：
  - 表示"唯一能说的""只能如此解释"，排除其他可能性。
  - 常与否定性结构 `〜としか〜ない` 呼应，不能说成 `〜としか〜ある`。

## 课文梳理

1. 一个小学生对着电视独自吃晚饭；中学生姐姐加热母亲早上备好的饭后急忙去学英语；父亲说今晚又会很晚；母亲最后一个人吃饭。
2. 这样的场景是"时代变了"的结果，家人围坐一桌的风景正在消失。
3. 过去的孩子们期待着"今天晚饭吃什么"，期待和家人一起吃饭。
4. 如今超市里食品充足、方便，但"妈妈花时间做饭"的场景越来越少。
5. 作者问：家人围坐一桌吃饭，已经只能说是"怀念的风景"了吗？

## 练习题

### 一、词汇练习

1. `朝の（　　　　　　　）に、買い物をすませておこう。`
   - 答案：`うち`
   - 解析：`朝のうちに` = 趁早上（还没过）。

2. `母は毎日、早めに夕食の（　　　　　　　）をしている。`
   - 答案：`準備`
   - 解析：`準備する` = 准备。

3. `スーパーには食べ物が（　　　　　　　）いる。`
   - 答案：`あふれて`
   - 解析：`あふれる` = 充满、溢出。

4. `家族でテーブルを（　　　　　　　）食事をする。`
   - 答案：`囲んで`
   - 解析：`囲む` = 围、围坐。

### 二、语法练习

5. `若い（　　　　　　　）に、いろいろな経験をしておこう。`
   - 答案：`うち`
   - 解析：趁着年轻，`〜うちに`。

6. `若い（　　　　　　　）はわからないが、年をとるとわかる。`
   - 答案：`うち`
   - 解析：在……期间是……，`〜うちは`。

7. `友達が辞書を持っていなかったので、使わ（　　　　　　　）あげた。`
   - 答案：`せて`
   - 解析：`使わせてあげた` = 让（对方）用了。

8. `試験は受けましたが、いっしょうけんめいやった（　　　　　　　）言えません。`
   - 答案：`としか`
   - 解析：`〜としか〜ない` = 只能说……。

9. `ドアが開いているのはどうしてだろう。誰かが入った（　　　　　　　）考えられない。`
   - 答案：`としか`
   - 解析：唯一的解释。

### 三、短答题

10. `本文中「なつかしい風景」指的是什么风景？`
    - 参考答案：家人全员围坐一桌，谈论当天发生的事，高声说笑的吃饭场景。

11. `用 〜うちに 写一句自己的例句。`
    - 参考答案：`忘れないうちに、メモしておこう。`

## 本课复习要点

- 能区分 `〜うちに`（趁……做某事 / 不知不觉发生）和 `〜うちは`（在……期间，某状态成立）。
- 能用 `〜(さ)せてあげる/やる` 表达"让（某人）做某事（为他好）"。
- 能用 `〜としか〜ない` 表达"只能……"的唯一可能性。
