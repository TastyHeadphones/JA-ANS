# 第2课 みつける ―「花の名前」

> 对应书内第2课（PDF 约第19–24页）。
> 这一课的主轴是「重新发现身边的事物」，重点学习"表达决心""习惯的持续""同时进行的动作""想做却没有做到"等表达。

## 本课主旨

叙述者来日本后开始不再走路，习惯了坐地铁。某天地铁停运，不得不步行，途中在公园里发现了许多认识的花名，看到了远处的山和沿途店铺，途中还有人微微点头打招呼。走到站后，感到"走路真好"，决定以后有时间就走路。

## 导入问题

课前会先围绕下面三个问题展开：

- 你经常乘坐哪种交通工具？
- 你平时走路的机会多吗？
- 你有没有在常走的路上发现了什么新鲜事物的经历？

## 文法要点

### 1. `〜もせずに〜てしまった`

- 用法：表示"连X都没做（却）做了Y"，常有"疏忽／违反常规"的语感。
- 接续：
  - `V（ます形词干）+ もせずに + V + てしまった`
- 例句：
  - 弟は昼ごはんを食べると、かたづけもせずに出かけてしまった。
  - 忙しくて、返事もせずに電話を切ってしまった。
- 说明：
  - `せず` 是 `する` 的古典否定形（`しないで`）。
  - `〜もせずに` 强调"连（理应做的事）都没做"。
  - 和 `〜ないで〜てしまった` 意思相近，`〜もせずに` 语气更强烈。

### 2. `〜ことにする／した`

- 用法：表示说话人主动做出决定。
- 接续：
  - `V辞書形 / Vない形 + ことにする`
- 例句：
  - あしたからはもっと早く起きることにします。
  - 大学の入学試験を受けるので、今年は日本にいることにしました。
- 说明：
  - `ことにした` 是"已做出决定"，`ことにする` 是"（现在）决定"。
  - 强调主观意志，区别于 `ことになる`（被动地被决定）。

### 3. `〜ことにしている／していた`

- 用法：表示已经养成的个人习惯或持续坚守的规则。
- 接续：
  - `V辞書形 / Vない形 + ことにしている`
- 例句：
  - 体のために、甘い物は食べないことにしています。
  - 学生時代には、毎日2時間図書館で勉強することにしていた。
- 说明：
  - 与 `ことにする` 的区别在于 `〜ことにしている` 强调持续性、习惯性。
  - `〜ようにしている` 也表示习惯，但更强调"努力去做"，`〜ことにしている` 更强调"规定自己做"。

### 4. `〜ながら`（同时进行）

- 用法：表示两个动作同时进行，主动作在后句。
- 接续：
  - `Vます形（去ます）+ ながら + メインの動作`
- 例句：
  - 父は昼間仕事をしながら、夜は大学で勉強していたそうです。
  - 音楽を聞きながら、宿題をする。
- 说明：
  - 两个动作的主语必须相同。
  - 后句才是主要动作，前句是次要的"伴随动作"。

### 5. `〜ようと思いながら、〜できない`

- 用法：表示"虽然想做……但一直没能做"，带有遗憾或无奈感。
- 接续：
  - `V意向形 + と思いながら + （まだ / なかなか）Vできない`
- 例句：
  - お世話になった先生にあいさつに行こうと思いながら、時間がなくて、なかなか行けません。
  - 返事を書こうと思いながら、ずっと書けないでいる。
- 说明：
  - 常与 `なかなか〜ない`、`〜ずにいる` 搭配，加强"无法完成"的感觉。
  - 区别于 `〜ようとしない`（故意不做），这里是"想做但没做到"。

## 课文梳理

1. 叙述者来日本后，因交通便利，渐渐不再走路。
2. 某天地铁停运，听到周围有人说"走到前面两站就通车了"，于是决定步行。
3. 走了一会儿，发现大公园，花坛里开着许多花，旁边标有名字，认出了学校学过的花名，感到很高兴。
4. 路上还看到了熟悉的电视广告店铺、远处的山景，和路人愉快的会釈。
5. 到站时略有些累，但感到走路真值得，决定以后有时间就步行。

## 练习题

### 一、词汇练习

1. `バスや電車などの（　　　　　　　）を利用する。`
   - 答案：`交通機関`
   - 解析：`交通機関` = 交通工具/设施的统称。

2. `方向の（　　　　　　　）をつけて歩き始めた。`
   - 答案：`見当`
   - 解析：固定搭配 `見当をつける`。

3. `初めは歩くのが（　　　　　　　）なかった。`
   - 答案：`気が進ま`
   - 解析：`気が進まない` = 不情愿，固定惯用语。

4. `通りすがりの人がにっこり笑って（　　　　　　　）してくれた。`
   - 答案：`会釈`
   - 解析：`会釈する` = 微微点头致意。

5. `公園にたくさんの花が（　　　　　　　）いた。`
   - 答案：`咲いて`
   - 解析：`咲く` = 花开，接 `〜ている` 表状态。

### 二、语法练习

6. `彼はごはんを食べると、かたづけも（　　　　　　　）出かけてしまった。`
   - 答案：`せずに`
   - 解析：`〜もせずに` = 连……都没做。

7. `健康のために、毎日30分歩く（　　　　　　　）にした。`
   - 答案：`こと`
   - 解析：主动决定用 `〜ことにする`。

8. `甘い物は食べない（　　　　　　　）にしている。`
   - 答案：`こと`
   - 解析：持续习惯用 `〜ことにしている`。

9. `音楽を聞き（　　　　　　　）、料理をしている。`
   - 答案：`ながら`
   - 解析：同时进行 `〜ながら`。

10. `手紙を書こうと思い（　　　　　　　）、なかなか書けない。`
    - 答案：`ながら`
    - 解析：`〜ようと思いながら〜できない` = 想做但没能做。

### 三、短答题

11. `为什么叙述者在公园里看到花名时感到高兴？`
    - 参考答案：因为认出了在学校学过的花名，感到学以致用、有了"找到"的发现感。

12. `用 ことにしている 写一句关于自己习惯的句子。`
    - 参考答案：`毎晩寝る前に日本語を30分勉強することにしています。`

## 本课复习要点

- 能区分 `〜もせずに〜てしまった`（连X都没做就做了Y）与 `〜ないで〜た`（没做X而做了Y）的语气差别。
- 能用 `〜ことにする` / `〜ことにしている` 表达一次性决定和持续习惯。
- 能用 `〜ながら` 描述两个同时进行的动作。
- 能用 `〜ようと思いながら〜できない` 表达"想做却没做到"的遗憾。
