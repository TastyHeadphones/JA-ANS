# 第1课 まなぶ -「なぞなぞ」

> 对应 PDF 第13-18页；本课正文在书内第1-6页。
> 这一课的主轴是「外国语学习」和「文化差异」，重点学习“状态变化”“第三者的情绪/愿望”“避免不良结果”“持续处于未完成状态”等表达。

## 本课主旨

这篇课文通过“孩子在英语课上讲到洗脸习惯差异”的小故事，说明学习外语不只是学单词和语法，也是在扩大看世界的方法。

## 导入问题

课前会先围绕下面三个问题展开：

- 除了日语，你还会说别的外语吗？
- 你喜欢学外语吗？
- 你在日语学习中，最近最困扰的是什么？

## 文法要点

### 1. `〜ようになる / 〜なくなる`

- 用法：表示能力、习惯、状态发生变化。
- 接续：
  - `V辞書形 + ようになる`
  - `Vない形 → なくなる`
- 例句：
  - 毎日練習して、漢字が少し読めるようになった。
  - 最近、夜遅くまで起きなくなった。
- 说明：
  - `〜ようになる` 强调“变成了某种状态”。
  - `〜なくなる` 强调“原来有的行为/状态消失了”。

### 2. `〜がる / 〜がっている`

- 用法：描述第三者表面上表现出来的情绪、愿望、反应。
- 接续：
  - `Vます形词干 + がる`
  - `い形容词词干 + がる`
  - 常见形式：`〜がっている`
- 例句：
  - 子どもは新しいゲームをほしがっている。
  - 友だちは暗い部屋をこわがっていた。
- 说明：
  - 不能直接用来描述说话人自己“想要/感觉”。
  - 常用于观察别人“看起来很想要”“看起来很害怕”。

### 3. `Vては / Vると + いけない`

- 用法：表示担心某种动作一发生，就会出现不好的结果。
- 接续：
  - `Vては + いけない`
  - `V辞書形 + と + いけない`
- 例句：
  - 子どもにいろいろ言いすぎては、いやがられるといけない。
  - 風邪をひくといけないから、早く寝た。
- 说明：
  - 这里不是单纯的“禁止”，而是“怕……就不好了”。
  - 常和 `と思って`、`から`、`ので` 一起使用。

### 4. `Vないで / Vずにいる`

- 用法：表示“持续处在没有做某事的状态中”。
- 接续：
  - `Vない形 + ないでいる`
  - `Vない形 + ずにいる`
- 例句：
  - 忙しくて、家族とゆっくり話せずにいる。
  - 何日も寝ないでいることはできない。
- 说明：
  - `〜ずにいる` 比 `〜ないでいる` 稍微书面一些。
  - 强调一种持续状态，不是一次性的“没有做”。

## 课文梳理

1. 母亲一开始担心孩子学英语会变得讨厌学习，也担心自己插嘴会让孩子不高兴。
2. 有一天，孩子提到英语课上老师让他们讨论“外国孩子睡前洗脸”这件事，母亲以为是又一个谜语。
3. 实际上，孩子是在讲一个文化差异的场景，老师借这个机会让学生注意不同国家的生活习惯。
4. 母亲听完后，意识到孩子不是只在学英语，而是在学习更宽广的世界。

## 练习题

### 一、词汇练习

1. `ある（　　）、子どもが学校でおもしろい話をしてくれた。`
   - 答案：`日`
   - 解析：`ある日` = 某一天。

2. `親があまり（　　）を出しすぎると、子どもはいやがる。`
   - 答案：`口`
   - 解析：固定表达 `口を出す`。

3. `来週の旅行が今から（　　）だ。`
   - 答案：`楽しみ`
   - 解析：`楽しみだ` = 期待、盼望。

4. `みんなが（　　）に意見を言ったので、話し合いが長くなった。`
   - 答案：`口々`
   - 解析：`口々に` = 你一言我一语地。

### 二、语法练习

5. `毎日練習して、漢字が少し読める（　　）なった。`
   - 答案：`ように`
   - 解析：能力变化用 `〜ようになる`。

6. `最近、夜遅くまでスマホを見ることが（　　）。`
   - 答案：`なくなった`
   - 解析：行为消失用 `〜なくなる`。

7. `子どもは新しいおもちゃをほし（　　）いる。`
   - 答案：`がって`
   - 解析：第三者的愿望用 `〜がる`。

8. `いろいろ言いすぎては、子どもにいやがられると（　　）と思う。`
   - 答案：`いけない`
   - 解析：担心不良结果时用 `Vては / Vると + いけない`。

9. `忙しくて、家族とゆっくり話せ（　　）いる。`
   - 答案：`ずに`
   - 解析：持续“没能做某事”的状态。

### 三、短答题

10. `这篇课文里，母亲最后为什么开始期待孩子继续讲故事？`
   - 参考答案：因为她意识到孩子不仅是在学英语，还在通过英语学习更广阔的世界，接触不同文化。

11. `你能用 `〜ようになる` 写一句话吗？`
   - 参考答案：`日本に来てから、少し日本語が話せるようになった。`

12. `你能用 `〜ないで / 〜ずにいる` 写一句话吗？`
   - 参考答案：`最近いそがしくて、友だちと会えずにいる。`

## 本课复习要点

- 能看懂并使用 `〜ようになる / 〜なくなる`。
- 能区分 `〜がる` 和说话人自己的主观感受。
- 能理解 `Vては / Vると + いけない` 的“担心不良结果”语气。
- 能用 `〜ないで / 〜ずにいる` 说出持续未完成状态。
