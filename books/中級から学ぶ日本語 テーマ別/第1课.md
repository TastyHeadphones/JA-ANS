# 第1课 まなぶ -「なぞなぞ」

> 对应 PDF 第13-18页；本课正文在书内第1-6页。
> 这一课的主轴是「外国语学习」和「文化差异」，重点学习“状态变化”“第三者的情绪/愿望”“避免不良结果”“持续处于未完成状态”等表达。

## 本课主旨

这篇课文通过“孩子在英语课上讲到洗脸习惯差异”的小故事，说明学习外语不只是学单词和语法，也是在扩大看世界的方法。

## 导入问题

课前会先围绕下面三个问题展开：

- 除了日语，你还会说别的外语吗？
- 你喜欢学外语吗？
- 你在日语学习中，最近最困扰的是什么？

## 文法要点

### 1. `〜ようになる / 〜なくなる`

- 用法：表示能力、习惯、状态发生变化。
- 接续：
  - `V辞書形 + ようになる`
  - `Vない形 → なくなる`
- 例句：
  - <ruby>毎日<rt>まいにち</rt></ruby><ruby>練習<rt>れんしゅう</rt></ruby>して、<ruby>漢字<rt>かんじ</rt></ruby>が<ruby>少し<rt>すこし</rt></ruby><ruby>読め<rt>よめ</rt></ruby>るようになった。
  - 最近、<ruby>夜遅く<rt>よるおそく</rt></ruby>まで<ruby>起き<rt>おき</rt></ruby>なくなった。
- 说明：
  - `〜ようになる` 强调“变成了某种状态”。
  - `〜なくなる` 强调“原来有的行为/状态消失了”。

### 2. `〜がる / 〜がっている`

- 用法：描述第三者表面上表现出来的情绪、愿望、反应。
- 接续：
  - `Vます形词干 + がる`
  - `い形容词词干 + がる`
  - 常见形式：`〜がっている`
- 例句：
  - <ruby>子ども<rt>こども</rt></ruby>は<ruby>新しい<rt>あたらしい</rt></ruby>ゲームをほしがっている。
  - <ruby>友<rt>とも</rt></ruby>だちは<ruby>暗い<rt>くらい</rt></ruby><ruby>部屋<rt>へや</rt></ruby>をこわがっていた。
- 说明：
  - 不能直接用来描述说话人自己“想要/感觉”。
  - 常用于观察别人“看起来很想要”“看起来很害怕”。

### 3. `Vては / Vると + いけない`

- 用法：表示担心某种动作一发生，就会出现不好的结果。
- 接续：
  - `Vては + いけない`
  - `V辞書形 + と + いけない`
- 例句：
  - <ruby>子ども<rt>こども</rt></ruby>にいろいろ<ruby>言い<rt>いい</rt></ruby>すぎては、いやがられるといけない。
  - <ruby>風邪<rt>かぜ</rt></ruby>をひくといけないから、<ruby>早く<rt>はやく</rt></ruby><ruby>寝た<rt>ねた</rt></ruby>。
- 说明：
  - 这里不是单纯的“禁止”，而是“怕……就不好了”。
  - 常和 `と思って`、`から`、`ので` 一起使用。

### 4. `Vないで / Vずにいる`

- 用法：表示“持续处在没有做某事的状态中”。
- 接续：
  - `Vない形 + ないでいる`
  - `Vない形 + ずにいる`
- 例句：
  - <ruby>忙しく<rt>いそがしく</rt></ruby>て、<ruby>家族<rt>かぞく</rt></ruby>とゆっくり<ruby>話せ<rt>はなせ</rt></ruby>ずにいる。
  - <ruby>何日<rt>なんにち</rt></ruby>も<ruby>寝な<rt>ねな</rt></ruby>いでいることはできない。
- 说明：
  - `〜ずにいる` 比 `〜ないでいる` 稍微书面一些。
  - 强调一种持续状态，不是一次性的“没有做”。

## 课文梳理

1. 母亲一开始担心孩子学英语会变得讨厌学习，也担心自己插嘴会让孩子不高兴。
2. 有一天，孩子提到英语课上老师让他们讨论“外国孩子睡前洗脸”这件事，母亲以为是又一个谜语。
3. 实际上，孩子是在讲一个文化差异的场景，老师借这个机会让学生注意不同国家的生活习惯。
4. 母亲听完后，意识到孩子不是只在学英语，而是在学习更宽广的世界。

## 练习题

### 一、词汇练习

1. `ある（　　）、子どもが学校でおもしろい話をしてくれた。`
   - 答案：`日`
   - 解析：`ある日` = 某一天。

2. `親があまり（　　）を出しすぎると、子どもはいやがる。`
   - 答案：`口`
   - 解析：固定表达 `口を出す`。

3. `来週の旅行が今から（　　）だ。`
   - 答案：`楽しみ`
   - 解析：`楽しみだ` = 期待、盼望。

4. `みんなが（　　）に意見を言ったので、話し合いが長くなった。`
   - 答案：`口々`
   - 解析：`口々に` = 你一言我一语地。

### 二、语法练习

5. `毎日練習して、漢字が少し読める（　　）なった。`
   - 答案：`ように`
   - 解析：能力变化用 `〜ようになる`。

6. `最近、夜遅くまでスマホを見ることが（　　）。`
   - 答案：`なくなった`
   - 解析：行为消失用 `〜なくなる`。

7. `子どもは新しいおもちゃをほし（　　）いる。`
   - 答案：`がって`
   - 解析：第三者的愿望用 `〜がる`。

8. `いろいろ言いすぎては、子どもにいやがられると（　　）と思う。`
   - 答案：`いけない`
   - 解析：担心不良结果时用 `Vては / Vると + いけない`。

9. `忙しくて、家族とゆっくり話せ（　　）いる。`
   - 答案：`ずに`
   - 解析：持续“没能做某事”的状态。

### 三、短答题

10. `这篇课文里，母亲最后为什么开始期待孩子继续讲故事？`
   - 参考答案：因为她意识到孩子不仅是在学英语，还在通过英语学习更广阔的世界，接触不同文化。

11. `你能用 `〜ようになる` 写一句话吗？`
   - 参考答案：`日本に来てから、少し日本語が話せるようになった。`

12. `你能用 `〜ないで / 〜ずにいる` 写一句话吗？`
   - 参考答案：`最近いそがしくて、友だちと会えずにいる。`

## 本课复习要点

- 能看懂并使用 `〜ようになる / 〜なくなる`。
- 能区分 `〜がる` 和说话人自己的主观感受。
- 能理解 `Vては / Vると + いけない` 的”担心不良结果”语气。
- 能用 `〜ないで / 〜ずにいる` 说出持续未完成状态。

### 課文：なぞなぞ

**①** <ruby>子ども<rt>こども</rt></ruby>が<ruby>英語<rt>えいご</rt></ruby>のクラスを<ruby>始め<rt>はじめ</rt></ruby>てから、<ruby>私<rt>わたし</rt></ruby>は<ruby>少し<rt>すこし</rt></ruby><ruby>心配<rt>しんぱい</rt></ruby>していた。

- **译文**：自从孩子开始上英语课，我就有点担心。
- **语法**：`〜てから`（自从……之后）；`〜ていた`（持续状态的过去进行）
- **单词**：`始める`（他动词，开始）；`心配する`（担心）

---

**②** <ruby>英語<rt>えいご</rt></ruby>がむずかしくて<ruby>子ども<rt>こども</rt></ruby>がいやがるようになると<ruby>困る<rt>こまる</rt></ruby>と<ruby>思った<rt>おもった</rt></ruby>のだ。

- **译文**：我担心英语太难，孩子可能会变得讨厌学习。
- **语法**：`〜ようになる`（状态变化，变得……）；`〜と`（条件假设，如果……就）；`〜のだ`（解释说明语气）
- **单词**：`いやがる`（自动词，厌恶、抗拒）；`困る`（自动词，苦恼、为难）

---

**③** それでも、<ruby>親<rt>おや</rt></ruby>があまり<ruby>口<rt>くち</rt></ruby>を<ruby>出し<rt>だし</rt></ruby>ては、いやがられるといけないと<ruby>思って<rt>おもって</rt></ruby>、だまって<ruby>見て<rt>みて</rt></ruby>いた。

- **译文**：尽管如此，我想着如果家长插嘴太多、惹孩子烦，就不好了，所以我默默地看着。
- **语法**：`Vては + いけない`（担心某行为导致不好结果）；`〜ていた`（持续状态）
- **单词**：`それでも`（副词，尽管如此）；`口を出す`（惯用语，插嘴）；`いやがられる`（`いやがる` 的被动形，被讨厌）

- **译文**：有一天，孩子从学校回来说：”喂，妈妈，来个谜语。”
- **语法**：`ある日`（某一天）；`〜てくる`（动作方向朝向说话人）
