## 使用建议

- 先看 `课程详细总结.md` 了解全书脉络与高频语法主题分组。
- 学具体一课时打开对应单课文件，按"重点词汇 → 文法要点 → 课文梳理 → 练习题 → 复习要点"逐节学习。
- 词汇均附两个例句；语法均附用法、接续、例句、说明四要素。
