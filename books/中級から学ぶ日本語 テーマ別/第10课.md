# 第10课 なれる ―「腕時計」

> 对应书内第10课（PDF 约第67–72页）。
> 这一课的主轴是「習慣と変化」，重点学习"无论……都""随着……""非但没有……反而""难怪……""并不是说……就……"等表达。

## 本课主旨

叙述者来日本后，一开始对日本人对时间的极度重视感到不适应——约定的时间分秒不差，迟到五分钟也要道歉。慢慢地，叙述者也养成了守时的习惯，开始戴上了手表（腕時計）。通过这个小小的变化，叙述者意识到：适应新文化不是放弃自己，而是扩展自己——人是可以改变的，而且随着时间推移，改变往往越来越自然。

## 导入问题

课前会先围绕下面三个问题展开：

- 你是一个守时的人吗？
- 来到日本后，你的生活习惯有什么变化吗？
- 你觉得"适应新环境"是容易的还是难的？

## 文法要点

### 1. `〜（よ）うと`（無論・不管）

- 用法：表示"无论（做什么）都……"，含有"即使……也不影响结果"的语气。
- 接续：
  - `V意向形 + と`
  - `V意向形 + と Vまい（否定意向）と`（无论做还是不做）
- 例句：
  - 何時間待とうと、彼は来ないだろう。
  - 雨が降ろうと、試合は行います。
  - 行こうと行くまいと、あなたの自由です。
- 说明：
  - `〜（よ）うと` 表示"即使做了某事，也无法改变结果"，含让步语气。
  - `〜（よ）うとも` 语气略强，更书面。
  - 与 `〜ても` 意思相近，`〜（よ）うと` 更正式。

### 2. `〜につれて`

- 用法：表示随着某事物的变化，另一事物也相应地变化，"随着……"。
- 接续：
  - `V辞書形 + につれて`
  - `N + につれて`
- 例句：
  - 日本での生活が長くなるにつれて、習慣も変わってきた。
  - 時間が経つにつれて、守時の大切さがわかってきた。
  - 経験が増えるにつれて、考え方も深くなる。
- 说明：
  - `〜につれて` 表示两件事同步变化，前件是基准变化，后件是随之发生的变化。
  - 与 `〜にしたがって` 意思相近，但 `〜にしたがって` 也有"按照……做"的意思，`〜につれて` 只用于变化。

### 3. `〜どころか、かえって〜`

- 用法：表示"非但不……，反而……"，结果与预期完全相反。
- 接续：
  - `V / い形 / な形 / N + どころか`
- 例句：
  - 薬を飲んだどころか、かえって具合が悪くなった。
  - 慣れようとしたどころか、かえってストレスが増えた。
  - 時間を節約できるどころか、かえって時間がかかった。
- 说明：
  - `どころか` 表示"非但如此……连……都不是"，后续的 `かえって` 强调反向结果。
  - 注意：`〜どころか` 也可单独用于"不仅如此，甚至……"，需根据上下文判断。

### 4. `〜わけだ`

- 用法：表示"难怪……""这就说明……"，对某事给出合理解释或当然结论。
- 接续：
  - `普通形 + わけだ`
  - `な形容詞（な）+ わけだ`
  - `N（の）+ わけだ`
- 例句：
  - 毎日練習したのだから、上手になったわけだ。
  - 5年も日本にいれば、日本語が上手なわけだ。
  - いつも早起きしているから、健康なわけだ。
- 说明：
  - 说话人基于已知信息推导出理所当然的结论，有"怪不得……"的语感。
  - 与 `〜はずだ`（预期推断）区分：`わけだ` 更强调"这就解释了"的理解感。

### 5. `〜からといって〜わけではない`

- 用法：表示"虽然……但并不因此……"，否定从某条件直接得出的必然结论。
- 接续：
  - `普通形 + からといって + V / い形 / な形 + わけではない`
- 例句：
  - 日本に慣れたからといって、自分の文化を忘れるわけではない。
  - 遅刻したからといって、悪い人というわけではない。
  - お金があるからといって、幸せなわけではない。
- 说明：
  - 用于提醒"不能因为A就断定B"，有打破简单因果逻辑的效果。
  - 常与 `だからといって`（即便如此）搭配使用。

## 课文梳理

1. 叙述者来日本后，发现日本人对时间极度重视，迟到哪怕五分钟也要认真道歉。
2. 起初感到不适应，觉得如此严格太拘束；尝试适应（适応しようとした）却一时难以改变。
3. 随着在日本生活时间增长（長くなるにつれて），叙述者不知不觉地开始注意时间，并买了一块手表。
4. 手表成为变化的象征：戴上手表，意味着内心也认可了"守时"这件事。
5. 结论：适应了日本的习惯，并不意味着放弃了自己原来的文化（からといって〜わけではない）；改变是扩展，不是替换。

## 练习题

### 一、词汇练习

1. `日本に来てから、時間を（　　　　　）するようになった。`
   - 答案：`意識`
   - 解析：`意識する` = 意识、注意。

2. `（　　　　　）のうちに、守時が自然になっていた。`
   - 答案：`いつの間`
   - 解析：`いつの間にか` = 不知不觉地。

3. `新しい習慣が（　　　　　）につくまで、時間がかかる。`
   - 答案：`身`
   - 解析：`身につく` = 养成（习惯）。

4. `異文化に（　　　　　）するには、まず観察することが大切だ。`
   - 答案：`適応`
   - 解析：`適応する` = 适应。

### 二、语法练习

5. `何時間待（　　　　　）と、彼は来ないと思う。`
   - 答案：`とう（＝待とう）`
   - 解析：`〜（よ）うと` = 无论……都……；`待つ` 的意向形是`待とう`。

6. `日本での生活が長くなる（　　　　　）、習慣も変わってきた。`
   - 答案：`につれて`
   - 解析：`〜につれて` = 随着……。

7. `薬を飲んだ（　　　　　）、かえって具合が悪くなった。`
   - 答案：`どころか`
   - 解析：`〜どころか、かえって〜` = 非但……，反而……。

8. `毎日練習していたのだから、上手になった（　　　　　）だ。`
   - 答案：`わけ`
   - 解析：`〜わけだ` = 难怪……、这就说明……。

9. `日本に慣れた（　　　　　）、自分の文化を忘れる（　　　　　）ではない。`
   - 答案：`からといって`、`わけ`
   - 解析：`〜からといって〜わけではない` = 虽然……但并不因此……。

### 三、短答题

10. `文中で、「腕時計」はどんな意味の象徴として使われていましたか？`
    - 参考答案：手表象征着叙述者适应了日本的"守时"文化、内心接受了"重视时间"这一价值观的变化。

11. `用 〜につれて 写一句话，描述某种随时间的变化。`
    - 参考答案：`日本語の勉強を続けるにつれて、少しずつ自信がついてきた。`

## 本课复习要点

- 能用 `〜（よ）うと` 表达"无论……都……"（让步语气）。
- 能用 `〜につれて` 表达"随着……也……"（两件事同步变化）。
- 能用 `〜どころか、かえって〜` 表达"非但没有……反而……"（结果与预期相反）。
- 能用 `〜わけだ` 表达"难怪……""这就说明……"（合理结论/理解）。
- 能用 `〜からといって〜わけではない` 表达"虽然……但并不因此……"（打破简单因果推断）。
