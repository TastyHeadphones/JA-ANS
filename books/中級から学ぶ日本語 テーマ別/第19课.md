# 第19课 かなえる ―「ふたつの夢」

> 对应书内第19课（PDF 约第113–118页）。
> 这一课的主轴是「本当の豊かさとは何か」，重点学习"……暂且不论""只要……就……""从……（推断）""有没有办法……？"等表达。

## 本课主旨

作者参加自治会与留学生的交流会。"来日本后吃惊的事"中，有人回答"日本孩子的梦想不同"——一句让会场不解的话。留学生解释：自己家乡的孩子梦想是"政治家、运动员、医生、老师"，日本小学生答的也差不多。但他感到"梦想不同"是因为——他家乡的孩子是"明知不可能也要追"的梦，而日本孩子是"觉得只要努力就能实现"的梦。在医疗设施缺乏的家乡，连救自家人的命都做不到；终日劳作的父母身影是孩子唯一的现实——所以"梦是贫困的反面"。作者反思：日本孩子物质丰富却没有那种"眼睛闪光、笑容明亮"的样子——真正的丰饶藏在留学生家乡孩子的眼神里。

## 导入问题

课前会先围绕下面三个问题展开：

- 你小时候的梦想是什么？
- 现在还有那个梦想吗？
- "丰饶"对你来说意味着什么？

## 文法要点

### 1. `〜はともかく（として）`

- 用法：「……暂且不论，先说……」。把一项放一边，先讨论另一项。
- 接续：`N + はともかく（として）`
- 例句：
  - どの大学に入るかはともかく、まず大学で何を学びたいか考えるべきだ。
  - 兄は外見はともかくとして、性格が良く信頼できる人です。
  - 費用はともかくとして、調子が悪いなら早く医者に見せたほうがいい。
- 说明：
  - 用于"把A的话题暂时搁置，重点谈B"。
  - 后续多为"重要的是B" 之意。

### 2. `〜さえ〜ば`

- 用法：「只要……就……」，强调一个最低条件即可达成结果。
- 接续：
  - `N + さえ + V ば形`
  - `Vます形（去ます）+ さえすれば`
  - `い形〜く + さえあれば`、`な形 / N + でさえあれば`
- 例句：
  - 体がじょうぶでさえあれば、自分のやりたいことができるはずだ。
  - 努力さえすれば夢がかなう可能性がある。
  - お金さえあれば、何でも買える。
- 说明：
  - 与一般"〜ば"的条件不同，"〜さえ〜ば" 强调"只这一点就足够"。

### 3. `〜ことから`

- 用法：「从……（这一点）来看／因为……」。基于某事实进行命名或推断。
- 接续：`普通形 + ことから`、`な形〜な / N + である + ことから`
- 例句：
  - この犬は毛が白いことから、「シロ」と名前がつけられました。
  - 医療施設がないことから、病気になっても身内の命を救えないことがある。
  - 約束を破ったことから、信用を失った。
- 说明：
  - 表示从某事实推出的结果、命名理由或评价。
  - 偏书面用法。

### 4. `〜ないものか／〜ないものだろうか`

- 用法：「难道没办法……吗？／能不能……？」，表达说话人强烈期望或希望寻找办法。
- 接续：`Vない形（可能形居多） + ないものか／ないものだろうか`
- 例句：
  - 最近ずっと残業が続いている。もう少し早く帰れないものだろうか。
  - もっと楽な生活はできないものか、と夢を描く。
  - この問題を解決する方法はないものだろうか。
- 说明：
  - 含强烈情绪，常用于书面或自言自语。
  - 后续多用 `〜だろうか` 增强反问感。

## 课文梳理

1. 作者参加自治会与留学生的交流会。司会役问"来日本后吃惊的事是什么？"
2. "交通方便""街道安静""没垃圾"之类的回答之后，一位留学生说："日本孩子的梦想（和我们家乡的）不同。"会场一头雾水，他便补充说明。
3. 自己家乡的孩子梦想是"政治家、运动员、医生、老师"——日本小学生答的差不多。但"不同"在于：自己家乡的孩子是"明知不可能也要追的梦"，日本孩子是"觉得只要努力就能实现的梦"。
4. 在医疗设施缺乏的家乡，连救亲人的命都做不到；只能看到不停劳作的父母——所以孩子靠"政治家或医生"的梦想来想象摆脱困境。教育也不够，没人伸出援手时，做政治家或医生的日子根本不会来。"梦是贫困的反面"。
5. 作者回想自己旅行过那些"贫困"地区，那里孩子的眼睛是闪光的、笑容是明亮的。日本孩子没有那样的表情；虽物质富足，却总有什么东西没被满足。
6. 真正的丰饶藏在那双闪闪发光的眼睛和明亮的笑容里。日本的丰饶只是表面，不是真品——这一念头不止一次袭来。留学生希望大家"严肃看待这种被眼神和笑容掩盖的贫困"——这份恳切让作者心痛。

## 练习题

### 一、词汇练习

1. `留学生の話を聞いて、首を（　　　　　　　）。`
   - 答案：`かしげた`
   - 解析：`首をかしげる` = 不解地歪头。

2. `物質的に（　　　　　　　）ていても、心が満たされない。`
   - 答案：`恵まれ`
   - 解析：`恵まれる` = 拥有、被惠及。

3. `夢は厳しさの（　　　　　　　）だ。`
   - 答案：`裏返し`
   - 解析：`裏返し` = 反面。

### 二、语法练习

4. `どの大学に入るか（　　　　　　　）、何を学ぶか考えるべきだ。`
   - 答案：`はともかく（として）`
   - 解析：`〜はともかく` = 暂且不论。

5. `努力（　　　　　　　）すれば、夢がかなう可能性がある。`
   - 答案：`さえ`
   - 解析：`〜さえ〜ば` = 只要……就……。

6. `毛が白い（　　　　　　　）、「シロ」と名づけられた。`
   - 答案：`ことから`
   - 解析：`〜ことから` = 由于……。

7. `もう少し早く帰れない（　　　　　　　）。`
   - 答案：`ものだろうか`
   - 解析：`〜ないものだろうか` = 难道不能……。

### 三、短答题

8. `「夢は、厳しさの裏返しです」とはどういう意味か？`
   - 参考答案：正因为生活艰苦、看不到光明，孩子们才会拼命想象"如果能成为政治家或医生"的梦想；越是严酷的现实，越激发出越大的梦想——梦其实是现实痛苦的镜像。

9. `用 〜さえ〜ば 写一个自己的例句。`
   - 参考答案：`時間さえあれば、毎日運動したいと思っている。`

## 本课复习要点

- 能用 `〜はともかく（として）` 暂时搁置一项、聚焦另一项。
- 能用 `〜さえ〜ば` 强调"只要这一点足够了"。
- 能用 `〜ことから` 基于事实做出推断或命名。
- 能用 `〜ないものか／〜ないものだろうか` 表达强烈期望或反问。
