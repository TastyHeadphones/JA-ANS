## 第1課 まなぶ ―「なぞなぞ」

### 本课核心目标

通过孩子在小学学习英语、回家讲述课堂上的「なぞなぞ」（谜语）这件事，理解中级阶段如何描述「状态变化」「想做某事的愿望」「不希望发生的担心」「在没做某事的状态中」。

### 重点词汇

| 词 | 词性 | 释义 | 例句 |
|---|---|---|---|
| 心配（しんぱい）スル | 名/サ变 | 担心 | 子供が遅れるんじゃないかと心配している。 |
| 口を出す（くちをだす） | 惯用 | 多嘴、插嘴 | 親が口を出すと、子供は嫌がる。 |
| いやがる | 五段 | 表现出讨厌、不愿意 | 子供は注射をいやがる。 |
| 場面（ばめん） | 名 | 场面、场景 | テレビでこわい場面が出てきた。 |
| 相手（あいて） | 名 | 对方、对象 | 小学生相手にむずかしい説明はできない。 |
| 口々に（くちぐちに） | 副 | 你一言我一语地 | 子供たちは口々に意見を言い始めた。 |
| 手を止める（てをとめる） | 惯用 | 停下手中的事 | 料理の手を止めて子供の話を聞いた。 |
| 学ぶ（まなぶ） | 五段 | 学习、学到 | この経験から多くのことを学んだ。 |
| 広げる（ひろげる） | 一段 | 拓展、扩大 | 旅行をすると、ものの見方が広げられる。 |
| なぞなぞ | 名 | 谜语 | 子供と一緒になぞなぞをするのは楽しい。 |
| 文化（ぶんか） | 名 | 文化 | 国によって食事の文化が違う。 |

### 文法详细总结

#### 1. `〜ようになる / 〜なくなる`

- 用法：表示能力或习惯发生变化，「变得能……／变得不再……」。
- 接续：`V辞書形 + ようになる`、`Vない形（去い）+ くなる` / `Vないように + なる`。
- 例句：友だちに教えてもらって、自転車にのれるようになりました。
- 翻译：朋友教了我之后，我学会骑自行车了。

#### 2. `〜ように + する／しない／なる`

- 用法：努力（不）让某种状态实现／发生。
- 接续：`V辞書形 / Vない形 + ようにする`。
- 例句：父は仕事をやめてから、運動するようになりました。
- 翻译：父亲退休后开始变得有锻炼的习惯。

#### 3. `〜たがる / 〜たがっている`

- 用法：客观叙述第三方"想做某事"的样子。
- 接续：`Vます形（去ます）+ たがる`。
- 例句：ランさんはご両親に会いたがっているんだ。
- 翻译：兰小姐想见父母。

#### 4. `〜てはいけない / 〜といけない`

- 用法：「（如果……）就糟糕了／不行」，常用于说明「为了避免某种坏结果而……」的语境。
- 接续：`Vて + は + いけない`、`V辞書形 + と + いけない`。
- 例句：かぜを引いてはいけないから、あたたかい服を着たほうがいい。
- 翻译：为了不感冒，最好穿暖和点儿。

#### 5. `〜ないで／ずに + いる`

- 用法：处于"没做某事"的持续状态。
- 接续：`Vない形 + ないで／ずに + いる`。
- 例句：いそがしくて、家族とゆっくり話す時間も持てずにいます。
- 翻译：因为太忙，没能和家人好好说话。

### 例题

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 1-1 | 友だちに教えてもらって、自転車にのれる（　　）なりました。 | ように | 能力的获得用「〜ようになる」。 |
| 1-2 | ジョンさんは新しいパソコンを買い（　　　　　）ている。 | たがっ | 第三人称愿望用「〜たがっている」。 |
| 1-3 | 子供がさびしがる（　　）いけないと思って、おもちゃで遊ばせておく。 | と | 「〜といけない」表示担心的结果。 |
| 1-4 | 何日も寝（　　）にいることはできません。 | ず | 状态延续「ずにいる」。 |
| 1-5 | 仕事をやめてから、運動する（　　　　　　）なりました。 | ように | 习惯变化用「〜ようになる」。 |

## 第2課 みつける ―「花の名前」

### 本课核心目标

通过"地铁停运后改走路回家、在路上发现了很多花"的故事，体会「在某段时间内做某事」「让别人有机会做某事」「除某种说法之外没别的话可说」等表达。

### 重点词汇

| 词 | 词性 | 释义 | 例句 |
|---|---|---|---|
| 便利（べんり）ナ | ナ形 | 方便 | 地下鉄ができて、町が便利になった。 |
| 困る（こまる） | 五段 | 为难、不知所措 | 道に迷って困っている。 |
| 疲れる（つかれる） | 一段 | 累 | 一日中歩いて、足が疲れた。 |
| 周り（まわり） | 名 | 周围 | 周りの人の話が耳に入った。 |
| 見当をつける（けんとうをつける） | 惯用 | 估摸、判断个大概 | 方向の見当をつけて歩き始めた。 |
| 咲く（さく） | 五段 | 开（花） | 公園にたくさんの花が咲いている。 |
| 横（よこ） | 名 | 旁边 | 花の横に名前が書いてあった。 |
| 会釈（えしゃく） | 名/サ变 | 微微点头致意 | すれちがう人ににっこり会釈する。 |
| 宣伝（せんでん）スル | 名/サ变 | 宣传 | テレビでよく宣伝している店。 |
| 景色（けしき） | 名 | 风景 | 山の上から見る景色はすばらしい。 |
| 気が進まない（きがすすまない） | 惯用 | 不太愿意 | はじめは歩くのが気が進まなかった。 |
| 利用（りよう）スル | 名/サ变 | 利用、使用 | バスや電車を利用する。 |

### 文法详细总结

#### 1. `〜のに、〜てしまった`

- 用法：「明明……却（不情愿地／意外地）发生了Y」，常带轻微遗憾或意外感。
- 接续：`普通形 + のに、〜てしまった`。
- 例句：いつもはいっしょにかたづけるのに、弟は私にかたづけさせてしまった。
- 翻译：明明平时是一起收拾的，弟弟却让我一个人收拾了。

#### 2. `〜うちに`

- 用法：在某种状态持续期间做（或发生了）某事。
- 接续：`V辞書形 + うちに`、`Vている + うちに`、`い形/な形/N + うちに`。
- 例句：朝のうちに、宿題を終わらせてしまおう。
- 翻译：趁早上把作业做完吧。

#### 3. `〜うちは`

- 用法：「在……期间是……（但变化后就不同了）」，常与对比的后句呼应。
- 接续：`V辞書形 + うちは`、`Vない形 + うちは`。
- 例句：漢字が読めないうちは大変でしたが、今は好きな本が読めるようになりました。
- 翻译：不会读汉字时很辛苦，现在能读自己喜欢的书了。

#### 4. `〜(さ)せてあげる／やる`

- 用法：让／允许某人做某事（带恩惠语气）。
- 接续：`Vさせて + あげる／やる`。
- 例句：将来のために、子供たちにいろいろなことを習わせてやりたい。
- 翻译：为了孩子的未来，想让他们学各种各样的东西。

### 例题

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 2-1 | 朝の（　　　）に、宿題をやってしまおう。 | うち | "在……期间"用「うちに」。 |
| 2-2 | 子供たちにいろいろなことを習わ（　　　　　　）てやりたい。 | せ（させ） | 使役形+てやる。 |
| 2-3 | 漢字が読めない（　　　　）は大変でしたが、今は好きな本が読めます。 | うち | 状态期间用「うちは」。 |
| 2-4 | 友だちが辞書を持っていなかったので、使（　　　　　）あげました。 | わせて | 使役てあげる。 |
| 2-5 | いつもはいっしょに帰るのに、急いで帰っ（　　　　　　）。 | てしまった | のに＋てしまった。 |

## 第3課 たべる ―「ごちそう」

### 本课核心目标

围绕"现代家庭一起吃饭的场景越来越少"展开，学习「下决心」「让对方做某事」「只能说……」等关键表达。

### 重点词汇

| 词 | 词性 | 释义 | 例句 |
|---|---|---|---|
| 準備（じゅんび）スル | 名/サ变 | 准备 | 母が朝のうちに夕食の準備をした。 |
| 温める（あたためる） | 一段 | 加热 | 冷めたごはんを温める。 |
| 変わる（かわる） | 五段 | 改变、变化 | 時代が変われば、食事の風景も変わる。 |
| 姿（すがた） | 名 | 身影、样子 | みんなで食卓を囲む姿が少なくなった。 |
| 昔（むかし） | 名 | 从前 | 昔は家族みんなで食事をしたものだ。 |
| 囲む（かこむ） | 五段 | 围、围着 | 家族でテーブルを囲む。 |
| 簡単（かんたん）ナ | ナ形 | 简单 | 簡単にできる料理を作る。 |
| あふれる | 一段 | 充满、溢出 | スーパーには食べ物があふれている。 |
| 栄養（えいよう） | 名 | 营养 | 栄養のバランスを考えた食事。 |
| 声を上げる（こえをあげる） | 惯用 | 发出（笑、喊等）声音 | 声を上げて笑う。 |
| 風景（ふうけい） | 名 | 风景、景象 | なつかしい風景。 |

### 文法详细总结

#### 1. `〜ことにする`

- 用法：说话人主动做出决定。
- 接续：`V辞書形 / Vない形 + ことにする`。
- 例句：あしたからはもっと早くおきることにします。
- 翻译：从明天起决定起更早。

#### 2. `〜ことにしている`

- 用法：持续保持的个人习惯、个人规则。
- 接续：`V辞書形 / Vない形 + ことにしている`。
- 例句：寝る前にスマホを見ないことにしている。
- 翻译：我决定（一直坚持）睡前不看手机。

#### 3. `〜(さ)せる`（再次复习——本课通过对比"过去 vs 现在"使用）

- 用法：使役，「让／使／命令／允许某人做某事」。
- 接续：`Vさせる`形。
- 例句：「食事を楽しみにする子供たちに何を食べさせてやろうか」と母親は考えた。
- 翻译：母亲想，"该给期待吃饭的孩子们吃些什么呢"。

#### 4. `〜としか〜ない`

- 用法：「只能说……，只能想成……」，表示其他可能性被排除。
- 接续：`普通形 + としか + V否定形`。
- 例句：試験は受けましたが、今はいっしょうけんめいやったとしか言えません。
- 翻译：我考完了，现在只能说尽力了。

### 例题

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 3-1 | 健康のために、毎朝歩く（　　　　　）にしました。 | こと | 主动决定用「〜ことにする」。 |
| 3-2 | 私は毎日6時に起きる（　　　　　）にしています。 | こと | 习惯用「〜ことにしている」。 |
| 3-3 | 子供にいろいろな経験を（　　　　　　）てやりたい。 | させ | 使役てやる。 |
| 3-4 | 「いっしょうけんめいやった」（　　　　　　）言えません。 | としか | 「〜としか〜ない」固定搭配。 |
| 3-5 | ドアがあいているのは、誰かが入った（　　　　　　）考えられない。 | としか | 同上。 |

## 第4課 たとえる ―「猫に小判」

### 本课核心目标

通过日语谚语（特别是与「猫」相关的谚语）介绍「比喻」，并学习「程度比较」「不能想象」「请求别人不做某事」等表达。

### 重点词汇

| 词 | 词性 | 释义 | 例句 |
|---|---|---|---|
| 紹介（しょうかい）スル | 名/サ变 | 介绍 | ことわざを学生に紹介する。 |
| 目が回る（めがまわる） | 惯用 | 忙得团团转 | 目が回るほど忙しい。 |
| 手を貸す（てをかす） | 惯用 | 帮忙、出手 | 困っている人に手を貸す。 |
| のんびりスル | 副/サ变 | 悠闲 | 公園で猫がのんびり昼寝している。 |
| 役に立つ（やくにたつ） | 惯用 | 有用 | 辞書はとても役に立つ。 |
| 額（ひたい） | 名 | 额头；引申"狭小的地方" | 猫の額ほどの庭。 |
| たとえる | 一段 | 比喻、比方 | 狭い土地を猫の額にたとえる。 |
| 小判（こばん） | 名 | 江户时代金币 | 「猫に小判」のことわざ。 |
| 価値（かち） | 名 | 价值 | 価値のある物。 |
| ずっと | 副 | 一直；远远地（比……） | 馬は人とずっと生活してきた動物だ。 |
| きっと | 副 | 一定 | みなさんの文化にも、きっと同じようなことわざがあると思う。 |
| ぜひ | 副 | 务必、一定 | ぜひ紹介してください。 |

### 文法详细总结

#### 1. `〜ほど〜はない／いない`

- 用法：「没有比……更……的」，最高级表达。
- 接续：`N + ほど + N + はない／いない`。
- 例句：母の作るケーキほどおいしい物はありません。
- 翻译：没有比妈妈做的蛋糕更好吃的了。

#### 2. `〜ほど（程度）`

- 用法：表示程度，「……（到了）……的程度」。
- 接续：`V普通形 / い形 / な形（〜な）+ ほど`。
- 例句：漢字をおぼえるために、目をとじても出てくるほどたくさん書いた。
- 翻译：为了记住汉字，我多到闭上眼也能浮现出来的程度反复写。

#### 3. `〜ないでほしい`

- 用法：希望对方不要做某事。
- 接续：`Vない形 + ないでほしい`。
- 例句：店の人に「忙しいから、まだ帰らないでほしい」とたのまれた。
- 翻译：店里的人请求我"很忙，希望你别走"。

#### 4. `〜とは思えない`

- 用法：无法相信／不觉得……。
- 接续：`普通形 + とは + 思えない`。
- 例句：練習しなければ、日本語が話せるようになるとは思えません。
- 翻译：不练习的话，我不觉得日语会变得能说。

### 例题

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 4-1 | 母が作るケーキ（　　　）おいしい物はありません。 | ほど | 最高级用「ほど〜ない」。 |
| 4-2 | 漢字をおぼえるために、目をとじても出てくる（　　　）書いた。 | ほど | 程度ほど。 |
| 4-3 | 「忙しいから、まだ帰ら（　　　　　）ほしい」とたのまれた。 | ないで | 〜ないでほしい。 |
| 4-4 | 練習しなければ、話せるようになる（　　　　）思えない。 | とは | 「とは思えない」。 |
| 4-5 | 猫に小判という言葉の意味は、何度説明されても役に立たない（　　　）言うことだ。 | と | 引用用「と」。 |

## 第5課 あきれる ―「満員電車」

### 本课核心目标

通过对日本"满员电车"中所见所闻的描写（小学生喧闹、老人无人让座、路上礼貌排队等矛盾景象），学习「刚……」「……得受不了」「连试都不试」「……什么的」等口语化表达。

### 重点词汇

| 词 | 词性 | 释义 | 例句 |
|---|---|---|---|
| 留学（りゅうがく）スル | 名/サ变 | 留学 | 日本に留学した友達に話を聞いた。 |
| 違う（ちがう） | 五段 | 不同 | 国によって習慣が違う。 |
| 様子（ようす） | 名 | 样子、情况 | 電車は社会の様子を知る場所だ。 |
| 満員（まんいん） | 名 | 满员、客满 | 朝の満員電車は本当に大変だ。 |
| がまんスル | 名/サ变 | 忍耐 | 痛くてがまんできない。 |
| 眠る（ねむる） | 五段 | 睡觉 | お酒を飲んで眠っている人。 |
| この前（このまえ） | 副 | 前几天 | この前、おもしろいことがあった。 |
| 席を取る（せきをとる） | 惯用 | 占座 | 急いで席を取る。 |
| 取り出す（とりだす） | 五段 | 拿出 | かばんからゲームを取り出す。 |
| 大声（おおごえ） | 名 | 大声 | 大声で騒ぎ始めた。 |
| 夢中（むちゅう） | 名/ナ形 | 入迷、专心 | ゲームに夢中になる。 |
| 礼儀正しい（れいぎただしい） | い形 | 有礼貌 | 礼儀正しい日本人。 |
| 列（れつ） | 名 | 队伍、行列 | 駅でじっと列に並ぶ。 |
| あきれる | 一段 | 愕然、傻眼 | あまりのひどさにあきれた。 |
| 信じる（しんじる） | 一段 | 相信 | 信じられないようなことだ。 |

### 文法详细总结

#### 1. `〜たばかり`

- 用法：刚刚……（动作或时间上的"刚"）。
- 接续：`Vた形 + ばかり`。
- 例句：日本に来たばかりなので、まだ携帯電話を持っていません。
- 翻译：因为刚到日本，所以还没有手机。

#### 2. `〜て／でたまらない`

- 用法：「……得受不了」，表示感情或感觉非常强烈。
- 接续：`Vて／い形容词くて／な形容词で + たまらない`。
- 例句：満員電車の中でどうして本や新聞が読めるのか、ふしぎでたまりません。
- 翻译：在挤满人的电车里居然能看书看报，让我不可思议得不行。

#### 3. `〜(よ)うともしない`

- 用法：「连试也不试着……」。
- 接续：`V意向形 + ともしない`。
- 例句：男の人はあやまろうともせずに、おりて行ってしまった。
- 翻译：那个男人连道歉的意思都没有就下车了。

#### 4. `〜なんて`

- 用法：「……什么的、居然……」（带轻蔑、惊讶、不可思议等情绪）。
- 接续：`普通形 / N + なんて`。
- 例句：結婚記念日をわすれるなんて、信じられない。
- 翻译：居然忘了结婚纪念日，真不敢相信。

### 例题

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 5-1 | 日本に来た（　　　　　）なので、まだ漢字があまり読めません。 | ばかり | 「〜たばかり」。 |

### 文法详细总结

#### 1. `〜とか〜とか`

- 用法：举例列举（不限于这些）。
- 接续：`V/N + とか + V/N + とか`。
- 例句：疲れたときは、ゆっくりおふろに入るとか、早く寝るとかしましょう。
- 翻译：累的时候，要么慢慢泡个澡，要么早点睡。

#### 2. `〜といっても`

- 用法：「虽说是……（其实没那么……）」，澄清对方的过度想象。
- 接续：`普通形 / N + といっても`。
- 例句：夏休みといっても、3日しかないんです。
- 翻译：虽说是暑假，但其实只有3天。

#### 3. `〜ものだ`（一般道理／常情）

- 用法：「按理说应该……／一般来说就该……」。
- 接续：`V辞書形 / い形 / な形〜な + ものだ`。
- 例句：人から物をもらったら、お礼を言うものですよ。
- 翻译：收到别人的东西，是应当道谢的。

#### 4. `〜ものではない`

- 用法：「不该……、不可以……」（道理上的禁止）。
- 接续：`V辞書形 + ものではない`。
- 例句：ノックもせずに、人の部屋に入るものではありません。
- 翻译：不可以连门都不敲就进别人房间。

#### 5. `〜たつもり`

- 用法：「自以为做了……（其实没有／对方没那么觉得）」。
- 接续：`Vた形 + つもり`。
- 例句：電気代を払ったつもりだったので、急に電気が止まっておどろいた。
- 翻译：以为已经付了电费，所以突然停电时吓了一跳。

### 文法详细总结

#### 1. `〜まで`（出乎意料的程度）

- 用法：「连……都……」，表示意想不到的程度。
- 接续：`N + まで`。
- 例句：100円で、時計まで買える店がある。
- 翻译：竟然有花100日元就能买到手表的店。

#### 2. `〜によって`（手段／方法）

- 用法：「通过／靠／用……」。
- 接续：`N + によって`。
- 例句：毎日運動することによって、じょうぶな体を作ることができる。

- 例句：台風によって、交通機関に影響が出ている。
- 翻译：由于台风，交通受到了影响。

#### 4. `せっかく〜`

- 用法：「难得……（如果不能好好享受／利用就遗憾了）」。常与「のに」「だから」呼应。
- 接续：`せっかく + N／V`。
- 例句：せっかくの旅行だから、ちょっといいホテルにとまりたい。
- 翻译：难得旅游一趟，想住稍好点儿的酒店。

#### 5. `〜ことだ`（建议／要点）

- 用法：「重要的就是……、关键是……」。
- 接续：`V辞書形 / Vない形 + ことだ`。
- 例句：信頼されるためには、まず、あいさつをすること、時間を守ることだ。
- 翻译：要想被信赖，关键就是打招呼、守时。

#### 1. `〜(よ)うと`

- 用法：「为了想要……」，主语主动的意图。
- 接续：`V意向形 + と`。
- 例句：少しでも運動をしようと、毎朝30分歩くことにした。
- 翻译：为了能活动活动身体，决定每天早上走30分钟。

#### 2. `〜につれて`

- 用法：「随着……（同时变化）」。
- 接续：`V辞書形 / N + につれて`。
- 例句：子供が大きくなるにつれて、学校や習い事にお金がかかるようになった。
- 翻译：随着孩子长大，学校和兴趣班的开销越来越大。

#### 3. `〜どころか、かえって〜`

- 用法：「岂止……，反而……」，与预期相反。
- 接续：`普通形 / い形 + どころか`、`N + どころか`。
- 例句：船を利用してみたところ、安いどころか、かえって費用が高くなってしまった。
- 翻译：试着坐船去出差，结果不仅没便宜，反而更贵了。

#### 4. `〜わけだ`

- 用法：「难怪／也就是说……」，解释原因／推出结论。
- 接续：`普通形 + わけだ`、`な形〜な / N + な / という + わけだ`。
- 例句：田中さんは学生時代からマラソンをやっているそうですよ。/ ああ、だから、あんなに体の動きが軽いわけですね。
- 翻译：田中先生学生时代就跑马拉松。/ 啊，难怪他动作那么轻快。

#### 5. `〜からといって、〜わけではない`

- 用法：「不能因为……就……」（否定笼统推论）。
- 接续：`普通形 + からといって、〜わけではない`。
- 例句：日本に長く住んでいるからといって、日本語がじょうずに話せるわけではない。
- 翻译：在日本住得久，并不意味着日语就一定说得好。

## 第11課 つながる ―「タテとヨコ」

### 本课核心目标

### 文法详细总结

#### 1. `〜ざるを得ない`

- 用法：「不得不……」，有"心里其实不愿意"的语气。
- 接续：`Vない形（去ない）+ ざるを得ない`（する → せざるを得ない）。
- 例句：台風で飛行機が飛ばず、空港で待たざるを得なかった。
- 翻译：由于台风航班停飞，不得不在机场等下去。

#### 2. `〜さえ`

- 用法：「连……都……」，举出极端例子，强调"出乎意料"。
- 接续：`N + さえ`、`Vて形 + さえ`。
- 例句：先週は忙しくて、家族からのメールさえ読めなかった。
- 翻译：上周太忙，连家人发来的邮件都没能读。

#### 3. `〜にしろ〜にしろ`

- 用法：「无论A还是B」，列举对比性的两项得出共同结论。
- 接续：`N／V辞書形 + にしろ + N／V辞書形 + にしろ`。
- 例句：個人にしろ会社にしろ、大切な情報は責任を持って管理しなければならない。
- 翻译：无论个人还是公司，重要信息都必须负起责任来管理。

#### 4. `〜た上で`

- 用法：「在……（之后）再」，强调先完成前一动作。
- 接续：`Vた形 + 上で`、`Nの + 上で`。
- 例句：両親や先生と相談した上で、大学を決めようと思います。
- 翻译：与父母和老师商量之后，再决定上哪所大学。

| 清潔（せいけつ）ナ | ナ形 | 干净 | 清潔な日本。 |
| 締めくくる（しめくくる） | 五段 | 收尾 | 拍手のうちに講演を締めくくった。 |

### 文法详细总结

#### 1. `〜をはじめ（として）`

- 用法：「以……为代表，还有……」。
- 接续：`N + をはじめ（として）`。
- 例句：卒業式には校長先生をはじめ、辞めた先生方も大勢いらっしゃった。
- 翻译：毕业典礼上以前校长为首，已退休的老师们也都来了。

#### 2. `さすが（に）／さすがの〜も`

- 用法：「果然不愧／连……都……」。
- 接续：`さすが（に） + V/形容詞`、`さすがの + N + も`。
- 例句：田中さんは学生時代スポーツをやっていたから、さすがに走るのが速い。
- 翻译：田中学生时代是运动员，果然跑得快。

#### 3. `〜わけがない`

- 用法：「不可能……」（强烈否定）。
- 接续：`普通形／な形〜な／N + である + わけがない`。
- 例句：けがで入院しているから、出られるわけがない。
- 翻译：他因伤住院，不可能出场比赛。

#### 4. `〜だけに`

- 用法：「正因为……（所以更……）」。
- 接续：`普通形／N／な形〜な + だけに`。
- 例句：思い込んでいただけに、若者の言葉がショックだった。
- 翻译：正因为之前坚信不疑，年轻人的话才让我受到很大冲击。

## 第16課 うたう ―「歌の力」

### 本课核心目标

通过作者在公司聚会上唱童谣的经历，学习「因为……（说明语气）」「正在／刚刚／马上……」「按自己的方式」「一定是……」等表达。

### 课文概要

作者不会唱歌，但歓迎会、送别会等聚会后被起哄唱歌。他借助童谣那"和人心、合众声"的不可思议力量度过。初唱时大家露出尴尬表情，但出于礼貌侧耳倾听；不知不觉合唱起来，唱完后会场陷入难以再起头的安静。歌真有把所有人的不同心思汇成一种声音的"歌の力"。

### 重点词汇

| 词 | 词性 | 释义 | 例句 |
|---|---|---|---|
| 勤める（つとめる）／なじむ | 一段／五段 | 工作／习惯 | 長く勤めているがなじめない。 |

### 文法详细总结

#### 1. `〜ものだから／〜もので（すから）`

- 用法：「因为……（带说明、辩解的语气）」。
- 接续：`普通形／な形〜な／N + な + ものだから`。
- 例句：事故で地下鉄が止まってしまったものですから、遅れました。
- 翻译：因为地铁出事故停了，所以迟到了。

- 用法：表示动作的"时间点"（要做／正在做／刚做完）。
- 接续：`V辞書形／Vている／Vた + ところだ`。
- 例句：今、お茶を飲んでいるところだ。
- 翻译：我现在正在喝茶。

#### 3. `〜なりに／〜なりの`

- 用法：「按自己的方式（量力而行）」。
- 接续：`N + なりに／なりの + N`。
- 例句：私は自分なりの勉強の仕方で漢字を覚えている。
- 翻译：我用自己的方式记汉字。

#### 4. `〜に違いない`

- 用法：「一定是……」（有把握的推断）。
- 接续：`普通形／な形／N + に違いない`。
- 例句：制服を着た学生たちを見て、高校生に違いないと思った。
- 翻译：看到穿校服的学生们，我想他们一定是高中生。

## 第17課 なおす ―「命」

### 本课核心目标

通过现代医疗与"活着"的伦理思考，学习「自从……以后没……」「这样下去就……」「一方面……另一方面」「不能不……」等表达。

### 课文概要

医疗的进步延长了寿命，许多过去无法救治的人被救活。但平均寿命延长的同时社会高龄化加深，"活着是什么"成了不得不重新追问的严峻问题。所谓"植物状态"的患者，靠药与器械每天延长一点生命，引发诸多问题。作者认为医疗应追求的不是"活得久"，而是"健康地活的时间长"——这是必须严肃重新审视的问题。

### 文法详细总结

#### 1. `〜たきり`

- 用法：「自从……以后再没……」。
- 接续：`Vた形 + きり`。
- 例句：友達にDVDを借りたきり、まだ返していない。
- 翻译：自从从朋友那借了 DVD，还没还。

#### 2. `〜ようでは`

- 用法：「这样下去就……（不行）」。
- 接续：`Vる／Vない + ようでは`。
- 例句：ひとりで起きられないようでは、会社で働くなんて無理だ。
- 翻译：连自己起床都做不到，怎么能上班。

- 用法：「一方面……另一方面……」。
- 接续：句子 + `。その一方（で）、` + 句子。
- 例句：兄は大学で研究している。その一方で、自分の会社も経営している。
- 翻译：哥哥一方面在大学做研究，一方面也经营自己的公司。

#### 4. `〜わけにはいかない`

- 用法：「不能（不去）」「不能那么做」（道义）。
- 接续：`V辞書形 + わけにはいかない`、`Vない形 + ないわけにはいかない`。
- 例句：今日は大切な仕事があるから、会社を休むわけにはいかない。
- 翻译：今天有重要工作，不能请假。

#### 3. `〜としても／たとえ〜としても`

- 接续：`Vない形 + ないことには`。
- 例句：今、手を打たないことには、大変なことになる。

## 第19課 かなえる ―「ふたつの夢」

### 本课核心目标

### 文法详细总结

#### 1. `〜はともかく（として）`

- 用法：「……暂且不论」。
- 接续：`N + はともかく（として）`。
- 例句：どの大学に入るかはともかく、まず何を学びたいか考えるべきだ。
- 翻译：上哪所大学暂且不论，应先考虑想学什么。

- 用法：「只要……就……」（最低条件即可）。
- 接续：`N + さえ + Vば`、`Vます形（去ます）+ さえすれば`、`い形〜く + さえあれば`。
- 例句：努力さえすれば、夢がかなう可能性がある。
- 翻译：只要努力，梦想就有可能实现。

#### 3. `〜ことから`

- 用法：「从……（这一点来看）」基于事实命名／推断。
- 接续：`普通形／な形〜な／N + である + ことから`。
- 例句：この犬は毛が白いことから、「シロ」と名づけられた。
- 翻译：因为这只狗毛是白的，所以取名"白"。

#### 4. `〜ないものか／〜ないものだろうか`

- 用法：「难道不能……？」（强烈期望／反问）。
- 接续：`V可能形ない + ないものか／ないものだろうか`。
- 例句：もう少し早く帰れないものだろうか。
- 翻译：就不能早一点回家吗？

### 文法详细总结

#### 1. `〜割に（は）`

- 用法：「相对于……出乎意料地……」。
- 接续：`普通形／な形〜な／N + の + 割に（は）`。
- 例句：姉は年齢の割に若く見えます。
- 翻译：姐姐相对她的年龄看起来很年轻。

#### 2. `〜にあたり／〜にあたって（は）`

- 用法：「在……之际」（重大场合）。
- 接续：`V辞書形／N + にあたり／にあたって（は）`。
- 例句：定年退職するにあたり、ごあいさつを申し上げます。
- 翻译：在退休之际，向各位致辞。

#### 3. `〜に限らず`

- 用法：「不仅……（也／还）」。
- 接续：`N + に限らず`。
- 例句：最近は女性に限らず、男性も化粧品を使うようになった。
- 翻译：最近不仅女性，男性也开始使用化妆品。

#### 4. `〜にかけては`

- 用法：「在……（这方面）（无人能及）」。
- 接续：`N + にかけては`。
- 例句：漢字を覚える速さにかけては、私に勝てる人はいない。
- 翻译：在记汉字的速度上，没人能赢我。

## 综合复习清单（第1〜20课）

### 高频语法主题分组

- **状态变化**：〜ようになる／なくなる、〜につれて、〜ば〜ほど、〜とともに
- **意志与决定**：〜ことにする／ことにしている、〜(よ)うと、〜ことだ
- **使役・被动**：〜(さ)せてあげる／やる、〜(さ)せられる
- **不得不・必须**：〜ざるを得ない、〜わけにはいかない、〜ないことには〜ない
- **请求／希望／反问**：〜ないでほしい、〜たがる、〜ないものか／ないものだろうか
- **常情／道理**：〜ものだ、〜ものではない、〜ことだ、〜べきだ／べきではない
- **推论**：〜はずだ、〜はずがない、〜わけだ、〜に違いない、〜のことだから、〜からといって〜わけではない
- **比较／程度**：〜ほど〜はない、〜ほど〜、〜まで、〜どころか、かえって〜、〜割に（は）
- **比喻／引用／举例**：〜とか〜とか、〜として、〜なんて、〜をはじめ、〜に限らず
- **意外発見・転換**：〜たところ、〜のに〜てしまった、〜といっても、〜。その一方（で）、〜
- **不可能・無法・否定推断**：〜ようがない、〜とは思えない、〜(よ)うともしない、〜としか〜ない、〜わけがない
- **怀旧／习惯／状态持续**：〜たものだ、〜うちに、〜うちは、〜まま、〜たきり、〜ようでは
- **强烈感觉**：〜てたまらない、〜たばかり、〜て／でならない、〜だけに
- **手段・原因・契机**：〜によって（手段／原因）、〜ことから、〜きっかけで／をきっかけに、〜たばかりに
- **立场・前提・基础**：〜にとって(は)、〜上で、〜た上で、〜にあたり／にあたって、〜次第（で）、〜はともかく
- **错觉・成见**：〜たつもり、〜思い込む
- **极端／强调／专长**：〜さえ、〜さえ〜ば、〜にしろ〜にしろ、〜にもかかわらず、〜にかけては、さすが（に）
- **理由说明**：〜ものだから／もので、〜というのも、〜通り
- **让步**：〜としても／たとえ〜としても
- **时点／时机**：〜ところだ（要做／正在做／刚做完）、〜とともに
- **认定／客观叙述**：〜とする／〜とされる、〜なりに／なりの
