# 第7课 かざる ―「名刺」

> 对应书内第7课（PDF 约第49–54页）。
> 这一课的主轴是「日本のビジネスマナー」，重点学习"直到……为止""根据……""难得……""应该……"等表达。

## 本课主旨

叙述者来到日本后第一次交换名片，发现日本人对名片的态度与自己国家完全不同——日本人双手递接名片、仔细阅读、放在桌上整齐摆放，把名片视为对方本人的象征。叙述者起初觉得麻烦，后来理解到：这些礼仪并不是多余的形式，而是向对方传达"我尊重你"的重要方式。不同文化对"礼仪"的理解各异，重要的是理解其背后的含义。

## 导入问题

课前会先围绕下面三个问题展开：

- 你有名片吗？你用过名片吗？
- 你觉得名片对商务场合有多重要？
- 你的国家有哪些独特的打招呼或见面礼仪？

## 文法要点

### 1. `〜まで`（持续到某时间/状态）

- 用法：表示某动作或状态持续到某个时间点或某事发生为止。
- 接续：
  - `N + まで`
  - `V辞書形 + まで`
- 例句：
  - 仕事が終わるまで、帰ることができなかった。
  - 日本語が上手になるまで、国に帰らないつもりだ。
  - 電車が来るまで、ベンチに座って待った。
- 说明：
  - `〜まで` 表示一个终止点，前文的状态或动作持续到这个终止点。
  - 与 `〜までに`（在……之前完成）区分：`〜まで` 强调持续，`〜までに` 强调截止时间。

### 2. `〜によって`①（根据、视……而定）

- 用法：表示因条件、对象不同而结果不同，"根据……、因……而异"。
- 接续：
  - `N + によって / により`
- 例句：
  - 文化によって、礼儀の形は違う。
  - 人によって、名刺に対する考え方は様々だ。
  - 国によって、ビジネスマナーが異なる。
- 说明：
  - 强调"以某个标准为依据"，前接的名词是判断、变化的依据。
  - 常见搭配：`〜によって違う / 異なる / 様々だ`。

### 3. `〜によって`②（通过……手段/由……造成）

- 用法：表示手段、方法，或动作的执行者（被动句中）。
- 接续：
  - `N + によって`
- 例句：
  - 礼儀によって、相手への尊重が伝わる。
  - この建物は有名な建築家によって設計された。
  - 言葉によってではなく、行動によって示す。
- 说明：
  - 与用法①区分：用法①表示"依据/因……而异"，用法②表示"通过某手段"或"由某人/某事造成"。
  - 在被动句中常表示行为的施动者，相当于 `〜に` 但语气更正式。

### 4. `せっかく〜のだから／せっかく〜のに`

- 用法：表示"难得……（所以应该好好利用）"或"难得……（却……，很可惜）"。
- 接续：
  - `せっかく + V / N + のだから / のに`
- 例句：
  - せっかく日本に来たのだから、日本のマナーをしっかり学びたい。
  - せっかくきれいに飾ったのに、誰も気づいてくれなかった。
  - せっかくの機会だから、ぜひ挑戦してみてください。
- 说明：
  - `せっかく〜のだから` 语气是"既然难得……，就要……"（正面利用）。
  - `せっかく〜のに` 语气是"难得……却……"（可惜、遗憾）。

### 5. `〜ことだ`

- 用法：对某人提出建议或劝告，表示"最重要的是……""应该……"。
- 接续：
  - `V辞書形 / Vない形 + ことだ`
- 例句：
  - 日本語を上手にしたければ、毎日練習することだ。
  - 病気を治したければ、薬を飲んでゆっくり休むことだ。
  - 相手に失礼にならないよう、まずマナーを学ぶことだ。
- 说明：
  - 用于给出建议或指出最关键的做法，带有一定的规劝语气。
  - 与 `〜ものだ`（一般常理）区分：`〜ことだ` 更针对具体对象提出具体建议。

## 课文梳理

1. 叙述者第一次在日本参加商务会面，见到了严格的名片交换礼仪：双手递接、仔细阅读、整齐摆放在桌上。
2. 叙述者起初觉得这些形式太繁琐，不明白为什么要如此郑重其事。
3. 日本同事解释：名片是对方的"分身"（象征），把它随意放置或写字是对对方的不尊重。
4. 叙述者理解到：难得来到日本做生意（せっかく日本に来た），就应该认真学习这里的礼仪。
5. 不同文化中礼仪形式各异（文化によって違う），但背后传达尊重的心意是共通的。

## 练习题

### 一、词汇练习

1. `名刺を（　　　　　）するときは、両手で渡すのがマナーだ。`
   - 答案：`交換`
   - 解析：`名刺を交換する` = 交换名片。

2. `丁寧に名刺を（　　　　　）、しっかり読んだ。`
   - 答案：`受け取って`
   - 解析：`受け取る` = 接受、收到。

3. `初対面では、まず相手に良い（　　　　　）を与えることが大切だ。`
   - 答案：`印象`
   - 解析：`印象を与える` = 给对方留下印象。

4. `その行動の（　　　　　）を理解するのに時間がかかった。`
   - 答案：`意味`
   - 解析：`意味` = 意义、含义。

### 二、语法练习

5. `日本語が上手になる（　　　　　）、帰国しないつもりだ。`
   - 答案：`まで`
   - 解析：`〜まで` = 持续到某个状态出现为止。

6. `文化（　　　　　）、礼儀の形は違う。`
   - 答案：`によって`
   - 解析：`〜によって` = 根据……、因……而异。

7. `せっかく日本に来た（　　　　　）、マナーをしっかり学びたい。`
   - 答案：`のだから`
   - 解析：`せっかく〜のだから` = 难得……，所以……（正面利用）。

8. `せっかく準備した（　　　　　）、急に中止になってしまった。`
   - 答案：`のに`
   - 解析：`せっかく〜のに` = 难得……却……（可惜）。

9. `日本語を上手にしたければ、毎日練習する（　　　　　）だ。`
   - 答案：`こと`
   - 解析：`〜ことだ` = 最重要的是……、应该……（建议/劝告）。

### 三、短答题

10. `日本では名刺を受け取った後、どうするのがマナーですか？`
    - 参考答案：双手接过后仔细阅读，开会时整齐摆放在桌上，不随意涂写或折叠，会议结束后妥善收入名片夹。

11. `用 〜によって 写一句话，说明某事因人/因文化而异。`
    - 参考答案：`人によって、時間の感覚は大きく違う。`

## 本课复习要点

- 能用 `〜まで` 表达"持续到……为止"，并与 `〜までに`（截止时间）区分。
- 能区分 `〜によって`① "因……而异"和 `〜によって`② "通过……手段/由……造成"两种用法。
- 能用 `せっかく〜のだから` 表达"难得……，应该好好利用"，以及 `せっかく〜のに` 表达"难得……却……（可惜）"。
- 能用 `〜ことだ` 对他人提出具体的建议或劝告。
