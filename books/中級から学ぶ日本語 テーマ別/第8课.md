# 第8课 おもいこむ ―「男の色・女の色」

> 对应书内第8课（PDF 约第55–60页）。
> 这一课的主轴是「先入観とジェンダー」，重点学习"无法……""对……来说""以前常常……""做了之后发现……""被……/让……"等表达。

## 本课主旨

叙述者在日本发现，粉色被视为"女性的颜色"，蓝色被视为"男性的颜色"，这与自己国家的习惯不同。进一步观察后发现，这种颜色与性别的对应关系因文化而异，并非放之四海而皆准的"常识"。通过这个例子，叙述者反思：很多我们习以为常的"理所当然"，其实只是文化上的先入观，重新审视这些固有印象，才能更客观地认识世界。

## 导入问题

课前会先围绕下面三个问题展开：

- 在你的国家，什么颜色被视为"女性的"？什么颜色被视为"男性的"？
- 你有没有发现某些事在日本和你的国家完全相反？
- 你认为"先入观"（先入为主的印象）是好事还是坏事？

### 补充表达

| 表达 | 中文释义 | 例句 |
|---|---|---|
| 〜とされている | 被认为是…… | ピンクは女性の色とされている。 |
| 〜にすぎない | 不过是…… | それは先入観にすぎない。 |

## 文法要点

### 1. `〜ようがない`

- 用法：表示完全没有办法做某事，"无从……""没有办法……"。
- 接续：
  - `Vます形（去ます）+ ようがない / ようがなかった`
- 例句：
  - 彼の連絡先を知らないので、連絡のしようがない。
  - こんなに壊れていたら、直しようがない。
  - 先入観は、気づかないと変えようがない。
- 说明：
  - 表示在客观条件或主观能力上都"没有做到的方法"。
  - 注意区分 `〜ようがない`（无方法可施）和 `〜ことができない`（没能力做）。

### 2. `〜にとって（は）`

- 用法：表示"对……来说"，标出评价或判断的立场。
- 接续：
  - `N + にとって（は）`
- 例句：
  - 日本人にとって、名刺はとても大切なものだ。
  - 私にとって、この色は特別な意味を持つ。
  - 子どもにとっては当たり前のことが、大人には難しいこともある。
- 说明：
  - 标出某人的立场或角度，常接表示感受、评价的句子。
  - 与 `〜にとっては` 相比，`〜にとって` 更中性，`〜にとっては` 有对比语气。

### 3. `〜たものだ`

- 用法：回忆过去曾经经常做某事，带有怀念或感慨的语气。
- 接续：
  - `Vた形 + ものだ`
- 例句：
  - 子どものころは、毎日外で遊んだものだ。
  - 学生時代は、よく友達と夜遅くまで語り合ったものだ。
  - 初めて日本に来たとき、何もわからなくて困ったものだ。
- 说明：
  - 用于说话人对过去习惯性行为的回忆，含有感慨或怀念。
  - 与 `〜ことにしていた`（过去持续的习惯规定）不同，`〜たものだ` 更强调情感性的回忆。

### 4. `〜たところ`

- 用法：表示做了某动作之后，出现了某结果（往往是意外或新发现的）。
- 接续：
  - `Vた形 + ところ`
- 例句：
  - 日本の本屋に行ったところ、ピンク色の表紙の本が多いことに気づいた。
  - 先生に聞いたところ、それは先入観だと教えてくれた。
  - 試してみたところ、思ったより簡単だった。
- 说明：
  - `〜たところ` 后句常表示"发现了"或"出现了意外结果"。
  - 与 `〜たら`（条件后果）意思相近，但 `〜たところ` 更强调"结果发现"的发现感。

### 5. `〜される / （さ）せられる`

- 用法①（受身 `〜される`）：表示被动，主语受到某动作的影响。
- 用法②（使役受身 `〜させられる`）：表示被迫做某事，即"被（强制）让……做"。
- 接续：
  - 受身：`V被动形（される）`
  - 使役受身：`V使役受身形（させられる）`
- 例句：
  - 先入観は、文化や環境によって作られる。（受身）
  - 子どものころ、毎日野菜を食べさせられた。（使役受身）
  - 仕事で残業をさせられて、疲れた。（使役受身）
- 说明：
  - 受身 `〜される` 表示主语受到动作影响，不一定带有负面情绪。
  - 使役受身 `〜させられる` 几乎总带有"被迫、不情愿"的语感。
  - 注意：五段动词的使役受身形为 `〜せられる` 可缩略为 `〜される`（与受身同形，靠上下文判断）。

## 课文梳理

1. 叙述者发现日本的颜色与性别的对应关系和自己国家不同，感到困惑。
2. 调查后发现（调べたところ），这种颜色-性别对应因文化而异，并没有统一标准。
3. 对日本人来说（日本人にとって），粉色=可爱=女性，这是理所当然的，但对其他文化的人来说并非如此。
4. 叙述者回忆，小时候曾经理所当然地以为某些事物是"全世界共通的常识"（思ったものだ）。
5. 结论：先入观是被文化和环境塑造出来的（作られる），意识到这一点，才能更客观地看待世界。

## 练习题

### 一、词汇练习

1. `（　　　　　）をなくして、物事を客観的に見ることが大切だ。`
   - 答案：`先入観`
   - 解析：`先入観` = 先入之见、固有印象。

2. `その考えは世界で（　　　　　）するものではない。`
   - 答案：`共通`
   - 解析：`共通する` = 共通、普遍适用。

3. `日本では当たり前なことが、外国では（　　　　　）のことがある。`
   - 答案：`逆`
   - 解析：`逆` = 相反。

4. `自分の意見を（　　　　　）に見直すことが成長につながる。`
   - 答案：`客観的`
   - 解析：`客観的に` = 客观地。

### 二、语法练习

5. `連絡先を知らないので、連絡の（　　　　　）ない。`
   - 答案：`しようが`
   - 解析：`〜ようがない` = 没有办法……。

6. `私（　　　　　）、これは大切な経験だった。`
   - 答案：`にとって`
   - 解析：`〜にとって` = 对……来说。

7. `子どものころは、毎日川で泳い（　　　　　）ものだ。`
   - 答案：`だ`
   - 解析：`〜たものだ` = （过去）曾经常常……（带怀念语气）。整体：`泳いだものだ`。

8. `先生に聞い（　　　　　）、その答えがわかった。`
   - 答案：`たところ`
   - 解析：`〜たところ` = 做了之后，（结果）发现……。

9. `仕事で毎日残業（　　　　　）、とても疲れた。`
   - 答案：`させられて`
   - 解析：`〜させられる` = 被迫做……（使役受身）。

### 三、短答题

10. `本文中，「先入観」の例として挙げられているのは何ですか？`
    - 参考答案：颜色与性别的对应关系——日本认为粉色是女性色、蓝色是男性色，但在其他文化中并非如此。

11. `用 〜たところ 写一句话，描述你做了某事后发现的结果。`
    - 参考答案：`友達に電話したところ、もう家を出たと言っていた。`

## 本课复习要点

- 能用 `〜ようがない` 表达"没有办法做……"，理解其与"没能力"的区别。
- 能用 `〜にとって（は）` 标出评价立场，表达"对……来说"。
- 能用 `〜たものだ` 带感情地回忆过去的习惯性行为。
- 能用 `〜たところ` 表达"做了之后发现……"，注意其"发现/结果"的语感。
- 能区分受身 `〜される`（被……）和使役受身 `〜させられる`（被迫做……）。
