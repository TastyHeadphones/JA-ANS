# 第11课 つながる ―「タテとヨコ」

> 对应书内第11课（PDF 约第63–68页）。
> 这一课的主轴是「人間関係（タテ社会 vs ヨコ社会）」，重点学习"不得不""连……都""无论A还是B""在……基础上"等表达。

## 本课主旨

随着时代变化，日本传统的严格上下关系（タテ社会）逐渐淡化，自由平等的横向连接（ヨコ社会）受到重视。亲子像朋友，学生对老师如同伙伴。但作者提出：横向关系虽轻松，却缺乏责任与角色担当，难以育出真正可信赖的人际关系。在追求平等的同时，我们也必须认真思考——「タテ社会消退之后正在失去的是什么」。

## 导入问题

课前会先围绕下面三个问题展开：

- 遇到困难时你会找谁商量？
- 你被父母严厉训过吗？是什么时候？
- 你和朋友吵过架吗？是什么时候、为什么？

## 文法要点

### 1. `〜ざるを得ない`

- 用法：表示"不得不……"，主观上虽不愿意，但客观上别无选择。
- 接续：
  - `Vない形（去ない）+ ざるを得ない`
  - 例外：`する → せざるを得ない`
- 例句：
  - 台風で飛行機が飛ばず、次の日まで空港で待たざるを得なかった。
  - 駅のトイレに携帯電話を忘れて、取りにもどらざるを得なかった。
- 说明：
  - 比 `〜なければならない` 多了一份"心里不情愿"的语气。
  - 书面色彩较强，常用于陈述客观情况。

### 2. `〜さえ`（举极端例子）

- 用法：举出极端例子，"连……都……"，强调连这都如此，更何况其他。
- 接续：`N + さえ`、`V て形 + さえ`
- 例句：
  - 先週は忙しくて、家族からのメールさえ読めなかった。
  - 人に会ったらあいさつをするということは、子供でさえ知っている。
  - 日本に来たばかりの頃は、何を質問されているのかさえ分からなかった。
- 说明：
  - 与 `〜も` 相比，`〜さえ` 更强调"出乎意料"。
  - 常和否定结构、消极内容搭配。

### 3. `〜にしろ〜にしろ`

- 用法：「无论A还是B（都……）」，举两个具有对比性的例子。
- 接续：`N / V辞書形 + にしろ + N / V辞書形 + にしろ`
- 例句：
  - 今は、個人にしろ会社にしろ、大切な情報は責任を持って管理しなければならない。
  - 外で食事をするにしろ、旅行に行くにしろ、その分お金がかかる。
  - 行くにしろ行かないにしろ、早く決めてください。
- 说明：
  - 与 `〜にせよ〜にせよ` 同义，后者更书面。
  - 常用于得出"无论哪种情况，结论都一样"的论断。

### 4. `〜た上で`

- 用法：「在做了……之后再（做某事）」，强调先完成前一动作作为基础。
- 接续：`Vた形 + 上で`、`Nの + 上で`
- 例句：
  - 両親や先生と相談した上で、大学を決めようと思います。
  - みんなに確認した上で、次に集まる日を決めましょう。
  - 子供や部下はそのことを理解した上で、タテの関係を受け入れた。
- 说明：
  - 强调"经过……之后再做"的顺序与慎重态度。
  - 区别于 `〜てから`（单纯时间先后）。

## 课文梳理

1. 现代亲子像朋友，学生像伙伴对待老师；连"年轻员工不会用敬语"这种事都不再被特别提起——日本进入了重视自由平等的「ヨコのつながり」时代。
2. 过去严格的「上下关系」社会里，人们不得不遵守关系而生活；身份不同就连随便说话都办不到，违反规则就会被仲间排斥。
3. 但作者指出：单靠手机短信确认"我和别人有联系"的轻便关系，培养出的常是无责任的人际关系。
4. 与之相比，过去虽严格，但上司、父母都有明确的责任和角色——把孩子和部下培养成才；孩子和部下理解这一点后接受タテ关系，因而形成了相互信赖的人际关系。
5. 横向社会中没有责任、役割的轻松关系，无法产生可信赖的人际网络。タテ的关系正在消失，作者呼吁我们应认真思考：在变化中正在失去的究竟是什么。

## 练习题

### 一、词汇练习

1. `身分が違えば気安く（　　　　）を利くこともできなかった。`
   - 答案：`口`
   - 解析：惯用语 `口を利く` = 说话、搭话。

2. `上に立つ者には明確な（　　　　）と役割が要求される。`
   - 答案：`責任`
   - 解析：`責任` = 责任。

3. `信頼できる人間関係を（　　　　）には時間がかかる。`
   - 答案：`築く`
   - 解析：`関係を築く` = 建立关系。

### 二、语法练习

4. `台風で飛行機が飛ばず、空港で一晩待た（　　　　）得なかった。`
   - 答案：`ざるを`
   - 解析：`〜ざるを得ない` = 不得不。

5. `忙しくて、家族からのメール（　　　　）読めなかった。`
   - 答案：`さえ`
   - 解析：`〜さえ` = 连……都。

6. `個人（　　　　）会社（　　　　）、情報の管理には責任が必要だ。`
   - 答案：`にしろ／にしろ`
   - 解析：`〜にしろ〜にしろ` = 无论A还是B。

7. `両親と相談した（　　　　）、進学先を決める。`
   - 答案：`上で`
   - 解析：`〜た上で` = 在……基础上。

### 三、短答题

8. `本文中、筆者がヨコ社会について警告しているのは何か？`
   - 参考答案：横向关系虽轻松平等，但若缺乏责任与角色担当，便培养不出真正可信赖的人际关系；纵向关系消失的同时，我们正在失去某种重要的东西。

9. `用 〜ざるを得ない 写一个自己的例句。`
   - 参考答案：`雨がひどかったので、外出を諦めざるを得なかった。`

## 本课复习要点

- 能用 `〜ざるを得ない` 表达"不得不（心里其实不愿意）"的语气。
- 能用 `〜さえ` 举出极端例子，强调出乎意料。
- 能用 `〜にしろ〜にしろ` 列举对比性事例并得出共同结论。
- 能用 `〜た上で` 表达"在……之后（再做某事）"的慎重顺序。
