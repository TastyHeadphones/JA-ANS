# 第16课 うたう ―「歌の力」

> 对应书内第16课（PDF 约第93–98页）。
> 这一课的主轴是「歌が人と人をつなぐ力」，重点学习"因为……（带辩解、说明语气）""刚……（动作状态）""按自己的方式""一定是……"等表达。

## 本课主旨

作者天生不会唱歌，但公司里常有歓迎会、送别会等场合被起哄逼着唱。无奈之下，他选择唱孩童时代的童谣「うみ」。一开始大家露出"竟然唱这个？"的表情，但出于礼貌还是侧耳倾听；不知不觉中众人也开始一起合唱，唱完之后场内陷入一种"难以再起头"的短暂安静。作者意识到：每个人在童谣里都唱进了自己的"海"和"远方的国家"，年轻人也借歌互相鼓励——歌真的有一种"把人和人连接起来"的不可思议的力量。

## 导入问题

课前会先围绕下面三个问题展开：

- 你喜欢唱歌吗？
- 你的国家有哪些大家从小都会唱的歌？
- 你通常在什么时候唱歌？

## 文法要点

### 1. `〜ものだから / 〜もので（すから）`

- 用法：「因为……（带说明、辩解的语气）」。
- 接续：`普通形 / な形〜な / N + な + ものだから`
- 例句：
  - 入院したのは初めてだったものですから、1日寝ているだけで耐え難い気持ちになりました。
  - 事故で地下鉄が止まってしまったものですから、遅れました。
  - 子供のことが心配なものですから、つい厳しく言ってしまう。
- 说明：
  - 多用于解释自己的行为或晚到等情况，含有"请理解我"的语气。
  - 比 `〜から` 更软化、更礼貌。

### 2. `〜ところだ`

- 用法：表示动作所处的"时间点"——刚要做／正在做／刚做完。
- 接续：
  - 即将做：`V辞書形 + ところだ`
  - 正在做：`Vている + ところだ`
  - 刚做完：`Vた + ところだ`
- 例句：
  - ちょうど授業の準備が終わったところだ。
  - 今、お茶を飲んでいるところだ。
  - これからお茶を飲むところだ。
  - 駅に着いたら、ちょうど電車が出たところだった。
- 说明：
  - 时点感很强：`〜辞書形 + ところ` 表"马上要"；`〜ている + ところ` 表"正在"；`〜た + ところ` 表"刚刚"。

### 3. `〜なりに / 〜なりの`

- 用法：「按自己的方式／程度（虽然有限，但尽力）」。
- 接续：`N + なりに / なりの + N`
- 例句：
  - 私は、自分なりの勉強の仕方で漢字を覚えている。
  - 若い人たちは自分なりの海や「よその国」を思って歌ったのだろうか。
  - 子供は子供なりに考えている。
- 说明：
  - 含有"虽然能力／程度有限，但有自己的方式"的语气。
  - 后接动词或名词皆可。

### 4. `〜に違いない`

- 用法：「一定是……、肯定……」，表示说话人很有把握的推断。
- 接续：`普通形 / な形 / N + に違いない`
- 例句：
  - 制服を着た学生たちを見て、この人たちは高校生に違いないと思った。
  - 若い人たちは「歌の力」を楽しんでいるに違いない。
  - 部屋に明かりがついている。誰かいるに違いない。
- 说明：
  - 比 `だろう / でしょう` 推断更强烈、更肯定。
  - 多用于书面或正式场合。

## 课文梳理

1. 作者长年在同一公司工作，但有件事始终融不进去：歓迎会、送别会等聚会后被起哄唱歌。
2. 他天生五音不全，被强行点歌时只能借助"童谣"——靠它「让大家心情放松、把场子拧成一股绳的不可思议的力量」来度过。
3. 第一次唱时，有人露出"怎么唱这个"的表情，有人露出"扫兴"的脸，但出于对上司的礼貌，大家都把脸转向他听。
4. 不知是不是想给他打气，开头几个人、最后所有人都合声一起唱起来；唱完后有一段难以再开口的安静。
5. 作者想：年轻人也在歌里怀念着各自心中的"海"和"远方的国家"；每个人带着不同的思绪，最后却汇成一个声音——这就是歌的力量。从前祭典、田植、捕鱼时人们都靠歌互相鼓劲；今天年轻人去 KTV，也是在不自觉地享受这种力量。

## 练习题

### 一、词汇练习

1. `生まれ（　　　　　　　）歌が苦手だ。`
   - 答案：`つき`
   - 解析：`生まれつき` = 天生。

2. `心を（　　　　　　　）歌だ。`
   - 答案：`和ませる`
   - 解析：`和ませる` = 让……缓和。

3. `みんなで（　　　　　　　）をそろえて歌った。`
   - 答案：`声`
   - 解析：`声をそろえる` = 齐声。

### 二、语法练习

4. `事故で地下鉄が止まってしまった（　　　　　　　）、遅れました。`
   - 答案：`ものですから`
   - 解析：`〜ものですから` = 因为……（说明）。

5. `今、ちょうどご飯を食べている（　　　　　　　）だ。`
   - 答案：`ところ`
   - 解析：正在做 `〜ているところだ`。

6. `子供は子供（　　　　　　　）にいろいろ考えている。`
   - 答案：`なり`
   - 解析：`〜なりに` = 按自己的方式。

7. `明かりがついている。誰かいる（　　　　　　　）。`
   - 答案：`に違いない`
   - 解析：`〜に違いない` = 一定是……。

### 三、短答题

8. `本文中、筆者が「歌の力」と呼ぶのはどんな力か？`
   - 参考答案：每个人各怀不同心思、不同回忆地唱同一首歌，最后却能把所有人的声音汇成一个、把心拉到一起的不可思议的力量。

9. `用 〜に違いない 写一句关于身边人的句子。`
   - 参考答案：`雨に濡れているから、彼は傘を持っていなかったに違いない。`

## 本课复习要点

- 能用 `〜ものだから／〜もので` 软语气地说明原因。
- 能区分 `〜ところだ` 的三种时点（要做／正在做／刚做完）。
- 能用 `〜なりに／〜なりの` 表达"按自己的方式"。
- 能用 `〜に違いない` 做出有把握的推断。

### 課文：歌の力

**①** 先日、会社の懇親会で「社歌を歌いましょう」と言われたとき、困ったところだった。

- **译文**：前几天在公司联谊会上，被要求"大家来唱社歌"，正好那时候我很为难。
- **语法**：`〜と言われる`（被说/被要求……）；`〜たとき`（……的时候）；`〜ところだった`（正好那时候是……）
- **单词**：`懇親会`（名词，联谊会/恳亲会）；`社歌`（名词，公司之歌）；`困る`（自动词，为难）

---

**②** 日本語が十分でないものだから、歌詞の意味がわからなくて歌えなかった。

- **译文**：因为日语还不够好，歌词的意思不懂，所以没能唱出来。
- **语法**：`〜ものだから`（软语气说明原因，因为……）；`〜て`（原因顺接）；`〜なくて`（否定形的顺接）
- **单词**：`歌詞`（名词，歌词）；`意味`（名词，意义）；`歌える`（`歌う` 的可能形）

---

**③** 上司は「知っているところだけでもいいよ」と優しく言ってくれた。

- **译文**：上司亲切地对我说："认识的地方唱出来就好了。"
- **语法**：`〜ところだけでも`（哪怕只是……的部分）；`〜てくれる`（对方为我做，表恩惠）
- **单词**：`上司`（名词，上司）；`優しく`（副词，亲切地/温柔地）

---

**④** 子どものころ、母になりに歌ってもらった子守歌を、今でも歌えるように覚えている。

- **译文**：小时候，妈妈用她自己的方式为我唱的摇篮曲，到现在我还记得能唱出来。
- **语法**：`〜なりに`（按……的方式/以……的风格）；`〜てもらう`（受到对方的恩惠）；`〜ように覚えている`（记得能……）
- **单词**：`子守歌`（名词，摇篮曲）；`なりに`（按照自己的方式）

---

**⑤** 歌は言語の壁を超えて、人の心に届く力があるに違いない。

- **译文**：歌曲一定有着超越语言隔阂、抵达人心的力量。
- **语法**：`〜に違いない`（一定……，有把握的推断）；`〜を超えて`（超越……）
- **单词**：`言語`（名词，语言）；`壁`（名词，隔阂/壁垒）；`超える`（他动词，超越）；`届く`（自动词，抵达/传达）

---

**⑥** 私なりの日本語の歌い方を見つけて、次の懇親会では声に出して歌いたい。

- **译文**：我想找到属于自己的日语歌唱方式，在下次联谊会上大声唱出来。
- **语法**：`〜なりの`（属于……的/按……风格的）；`〜たい`（说话人的愿望）
- **单词**：`声に出す`（惯用，出声说/唱）；`見つける`（他动词，找到）
