# 第16课 うたう ―「<ruby>歌<rt>うた</rt></ruby>の<ruby>力<rt>ちから</rt></ruby>」

> 对应书内第16课（PDF 约第93–98页）。
> 这一课的主轴是「<ruby>歌<rt>うた</rt></ruby>が<ruby>人<rt>ひと</rt></ruby>と<ruby>人<rt>ひと</rt></ruby>をつなぐ<ruby>力<rt>ちから</rt></ruby>」，重点学习"因为……（带辩解、说明语气）""刚……（动作状态）""按自己的方式""一定是……"等表达。

## 本课主旨

作者天生不会唱歌，但公司里常有歓迎会、送别会等场合被起哄逼着唱。无奈之下，他选择唱孩童时代的童谣「うみ」。一开始大家露出"竟然唱这个？"的表情，但出于礼貌还是侧耳倾听；不知不觉中众人也开始一起合唱，唱完之后场内陷入一种"难以再起头"的短暂安静。作者意识到：每个人在童谣里都唱进了自己的"海"和"远方的国家"，年轻人也借歌互相鼓励——歌真的有一种"把人和人连接起来"的不可思议的力量。

## 导入问题

课前会先围绕下面三个问题展开：

- 你喜欢唱歌吗？
- 你的国家有哪些大家从小都会唱的歌？
- 你通常在什么时候唱歌？

## 文法要点

### 1. `〜ものだから / 〜もので（すから）`

- 用法：「因为……（带说明、辩解的语气）」。
- 接续：`普通形 / な形〜な / N + な + ものだから`
- 例句：
  - <ruby>入院<rt>にゅういん</rt></ruby>したのは<ruby>初めて<rt>はじめて</rt></ruby>だったものですから、1<ruby>日<rt>にち</rt></ruby><ruby>寝て<rt>ねて</rt></ruby>いるだけで<ruby>耐え難い<rt>たえがたい</rt></ruby><ruby>気持ち<rt>きもち</rt></ruby>になりました。
  - <ruby>事故<rt>じこ</rt></ruby>で<ruby>地下鉄<rt>ちかてつ</rt></ruby>が<ruby>止ま<rt>とま</rt></ruby>ってしまったものですから、<ruby>遅れ<rt>おくれ</rt></ruby>ました。
  - <ruby>子供<rt>こども</rt></ruby>のことが<ruby>心配<rt>しんぱい</rt></ruby>なものですから、つい<ruby>厳しく<rt>きびしく</rt></ruby><ruby>言って<rt>いって</rt></ruby>しまう。
- 说明：
  - 多用于解释自己的行为或晚到等情况，含有"请理解我"的语气。
  - 比 `〜から` 更软化、更礼貌。

### 2. `〜ところだ`

- 用法：表示动作所处的"时间点"——刚要做／正在做／刚做完。
- 接续：
  - 即将做：`V辞書形 + ところだ`
  - 正在做：`Vている + ところだ`
  - 刚做完：`Vた + ところだ`
- 例句：
  - ちょうど<ruby>授業<rt>じゅぎょう</rt></ruby>の<ruby>準備<rt>じゅんび</rt></ruby>が<ruby>終わ<rt>おわ</rt></ruby>ったところだ。
  - 今、お<ruby>茶<rt>ちゃ</rt></ruby>を<ruby>飲ん<rt>のん</rt></ruby>でいるところだ。
  - これからお<ruby>茶<rt>ちゃ</rt></ruby>を<ruby>飲む<rt>のむ</rt></ruby>ところだ。
  - <ruby>駅<rt>えき</rt></ruby>に<ruby>着い<rt>つい</rt></ruby>たら、ちょうど<ruby>電車<rt>でんしゃ</rt></ruby>が<ruby>出た<rt>でた</rt></ruby>ところだった。
- 说明：
  - 时点感很强：`〜辞書形 + ところ` 表"马上要"；`〜ている + ところ` 表"正在"；`〜た + ところ` 表"刚刚"。

### 3. `〜なりに / 〜なりの`

- 用法：「按自己的方式／程度（虽然有限，但尽力）」。
- 接续：`N + なりに / なりの + N`
- 例句：
  - <ruby>私<rt>わたし</rt></ruby>は、<ruby>自分<rt>じぶん</rt></ruby>なりの<ruby>勉強<rt>べんきょう</rt></ruby>の<ruby>仕方<rt>しかた</rt></ruby>で<ruby>漢字<rt>かんじ</rt></ruby>を<ruby>覚え<rt>おぼえ</rt></ruby>ている。
  - <ruby>若い<rt>わかい</rt></ruby><ruby>人<rt>ひと</rt></ruby>たちは<ruby>自分<rt>じぶん</rt></ruby>なりの<ruby>海<rt>うみ</rt></ruby>や「よその<ruby>国<rt>くに</rt></ruby>」を<ruby>思って<rt>おもって</rt></ruby><ruby>歌っ<rt>うたっ</rt></ruby>たのだろうか。
  - <ruby>子供<rt>こども</rt></ruby>は<ruby>子供<rt>こども</rt></ruby>なりに<ruby>考え<rt>かんがえ</rt></ruby>ている。
- 说明：
  - 含有"虽然能力／程度有限，但有自己的方式"的语气。
  - 后接动词或名词皆可。

### 4. `〜に違いない`

- 用法：「一定是……、肯定……」，表示说话人很有把握的推断。
- 接续：`普通形 / な形 / N + に違いない`
- 例句：
  - <ruby>制服<rt>せいふく</rt></ruby>を<ruby>着た<rt>きた</rt></ruby><ruby>学生<rt>がくせい</rt></ruby>たちを<ruby>見て<rt>みて</rt></ruby>、この<ruby>人<rt>ひと</rt></ruby>たちは<ruby>高校生<rt>こうこうせい</rt></ruby>に<ruby>違い<rt>ちがい</rt></ruby>ないと<ruby>思った<rt>おもった</rt></ruby>。
  - <ruby>若い<rt>わかい</rt></ruby><ruby>人<rt>ひと</rt></ruby>たちは「<ruby>歌<rt>うた</rt></ruby>の<ruby>力<rt>ちから</rt></ruby>」を<ruby>楽し<rt>たのし</rt></ruby>んでいるに<ruby>違い<rt>ちがい</rt></ruby>ない。
  - <ruby>部屋<rt>へや</rt></ruby>に<ruby>明かり<rt>あかり</rt></ruby>がついている。<ruby>誰か<rt>だれか</rt></ruby>いるに<ruby>違い<rt>ちがい</rt></ruby>ない。
- 说明：
  - 比 `だろう / でしょう` 推断更强烈、更肯定。
  - 多用于书面或正式场合。

## 课文梳理

1. 作者长年在同一公司工作，但有件事始终融不进去：歓迎会、送别会等聚会后被起哄唱歌。
2. 他天生五音不全，被强行点歌时只能借助"童谣"——靠它「让大家心情放松、把场子拧成一股绳的不可思议的力量」来度过。
3. 第一次唱时，有人露出"怎么唱这个"的表情，有人露出"扫兴"的脸，但出于对上司的礼貌，大家都把脸转向他听。
4. 不知是不是想给他打气，开头几个人、最后所有人都合声一起唱起来；唱完后有一段难以再开口的安静。
5. 作者想：年轻人也在歌里怀念着各自心中的"海"和"远方的国家"；每个人带着不同的思绪，最后却汇成一个声音——这就是歌的力量。从前祭典、田植、捕鱼时人们都靠歌互相鼓劲；今天年轻人去 KTV，也是在不自觉地享受这种力量。

## 练习题

### 一、词汇练习

1. `生まれ（　　　　　　　）歌が苦手だ。`
   - 答案：`つき`
   - 解析：`生まれつき` = 天生。

2. `心を（　　　　　　　）歌だ。`
   - 答案：`和ませる`
   - 解析：`和ませる` = 让……缓和。

3. `みんなで（　　　　　　　）をそろえて歌った。`
   - 答案：`声`
   - 解析：`声をそろえる` = 齐声。

### 二、语法练习

4. `事故で地下鉄が止まってしまった（　　　　　　　）、遅れました。`
   - 答案：`ものですから`
   - 解析：`〜ものですから` = 因为……（说明）。

5. `今、ちょうどご飯を食べている（　　　　　　　）だ。`
   - 答案：`ところ`
   - 解析：正在做 `〜ているところだ`。

6. `子供は子供（　　　　　　　）にいろいろ考えている。`
   - 答案：`なり`
   - 解析：`〜なりに` = 按自己的方式。

7. `明かりがついている。誰かいる（　　　　　　　）。`
   - 答案：`に違いない`
   - 解析：`〜に違いない` = 一定是……。

### 三、短答题

8. `本文中、筆者が「歌の力」と呼ぶのはどんな力か？`
   - 参考答案：每个人各怀不同心思、不同回忆地唱同一首歌，最后却能把所有人的声音汇成一个、把心拉到一起的不可思议的力量。

9. `用 〜に違いない 写一句关于身边人的句子。`
   - 参考答案：`雨に濡れているから、彼は傘を持っていなかったに違いない。`

## 本课复习要点

- 能用 `〜ものだから／〜もので` 软语气地说明原因。
- 能区分 `〜ところだ` 的三种时点（要做／正在做／刚做完）。
- 能用 `〜なりに／〜なりの` 表达"按自己的方式"。
- 能用 `〜に違いない` 做出有把握的推断。
