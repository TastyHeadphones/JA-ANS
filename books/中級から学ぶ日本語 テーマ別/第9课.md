# 第9课 まもる ―「見えない相手」

> 对应书内第9课（PDF 约第61–66页）。
> 这一课的主轴是「ルールとモラル」，重点学习"理应如此""不可能如此""作为……""越……越……"等表达。

## 本课主旨

叙述者观察到，在日本，即使没有人监督，人们也会严格遵守交通规则——凌晨的十字路口，没有车也没有人，行人依然等红灯。叙述者一开始觉得这太拘谨，但后来理解到：日本的"规则遵守"不是出于恐惧被罚，而是对规则本身的信任，以及对"作为社会成员应该做的事"的自觉。与此同时，叙述者也思考，当规则与实际情况产生矛盾时，如何判断才是真正的"守规则"。

## 导入问题

课前会先围绕下面三个问题展开：

- 在你的国家，人们遵守交通规则吗？
- 没有人看的时候，你会遵守规则吗？
- 你觉得"规则"对社会来说有多重要？

### 补充表达

| 表达 | 中文释义 | 例句 |
|---|---|---|
| 〜のではないか | 是不是……吧（推测/质疑） | これが本当のモラルなのではないか。 |
| どんな〜でも | 无论……都…… | どんな状況でもルールを守るのは難しい。 |

## 文法要点

### 1. `〜はずだ`

- 用法：根据已知情况或逻辑推断，表示"理应如此""应该是这样"。
- 接续：
  - `V辞書形 / Vない形 / Vた形 + はずだ`
  - `い形容詞 + はずだ`
  - `な形容詞（な）+ はずだ`
  - `N（の）+ はずだ`
- 例句：
  - 電車は5分後に来るはずだ。
  - 毎日練習しているから、日本語が上手になっているはずだ。
  - 彼はもう知っているはずなのに、何も言わない。
- 说明：
  - 表示说话人基于某种根据推断出应有的结果。
  - 与 `〜だろう`（推测，主观判断）区分：`はずだ` 更强调有逻辑依据。

### 2. `〜はずがない`

- 用法：根据推断，表示"不可能是这样""绝对不会……"。
- 接续：
  - `V辞書形 / Vた形 / Vない形 + はずがない`
  - `い形容詞 + はずがない`
  - `な形容詞（な）+ はずがない`
  - `N（の）+ はずがない`
- 例句：
  - 彼がそんなことを言うはずがない。
  - 一日で日本語が上手になるはずがない。
  - そんなに安いはずがない。何か問題があるはずだ。
- 说明：
  - 是 `〜はずだ` 的否定，表示逻辑上不可能。
  - 语气比 `〜わけがない` 稍客观，`〜わけがない` 更情绪化。

### 3. `〜として`

- 用法：表示身份、立场或角色，"作为……""以……身份"。
- 接续：
  - `N + として`
- 例句：
  - 社会の一員として、ルールを守る義務がある。
  - 留学生として日本で生活している。
  - 友達として、正直に話してあげた。
- 说明：
  - 表示以某种身份或立场来做事。
  - 注意与 `〜にとって`（对……来说）区分：`として` 表示角色，`にとって` 表示评判立场。

### 4. `〜ば〜ほど`

- 用法：表示随着条件程度的加深，另一个结果也相应变化，"越……越……"。
- 接续：
  - `Vば形 + V辞書形 + ほど`
  - `い形容詞（〜ければ）+ い形容詞 + ほど`
  - `な形容詞（〜なら/であれば）+ な形容詞 + ほど`
- 例句：
  - 日本語は、勉強すればするほど面白くなる。
  - 規則は、厳しければ厳しいほど守りにくくなることもある。
  - 社会のモラルは、多くの人が守れば守るほど強くなる。
- 说明：
  - 前后两个动词或形容词通常是同一词，或有因果关联的两个词。
  - 常见格式：`Vば + V辞書形 + ほど`（两次出现同一动词）。

## 课文梳理

1. 深夜的十字路口，没有车也没有人，行人仍然等红灯——叙述者目睹这一幕，感到既惊讶又困惑。
2. 叙述者起初认为，这种情况下遵守规则似乎没有必要，甚至有些拘谨。
3. 思考后理解：日本人遵守规则不是因为有人监视，而是出于"作为社会成员的自觉"（社会の一員として）。
4. 越是大家都遵守规则，社会的信赖就越强（守れば守るほど）；规则不是外部约束，而是内部信念。
5. 叙述者也提出疑问：当规则与现实发生矛盾时，应该如何判断，才是真正意义上的"守规则"？

## 练习题

### 一、词汇练习

1. `深夜の（　　　　　）でも、きちんと信号を守っていた。`
   - 答案：`交差点`
   - 解析：`交差点` = 十字路口。

2. `社会の（　　　　　）として、責任ある行動をすることが大切だ。`
   - 答案：`一員`
   - 解析：`社会の一員` = 社会的一员。

3. `法律に（　　　　　）すると、罰せられることがある。`
   - 答案：`違反`
   - 解析：`違反する` = 违反。

4. `ルールへの（　　　　　）があるからこそ、守ることができる。`
   - 答案：`信頼`
   - 解析：`信頼` = 信任、信赖。

### 二、语法练习

5. `毎日練習しているから、もう上手になっている（　　　　　）だ。`
   - 答案：`はず`
   - 解析：`〜はずだ` = 理应如此（有依据的推断）。

6. `一日で上手になる（　　　　　）がない。`
   - 答案：`はず`
   - 解析：`〜はずがない` = 不可能……。

7. `留学生（　　　　　）、日本の文化をたくさん学びたい。`
   - 答案：`として`
   - 解析：`〜として` = 作为……（身份/立场）。

8. `日本語は、勉強すれ（　　　　　）するほど面白くなる。`
   - 答案：`ば`
   - 解析：`〜ば〜ほど` = 越……越……。

9. `規則は、みんなが守れ（　　　　　）守るほど、社会がよくなる。`
   - 答案：`ば`
   - 解析：`〜ば〜ほど` = 越……越……。

### 三、短答题

10. `文中で、筆者が「日本人がルールを守る理由」についてどう理解しましたか？`
    - 参考答案：不是因为害怕被惩罚或有人监视，而是出于"作为社会成员的自觉"——认为遵守规则是应尽的义务，也是对社会信赖的维护。

11. `用 〜ば〜ほど 写一句话。`
    - 参考答案：`人との交流は、続ければ続けるほど関係が深まる。`

## 本课复习要点

- 能用 `〜はずだ` 表达基于逻辑或根据的推断，能用 `〜はずがない` 表达"不可能……"。
- 能用 `〜として` 表达"作为……身份/立场"，并与 `〜にとって` 区分。
- 能用 `〜ば〜ほど` 表达"越……越……"的递进关系。
