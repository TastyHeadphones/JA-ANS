# 第14课 みなおす ―「てるてるぼうず」

> 对应书内第14课（PDF 约第81–86页）。
> 这一课的主轴是「天気予報と人間の自然観」，重点学习"在……方面""根据……（情况）""伴随……""尽管……（也）"等表达。

## 本课主旨

预测天气与气候自古以来是人类生活不可或缺的本领。日本各地至今仍有"夕烧次日必晴""山雪化到哪儿就开始耕田"之类的谚语；雨天人们做"晴天娃娃"（てるてるぼうず）祈祷天晴。明治以来天气预报开始正式发布，随着计算机的使用，气象观测技术大幅进步，预报越发精准，今天人们出门带不带伞都靠它决定。课文反思：科学预报方便的同时，是不是也让我们不再观察天空、不再与自然对话？

## 导入问题

课前会先围绕下面三个问题展开：

- 你每天看天气预报吗？
- 你的家乡有没有关于天气的谚语？
- 不看预报的话，你能从什么迹象判断天气？

## 文法要点

### 1. `〜上で`（在……方面/前提下）

- 用法：「在做某事的过程中／在……方面」。表示从事某活动时所必需的方面。
- 接续：
  - `V辞書形 + 上で`
  - `Nの + 上で`
- 例句：
  - 外国語を学ぶ上で、その国の文化を知ることはとても大切だ。
  - 人間が毎日の生活を続けていく上で、欠かせない営みだ。
  - 仕事を進める上で、相手とのコミュニケーションが必要だ。
- 说明：
  - 与 `〜た上で`（先做……再做）不同：这里强调"做某事的过程中/作为前提"。
  - 后续常是 "重要だ／必要だ／大切だ" 等评价。

### 2. `〜次第（で）／〜次第だ`

- 用法：「根据……（来决定）」「全看……如何」。
- 接续：`N + 次第（で）／次第だ`
- 例句：
  - 試験の結果次第では、進学先を変えざるを得ない。
  - 天気次第で、収穫が大きく影響を受ける。
  - 結果がどうなるかは、あなたの努力次第だ。
- 说明：
  - 后接 `で` 表示"取决于"；接 `だ` 表示"全在于……"。
  - 与"〜によって"相比，更强调"决定性影响"。

### 3. `〜とともに`

- 用法：「随着……（同时）」「与……一起」。
- 接续：
  - `V辞書形 / N + とともに`
- 例句：
  - コンピュータが使われ始めるとともに、気象観測の技術が大きく進歩した。
  - この村では、季節の変化とともに様々な景色が見られる。
  - 経済の発展とともに、生活が便利になった。
- 说明：
  - 表示同时变化（伴随关系），书面色彩浓。
  - 也可表示"和……一起"（伴随对象）。

### 4. `〜にもかかわらず`

- 用法：「尽管……（也／却）」，表示出乎预料的结果。
- 接续：`普通形 / N / な形容詞〜である + にもかかわらず`
- 例句：
  - 親の反対にもかかわらず、兄は外国へ行って仕事を探している。
  - 雨にもかかわらず、試合は予定通り行われた。
  - 努力したにもかかわらず、合格できなかった。
- 说明：
  - 比 `〜のに` 更书面、更正式。
  - 用于强调"虽然……，但仍/却……"。

## 课文梳理

1. 自古以来，预测天气和气候，对人类的日常生活是不可或缺的本领。
2. 收成受天气左右，进山或出海找食物时也常因天气丢命，因此人们仔细观察天空的颜色、云的流向、风向、土水的温度，努力与自然协调而生活。
3. 至今日本各地仍保留着许多关于天气的"言い伝え"——"夕烧次日必晴""山雪到哪里融化就开始耕田"等等。这些经验未必准确，但反映了人们对天气的关注。
4. 雨天里农民向神祈雨，孩子做晴天娃娃祈晴。1884 年日本第一次发布天气预报，起初很简单。
5. 随着计算机的使用，气象观测技术大幅进步，预报越来越准确，今天人们决定农活、旅行、穿什么、带不带伞，都参考天气预报。

## 练习题

### 一、词汇练习

1. `自然の（　　　　　　　）を観察し続けてきた。`
   - 答案：`法則`
   - 解析：`法則` = 法则、规律。

2. `天気予報は今や生活に（　　　　　　　）情報だ。`
   - 答案：`欠かせない`
   - 解析：`欠かせない` = 不可缺少的。

3. `各地に天気の（　　　　　　　）が残っている。`
   - 答案：`言い伝え`
   - 解析：`言い伝え` = 口耳相传的话。

### 二、语法练习

4. `外国語を学ぶ（　　　　　　　）、文化を知ることは大切だ。`
   - 答案：`上で`
   - 解析：`〜上で` = 在……方面。

5. `天気（　　　　　　　）で、出かけるかどうかを決める。`
   - 答案：`次第`
   - 解析：`〜次第で` = 根据……。

6. `経済の発展（　　　　　　　）、生活が変わった。`
   - 答案：`とともに`
   - 解析：`〜とともに` = 随着……。

7. `親の反対（　　　　　　　）、留学を決めた。`
   - 答案：`にもかかわらず`
   - 解析：`〜にもかかわらず` = 尽管……。

### 三、短答题

8. `本文では、天気予報の発達がもたらしたものは何か？`
   - 参考答案：使收成、旅行日期、出门衣物乃至带不带伞这些日常决定都变得可以"科学地"判断；同时也让人们不再像过去那样仔细观察天空、与自然对话。

9. `用 〜にもかかわらず 写一个关于自己生活的例句。`
   - 参考答案：`雨にもかかわらず、彼女は時間通りに来てくれた。`

## 本课复习要点

- 能用 `〜上で` 表达"在做某事的过程中／作为前提"。
- 能用 `〜次第（で）` 表达"根据……（结果会变）"。
- 能用 `〜とともに` 描述"随着……变化"。
- 能用 `〜にもかかわらず` 表达"尽管……仍／却……"。
