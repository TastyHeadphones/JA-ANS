# 第14课 みなおす ―「てるてるぼうず」

> 对应书内第14课（PDF 约第81–86页）。
> 这一课的主轴是「<ruby>天気予報<rt>てんきよほう</rt></ruby>と<ruby>人<rt>ひと</rt></ruby><ruby>間の<rt>まの</rt></ruby><ruby>自然観<rt>しぜんかん</rt></ruby>」，重点学习"在……方面""根据……（情况）""伴随……""尽管……（也）"等表达。

## 本课主旨

预测天气与气候自古以来是人类生活不可或缺的本领。日本各地至今仍有"夕烧次日必晴""山雪化到哪儿就开始耕田"之类的谚语；雨天人们做"晴天娃娃"（てるてるぼうず）祈祷天晴。明治以来天气预报开始正式发布，随着计算机的使用，气象观测技术大幅进步，预报越发精准，今天人们出门带不带伞都靠它决定。课文反思：科学预报方便的同时，是不是也让我们不再观察天空、不再与自然对话？

## 导入问题

课前会先围绕下面三个问题展开：

- 你每天看天气预报吗？
- 你的家乡有没有关于天气的谚语？
- 不看预报的话，你能从什么迹象判断天气？

## 文法要点

### 1. `〜上で`（在……方面/前提下）

- 用法：「在做某事的过程中／在……方面」。表示从事某活动时所必需的方面。
- 接续：
  - `V辞書形 + 上で`
  - `Nの + 上で`
- 例句：
  - <ruby>外国語<rt>がいこくご</rt></ruby>を<ruby>学ぶ<rt>まなぶ</rt></ruby><ruby>上で<rt>うえで</rt></ruby>、その<ruby>国<rt>くに</rt></ruby>の<ruby>文化<rt>ぶんか</rt></ruby>を<ruby>知る<rt>しる</rt></ruby>ことはとても<ruby>大切<rt>たいせつ</rt></ruby>だ。
  - <ruby>人<rt>ひと</rt></ruby><ruby>間<rt>かん</rt></ruby>が<ruby>毎日<rt>まいにち</rt></ruby>の<ruby>生活<rt>せいかつ</rt></ruby>を<ruby>続け<rt>つづけ</rt></ruby>ていく<ruby>上で<rt>うえで</rt></ruby>、<ruby>欠か<rt>かか</rt></ruby>せない<ruby>営み<rt>いとなみ</rt></ruby>だ。
  - <ruby>仕事<rt>しごと</rt></ruby>を<ruby>進め<rt>すすめ</rt></ruby>る<ruby>上で<rt>うえで</rt></ruby>、<ruby>相手<rt>あいて</rt></ruby>とのコミュニケーションが<ruby>必要<rt>ひつよう</rt></ruby>だ。
- 说明：
  - 与 `〜た上で`（先做……再做）不同：这里强调"做某事的过程中/作为前提"。
  - 后续常是 "<ruby>重要<rt>じゅうよう</rt></ruby>だ／<ruby>必要<rt>ひつよう</rt></ruby>だ／<ruby>大切<rt>たいせつ</rt></ruby>だ" 等评价。

### 2. `〜次第（で）／〜次第だ`

- 用法：「根据……（来决定）」「全看……如何」。
- 接续：`N + 次第（で）／次第だ`
- 例句：
  - <ruby>試験<rt>しけん</rt></ruby>の<ruby>結果次第<rt>けっかしだい</rt></ruby>では、<ruby>進学先<rt>しんがくさき</rt></ruby>を<ruby>変え<rt>かえ</rt></ruby>ざるを<ruby>得な<rt>えな</rt></ruby>い。
  - <ruby>天気<rt>てんき</rt></ruby><ruby>次第<rt>しだい</rt></ruby>で、<ruby>収穫<rt>しゅうかく</rt></ruby>が<ruby>大きく<rt>おおきく</rt></ruby><ruby>影響を受ける<rt>えいきょうをうける</rt></ruby>。
  - <ruby>結果<rt>けっか</rt></ruby>がどうなるかは、あなたの<ruby>努力<rt>どりょく</rt></ruby><ruby>次第<rt>しだい</rt></ruby>だ。
- 说明：
  - 后接 `で` 表示"取决于"；接 `だ` 表示"全在于……"。
  - 与"〜によって"相比，更强调"决定性影响"。

### 3. `〜とともに`

- 用法：「随着……（同时）」「与……一起」。
- 接续：
  - `V辞書形 / N + とともに`
- 例句：
  - コンピュータが<ruby>使わ<rt>つかわ</rt></ruby>れ<ruby>始め<rt>はじめ</rt></ruby>るとともに、<ruby>気象<rt>きしょう</rt></ruby><ruby>観測<rt>かんそく</rt></ruby>の<ruby>技術<rt>ぎじゅつ</rt></ruby>が<ruby>大きく<rt>おおきく</rt></ruby><ruby>進歩<rt>しんぽ</rt></ruby>した。
  - この<ruby>村<rt>むら</rt></ruby>では、<ruby>季節<rt>きせつ</rt></ruby>の<ruby>変化<rt>へんか</rt></ruby>とともに<ruby>様々な<rt>さまざまな</rt></ruby><ruby>景色<rt>けしき</rt></ruby>が<ruby>見ら<rt>みら</rt></ruby>れる。
  - <ruby>経済<rt>けいざい</rt></ruby>の<ruby>発展<rt>はってん</rt></ruby>とともに、<ruby>生活<rt>せいかつ</rt></ruby>が<ruby>便利<rt>べんり</rt></ruby>になった。
- 说明：
  - 表示同时变化（伴随关系），书面色彩浓。
  - 也可表示"和……一起"（伴随对象）。

### 4. `〜にもかかわらず`

- 用法：「尽管……（也／却）」，表示出乎预料的结果。
- 接续：`普通形 / N / な形容詞〜である + にもかかわらず`
- 例句：
  - <ruby>親<rt>おや</rt></ruby>の<ruby>反対<rt>はんたい</rt></ruby>にもかかわらず、<ruby>兄<rt>あに</rt></ruby>は<ruby>外国<rt>がいこく</rt></ruby>へ<ruby>行って<rt>いって</rt></ruby><ruby>仕事<rt>しごと</rt></ruby>を<ruby>探し<rt>さがし</rt></ruby>ている。
  - <ruby>雨<rt>あめ</rt></ruby>にもかかわらず、<ruby>試合<rt>しあい</rt></ruby>は<ruby>予定<rt>よてい</rt></ruby><ruby>通り<rt>とおり</rt></ruby><ruby>行わ<rt>おこなわ</rt></ruby>れた。
  - <ruby>努力<rt>どりょく</rt></ruby>したにもかかわらず、<ruby>合格<rt>ごうかく</rt></ruby>できなかった。
- 说明：
  - 比 `〜のに` 更书面、更正式。
  - 用于强调"虽然……，但仍/却……"。

## 课文梳理

1. 自古以来，预测天气和气候，对人类的日常生活是不可或缺的本领。
2. 收成受天气左右，进山或出海找食物时也常因天气丢命，因此人们仔细观察天空的颜色、云的流向、风向、土水的温度，努力与自然协调而生活。
3. 至今日本各地仍保留着许多关于天气的"<ruby>言い伝え<rt>いいつたえ</rt></ruby>"——"夕烧次日必晴""山雪到哪里融化就开始耕田"等等。这些经验未必准确，但反映了人们对天气的关注。
4. 雨天里农民向神祈雨，孩子做晴天娃娃祈晴。1884 年日本第一次发布天气预报，起初很简单。
5. 随着计算机的使用，气象观测技术大幅进步，预报越来越准确，今天人们决定农活、旅行、穿什么、带不带伞，都参考天气预报。

### 一、词汇练习

   - 解析：`法則` = 法则、规律。

   - 答案：`欠かせない`

### 二、语法练习

4. `外国語を学ぶ（　　　　　　　）、文化を知ることは大切だ。`
   - 答案：`上で`
   - 解析：`〜上で` = 在……方面。

5. `天気（　　　　　　　）で、出かけるかどうかを決める。`
   - 答案：`次第`
   - 解析：`〜次第で` = 根据……。

6. `経済の発展（　　　　　　　）、生活が変わった。`
   - 答案：`とともに`
   - 解析：`〜とともに` = 随着……。

7. `親の反対（　　　　　　　）、留学を決めた。`
   - 答案：`にもかかわらず`
   - 解析：`〜にもかかわらず` = 尽管……。

### 三、短答题

8. `本文では、天気予報の発達がもたらしたものは何か？`
   - 参考答案：使收成、旅行日期、出门衣物乃至带不带伞这些日常决定都变得可以"科学地"判断；同时也让人们不再像过去那样仔细观察天空、与自然对话。

9. `用 〜にもかかわらず 写一个关于自己生活的例句。`
   - 参考答案：`雨にもかかわらず、彼女は時間通りに来てくれた。`

## 本课复习要点

- 能用 `〜上で` 表达"在做某事的过程中／作为前提"。
- 能用 `〜次第（で）` 表达"根据……（结果会变）"。
- 能用 `〜とともに` 描述"随着……变化"。
- 能用 `〜にもかかわらず` 表达"尽管……仍／却……"。

- **译文**：小时候，我曾经为了希望明天春游是晴天，而做过"晴天娃娃"。
- **语法**：`〜たことがある`（曾经……过）；`〜ことを願う`（祈愿……）
- **单词**：`遠足`（名词，郊游/远足）；`晴れる`（自动词，放晴）；`願う`（他动词，祈愿）；`てるてるぼうず`（名词，晴天娃娃）

---

**②** てるてるぼうずは、<ruby>昔<rt>むかし</rt></ruby>の<ruby>日本<rt>にほん</rt></ruby><ruby>人<rt>ひと</rt></ruby>が<ruby>天気<rt>てんき</rt></ruby>を<ruby>予測する<rt>よそくする</rt></ruby><ruby>上で<rt>うえで</rt></ruby>、<ruby>自然<rt>しぜん</rt></ruby>への<ruby>感謝<rt>かんしゃ</rt></ruby>を<ruby>込め<rt>こめ</rt></ruby>て<ruby>作り<rt>つくり</rt></ruby><ruby>始め<rt>はじめ</rt></ruby>たものだと<ruby>言わ<rt>いわ</rt></ruby>れている。

- **译文**：据说晴天娃娃是古代日本人在预测天气时，为表达对自然的感谢而开始制作的。
- **语法**：`〜上で`（在……的过程中/作为前提）；`〜と言われている`（据说……）；`〜を込めて`（怀着……）
- **单词**：`予測する`（サ变，预测）；`感謝`（名词，感谢）；`込める`（他动词，含有/寄寓）

---

**③** <ruby>現代<rt>げんだい</rt></ruby>の<ruby>天気予報<rt>てんきよほう</rt></ruby>の<ruby>技術<rt>ぎじゅつ</rt></ruby>が<ruby>発達<rt>はったつ</rt></ruby>するとともに、てるてるぼうずを<ruby>作る<rt>つくる</rt></ruby><ruby>習慣<rt>しゅうかん</rt></ruby>は<ruby>薄れ<rt>うすれ</rt></ruby>てきた。

- **译文**：随着现代天气预报技术的发展，制作晴天娃娃的习惯也在渐渐淡薄。
- **语法**：`〜とともに`（随着……）；`〜てくる`（变化朝向现在）
- **单词**：`天気予報`（名词，天气预报）；`技術`（名词，技术）；`発達する`（サ变，发展）；`薄れる`（自动词，淡薄）

---

**④** <ruby>天気予報<rt>てんきよほう</rt></ruby>の<ruby>精度<rt>せいど</rt></ruby><ruby>次第<rt>しだい</rt></ruby>で、<ruby>人<rt>ひと</rt></ruby>々の<ruby>行動<rt>こうどう</rt></ruby>が<ruby>変わ<rt>かわ</rt></ruby>る<ruby>現代<rt>げんだい</rt></ruby>でも、てるてるぼうずの<ruby>文化<rt>ぶんか</rt></ruby>は<ruby>消え<rt>きえ</rt></ruby>ていない。

- **译文**：即便在如今天气预报精度左右人们行动的时代，晴天娃娃的文化也没有消失。
- **语法**：`〜次第で`（根据……，结果会变）；`〜でも`（即使……也）
- **单词**：`精度`（名词，精度/准确度）；`次第で`（根据……）；`行動`（名词，行动）

---

**⑤** <ruby>科学<rt>かがく</rt></ruby>が<ruby>発達<rt>はったつ</rt></ruby>しているにもかかわらず、<ruby>昔<rt>むかし</rt></ruby>からの<ruby>習慣<rt>しゅうかん</rt></ruby>を<ruby>守り<rt>まもり</rt></ruby><ruby>続け<rt>つづけ</rt></ruby>る<ruby>人<rt>ひと</rt></ruby>が<ruby>多い<rt>おおい</rt></ruby>のは、それに<ruby>意味<rt>いみ</rt></ruby>を<ruby>感じ<rt>かんじ</rt></ruby>ているからだろう。

- **译文**：尽管科学已经发展，却仍有很多人坚守古老习惯——这大概是因为他们在其中感受到了意义。
- **语法**：`〜にもかかわらず`（尽管……仍/却）；`〜からだろう`（大概是因为……）；`〜続ける`（持续……）
- **单词**：`科学`（名词，科学）；`守り続ける`（持续守护）；`意味`（名词，意义）

---

**⑥** <ruby>伝統<rt>でんとう</rt></ruby>とは、<ruby>科学的<rt>かがくてき</rt></ruby>な<ruby>根拠<rt>こんきょ</rt></ruby>の<ruby>有無<rt>うむ</rt></ruby>にかかわらず、<ruby>人<rt>ひと</rt></ruby>々の<ruby>心<rt>こころ</rt></ruby>の<ruby>中<rt>なか</rt></ruby>で<ruby>生き続け<rt>いきつづけ</rt></ruby>るものだ。

- **译文**：传统就是这样一种东西——无论有没有科学依据，它都在人们心中一直活下去。
- **语法**：`〜にかかわらず`（无论……都）；`〜ものだ`（一般性道理）；`〜続ける`（持续……）
- **单词**：`伝統`（名词，传统）；`根拠`（名词，根据）；`有無`（名词，有无）
