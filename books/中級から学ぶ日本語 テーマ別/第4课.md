# 第4课 たとえる ―「<ruby>猫<rt>ねこ</rt></ruby>に<ruby>小判<rt>こばん</rt></ruby>」

> 对应书内第4课（PDF 约第31–36页）。
> 这一课的主轴是「ことわざとたとえ」，重点学习"程度最高级表达""表示程度""希望对方不要做某事""觉得不可能"等表达。

## 本课主旨

介绍日语中用"猫"做的几个谚语和比喻表达：「<ruby>猫<rt>ねこ</rt></ruby>の<ruby>手<rt>て</rt></ruby>も<ruby>借り<rt>かり</rt></ruby>たい」（忙得连猫的手都想借）、「<ruby>猫<rt>ねこ</rt></ruby>の<ruby>額<rt>ひたい</rt></ruby>ほどの<ruby>庭<rt>にわ</rt></ruby>」（和猫额头一样小的院子）、「<ruby>猫<rt>ねこ</rt></ruby>に<ruby>小判<rt>こばん</rt></ruby>」（对猫展示金币毫无意义）。由此引申：不只是猫，狗、牛、马等与人类长期共生的动物也被用于谚语，因为比喻能用简洁有趣的方式传达意思。

## 导入问题

课前会先围绕下面三个问题展开：

- 你觉得狗是一种什么样的动物？
- 猫呢？
- 你认为什么动物是"不好"的动物？

## 文法要点

### 1. `〜ほど〜はない／いない`

- 用法：表示"没有比……更……"，是最高级的表达。
- 接续：
  - `N + ほど + N + はない / いない`
  - `〜ほど + い形容詞 + ものはない`
- 例句：
  - <ruby>今まで<rt>いままで</rt></ruby><ruby>見た<rt>みた</rt></ruby><ruby>映画<rt>えいが</rt></ruby>の<ruby>中<rt>なか</rt></ruby>で、「道」ほどいい<ruby>映画<rt>えいが</rt></ruby>はありませんでした。
  - この<ruby>学校<rt>がっこう</rt></ruby>では、チンさんほどよく<ruby>勉強<rt>べんきょう</rt></ruby>する<ruby>学生<rt>がくせい</rt></ruby>はいないでしょう。
- 说明：
  - 相当于"……是最……"，但用否定形式表达，语气更自然。
  - 注意：`〜ほど` 后接肯定句是"到……程度"（见下条），接否定句才是"最高级"。

### 2. `〜ほど`（程度）

- 用法：表示"达到……的程度"，程度越高越强烈。
- 接续：
  - `V辞書形 / Vた形 + ほど`
  - `い形容詞 / な形容詞（〜な）+ ほど`
- 例句：
  - <ruby>漢字<rt>かんじ</rt></ruby>をおぼえるために、<ruby>目<rt>め</rt></ruby>をとじても<ruby>出て<rt>でて</rt></ruby>くるほどたくさん<ruby>書き<rt>かき</rt></ruby>ました。
  - ジョンさんの<ruby>話<rt>はなし</rt></ruby>を<ruby>聞い<rt>きい</rt></ruby>て、クラスの<ruby>友達<rt>ともだち</rt></ruby>はみんなおなかが<ruby>痛く<rt>いたく</rt></ruby>なるほど<ruby>笑い<rt>わらい</rt></ruby>ました。
- 说明：
  - 常用于夸张表达，描述程度极深。
  - 和 `〜くらい` 意思相近，但 `ほど` 程度感更强。

### 3. `〜ないでほしい`

- 用法：希望对方不要做某事。
- 接续：
  - `Vない形 + ないでほしい`
- 例句：
  - <ruby>母<rt>はは</rt></ruby>に<ruby>新しい<rt>あたらしい</rt></ruby>パソコンを<ruby>買っ<rt>かっ</rt></ruby>てほしいとたのんだが、まだ<ruby>使え<rt>つかえ</rt></ruby>るから<ruby>買わ<rt>かわ</rt></ruby>ないと<ruby>言った<rt>いった</rt></ruby>。
  - <ruby>仕事<rt>しごと</rt></ruby>は<ruby>午後<rt>ごご</rt></ruby>5<ruby>時<rt>とき</rt></ruby>までだが、<ruby>店<rt>みせ</rt></ruby>の<ruby>人<rt>ひと</rt></ruby>に「<ruby>今日<rt>きょう</rt></ruby>は<ruby>忙しい<rt>いそがしい</rt></ruby>から、まだ<ruby>帰ら<rt>かえら</rt></ruby>ないでほしい」とたのまれて、午後7<ruby>時<rt>とき</rt></ruby>まで<ruby>働い<rt>はたらい</rt></ruby>た。
- 说明：
  - `〜ないでほしい` 是委婉表达，比命令句 `〜ないでください` 语气更温和。
  - `〜てほしい`（希望对方做）的否定形式。

### 4. `〜とは思えない`

- 用法：表示"无法相信／不觉得……"，强调说话人的主观否定。
- 接续：
  - `普通形 + とは思えない`
  - `N + とは思えない`
- 例句：
  - <ruby>私<rt>わたし</rt></ruby>はすぐに<ruby>月<rt>がつ</rt></ruby>に<ruby>住め<rt>すめ</rt></ruby>るようになるとは<ruby>思え<rt>おもえ</rt></ruby>ません。
  - たとえ<ruby>私<rt>わたし</rt></ruby>がたのんでみても、あの<ruby>人<rt>ひと</rt></ruby>がこの<ruby>仕事<rt>しごと</rt></ruby>をやってくれるとは<ruby>思え<rt>おもえ</rt></ruby>ません。
  - <ruby>練習<rt>れんしゅう</rt></ruby>しなければ、<ruby>日本<rt>にほん</rt></ruby><ruby>語<rt>ご</rt></ruby>が<ruby>話せ<rt>はなせ</rt></ruby>るようになるとは<ruby>思え<rt>おもえ</rt></ruby>ません。
- 说明：
  - 比 `〜と思わない` 语气更强，更强调"无论如何都无法相信"。
  - 常用 `いくら〜ても`、`たとえ〜ても` 等让步表达做前提。

## 课文梳理

1. 日语中有很多以动物为主题的谚语，尤以"猫"相关的最有趣。
2. 「<ruby>猫<rt>ねこ</rt></ruby>の<ruby>手<rt>て</rt></ruby>も<ruby>借り<rt>かり</rt></ruby>たい」：太忙了，连猫的帮助都想要——虽然猫不怎么帮得上忙，但正是这种夸张才有趣。
3. 「<ruby>猫<rt>ねこ</rt></ruby>の<ruby>額<rt>ひたい</rt></ruby>ほどの<ruby>庭<rt>にわ</rt></ruby>」：猫的额头很小，用来比喻土地/庭院狭小。
4. 「<ruby>猫<rt>ねこ</rt></ruby>に<ruby>小判<rt>こばん</rt></ruby>」：给猫价值连城的小判金币也没用，用来比喻"再好的东西对不懂的人也没有意义"。
5. 除猫外，狗、牛、马等与人长期生活的动物也被用于谚语，因为谚语和比喻能简洁有趣地传达意思，所以世代流传。

## 练习题

### 一、词汇练习

1. `今まで見た映画の中で、この映画（　　　）面白い映画はありませんでした。`
   - 答案：`ほど`
   - 解析：`〜ほど〜ない` 表示最高级。

2. `「猫に小判」は、（　　　　　）のある物でも、分からない相手には役に立たないという意味だ。`
   - 答案：`価値`
   - 解析：`価値` = 价值。

3. `忙しくて、誰かに手を（　　　　　）ほどだ。`
   - 答案：`借りたい`
   - 解析：`手を借りたい` = 想借助他人之手。

4. `皆さんの国にも同じようなことわざが（　　　　　）あると思います。`
   - 答案：`きっと`
   - 解析：`きっと` = 一定。

### 二、语法练習

5. `漢字をおぼえるために、目をとじても出てくる（　　　）書いた。`
   - 答案：`ほど`
   - 解析：程度极深，`〜ほど`。

6. `母に「まだ帰ら（　　　　　　　　　）」とたのまれた。`
   - 答案：`ないでほしい`
   - 解析：希望对方不做某事 `〜ないでほしい`。

7. `練習しなければ、上手になる（　　　　　　　）。`
   - 答案：`とは思えない`
   - 解析：无法相信会发生，`〜とは思えない`。

8. `彼女（　　　）頭のいい学生はいない。`
   - 答案：`ほど`
   - 解析：`〜ほど〜はない` 最高级。

### 三、短答題

9. `「猫に小判」は日常のどんな場面で使いますか？例を挙げてください。`
   - 参考答案：比如：给不懂音乐的人高级乐器、给不懂艺术的人名画——对方感受不到其价值，就是"<ruby>猫<rt>ねこ</rt></ruby>に<ruby>小判<rt>こばん</rt></ruby>"。

10. `用 〜ほど〜はない 写一句话，形容自己最喜欢的事物。`
    - 参考答案：`私の国の料理ほどおいしい料理はないと思っています。`

## 本课复习要点

- 能用 `〜ほど〜はない/いない` 表达最高级，并与程度用法 `〜ほど` 区分。
- 能用 `〜ないでほしい` 委婉表达"希望对方不要做某事"。
