# 第15课 ふれあう ―「旅行かばん」

> 对应书内第15课（PDF 约第87–92页）。
> 这一课的主轴是「自分を知ることと他者とのふれ合い」，重点学习"以……为首""果然不愧是""不可能……""正因为……（所以更……）"等表达。

## 本课主旨

畅销书《世界ふれ合いの旅》的作者在演讲会上回忆：他在印度遇到一位曾留学日本的年轻人，自夸"我去过亚洲各国，每个地方都有独特的气味"，年轻人却回道"对，日本也是"——这一句让作者哑口无言。他原本固执地以为整洁的日本没有气味，但回头一想，日本人自己闻不到的"日本的气味"对外人来说一定存在。年轻人的话其实是在问："不了解自己（自己的气味），怎么可能与别人真正『ふれ合う』呢？" 从此他把"自己也是有气味的"这种自觉装进了旅行箱，继续旅行。

## 导入问题

课前会先围绕下面三个问题展开：

- 你喜欢旅行吗？
- 决定旅行目的地时，你会参考什么？
- 至今为止的旅行中，哪些人或地方给你留下了深刻印象？

## 文法要点

### 1. `〜をはじめ（として）`

- 用法：「以……为代表（首），还有……」。举出最有代表性的一项后列举其他。
- 接续：`N + をはじめ（として）`
- 例句：
  - 卒業式には、前の校長先生をはじめ、定年で辞めた先生方も大勢いらっしゃった。
  - 世界には人口問題をはじめ、急がなければならない問題がたくさんある。
  - アジアをはじめ、世界中を歩き回ってきた。
- 说明：
  - 用于书面或正式语境，强调"以这个最有代表性的为首"。
  - 后续多为同类成员的列举。

### 2. `さすが（に）／さすがの〜も`

- 用法：「果然不愧是……／连……（以为不会的他/她）也……」。
- 接续：
  - `さすが（に） + V/形容詞`
  - `さすがの + N + も`
- 例句：
  - 田中さんは学生時代にスポーツをやっていたから、さすがに走るのが速い。
  - きのうの会議では、さすがの山本さんも社長に対しては我慢していた。
  - 「世界を旅した人だ」とさすがに感心した表情で聞いていた。
- 说明：
  - "さすが"含有"果然如所预期"的感叹语气。
  - "さすがの〜も"则强调"连（一向如此的）他都……"。

### 3. `〜わけがない`

- 用法：「不可能……」，强烈否定推论。
- 接续：`普通形 / な形〜な / N + である + わけがない`
- 例句：
  - 田中選手はけがで入院しているから、出られるわけがないよ。
  - カラオケが嫌いだから、来るわけがない。
  - 清潔な日本でにおいなどするわけがないと思い込んでいた。
- 说明：
  - 与 `〜はずがない` 意思接近，`〜わけがない` 更口语、更强烈。
  - 多用于否定常识性、出乎意料的事。

### 4. `〜だけに`

- 用法：「正因为……（所以更……）」，前后含因果关系，强调"格外、特别"。
- 接续：`普通形 / N / な形〜な + だけに`
- 例句：
  - ナイフやフォークは、毎日のように使う物だけに、時間をかけて磨く。
  - ここは、外国からのお客さんが多いだけに、外国人の店員もたくさんいる。
  - 思い込んでいただけに、若者の言葉がショックだった。
- 说明：
  - 用于"前句的状况是后句的强化原因"的语境。
  - 与 `だけあって`（因为……所以果然……）相比，`だけに` 更强调"格外"。

## 课文梳理

1. 一场演讲会上，《世界ふれ合いの旅》的作者以"你被人说过『你也很臭哦』吗？"这一意外的开场切入。
2. 他讲述了在印度遇到曾留日的年轻人，自己得意地说"每个地方都有独特的气味"，年轻人附和"对，日本也是"。
3. 这一句让他无言以对——他一直以为干净的日本没有气味，但其实日本人闻不出，外人却能闻到。
4. 他羞愧的是：自己根本没想过"自己生活的地方在他人眼里是什么样的"。年轻人那句话仿佛在问：不了解自己，怎么可能与别人真正"ふれ合う"？
5. 演讲尾声，他说："如今，我把『自己也是有气味的』这份自觉装进旅行箱，继续旅行。" 全场报以热烈掌声。

## 练习题

### 一、词汇练习

1. `若者の（　　　　　　　）一言が私を変えた。`
   - 答案：`何気ない`
   - 解析：`何気ない` = 不经意的。

2. `ほめられて、つい（　　　　　　　）になってしまった。`
   - 答案：`いい気`
   - 解析：`いい気になる` = 得意忘形。

3. `説明を聞いて（　　　　　　　）した。`
   - 答案：`納得`
   - 解析：`納得する` = 理解、信服。

### 二、语法练习

4. `卒業式には校長先生（　　　　　　　）、多くの先生がいた。`
   - 答案：`をはじめ`
   - 解析：`〜をはじめ` = 以……为首。

5. `田中さんは（　　　　　　　）走るのが速い。`
   - 答案：`さすがに`
   - 解析：`さすが（に）` = 果然不愧是。

6. `カラオケがきらいだから、来る（　　　　　　　）。`
   - 答案：`わけがない`
   - 解析：`〜わけがない` = 不可能。

7. `毎日使うものだ（　　　　　　　）、ていねいに磨く。`
   - 答案：`けに`
   - 解析：`〜だけに` = 正因为……。

### 三、短答题

8. `若者の言葉「日本もにおいがする」が講演者をどう変えたか。`
   - 参考答案：他意识到自己从未思考过"自己生活的地方在他人看来是什么样"，对自己的无自觉感到羞愧；从此他怀着"自己也是有气味的"这种自觉继续旅行，把这份自觉装进旅行箱。

9. `用 〜だけに 写一个自己的例句。`
   - 参考答案：`日本に住んでいただけに、日本語の発音がきれいだ。`

## 本课复习要点

- 能用 `〜をはじめ` 列举代表性成员。
- 能用 `さすが（に）／さすがの〜も` 表达"果然不愧""连他也……"。
- 能用 `〜わけがない` 强烈否定可能性。
- 能用 `〜だけに` 表达"正因为……（所以更……）"。
