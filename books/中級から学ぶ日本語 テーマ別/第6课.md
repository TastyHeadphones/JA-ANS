# 第6课 つたえる ―「思いやり」

> 对应书内第6课（PDF 约第43–48页）。
> 这一课的主轴是「言葉と思いやり」，重点学习"举例列举""虽说……但""理应如此""不该如此""自以为……"等表达。

## 本课主旨

叙述者在日语课上学到，日本人认为"不麻烦别人"是美德，所以即使需要帮助也往往不开口。叙述者对此感到费解——如果不说出来，对方怎么知道你需要帮助？通过和日本朋友的对话，叙述者了解到：日本文化中，察觉对方的需求（思いやり）比直接说出来更被重视。叙述者最终也思考：自己文化中的"直接表达"和日本式的"不言而喻"，哪种更能传达真正的关心？

## 导入问题

课前会先围绕下面三个问题展开：

- 你能感觉到别人对你的关心吗？你是怎么感受到的？
- 需要帮助时，你会直接开口说吗？
- 你认为"思いやり"（体贴、为对方着想）是什么意思？

## 文法要点

### 1. `〜とか〜とか`

- 用法：从多个事物中举出几个代表性的例子，表示"……啊……啊之类的"。
- 接续：
  - `N / 普通形 + とか + N / 普通形 + とか`
- 例句：
  - 日本語を勉強するには、単語とか文法とかをしっかり覚える必要がある。
  - 週末は友達と映画を見るとか、カラオケに行くとかして過ごします。
  - 彼女は「疲れた」とか「忙しい」とか言って、なかなか会ってくれない。
- 说明：
  - `〜とか〜とか` 用于非穷举式列举，语气比 `〜や〜` 更口语。
  - 可以接名词，也可以接句子（普通形）。

### 2. `〜といっても`

- 用法：提出某个前提，随即对其加以限定或修正，表示"虽说……但（实际上没那么……）"。
- 接续：
  - `N / 普通形 + といっても`
- 例句：
  - 日本語が話せるといっても、まだ敬語はあまりわかりません。
  - 旅行に行くといっても、一泊二日の近い所だ。
  - 家族といっても、もう何年も会っていない。
- 说明：
  - 用来纠正对方可能产生的过高期待或误解。
  - 前半是承认的前提，后半是实际限制或补充说明。

### 3. `〜ものだ`

- 用法①：表示一般性的常识、理所当然的道理，带有"本来就是这样的"语感。
- 接续：
  - `V辞書形 / V ない形 + ものだ`
  - `い形容詞 / な形容詞 + ものだ`
- 例句：
  - 困ったときには、助け合うものだ。
  - 人の話を聞くときは、ちゃんと顔を向けるものだ。
  - 子どもは外で元気に遊ぶものだ。
- 说明：
  - 用于说明常理、习惯或社会规范，有时带有轻微的说教意味。
  - 不能用于自己当下正在做的事，而是讲一般性的道理。

### 4. `〜ものではない`

- 用法：表示"不应该做……""……是不对的"，是对行为的否定性评价。
- 接续：
  - `V辞書形 + ものではない`
- 例句：
  - 人の悪口を言うものではない。
  - 知らない人から物をもらうものではないと、親に言われた。
  - 人の前で大きな声で怒鳴るものではない。
- 说明：
  - 是 `〜ものだ` 的否定，表示"按道理不该这样做"。
  - 带有强烈的规范性，常用于忠告、教导或批评。

### 5. `〜たつもり`

- 用法：表示说话人"自以为做了某事"或"自认为处于某种状态"，但实际上未必如此。
- 接续：
  - `Vた形 + つもり`
  - `V辞書形 / Vない形 + つもり`（表意图时）
- 例句：
  - 気持ちを伝えたつもりだったが、相手には伝わっていなかった。
  - 日本語の敬語はわかっているつもりだが、まだ間違えることがある。
  - 十分説明したつもりだったが、みんなに理解してもらえなかった。
- 说明：
  - `〜たつもり` 强调主观认为已经做了，实际结果可能不符合。
  - 与表示意图的 `〜つもりだ`（打算做）区分：`〜たつもり` 是回顾已做之事。

## 课文梳理

1. 叙述者在日语课上学到，日本人认为"不给别人添麻烦"是美德，经常忍住不说出自己的需求。
2. 叙述者感到费解：如果不说，对方怎么知道你需要帮助？直接表达不是更省事吗？
3. 日本朋友解释：比起说出来，日本文化更重视"察觉"——看出对方的需要，不等对方开口就行动，才叫真正的思いやり。
4. 叙述者回想，自己有时以为自己表达了关心（伝えたつもり），但对方其实没有感受到。
5. 最后的思考：直接表达也好，不言而喻也好，重要的是"对方有没有感受到你的关心"。

## 练习题

### 一、词汇练习

1. `困ったときには、お互いに（　　　　　）を使って助け合うことが大切だ。`
   - 答案：`気`
   - 解析：`気を使う` = 顾虑、费心。

2. `自分の気持ちをうまく（　　　　　）にするのは難しい。`
   - 答案：`言葉`
   - 解析：`言葉にする` = 用语言表达出来。

3. `人に（　　　　　）をかけないようにするのが礼儀だと言われている。`
   - 答案：`迷惑`
   - 解析：`迷惑をかける` = 给别人添麻烦。

4. `相手の気持ちを（　　　　　）ことが、思いやりの第一歩だ。`
   - 答案：`察する`
   - 解析：`察する` = 察觉、揣摩。

### 二、语法练习

5. `日本語が話せる（　　　　　）、まだ敬語は難しい。`
   - 答案：`といっても`
   - 解析：`〜といっても` = 虽说……但实际上……。

6. `困ったときには、助け合う（　　　　　）だ。`
   - 答案：`もの`
   - 解析：`〜ものだ` = 理所当然、本来如此。

7. `人の悪口を言う（　　　　　）ではない。`
   - 答案：`もの`
   - 解析：`〜ものではない` = 不应该……。

8. `気持ちを伝え（　　　　　）だったが、相手には届いていなかった。`
   - 答案：`たつもり`
   - 解析：`〜たつもり` = 自以为做了……。

9. `週末は本を読む（　　　　　）、散歩する（　　　　　）して過ごした。`
   - 答案：`とか`、`とか`
   - 解析：`〜とか〜とか` = ……啊……啊（举例列举）。

### 三、短答题

10. `「思いやり」という言葉の意味を、课文的内容で説明してください。`
    - 参考答案：察觉对方的需求，不等对方开口就主动给予帮助或关心，是日本文化中的一种美德。

11. `用 〜ものだ 写一句关于礼仪或习惯的句子。`
    - 参考答案：`お世話になった人には、お礼を言うものだ。`

## 本课复习要点

- 能用 `〜とか〜とか` 进行非穷举式的举例列举。
- 能用 `〜といっても` 对前提做限定说明，表示"虽说……但实际上……"。
- 能区分 `〜ものだ`（理所当然）和 `〜ものではない`（不应该），理解其规范性用法。
- 能用 `〜たつもり` 表达"自以为做了某事"，并理解与现实可能存在落差。
