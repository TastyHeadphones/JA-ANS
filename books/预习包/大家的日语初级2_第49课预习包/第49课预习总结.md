# 大家的日语 初级2 第49课预习总结

> 用途：课前预习、考试前速查。  
> 整理方式：以本仓库现有第49课习题、总复习包词表和语法总结做交叉整理。  
> 本课核心：`尊敬表达（被动形）`、`お/ご〜になる`、`〜ていらっしゃる`、`特殊尊敬动词`、`お/ご〜ください`。  
> 这课最容易出错的地方：`〜られる` 不是只表示被动；尊敬语只用于对方/第三方；`お` 和 `ご` 前缀要分清；`お/ご〜になる` 和 `お/ご〜ください` 方向相反。

---

## 1. 本课你要先抓住什么

| 项目 | 内容 |
|---|---|
| 主题重点 | 对上位者、客户、老师、长辈说话时，把对方动作说得更郑重 |
| 核心文法 | `V（ら）れます`、`お/ご〜になる`、`Vていらっしゃいます`、`特殊尊敬动词`、`お/ご〜ください` |
| 会话场景 | 见面、接待、电话、询问时间、请稍等、请填写/确认 |
| 词汇重点 | `いらっしゃいます`、`お戻りになります`、`ご覧になります`、`召し上がります`、`お待ちください`、`ご確認ください` |
| 考试高频点 | 尊敬形变换、`お` / `ご` 的前缀区别、特殊尊敬动词对应、`られる` 和 `になる` 的替换 |

---

## 2. 文法总览

| 句型 | 基本意思 | 核心感觉 | 例句 |
|---|---|---|---|
| `V（ら）れます` | 把对方动作说得更郑重 | 表尊敬时和被动同形 | `先生はどちらで電車を降りられますか。` |
| `お/ご〜になる` | 更直接的尊敬表达 | 说话人抬高对方动作 | `部長は今お戻りになります。` |
| `Vていらっしゃいます` | 正在……；处于…… | 尊敬的进行态 | `課長は今席を外していらっしゃいます。` |
| `特殊尊敬动词` | 固定尊敬说法 | 最常见的高频尊敬 | `ご覧になります。` |
| `お/ご〜ください` | 请您…… | 礼貌请求对方做 | `少々お待ちください。` |

## 3. 文法1：`V（ら）れます`

## 3.1 基本意思

`V（ら）れます` 是第49课里最基础的尊敬形式之一。  
它的表面和被动形很像，但在本课里主要按“尊敬”来理解。

## 3.2 变形规则

| 动词类型 | 尊敬形 | 例子 |
|---|---|---|
| 一类动词 | `〜(あ段)れる` | `行く → 行かれる`、`読む → 読まれる`、`出かける → 出かけられる` |
| 二类动词 | `〜られる` | `食べる → 食べられる`、`見る → 見られる` |
| 三类动词 | 特殊变化 | `する → される`、`来る → 来られる` |

## 3.3 高频例句

| 例句 | 中文 |
|---|---|
| `先生はどちらで電車を降りられますか。` | 老师您从哪里下车？ |
| `いつかぎをなくされたんですか。` | 您是什么时候把钥匙弄丢的？ |
| `部長はたった今出かけられたところです。` | 部长刚刚出去了。 |
| `シュミットさんは10時に来られるそうです。` | 听说施密特先生 10 点会来。 |

## 3.4 易错点

| 易错点 | 正确理解 |
|---|---|
| `〜られる` 一定是被动 | 不对。这里也可能是尊敬。 |
| 看到 `先生は〜られます` 就先按被动翻 | 先看主语和语境，通常是尊敬。 |
| 觉得尊敬语可以随便和 `お/ご〜になる` 叠加 | 不要叠太多层，句子会别扭。 |
| 把 `説明されたとおりに` 看成普通被动 | 这里也可能是在转述“老师说明过的内容”。 |

## 4. 文法2：`お/ご〜になる` 和 `〜ていらっしゃる`

## 4.1 `お/ご〜になる`

这个句型是第49课最常用、也最稳妥的尊敬表达之一。  
一般是把动词的 `ます` 形去掉，再加 `お/ご〜になる`。

| 例子 | 中文 |
|---|---|
| `お疲れになる` | 您辛苦了；累了 |
| `お勤めになる` | 任职；工作 |
| `お戻りになる` | 回来 |
| `お買いになる` | 您买 |
| `お待ちになる` | 您等 |
| `お泊まりになる` | 您住宿 |
| `お吸いになる` | 您吸烟 |
| `おかけになる` | 您打电话 |
| `お話しになる` | 您说话 |

### `お` 和 `ご` 的前缀感觉

| 前缀 | 常见对象 | 例子 |
|---|---|---|
| `お` | 和语词、日常词 | `お名前`、`お電話`、`お待ちください` |
| `ご` | 汉语词、书面词 | `ご住所`、`ご利用`、`ご確認` |

## 4.2 `〜ていらっしゃる`

`〜ていらっしゃる` 是 `〜ている` 的尊敬说法。  
它常用来表示“正在做”或“处于某种状态”。

| 例句 | 中文 |
|---|---|
| `課長は今席を外していらっしゃいます。` | 课长现在不在座位上。 |
| `社長はただ今、会議室にいらっしゃいます。` | 社长现在在会议室。 |
| `部長は今、電話で話していらっしゃいます。` | 部长现在正在打电话。 |

## 4.3 这一组最常见的替换关系

| 普通说法 | 尊敬说法 |
|---|---|
| `います` | `いらっしゃいます` |
| `行きます` | `いらっしゃいます` |
| `来ます` | `いらっしゃいます` |
| `〜ています` | `〜ていらっしゃいます` |

## 5. 文法3：特殊尊敬动词

这组动词一定要单独背。  
它们比 `V（ら）れます` 更自然，也更常出现在考试里。

| 普通形 | 尊敬形 | 例句 | 中文 |
|---|---|---|---|
| `来る / 行く / いる` | `いらっしゃる` | `先生はどちらにいらっしゃいますか。` | 老师您在哪里？ |
| `言う` | `おっしゃる` | `何とおっしゃいましたか。` | 您说什么了？ |
| `食べる / 飲む` | `召し上がる` | `社長は何を召し上がりますか。` | 社长您吃什么？ |
| `見る` | `ご覧になる` | `その映画をご覧になりましたか。` | 您看那部电影了吗？ |
| `する` | `なさる` | `何をなさいますか。` | 您要做什么？ |
| `知っている` | `ご存じです` | `あの方をご存じですか。` | 您认识那位吗？ |

## 5.1 记忆顺序建议

先背这 6 个最常见的对照：

1. `いらっしゃる`
2. `おっしゃる`
3. `召し上がる`
4. `ご覧になる`
5. `なさる`
6. `ご存じです`

## 6. 文法4：`お/ご〜ください`

这个句型是礼貌请求，不是命令。  
常见于 안내牌、服务场景、表格填写和接待用语。

| 例句 | 中文 |
|---|---|
| `少々お待ちください。` | 请稍等。 |
| `こちらにお名前とご住所をお書きください。` | 请在这里填写姓名和住址。 |
| `あちらのエレベーターをご利用ください。` | 请使用那边的电梯。 |
| `忘れ物にご注意ください。` | 请注意随身物品。 |
| `あしたは8時までにロビーにお集まりください。` | 请明天 8 点前在大厅集合。 |
| `出口からお入りください。` | 请从出口进入。 |
| `コンサートをどうぞお楽しみください。` | 请尽情欣赏音乐会。 |
| `あと10分ほどお待ちください。` | 请再等 10 分钟左右。 |

## 6.1 这个句型里最常用的搭配

| 搭配 | 中文 |
|---|---|
| `お待ちください` | 请稍等 |
| `ご利用ください` | 请使用 |
| `ご確認ください` | 请确认 |
| `お書きください` | 请填写 |
| `ご注意ください` | 请注意 |
| `お楽しみください` | 请尽情享受 |

## 9. 会话预习：你大概率会碰到的表达

## 9.1 会话主题

第49课最常见的场景是：

- 在公司或学校里询问上位者是否在
- 电话转接时说“请稍等”
- 让对方填写姓名、住址或确认信息
- 送别时请对方向别人转达问候

## 9.2 预习时先记住这几句

| 日语 | 中文 |
|---|---|
| `先生はいらっしゃいますか。` | 老师在吗？ |
| `ただ今、会議室にいらっしゃいます。` | 现在在会议室。 |
| `少々お待ちください。` | 请稍等。 |
| `こちらにお名前とご住所をご記入ください。` | 请在这里填写姓名和住址。 |
| `部長はもうお戻りになりましたか。` | 部长已经回来了吗？ |
| `では、よろしくお伝えください。` | 那么，请代我问好。 |

## 10. 这课最容易混的点

## 10.1 `〜られる` 不只是被动

| 误区 | 正确理解 |
|---|---|
| `先生が帰られました` 一定是被动 | 不一定。语境里也可能是尊敬。 |
| `降りられる` 只能表示“能下车” | 这里也常用来表示“您下车”。 |

## 10.2 不要把敬语叠太多层

| 误区 | 更自然的说法 |
|---|---|
| `お帰りになられました` | `お帰りになりました` / `帰られました` |
| `ご覧になられました` | `ご覧になりました` |
| `お召し上がりになられました` | `召し上がりました` |

## 10.3 `お/ご` 前缀不是随便加

| 误区 | 正确理解 |
|---|---|
| 什么词前面都加 `お` | 不对。`お` 和 `ご` 要按常见搭配记。 |
| `お住所` | 更常见的是 `ご住所`。 |
| `ご名前` | 更常见的是 `お名前`。 |

## 10.4 `お/ご〜になる` 和 `お/ご〜ください` 方向相反

| 句型 | 谁做动作 | 作用 |
|---|---|---|
| `お/ご〜になる` | 对方做动作 | 抬高对方 |
| `お/ご〜ください` | 请对方做动作 | 礼貌请求 |

## 10.5 `〜ていらっしゃる` 是 `〜ている` 的尊敬版

| 误区 | 正确理解 |
|---|---|
| `いらっしゃる` 和 `〜ていらっしゃる` 没区别 | 不对。前者多对应 `いる/来る/行く`，后者对应 `〜ている`。 |

## 11. 课前自测

## 11.1 先自己做

1. `先生はどちらで電車を（　　　　　　）か。`
2. `その映画を（　　　　　　）か。`
3. `毎晩、何時ごろ（　　　　　　）か。`
4. `部長は今、会議室に（　　　　　　）。`
5. `部長は今席を外して（　　　　　　）。`
6. `社長、昼ごはんは（　　　　　　）か。`

8. `こちらにお名前とご住所を（　　　　　　）ください。`
9. `あちらのエレベーターを（　　　　　　）ください。`

## 11.2 答案

| 题号 | 答案 | 解析 |
|---|---|---|
| 1 | `降りられます` | `降りる` 的尊敬形。 |
| 2 | `ご覧になりました` | `見る` 的尊敬形。 |
| 3 | `お休みになります` | `休む` 的尊敬表达。 |
| 4 | `いらっしゃいます` | `いる` 的尊敬形。 |
| 5 | `いらっしゃいます` | `外していらっしゃいます` 的完成句。 |
| 6 | `召し上がりました` | `食べる / 飲む` 的尊敬特例。 |
| 7 | `なさいます` | `する` 的尊敬特例。 |
| 8 | `お書き` | `書く` 的礼貌请求形式。 |
| 9 | `ご利用` | `利用する` 的礼貌请求形式。 |
| 10 | `お待ち` | `待つ` 的礼貌请求形式。 |

## 12. 预习建议

| 先做什么 | 怎么做 |
|---|---|
| 第一步 | 先把 6 个特殊尊敬动词背熟。 |
| 第二步 | 再区分 `V（ら）れます` 和 `お/ご〜になる`。 |
| 第三步 | 把 `お` / `ご` 前缀的常见搭配记成块。 |
| 第四步 | 把 `お/ご〜ください` 当成服务场景的固定句型来背。 |
| 第五步 | 第50课再系统整理自谦语，不要在第49课里把两课混在一起。 |
