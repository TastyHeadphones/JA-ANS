# 大家的日语 初级2 第50课预习总结

> 用途：课前预习、考试前速查。  
> 整理方式：以本仓库现有第50课习题、总复习包语法总结和词表做交叉整理。  
> 本课核心：`自谦语特殊动词`、`お/ご〜する`、`Vていただく`、`〜ております`、`でございます`、`书信/商务固定表达`。  
> 这课最容易出错的地方：尊敬语和自谦语方向相反；`お` / `ご` 前缀要按词固定记；`〜ている` 和 `〜ております` 不是随便互换；`いただく` 是“承蒙别人”，不是“自己去做”。

---

## 1. 本课你要先抓住什么

| 项目 | 内容 |
|---|---|
| 主题重点 | 对外打交道时，把自己一方的动作说得更谦逊、更正式 |
| 核心文法 | `まいります`、`おる / おります`、`申します`、`伺います`、`拝見します`、`存じます`、`お目にかかります`、`お/ご〜する`、`Vていただく`、`〜ております`、`でございます` |
| 会话场景 | 自我介绍、电话接待、拜访、转接、说明情况、请对方稍等 |
| 词汇重点 | `ただ今`、`わたくし`、`お宅`、`再来週`、`半年`、`緊張します`、`協力します`、`感謝します`、`拝啓`、`敬具` |
| 考试高频点 | 自谦特殊动词对应、`お/ご` 的前缀搭配、`〜ていただく` 的方向、`〜ております` 和 `でございます` 的商务用法 |

## 2. 第49课和第50课的区别

第49课学的是尊敬语，第50课学的是自谦语。  
判断原则很简单：

- 说的是“对方 / 第三方”的动作，就抬高对方，用尊敬语
- 说的是“自己 / 自己一方”的动作，就降低自己，用自谦语

| 普通动词 | 第49课：尊敬语 | 第50课：自谦语 |
|---|---|---|
| 来る / 行く / いる | `いらっしゃる` | `まいる` / `伺う` / `おる` |
| 言う | `おっしゃる` | `申す` |
| 見る | `ご覧になる` | `拝見する` |
| 会う | `お会いになる` | `お目にかかる` |
| する | `なさる` | `いたす` |
| 知る | `ご存じです` | `存じる` |
| 食べる / 飲む | `召し上がる` | `いただく` |

## 3. 文法总览

| 句型 | 基本意思 | 核心感觉 | 例句 |
|---|---|---|---|
| `まいります` / `伺います` | 我去 / 我来 / 我拜访 | 对外说自己动作 | `来週、先生のお宅へ伺います。` |
| `申します` | 我叫；我说 | 自我介绍、说明自己 | `山田と申します。` |
| `拝見します` | 我看 / 我拜读 | 谦逊地说“看” | `資料を拝見しました。` |
| `Vていただく` | 承蒙别人替我做…… | “我得到帮助” | `教えていただきました。` |
| `〜ております` | 正在……；一直…… | `〜ています` 的更郑重说法 | `ただ今、外出しております。` |
| `でございます` | 是…… | 比 `です` 更正式 | `こちらはパワー電気でございます。` |

## 4. 文法1：自谦语特殊动词

这组词一定要单独背。  
它们是第50课最核心的“对外商务日语”。

| 普通形 | 自谦形 | 例句 | 中文 |
|---|---|---|---|
| `行く / 来る` | `まいります` | `それでは、まいります。` | 我这就去 / 来 |
| `行く / 来る / 訪ねる` | `伺います` | `来週の火曜日にご自宅へ伺います。` | 我去拜访 / 前往 |
| `いる` | `おる / おります` | `係の者はただ今席を外しております。` | 我在 / 我方在 |
| `言う` | `申します` | `林と申します。` | 我叫 / 我说 |
| `する` | `いたす` | `よろしくお願いいたします。` | 我做 / 我来做 |
| `見る` | `拝見する` | `先ほど資料を拝見しました。` | 我看 / 我拜读 |
| `知る` | `存じる` | `その件は存じません。` | 我知道 / 我不清楚 |
| `会う` | `お目にかかる` | `来週、直接お目にかかれるのを楽しみにしております。` | 我拜见 / 见面 |
| `食べる / 飲む` | `いただく` | `お茶をいただきました。` | 我吃 / 我喝 / 我领受 |

## 4.1 记忆顺序建议

先背这 8 个最常见的对照：

1. `まいります`
2. `伺います`
3. `申します`
4. `いたします`
5. `拝見します`
6. `存じます`
7. `お目にかかります`
8. `いただきます`

## 4.2 `まいります` 和 `伺います` 的差别

| 词语 | 常见用法 | 感觉 |
|---|---|---|
| `まいります` | 去、来、前来、前往 | 比较通用，商务场景常用 |
| `伺います` | 拜访、前往、询问 | 更正式，常见于“去您那边”或“请教” |

## 4.3 `おる / おります`

`おります` 是 `いる` 的自谦/礼貌说法。  
电话、邮件、接待里特别常见。

| 例句 | 中文 |
|---|---|
| `係の者はただ今席を外しております。` | 负责的人现在不在座位上。 |
| `私は東京におります。` | 我在东京。 |

---

## 5. 文法2：`お/ご〜する` 的一般自谦表达

这个句型是把自己的动作说得更谦逊。  
常见规则是：

- 和语词多用 `お`
- 汉语词、サ变词多用 `ご`
- 更正式时常换成 `〜いたします`

| 表达 | 例句 | 中文 |
|---|---|---|
| `ご連絡します` / `ご連絡いたします` | `あとでご連絡いたします。` | 我稍后联系您。 |
| `ご説明します` / `ご説明いたします` | `私がご説明いたします。` | 我来说明。 |
| `お伝えします` / `お伝えいたします` | `部長にお伝えしておきます。` | 我会转达。 |
| `お電話します` / `お電話いたします` | `後ほどお電話いたします。` | 我稍后打电话。 |
| `お待ちします` / `お待ちいたします` | `ここでお待ちします。` | 我在这里等。 |
| `お入れします` / `お入れいたします` | `お茶をお入れします。` | 我来泡茶。 |
| `お取り替えします` / `お取り替えいたします` | `サイズが合わなければお取り替えします。` | 我来更换。 |
| `お貸しします` / `お貸しいたします` | `傘をお貸ししましょうか。` | 我借给您。 |

## 5.1 常见搭配块

| 搭配 | 中文 |
|---|---|
| `ご案内します` | 我来带路 / 我来说明 |
| `ご連絡します` | 我来联系 |
| `ご説明します` | 我来说明 |
| `お手伝いします` | 我来帮忙 |
| `お知らせします` | 我来通知 |

## 5.2 这组的重点

`お/ご〜する` 不是随便给动词前面加个前缀。  
要么是固定搭配，要么是本来就能变成自谦表达的动作。

## 6. 文法3：`Vていただく`

这个句型表示“承蒙对方替我做某事”。  
说话人是受惠者，动作是别人帮我完成的。

| 例句 | 中文 |
|---|---|
| `先生に作文を見ていただきました。` | 承蒙老师帮我看了作文。 |
| `部長の奥様にすき焼きの作り方を教えていただきました。` | 承蒙部长夫人教我寿喜烧的做法。 |
| `先ほどお送りいただいた資料を拝見しました。` | 我看了刚才您发来的资料。 |
| `少しお待ちいただけませんか。` | 能请您稍等一下吗？ |

## 6.1 和 `Vてください` 的区别

| 句型 | 谁做动作 | 意思 |
|---|---|---|
| `Vてください` | 请对方做 | 请您…… |
| `Vていただく` | 对方替我做 | 承蒙您……，我得以…… |

## 6.2 易混点

| 易错点 | 正确理解 |
|---|---|
| 把 `いただく` 当成自己主动做 | 不对。它是“从别人那里得到好处”。 |
| 把 `いただく` 和 `くださる` 混掉 | 方向相反：`いただく` 是我方受惠，`くださる` 是对方给我方。 |

---

## 7. 文法4：`〜ております`、`でございます`

## 7.1 `〜ております`

`〜ております` 是 `〜ています` 的更郑重、更商务的说法。  
常用来表示自己一方的动作、状态或持续情况。

| 例句 | 中文 |
|---|---|
| `ただ今、外出しております。` | 现在外出了。 |
| `担当の者がまいりますので、少々お待ちください。` | 负责人员马上过来，请稍等。 |
| `この件は確認しております。` | 这件事我已经确认了。 |

## 7.2 `でございます`

`でございます` 是比 `です` 更正式的判断表达。  
电话、接待、公告里很常见。

| 例句 | 中文 |
|---|---|
| `こちらはパワー電気でございます。` | 这里是 Power 电气。 |
| `私どもは○○商事でございます。` | 我们是 XX 商事。 |

## 7.3 书信/致辞里的固定表达

| 表达 | 用法 |
|---|---|
| `拝啓` | 书信开头 |
| `敬具` | 书信结尾 |
| `お元気でいらっしゃいますか` | 很郑重的问候 |
| `ご多忙のところ恐れ入りますが` | 打扰您繁忙之中，敬请…… |
| `よろしくお願いいたします` | 请多关照 / 拜托了 |
| `心から感謝します` | 衷心感谢 |

## 10. 会话预习：你大概率会碰到的表达

## 10.1 会话主题

第50课最常见的场景是：

- 电话中自我介绍
- 说明自己一方的情况
- 说“稍等，我马上去”
- 拜访客户、老师、对方住处
- 邮件、信件、致辞里的正式表达

## 10.2 预习时先记住这几句

| 日语 | 中文 |
|---|---|
| `初めまして。林と申します。` | 初次见面，我叫林。 |
| `ただ今、担当の者がまいります。` | 现在负责的人马上过去。 |
| `少々お待ちください。` | 请稍等。 |
| `係の者はただ今席を外しております。` | 负责的人现在不在座位上。 |
| `先ほどお送りいただいた資料を拝見しました。` | 我已经看了您刚发来的资料。 |
| `では、失礼いたします。` | 那么，告辞了。 |

## 11. 这课最容易混的点

## 11.1 `お/ご〜する` 和 `〜いたす`

| 误区 | 正确理解 |
|---|---|
| 看到动词都能随便加 `お` / `ご` | 不对，要看固定搭配。 |
| `ご説明します` 和 `ご説明いたします` 差不多 | 都是自谦，后者更正式。 |

## 11.2 `いただく` 和 `くださる`

| 误区 | 正确理解 |
|---|---|
| 都是“给” | 不对。方向相反。 |
| `先生に教えていただきました` | 老师帮我教了，我受惠。 |
| `先生が教えてくださいました` | 老师给我教了，强调对方给予。 |

## 11.3 `〜ております` 和 `〜ています`

| 误区 | 正确理解 |
|---|---|
| 完全一样 | 含义接近，但 `〜ております` 更正式，更适合商务场景。 |
| 任何人都能随便混用 | 不是。通常用于自己一方或己方状态。 |

## 11.4 `です` 和 `でございます`

| 误区 | 正确理解 |
|---|---|
| 只是更长一点 | 不只如此。它更正式，常见于接待/电话。 |
| 所有句子都能换成 `でございます` | 不自然。主要用于名词判断和正式场景。 |

## 11.5 `まいります` 和 `伺います`

| 误区 | 正确理解 |
|---|---|
| 两个词完全一样 | 都是自谦，但使用场景不同。 |
| 见面、拜访场景常优先用 `伺います` | 对，尤其是“去您那边”更常见。 |

## 12.1 先自己做

1. `初めまして。山田（　　　　　）申します。`
2. `先ほどお送りいただいた資料を（　　　　　）しました。`
3. `先生のお宅へ（　　　　　）つもりです。`
4. `係の者はただ今席を外して（　　　　　）。`
5. `こちらはパワー電気で（　　　　　）。`
6. `先生に作文を見て（　　　　　）ました。`
7. `あの方を（　　　　　）か。`

## 12.2 答案

| 题号 | 答案 | 解析 |

| 1 | `と` | `〜と申します`。 |
| 2 | `拝見` | `拝見します`。 |

| 5 | `ございます` | `でございます`。 |
| 6 | `いただき` | `見ていただきました`。 |
| 7 | `ご存じです` | `ご存じですか`。 |
| 8 | `お待ち` | `お待ちください`。 |

## 13. 预习建议

| 先做什么 | 怎么做 |
|---|---|
| 第一步 | 先把 `まいります / 伺います / 申します / 拝見します / 存じます / お目にかかります` 背熟。 |
| 第二步 | 再把尊敬语和自谦语做成对照表，看到主语先判断方向。 |
| 第三步 | 把 `ご連絡します / ご説明します / お伝えします / お電話します` 这种商务句块整块记住。 |
| 第四步 | 把 `〜ております`、`でございます`、`拝啓 / 敬具` 当成固定表达背。 |
| 第五步 | 练习电话自我介绍和“请稍等”的完整流程。 |
