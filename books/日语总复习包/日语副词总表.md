# 日语副词总表

> 用途：系统整理日语副词。  
> 说明：日语副词数量非常大，不可能靠一页穷尽所有冷门词。本文按现代日语和初中级到中高级考试中最常见、最有系统的一批副词来整理，重点放在“词义、常见搭配、易错限制”。

---

## 1. 什么是副词

副词一般用来修饰：

- 动词：`ゆっくり歩きます。`
- 形容词：`とても暑いです。`
- 其他副词：`ほんとうにとても便利です。`
- 全句：`たぶん来ないと思います。`

---

## 2. 按功能总览

| 类别 | 作用 | 代表词 |
|---|---|---|
| 时间顺序 | 表示时间先后、刚才、马上 | `もう` `まだ` `すぐ` `さっき` `たった今` |
| 频率 | 表示经常、偶尔、很少 | `いつも` `よく` `ときどき` `たまに` `めったに` |
| 程度 | 表示非常、稍微、更加 | `とても` `かなり` `ずいぶん` `ちょっと` `もっと` |
| 数量 | 表示很多、几乎、完全 | `たくさん` `いっぱい` `ほとんど` `ぜんぜん` |
| 状态方式 | 表示怎么做、做成什么样 | `ゆっくり` `はっきり` `しっかり` `ちゃんと` |
| 心情态度 | 表示一定、大概、也许、难得 | `きっと` `たぶん` `もしかすると` `せっかく` |
| 连接展开 | 表示顺序、补充、转折 | `まず` `そして` `それから` `それに` `ところで` |

## 3. 时间副词

### 3.1 现在、过去、未来相关

| 副词 | 读音 | 中文 | 例句 |
|---|---|---|---|
| `今` | いま | 现在 | `今勉強しています。` |
| `さっき` | さっき | 刚才 | `さっき帰りました。` |
| `たった今` | たったいま | 刚刚 | `たった今来たところです。` |
| `もう` | もう | 已经；马上就 | `もう食べました。` |
| `まだ` | まだ | 还，尚未 | `まだ終わっていません。` |
| `すぐ` | すぐ | 马上；立刻 | `すぐ行きます。` |
| `まもなく` | まもなく | 不久；马上 | `まもなく電車が来ます。` |
| `これから` | これから | 从现在开始 | `これから出かけます。` |
| `そのうち` | そのうち | 过些时候 | `そのうち慣れるでしょう。` |
| `やがて` | やがて | 不久后，最终 | `やがて春になります。` |

### 3.2 时间顺序副词

| 副词 | 中文 | 例句 |
|---|---|---|
| `まず` | 首先 | `まず名前を書いてください。` |
| `次(つぎ)に` | 接着 | `次に番号を入れます。` |
| `それから` | 然后 | `朝ごはんを食べて、それから会社へ行きます。` |
| `最後(さいご)に` | 最后 | `最後にもう一度確認します。` |

## 4. 频率副词

| 副词 | 读音 | 中文 | 常见搭配 |
|---|---|---|---|
| `いつも` | いつも | 总是 | `いつも早く起きます。` |
| `たいてい` | たいてい | 大多；通常 | `たいてい7時に帰ります。` |
| `よく` | よく | 经常；好好地；很 | `よく行きます。` |
| `しょっちゅう` | しょっちゅう | 老是，总是 | 口语较多 |
| `ときどき` | ときどき | 有时 | `ときどき映画を見ます。` |
| `たまに` | たまに | 偶尔 | 口语常见 |
| `あまり` | あまり | 不太…… | 常接否定 |
| `めったに` | めったに | 很少…… | 常接否定 |
| `ぜんぜん` | ぜんぜん | 完全不…… | 传统上接否定 |

### 4.1 这一组要特别记住的限制

| 副词 | 典型搭配 | 例句 |
|---|---|---|
| `あまり` | `あまり + 否定` | `あまり食べません。` |
| `めったに` | `めったに + 否定` | `めったに行きません。` |
| `ぜんぜん` | `ぜんぜん + 否定` | `ぜんぜんわかりません。` |

## 5. 程度副词

| 副词 | 读音 | 中文 | 程度感觉 |
|---|---|---|---|
| `とても` | とても | 很，非常 | 标准高程度 |
| `たいへん` | たいへん | 非常 | 稍正式 |
| `すごく` | すごく | 非常，特别 | 口语很常见 |
| `かなり` | かなり | 相当，颇 | 比较高 |
| `ずいぶん` | ずいぶん | 相当，很 | 带说话人感受 |
| `だいぶ` | だいぶ | 很大程度上 | 常表示变化后程度 |
| `もっと` | もっと | 更…… | 比较级常见 |
| `いちばん` | いちばん | 最…… | 最高级常见 |
| `ちょっと` | ちょっと | 稍微；有点 | 低程度 |
| `少(すこ)し` | すこし | 一点儿 | 比较中性 |
| `やや` | やや | 稍微，略微 | 书面感更强 |

### 5.1 高频例句

- `この本はとてもおもしろいです。`
- `今日はかなり寒いです。`
- `日本語がだいぶ上手になりました。`
- `もう少し待ってください。`
- `もっと大きいのはありませんか。`

## 8. 推测、判断、态度副词

| 副词 | 中文 | 常见搭配 | 例句 |
|---|---|---|---|
| `たぶん` | 大概 | `でしょう / と思います` | `たぶん雨が降るでしょう。` |
| `おそらく` | 恐怕，大概 | 书面感更强 | `おそらく間に合わないでしょう。` |
| `もしかすると` | 也许，说不定 | 推测偏不确定 | `もしかすると来ないかもしれません。` |
| `きっと` | 一定 | 说话人确信较强 | `きっと大丈夫です。` |
| `必(かなら)ず` | 一定，务必 | 意志、规则都常见 | `必ず返してください。` |
| `絶対(ぜったい)に` | 绝对 | 常与否定或强意志搭配 | `絶対に忘れません。` |
| `ぜひ` | 务必；一定要 | 请求、邀请常见 | `ぜひ来てください。` |
| `どうか` | 请务必；总之 | 正式请求常见 | `どうかよろしくお願いします。` |
| `せっかく` | 难得；特意 | 常和 `のに`、`から` 连用 | `せっかく来たのに。` |
| `やはり / やっぱり` | 果然；还是 | 表符合预期 | `やっぱり日本語は難しいです。` |

## 9. 否定里特别高频的副词

| 副词 | 中文 | 典型结构 | 例句 |
|---|---|---|---|
| `あまり` | 不太 | `あまり + 否定` | `あまり元気ではありません。` |
| `ぜんぜん` | 完全不 | `ぜんぜん + 否定` | `ぜんぜん知りません。` |
| `ほとんど` | 几乎不；几乎都 | 视句子而定 | `ほとんど食べません。` |
| `なかなか` | 轻易不；相当 | `なかなか + 否定` / `なかなか + 积极评价` | `なかなか覚えられません。` |
| `まったく` | 完全，真是 | 常带批评或强调 | `まったくわかりません。` |
| `別(べつ)に` | 也不……；并不…… | 常接否定 | `別に困っていません。` |

### 9.1 `なかなか` 很重要

| 用法 | 意思 | 例句 |
|---|---|---|
| `なかなか + 否定` | 怎么也不……；不容易…… | `なかなか寝られません。` |
| `なかなか + 肯定` | 相当，很不错 | `この料理はなかなかおいしいです。` |

## 14. 最容易混的副词对比

### 14.1 `もう` vs `まだ`

| 副词 | 中文 | 例句 |
|---|---|---|
| `もう` | 已经 | `もう食べました。` |
| `まだ` | 还没；还 | `まだ食べていません。` |

### 14.2 `よく` vs `たいてい` vs `いつも`

| 副词 | 大致频率 | 说明 |
|---|---|---|
| `いつも` | 最高 | 几乎每次都如此 |
| `たいてい` | 较高 | 大多数时候 |
| `よく` | 经常 | 比 `たいてい` 更灵活 |

### 14.3 `ちょっと` vs `少し`

| 副词 | 语感 | 例句 |
|---|---|---|
| `ちょっと` | 更口语，也能委婉 | `ちょっと難しいです。` |
| `少し` | 更中性 | `少し時間があります。` |

### 14.4 `たぶん` vs `きっと`

| 副词 | 确信度 |
|---|---|
| `たぶん` | 大概，比较保留 |
| `きっと` | 一定，确信更强 |

## 16. 记忆建议

| 先背什么 | 原因 |
|---|---|
| 时间副词：`もう` `まだ` `すぐ` `さっき` | 使用频率最高 |
| 频率副词：`いつも` `よく` `ときどき` `たまに` | 日常表达极常见 |
| 程度副词：`とても` `かなり` `ちょっと` `もっと` | 会话和作文都高频 |
| 限制型副词：`あまり` `ぜんぜん` `めったに` | 经常和否定一起考 |
| 推测副词：`たぶん` `きっと` `もしかすると` | 和句末形式关系很强 |
| 状态副词：`ゆっくり` `はっきり` `しっかり` `ちゃんと` | 听力和口语都常见 |
