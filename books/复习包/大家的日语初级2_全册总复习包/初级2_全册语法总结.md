## 2. 重点语法分组复习

### 2.1 说明、原因、请求（26-33）

1. `普通形 + んです`
- 用法：解释原因、补充背景。
- 例句：どうしたんですか。頭が痛いんです。
- 易错：`な形容词/名词`前要接`な`（静かなんです、学生なんです）。

2. `〜ていただけませんか`
- 用法：高礼貌请求。
- 例句：この文を読んでいただけませんか。
- 易错：不要写成 `いただきませんか`（语气不同，通常不如固定句自然）。

3. 可能形
- I组：書く→書ける、読む→読める
- II组：食べる→食べられる
- III组：する→できる、来る→来られる
- 例句：車が運転できます。

4. `〜し、〜し`
- 用法：并列理由。
- 例句：この町は静かだし、便利だし、住みやすいです。

5. `Vてしまう`
- 用法：完成/遗憾。
- 例句：財布をなくしてしまいました。

### 2.2 条件、意向、习惯控制（34-39）

1. `〜とおりに`
- 用法：按…那样做。
- 例句：先生が言ったとおりに書きました。

2. 条件四套：`と / たら / ば / なら`
- `と`：恒常结果（春になると暖かくなる）
- `たら`：一次性条件（雨が降ったら、行きません）
- `ば`：书面、一般条件（時間があれば）
- `なら`：承接话题条件（京都へ行くなら）

3. `〜ようにする / 〜ないようにする`
- 用法：表示努力形成习惯。
- 例句：毎日日本語を読むようにしています。

4. 被动（受身）
- 基本：`AはBにVられます`
- 被害：`私は弟にケーキを食べられました。`
- 易错：施事者多用`に`，材料/手段注意`で/から`。

### 2.3 名词化、间接疑问、授受、目的（38-43）

1. 名词化 `〜の`
- `Vるのは`（作为主题）
- `Vるのが`（作为主语/好恶对象）
- `Vるのを`（作为宾语）
- 例句：漢字を覚えるのは難しいです。

2. 间接疑问
- 有疑问词：`どこで買ったか`
- 是否判断：`行くかどうか`
- 例句：会議が何時に終わるか知っていますか。

3. 授受
- `あげる`（我方给他人）
- `くれる`（他人给我方）
- `もらう`（我方得到）
- 例句：友達が辞書をくれました。

4. 目的表达
- `V辞書形 + ために`
- `Nの + ために`
- `N + に使う`
- 例句：家を買うために貯金しています。

### 2.4 程度、推测、状态阶段（43-47）

1. `〜そうです`
- 样态：おいしそうです（看起来）
- 传闻：天気予報によると、雨だそうです（听说）

2. `〜すぎる`
- 例句：飲みすぎました。

3. `〜やすい / 〜にくい`
- 例句：このペンは書きやすいです。

4. `〜ところです`
- `Vるところ`（正要）
- `Vているところ`（正在）
- `Vたところ`（刚刚）

5. `〜ばかり`
- 刚刚做完：帰ってきたばかりです。

6. `〜はずです`
- 有依据推测：田中さんはもう着いたはずです。

7. 使役
- `AはBにVさせる`
- 例句：先生は学生に本を読ませます。

### 2.5 敬语系统（48-50）

1. 尊敬语（抬高对方）
- 行く/来る/いる → いらっしゃる
- 言う → おっしゃる
- 食べる/飲む → 召し上がる
- する → なさる
- 看 → ご覧になる

2. 自谦语（降低自己）
- 行く/来る → 参る
- いる → おる
- 言う → 申す
- する → いたす
- 见面 → お目にかかる
- 看/读 → 拝見する

3. 郑重语（です・ます体系）
- 〜でございます
- 〜いたします

## 3. 高频易错点清单

1. `かどうか` 只用于“是否”判断；有疑问词时用 `〜か`。
2. `そうです` 有两种：样态（看起来）和传闻（听说）。
3. `に` / `で`：存在地点用`に`，动作场所用`で`。
4. `受身` 与 `使役受身` 不要混：
- 受身：先生にほめられました。
- 使役受身：先生に立たされました。
5. `ために` 与 `ように`：
- 明确意志目的常用 `ために`。
- 能力变化/习惯控制常用 `ように`。
6. 授受方向必须先定“谁是我方”：
- 他人给我方：`くれる`。
- 我方给他人：`あげる/やる`。

## 4. 考前速记（建议）

1. 先背：`受身/使役/敬语`三大变形。  
2. 再背：`かどうか、ために、ように、そう、はず、ところ、ばかり`的区别。  
3. 最后做题：优先做复习页与文法チェックシート。
