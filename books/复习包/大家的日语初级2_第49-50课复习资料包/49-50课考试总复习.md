# 大家的日语 初级2 第49-50课考试总复习

> 用途：考前速查 + 例题训练。  
> 说明：第49课是尊敬语，第50课是自谦语。做题时先看“动作是谁做的”，再决定用抬高对方还是降低自己。  
> 这两课最容易考错的地方：`〜られる` 既可能是被动，也可能是尊敬；`お/ご` 前缀要按词固定记；`いただく` 和 `くださる` 方向相反。

---

## 1. 这两课最容易考什么

| 课次 | 高频考点 | 典型题型 |
|---|---|---|
| 第49课 | `V（ら）れます`、`お/ご〜になる`、`〜ていらっしゃる`、特殊尊敬动词、`お/ご〜ください` | 改写尊敬句、填空、礼貌请求 |
| 第50课 | 自谦特殊动词、`お/ご〜する`、`Vていただく`、`〜ております`、`でございます` | 自我介绍、电话接待、拜访、商务会话 |
| 综合 | 尊敬 vs 自谦、`お` vs `ご`、`いただく` vs `くださる` | 对照题、会话补全、改写题 |

---

## 2. 一眼分清尊敬和自谦

| 先看什么 | 你该怎么想 |
|---|---|
| 动作是对方/第三方做 | 用尊敬语，把对方抬高 |
| 动作是自己/自己一方做 | 用自谦语，把自己放低 |
| 句子里有“请您……” | 多半是 `お/ご〜ください` |
| 句子里有“承蒙您……” | 多半是 `Vていただく` |

---

## 3. 第49课：尊敬语速记

## 3.1 核心句型

| 句型 | 基本意思 | 例句 |
|---|---|---|
| `V（ら）れます` | 把对方动作说得更郑重 | `先生はどちらで電車を降りられますか。` |
| `お/ご〜になる` | 更直接的尊敬表达 | `部長は今お戻りになります。` |
| `〜ていらっしゃる` | `〜ている` 的尊敬说法 | `課長は今席を外していらっしゃいます。` |
| `特殊尊敬动词` | 固定尊敬说法 | `ご覧になります。` |
| `お/ご〜ください` | 请对方做某事 | `少々お待ちください。` |

## 3.2 高频尊敬动词

| 普通形 | 尊敬形 | 例句 |
|---|---|---|
| `来る / 行く / いる` | `いらっしゃる` | `先生はどちらにいらっしゃいますか。` |
| `言う` | `おっしゃる` | `何とおっしゃいましたか。` |
| `食べる / 飲む` | `召し上がる` | `社長は何を召し上がりますか。` |
| `見る` | `ご覧になる` | `その映画をご覧になりましたか。` |
| `する` | `なさる` | `何をなさいますか。` |
| `知っている` | `ご存じです` | `あの方をご存じですか。` |

## 3.3 `お` / `ご` 前缀怎么记

| 前缀 | 常见对象 | 例子 |
|---|---|---|
| `お` | 和语词、日常词 | `お名前`、`お電話`、`お待ちください` |
| `ご` | 汉语词、书面词 | `ご住所`、`ご利用`、`ご確認` |

## 3.4 第49课易错点

| 易错点 | 正确理解 |
|---|---|
| `〜られる` 一定是被动 | 不对，这里也可能是尊敬。 |
| 尊敬句里把 `お/ご` 叠太多层 | 不自然，尽量用一层最稳妥的形式。 |
| `お名前` / `ご名前` 随便换 | 不对，`お名前` 常见，`ご住所` 常见。 |
| `お/ご〜になる` 和 `お/ご〜ください` 混掉 | 前者抬高对方动作，后者是请对方做动作。 |

## 3.5 第49课考前必背句

- `先生はいらっしゃいますか。`
- `少々お待ちください。`
- `こちらにお名前とご住所をお書きください。`
- `あちらのエレベーターをご利用ください。`
- `忘れ物にご注意ください。`
- `その映画をご覧になりましたか。`

## 4. 第50课：自谦语速记

## 4.1 核心句型

| 句型 | 基本意思 | 例句 |
|---|---|---|
| `自谦特殊动词` | 说自己的动作更谦逊 | `林と申します。` |
| `お/ご〜する` | 把自己的动作说得更谦逊 | `ご説明します。` |
| `Vていただく` | 承蒙对方替我做某事 | `先生に見ていただきました。` |
| `〜ております` | `〜ています` 的商务敬语 | `ただ今、外出しております。` |
| `でございます` | `です` 的郑重表达 | `こちらはパワー電気でございます。` |

## 4.2 高频自谦动词

| 普通形 | 自谦形 | 例句 |
|---|---|---|
| `行く / 来る` | `まいります` | `それでは、まいります。` |
| `行く / 来る / 訪ねる` | `伺います` | `来週、先生のお宅へ伺います。` |
| `いる` | `おる / おります` | `係の者はただ今席を外しております。` |
| `言う` | `申します` | `山田と申します。` |
| `する` | `いたす` | `よろしくお願いいたします。` |
| `見る` | `拝見する` | `資料を拝見しました。` |
| `知る` | `存じる` | `その件は存じません。` |
| `会う` | `お目にかかる` | `また先生にお目にかかりたいと思います。` |
| `食べる / 飲む` | `いただく` | `お茶をいただきました。` |

## 4.3 `お/ご〜する` 的常见搭配

| 搭配 | 中文 |
|---|---|
| `ご連絡します` / `ご連絡いたします` | 我联系您 |
| `ご説明します` / `ご説明いたします` | 我说明 |
| `お伝えします` / `お伝えいたします` | 我转达 |
| `お電話します` / `お電話いたします` | 我打电话 |
| `お待ちします` / `お待ちいたします` | 我等 |
| `お手伝いします` | 我帮忙 |

## 4.4 第50课易错点

| 易错点 | 正确理解 |
|---|---|
| `いただく` 当成“自己主动做” | 不对，它是“承蒙别人替我做”。 |
| `くださる` 和 `いただく` 混掉 | `くださる` 是对方给我方，`いただく` 是我方得到。 |
| `おります` 和 `います` 随便换 | `おります` 更正式，常用于自己一方。 |
| `です` 和 `でございます` 随便换 | `でございます` 更郑重，常用于接待/电话。 |
| `まいります` 和 `伺います` 完全一样 | 都是自谦，但 `伺います` 更常见于拜访、前往。 |

## 4.5 第50课考前必背句

- `初めまして。林と申します。`
- `先生のお宅へ伺います。`
- `先ほどお送りいただいた資料を拝見しました。`
- `係の者はただ今席を外しております。`
- `こちらはパワー電気でございます。`
- `少々お待ちいただけませんか。`
- `よろしくお願いいたします。`

## 6. 例题训练

## 6.1 第49课填空

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 49-1 | 先生はどちらで電車を（降ります…）か。 | 降りられますか | `降りる` 的尊敬形。 |
| 49-2 | その映画を（見ます…）か。 | ご覧になりましたか | `見る` 的尊敬特例。 |
| 49-3 | 毎晩、何時ごろ（休みます…）か。 | お休みになりますか | `休む` 的尊敬表达。 |
| 49-4 | こちらにお名前とご住所を（書きます…）ください。 | お書き | `書く` 的礼貌请求。 |
| 49-5 | あちらのエレベーターを（利用します…）ください。 | ご利用 | `利用する` 的礼貌请求。 |
| 49-6 | 忘れ物に（注意します…）ください。 | ご注意 | `注意する` 的礼貌请求。 |

## 6.2 第50课填空

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 50-1 | 初めまして。林（　）申します。 | と | `〜と申します`。 |
| 50-2 | 先生のお宅へ（　）つもりです。 | 伺う | `行く / 訪ねる` 的自谦。 |
| 50-3 | 先ほどお送りいただいた資料を（　）しました。 | 拝見 | `見る` 的自谦。 |
| 50-4 | 係の者はただ今席を外して（　）。 | おります | `いる` 的自谦。 |
| 50-5 | こちらはパワー電気で（　）。 | ございます | `です` 的郑重表达。 |
| 50-6 | 先生に作文を見て（　）ました。 | いただき | `Vていただく`。 |
| 50-7 | また先生に（会います…）たいと思います。 | お目にかかり | `会う` 的自谦。 |
| 50-8 | この件は私が（説明します…）ます。 | ご説明し | `お/ご〜する`。 |

## 6.3 综合改写

| 题号 | 原句 | 参考答案 | 解析 |
|---|---|---|---|
| 6-1 | わたしは林です。 | わたしは林と申します。 | 自我介绍用自谦。 |
| 6-2 | わたしは明日、先生の家へ行きます。 | わたしは明日、先生のお宅へ伺います。 | 去对方处用 `伺う`。 |
| 6-3 | 先生は今、会議室にいます。 | 先生は今、会議室にいらっしゃいます。 | `いる` 的尊敬。 |
| 6-4 | わたしはその本を見ました。 | その本を拝見しました。 | `見る` 的自谦。 |
| 6-5 | 先生が教えてくれました。 | 先生に教えていただきました。 | `Vていただく`。 |
| 6-6 | どうぞ座ってください。 | どうぞお掛けください。 | 礼貌请求。 |
| 6-7 | こちらは○○商事です。 | こちらは○○商事でございます。 | 郑重表达。 |

## 7. 考前 5 分钟速记

1. 先看动作主体。对方做动作，用尊敬语；自己做动作，用自谦语。  
2. 第49课先背 6 个尊敬动词：`いらっしゃる`、`おっしゃる`、`なさる`、`召し上がる`、`ご覧になる`、`ご存じです`。  
3. 第50课先背 6 个自谦动词：`まいる`、`伺う`、`申す`、`拝見する`、`存じる`、`お目にかかる`。  
4. 看到 `〜ていただく`，就想“承蒙对方替我做”。  
5. 看到 `〜ております`，就想“更正式的 `〜ています`”。  
6. 看到 `でございます`，就想“更郑重的 `です`”。  
7. 看到 `お/ご〜ください`，就想“请您……”。
