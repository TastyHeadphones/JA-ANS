# JLPT N2/N3 文法対策 一页速査表

> **N3 核心**：句间逻辑 + 时态区分 + 授受敬语  
> **N2 核心**：语气强弱 + 书面程度 + 逻辑关系的精准选择

---

## 一、推量・推断（N3→N2 升级对比）

| 文法 | 含义 | 接续 | 书面度 | 易错点 |
|---|---|---|---|---|
| `〜そうだ`（样态） | 看起来要… | Vます形 | ☆ | ≠传闻そうだ |
| `〜そうだ`（传闻） | 听说… | 普通形 | ☆ | ≠样态そうだ |
| `〜ようだ` | 好像…（有依据） | 普通形 | ☆☆ | 比みたい书面 |
| `〜みたいだ` | 好像…（口语） | 普通形 | ☆ | 口语版ようだ |
| `〜らしい` | 听说/具有…特点 | 普通形 | ☆☆ | 兼含传闻+典型性 |
| `〜かもしれない` | 可能… | 普通形 | ☆ | 概率最低 |
| `〜はずだ` | 理应… | 普通形 | ☆☆ | 基于前提逻辑 |
| `〜はずがない` | 不可能… | 普通形 | ☆☆ | 强否定推断 |
| `〜にちがいない` | 一定是… | 普通形 | ☆☆☆ | 主观确信强 |
| `〜に決まっている` | 肯定是… | 普通形 | ☆ | 口语反驳语气 |
| `〜わけだ` | 难怪/原来如此 | 普通形 | ☆☆☆ | ≠单纯"所以" |
| `〜わけではない` | 并非都… | 普通形 | ☆☆ | 部分否定 |
| `〜わけがない` | 不可能… | 普通形 | ☆☆ | 比はずがない更主观 |

## 二、逆接・让步・对比

| 文法 | 含义 | 书面度 | 易错点 |
|---|---|---|---|
| `〜のに` | 明明…却… | ☆ | ≠用途のに |
| `〜ても` | 即使…也… | ☆ | 通用让步 |
| `〜くせに` | 明明…却…（责备） | ☆ | 勿对上级用 |
| `〜ながら`（逆接） | 虽然…却… | ☆☆ | ≠并行ながら |
| `〜ものの` | 虽然…但是… | ☆☆☆ | 书面逆接 |
| `〜つつ（も）` | 虽然…但… | ☆☆☆ | 书面反省 |
| `〜とはいえ` | 虽说… | ☆☆☆ | 后项有保留 |
| `〜にもかかわらず` | 尽管…仍… | ☆☆☆ | 正式逆接 |
| `〜にしては` | 以…来说却… | ☆☆ | "和预期不符" |
| `〜わりに` | 虽…却… | ☆ | 口语化对比 |
| `〜反面` | 另一方面… | ☆☆☆ | 双面评价 |
| `〜一方（で）` | 一方面…另一方面… | ☆☆☆ | 对比分析 |
| `〜どころか` | 别说…连…都… | ☆☆ | 程度更甚 |

## 三、原因・结果

| 文法 | 含义 | 极性 | 书面度 |
|---|---|---|---|
| `〜ので` | 因为…（客观礼貌） | 中 | ☆☆ |
| `〜から` | 因为…（主观） | 中 | ☆ |
| `〜ため（に）` | 因为…/为了… | 中 | ☆☆☆ |
| `〜おかげで` | 多亏… | ✅正面 | ☆☆ |
| `〜せいで` | 都怪… | ❌负面 | ☆☆ |
| `〜せいか` | 也许因为… | ❌偏负 | ☆☆ |
| `〜ばかりに` | 正因为…才…（坏结果） | ❌负面 | ☆☆☆ |
| `〜あげく（に）` | 折腾到最后 | ❌负面 | ☆☆☆ |
| `〜末（に）` | 最终… | 中 | ☆☆☆ |
| `〜結果` | 结果… | 中 | ☆☆☆ |
| `〜ことから` | 由此可知/因为 | 中 | ☆☆☆ |
| `〜かねない` | 恐怕会…（坏事） | ❌负面 | ☆☆☆ |

## 四、条件・假定

| 文法 | 含义 | 用法要点 |
|---|---|---|
| `Vば` | 如果… | 书面逻辑条件 |
| `Vたら` | 如果…就… | 口语最通用 |
| `Vと` | 一…就…（规律） | 不接命令/请求 |
| `Nなら` | 如果是… | 接前文话题 |
| `〜限り` | 只要…就… | 辞书形 |
| `〜ない限り` | 除非…否则不… | 双重否定常考 |
| `たとえ〜ても` | 即使…也… | 强让步 |
| `〜にしろ/にしても` | 不管…都/即使… | 常成对 `AにしろBにしろ` |
| `〜さえ〜ば` | 只要…就… | 最低条件 |
| `〜ならでは` | 只有…才有 | 后接名词高频 |
| `〜としたら` | 如果说…的话 | 假设推理 |
| `〜以上（は）` | 既然…就… | 后项责任义务 |
| `〜からには` | 既然… | 决心表达 |

## 五、时间轴

| 文法 | 含义 | 对比要点 |
|---|---|---|
| `Vるところだ` | 正要… | 三个ところ要区分时态 |
| `Vているところだ` | 正在… | 瞬间截面 |
| `Vたところだ` | 刚刚… | ≠一直持续 |
| `Vたばかりだ` | 刚…（主观） | 时间可不"刚刚" |
| `〜うちに` | 趁着… | 变化前窗口 |
| `〜間に` | 在…期间（某时点） | 一次性动作 |
| `〜間` | 在…整个期间 | 持续背景 |
| `〜たびに` | 每当…就… | 周期性反复 |
| `〜次第` | 一…就…（书面） | 后项多未来动作 |
| `〜にあたって` | 值此…之际 | 庄重书面 |
| `〜に際して` | 临…之际 | ≈にあたって |
| `〜に先立って` | 在…之前（正式） | 仪式/流程 |
| `〜て以来` | 自从…以来 | 后项多持续变化 |

## 六、目的・变化・习惯

| 文法 | 含义 | 对比要点 |
|---|---|---|
| `Vるために` | 为了…（目的强） | 意志动词前项 |
| `Vるように` | 为了能…（状态目标） | 能力/状态类 |
| `Vるようになる` | 变得会… | 结果渐变 |
| `Vるようにする` | 努力做到… | 主观持续努力 |
| `〜ことになる` | 决定为…（外部） | ≠ことにする |
| `〜ことにする` | 决定…（自己） | 自我意志 |
| `〜ことにしている` | 一直坚持… | 长期自律规则 |

## 七、授受・受身・使役（N3核心）

| 文法 | 视角 | 例句 |
|---|---|---|
| `AはBにNをあげる` | 给出方 | `私は妹に本をあげた。` |
| `AはBにNをもらう` | 收到方 | `先生に辞書をもらった。` |
| `Aは私にNをくれる` | 我方受益 | `友達が傘をくれた。` |
| `Vてあげる` | 为别人做 | 对上级慎用 |
| `Vてくれる` | 别人为我做 | 主语是施惠者 |
| `Vてもらう` | 请别人做 | 主语是受益者 |
| `受身形` | 被… | 助词 `に` 常考 |
| `使役形` | 让/使… | 一类变形易错 |
| `使役受身` | 被迫… | 含不情愿语感 |
| `させていただく` | 谨让我… | 过度用显啰嗦 |

## 十、限定・强调

| 文法 | 含义 | 对比要点 |
|---|---|---|
| `〜しか〜ない` | 只有… | 后项必须否定 |
| `〜だけ` | 仅… | 比しか中性 |
| `〜ばかり` | 尽是…/刚… | 含负面语感 |
| `〜さえ` | 连…都 | 最低要素 |
| `〜すら` | 连…都（书面） | 比さえ更书面 |
| `〜こそ` | 正是… | 强调焦点 |
| `〜にすぎない` | 只不过… | 书面降级 |
| `〜にほかならない` | 正是… | 书面断定 |
| `〜ばかりか` | 不但…反而… | 后项程度更高 |
| `〜のみならず` | 不仅…还… | 书面 |
| `〜に限らず` | 不限于… | 范围扩展 |
| `〜を問わず` | 不论… | 常接二元名词 |
| `〜にかかわらず` | 不管…都… | ≠にもかかわらず（逆接） |

## 十一、评价・规范（N2书面）

| 文法 | 含义 | 场景 |
|---|---|---|
| `〜ものだ` | 本来就…/感慨 | 常识/感叹/回忆 |
| `〜ものではない` | 不该… | 道德规劝 |
| `〜べきだ` | 应该… | 论述立场 |

## 十二、N3高频易混 vs N2高频易混

### N3 关键区别
| 易混项 | 区别核心 |
|---|---|
| `ようだ` vs `らしい` | 眼前证据 vs 传闻/典型性 |
| `ことになる` vs `ことにする` | 外部决定 vs 自我决定 |
| `ために` vs `ように` | 目的行为 vs 能力/状态目标 |
| `ば` vs `たら` | 书面逻辑 vs 口语通用 |
| `ので` vs `から` | 客观礼貌 vs 主观直接 |
| `受身` vs `てもらう` | 被动受害 vs 受益 |
| `ばかり` vs `だけ` | 偏多/不满 vs 中性限定 |

### N2 关键区别
| 易混项 | 区别核心 |
|---|---|
| `わけではない` vs `わけがない` | 部分否定 vs "不可能" |
| `にしては` vs `わりに` | 基准不符 vs 对比落差 |
| `ものの` vs `とはいえ` | 叙述逆接 vs 评论逆接 |
| `ばかりに` vs `せいで` | 懊悔感 vs 一般负面原因 |
| `ざるを得ない` vs `ないわけにはいかない` | 书面强硬 vs 书面柔和 |
| `に基づいて` vs `をもとに` | 正式客观（法律） vs 数据分析 |
| `につれて` vs `に従って` | 自然渐变 vs 渐变+规则遵守 |

## 速记口诀

```
推断强弱：かもしれない < でしょう < らしい < はずだ < にちがいない
负面原因：せいで（一般）< ばかりに（懊悔）
逆接程度：が < けど < のに < ものの < とはいえ < にもかかわらず
书面程度：口语 ☆ → 一般 ☆☆ → 正式书面 ☆☆☆
```
