# JLPT N2/N3 文法対策 — 按课别速查表
## Minna no Nihongo Intermediate 1（第1课～第12课）

---

## 第1课｜请求・比喻・命名・让步
**会话主题**：お願いがあるんですが　／　**读写**：畳

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `Vて + もらえませんか` | 能帮我…吗？（礼貌请求） | `駅までの行き方を教えてもらえませんか。` | 不是命令句 |
| `Vて + いただけませんか` | 能否请您…？（更礼貌） | `資料を確認していただけませんか。` | 比もらえませんか更正式 |
| `Vて + もらえないでしょうか` | 可否请…？（更委婉） | `少し待ってもらえないでしょうか。` | 礼貌度递增 |
| `Vて + いただけないでしょうか` | 能否劳驾您…？（最礼貌） | `駅までの行き方を教えていただけないでしょうか。` | 最高礼貌度 |
| `Nのようだ` | 好像…（比喻/推测） | `彼女はモデルのようだ。` | 与みたいだ区分书面度 |
| `NのようなN` | 像…一样的… | `彼はロボットのように正確に作業する。` | 修饰名词用のような |
| `NのようにV` | 像…那样地… | `彼女のように上手に話したい。` | 修饰动词用のように |
| `V辞書形 + ことは/が/を` | 把动词名词化 | `毎日読むことは発音練習になる。` | こと后接不同助词功能不同 |
| `NをNと言う` | 把A叫做B | `この料理を「お好み焼き」と言います。` | 定义/命名用法 |
| `〜という + N` | 叫做…的… | `「和紙」という素材があります。` | 说明名称/属性 |
| `疑問詞 + ても` | 无论…都… | `どんなに忙しくても朝ごはんを食べる。` | 后项必须有结论句 |

## 第2课｜定义・转述・口语比喻
**会话主题**：何のことですか　／　**读写**：外来語

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜たら、〜た` | …之后发现… | `家に帰ったら、荷物が届いていた。` | 后项是"结果发现"，非预期 |
| `〜というのは〜のことだ` | 所谓…就是… | `外来語というのは外国から来た言葉のことだ。` | 下定义专用 |
| `〜ということだ` | 也就是说…/听说… | `天気予報によると台風が来るということだ。` | 兼含总结和传闻 |
| `〜という + N` | 叫做…的… | `「リスキリング」という考え方が注目されている。` | 具体化名词 |
| `普通形 + ように言う` | 叫/让某人… | `先生は遅刻しないように注意した。` | 间接命令，前接辞书/ない形 |
| `普通形 + ように伝える` | 转告某人… | `9時に来るように伝えてください。` | 转述指令 |
| `普通形 + ように頼む` | 拜托某人… | `友達に手伝うように頼んだ。` | 请求传达 |
| `〜みたいだ` | 好像…（口语） | `今日は春みたいに暖かい。` | 比ようだ更口语，勿用于正式文 |
| `〜みたいなN` | 像…一样的… | `夢みたいな話だ。` | 口语口吻 |
| `〜みたいにV` | 像…那样地… | `彼みたいに上手くなりたい。` | 口语版のように |

## 第3课｜许可・决定・样态否定
**会话主题**：遅れそうなんです　／　**读写**：時間よ、止まれ！

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜(さ)せていただけませんか` | 请允许我… | `最初に発表させていただけませんか。` | 谦让请求许可 |
| `〜(さ)せてもらえませんか` | 可以让我…吗？ | `先に帰らせてもらえませんか。` | 比いただく略低礼貌 |
| `〜ことにする` | 决定…（自己决定） | `毎朝30分歩くことにした。` | 主观意志，≠ことになる |
| `〜ことにしている` | 一直坚持… | `寝る前にスマホを見ないことにしている。` | 自律习惯/规则 |
| `〜ことになる` | 被决定/结果变成… | `来月大阪へ転勤することになった。` | 外部决定，≠ことにする |
| `〜ことになっている` | 规定/既定安排 | `図書館では飲食してはいけないことになっている。` | 制度规定 |
| `〜てほしい` | 希望（别人）做… | `明日は早めに来てほしい。` | 主语≠自己 |
| `〜ないでほしい` | 希望（别人）不做… | `そこには触らないでほしい。` | 希望对方别做 |
| `Vます形去ます + そうだ` | 看起来要…（样态） | `雨が降りそうな空だ。` | ≠传闻そうだ |
| `〜なさそうだ` | 看起来不… | `彼は今日来なさそうだ。` | 样态否定 |
| `〜そうもない` | 怎么看都不太可能… | `今夜中には終わりそうもない。` | 强否定样态 |

## 第4课｜转述・口语缩约・书面体・愿望
**会话主题**：伝言、お願いできますか　／　**读写**：電話嫌い

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜ということだ` | 据说…/也就是说… | `台風は明日上陸するということだ。` | 兼传闻与总结 |
| `〜の/〜の？` | 口语确认/问原因 | `どうしてそんなに急いでいるの？` | 升调=疑问 |
| `〜ちゃう`（←てしまう） | 不小心…/做完了 | `財布を忘れちゃった。` | 口语缩约，勿用正式文 |
| `〜とく`（←ておく） | 预先做 | `資料を印刷しといてください。` | 口语缩约 |
| `〜てる`（←ている） | 正在…/一直… | `今何してる？` | 口语缩约 |
| `〜(さ)せられる` | 被迫… | `毎日ピアノを練習させられた。` | 使役受身，含不情愿 |
| `〜である` | …是…（书面体） | `評価基準は明確であるべきだ。` | 书面正式，≠です |
| `Vます形 + 中止形` | 并列动作（书面） | `朝起きて顔を洗い、家を出る。` | 书面并列，比てから正式 |
| `Vます形去ます + たがる` | （他人）表现出想… | `子どもは新しいゲームをやりたがっている。` | 第三人称愿望，≠たい |
| `〜(た)がっている` | 持续表现出想… | `妹はずっと留学したがっている。` | 强调持续状态 |
| `普通形 + こと/ということ` | 把内容抽象为"这件事" | `約束を守ることは信頼につながる。` | 名词化强调 |

## 第5课｜文脉指示・推量・路线・助词连体
**会话主题**：どう行ったらいいでしょうか　／　**读写**：地図

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `あの/あれ/あそこ` | 双方都知道的信息 | `あの人、昨日すごく緊張していたね。` | ≠そ系（一方知道） |
| `その/それ/そこ` | 前文已提信息/一方知道 | `来月北海道へ行く。そのとき雪祭りを見たい。` | 文章/会话中的回指 |
| `〜んじゃない？` | 口语推测/轻提醒 | `彼、今日来ないんじゃない？` | 口语用，≠正式疑问 |
| `〜たところに/で` | 刚好在…时点 | `出かけたところに、友達から電話が来た。` | 表时间节点"恰巧" |
| `〜(よ)うとする` | 正要做/试图做 | `電車に乗ろうとしたとき財布がなかった。` | 后项常被打断 |
| `〜(よ)うとしない` | 不想做/不肯做 | `彼は失敗を認めようとしない。` | 主观拒绝意志 |
| `〜のだろうか` | 到底会…吗（思考感） | `この方法で本当に間に合うのだろうか。` | 书面内省/思考 |
| `〜だろう` | 大概…吧（推量） | `明日は雨になるだろう。` | 比でしょう略书面 |
| `N + 助词 + の + N` | 助词短语→连体 | `駅からの道を地図に書いてください。` | との/への/からの等 |

## 第6课｜引用・主题・つもり・てくる/いく
**会话主题**：行かせていただきたいんですが　／　**读写**：メンタルトレーニング

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `普通形 + って` | 口语引用"听说…" | `先生が明日テストだって言っていた。` | ≠話題提示のって |
| `N + って` | 口语话题提示（≈は） | `日本の生活って最初は大変だよね。` | ≠引用のって |
| `〜つもりはない` | 不打算… | `もう言い訳するつもりはない。` | 明确否定意志 |
| `〜つもりだった` | 本来打算…（但未实现） | `週末は勉強するつもりだったが熱を出した。` | 含遗憾/说明 |
| `〜たつもり` | 自以为已经… | `説明したつもりだったが伝わっていなかった。` | 主观认为≠客观事实 |
| `〜ているつもり` | 自认一直在… | `気をつけているつもりだが間違える。` | 主观认知偏差 |
| `〜てばかりいる` | 老是（一直）做… | `最近、甘い物を食べてばかりいる。` | 含负面/偏颇语感 |
| `〜とか〜とか` | …啊…啊（不完全列举） | `休日は映画を見るとか散歩するとかします。` | 口语，非穷尽 |
| `〜てくる`（事态出现） | 某变化从无到有出现 | `日本語が少しずつ聞き取れるようになってくる。` | 变化朝现在累积 |
| `〜てくる/〜ていく` | 朝现在靠近/从现在延伸 | `だんだん暑くなってきた。` | てくる=靠近，ていく=延伸 |

## 第7课｜义务・许可・语气词・感情使役
**会话主題**：楽しみにしています・遠慮させてください　／　**读写**：まんじゅう、怖い

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜なくてはならない` | 必须… | `明日は7時までに来なくてはならない。` | 与なければならない可互换 |
| `〜なくてもかまわない` | 不做也可以 | `今日は来なくてもかまいません。` | 许可的否定表达 |
| `〜なくちゃ/なきゃ` | 必须…（口语） | `もう行かなきゃ。` | 口语缩约，勿正式场合 |
| `〜だけだ` | 只…而已 | `必要なところだけ読めばいい。` | 限定，中性语气 |
| `〜だけでいい` | 只要…就行 | `名前を書くだけでいいです。` | 最低限要求 |
| `〜かな` | 自问/不确定 | `明日、雨かな。` | 句末，自语/柔和疑问 |
| `〜なんか` | 诶…什么的（轻视/谦逊） | `私なんかまだまだです。` | 勿正式场合，含轻视感 |
| `〜なんて` | 竟然…（惊讶/评价） | `一人で富士山に登るなんてすごいですね。` | 比なんか更主观强烈 |
| `〜(さ)せる`（感情使役） | 使（某人）产生感受 | `そのニュースは家族を安心させた。` | 主语是使其产生情绪之事物 |
| `〜(さ)せられる`（感情） | 被（某事）弄得… | `彼の一言には考えさせられた。` | 受影响/引发思考 |
| `〜なら、〜` | 如果是…的话 | `時間がないなら要点だけ確認しよう。` | 接前文话题，给建议 |

## 第8课｜时间边界・状态保持・原因
**会话主题**：迷子になっちゃったんです　／　**读写**：科学者ってどう見える？

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜あいだ` | 在…期间（持续） | `電車に乗っているあいだずっと本を読んでいた。` | Y也持续进行 |
| `〜あいだに` | 在…期间（某时点发生） | `料理しているあいだに友達が来た。` | Y是一次性事件 |
| `〜まで` | 直到…（动作持续到终点） | `会議が終わるまでここで待ってください。` | 持续到某时点 |
| `〜までに` | 在…之前（截止时点） | `明日の朝までにレポートを送ってください。` | ≠まで（截止≠持续） |
| `Vた形 + N` | 已完成状态修饰名词 | `眼鏡をかけた人が受付にいます。` | 表完成结果 |
| `〜によって` | 因…而异/由…完成 | `仕事の進め方は会社によって違う。` | 多义：因/由/通过 |
| `〜たまま` | 保持…状态不变地 | `窓を開けたまま寝てしまった。` | 前项是已完成动作 |
| `〜のまま` | 保持…状态不变 | `子どものまま大人になった。` | 前接名词/状态 |
| `〜からだ` | 正是因为…（明确原因） | `日本語を学ぶのは日本で働きたいからだ。` | 强调原因，较书面 |

## 第9课｜尊敬・比较・反事实
**会話主题**：どこが違うんですか　／　**读写**：カラオケ

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `お + Vます形 + です` | （您）正在…（尊敬） | `社長は今、応接室でお待ちです。` | 只用于对方动作，≠自称 |
| `〜てもかまわない` | …也没关系 | `この席に座ってもかまいません。` | 比てもいい略正式 |
| `AはBほど〜ない` | A不如B那么… | `この町は東京ほど人が多くない。` | 后项必须否定 |
| `AはBほどではない` | A比不上B | `去年ほどではないが今年も寒い。` | 略感叹，部分比较 |
| `XほどYはない` | 没有比X更Y的 | `春の京都ほど美しい景色はない。` | 主观最上级 |
| `〜ため[に]〜` | 因为…（书面原因） | `大雨のため、試合は中止になった。` | 书面≠から，兼原因/目的 |
| `〜ためだ` | 正是因为…（书面结论） | `事故が起きたのは確認を怠ったためだ。` | 书面理由陈述 |
| `〜たら/ば、〜た` | 如果当时…就… | `もっと早く出発していれば間に合った。` | 反事实假设，后项是过去 |

## 第10课｜高确信判断・结果・复合动词
**会话主题**：そんなはずはありません　／　**读写**：記憶型と注意型

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜はずだ` | 理应…/按理说… | `田中さんは真面目だから約束を守るはずだ。` | 基于逻辑前提推断 |
| `〜はずがない` | 不可能… | `あの人がうそをつくはずがない。` | 强否定推断 |
| `〜はずだった` | 原本应该…（但未实现） | `今日の面談は10時のはずだった。` | 预期落空 |
| `〜ことがある` | 有时会… | `忙しいと昼ご飯を食べないことがある。` | 低频率 |
| `〜こともある` | 偶尔也有… | `たまに失敗することもある。` | 强调偶尔 |
| `〜た結果` | 结果… | `十分に話し合った結果、計画を見直すことにした。` | 后项是客观结果，≠意图 |
| `〜の結果` | …的结果 | `長年の研究の結果、新薬が完成した。` | 名词接続 |
| `Vます形 + 出す` | 突然开始… | `空が急に暗くなり、雨が降り出した。` | 突发性开始 |
| `Vます形 + 始める` | 开始… | `9時から会議が始まります。` | 动作起点 |
| `Vます形 + 終わる` | 做完… | `レポートを書き終わったら送ります。` | 动作终点 |
| `Vます形 + 続ける` | 继续做… | `彼は休まず働き続けた。` | 持续 |
| `Vます形 + 忘れる` | 忘记做… | `鍵をかけ忘れて出かけた。` | 疏忽遗漏 |
| `Vます形 + 合う` | 互相… | `二人で話し合った。` | 相互动作 |
| `Vます形 + 換える` | 更换/改换… | `携帯を買い換えた。` | 更换对象 |

## 第11课｜渐变方向・比较・らしい・ず形
**会话主题**：お勧めのところ、ありませんか　／　**读写**：白川郷の黄金伝説

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜てくる`（渐变） | 变化累积到现在 | `最近、気温が下がってきた。` | 朝现在方向变化 |
| `〜ていく`（渐变） | 从现在继续往后变化 | `これからもっと暑くなっていくだろう。` | 从现在向未来延伸 |
| `〜たらどう？` | 你要不要…？（建议） | `疲れているなら少し休んだらどう？` | 比ほうがいい更柔和 |
| `AよりBのほうが〜` | 比起A，B更… | `電車よりバスのほうが安い。` | 两项比较基本句型 |
| `〜らしい`（典型） | 像…该有的样子 | `今日は春らしい天気ですね。` | 表典型性质，接名词 |
| `〜らしい`（传闻） | 听说/好像… | `天気予報によると明日は雪らしい。` | 间接信息推测 |
| `〜として` | 作为…/以…身份 | `留学生として日本に来ました。` | 立场/资格/观点 |
| `〜ず[に]` | 不…地（书面） | `朝ご飯を食べずに学校へ行った。` | 书面版ないで，する→せず |
| `〜ず、〜` | 不做…，而…（书面并列） | `彼は謝らず、部屋を出ていった。` | 书面逻辑连接 |
| `〜ている`（经历/经历持续） | 有过…的经历/一直处于 | `彼は十年日本で働いている。` | 表经历积累或状态持续 |

## 第12课｜理由深化・间接受身・状态放置
**会话主题**：ご迷惑をかけてすみませんでした　／　**读写**：〖座談会〗日本で暮らす

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `〜ものだから` | 因为…（辩解语气） | `電車が遅れたものですから遅刻しました。` | 带辩解/说明自己语气 |
| `〜もんだから` | 因为…（口语辩解） | `急いでいたもんだから忘れちゃった。` | もんだから更口语 |
| `〜(ら)れる`（间接受身・自动词） | 被（外界）影响，受害 | `急に雨に降られて服が濡れた。` | 自动词受身，受害视角 |
| `〜(ら)れる`（间接受身・他动词） | （他人做某事）连累我 | `隣で夜中まで騒がれて眠れなかった。` | 他动词受身，间接受害 |
| `〜たり〜たり` | 时而…时而… | `最近、寒かったり暖かかったりしている。` | 交替/不稳定状态 |
| `〜っぱなし` | 一直保持…状态不管 | `ドアを開けっぱなしにしないで。` | 多含负面/放任语感 |
| `〜おかげで` | 多亏…（正面结果） | `先生のおかげで試験に合格できた。` | 后项正面 |
| `〜おかげだ` | 都是因为…的功劳 | `合格できたのは先生のおかげだ。` | 结论句型 |
| `〜せいで` | 都怪…（负面结果） | `渋滞のせいで約束に遅れた。` | 后项负面，≠おかげで |
| `〜せいだ` | 都是因为…的错 | `失敗したのは準備不足のせいだ。` | 负面结论 |

## 全册重点对比速查

| 易混项 | 区别核心 | 对照例句 |
|---|---|---|
| `もらえませんか` vs `いただけませんか` | 后者更正式礼貌 | `教えてもらえませんか。/ 教えていただけませんか。` |
| `ことにする` vs `ことになる` | 自我决定 vs 外部决定 | `留学することにした。/ 転勤することになった。` |
| `たいです` vs `たがる` | 第一人称 vs 第三人称 | `私は行きたい。/ 彼は行きたがっている。` |
| `そうだ`（様態） vs `そうだ`（伝聞） | Vます形+そうだ vs 普通形+そうだ | `雨が降りそうだ。/ 雨が降るそうだ。` |
| `らしい`（典型） vs `らしい`（伝聞） | N+らしい vs 普通形+らしい | `春らしい天気。/ 明日は雪らしい。` |
| `あいだ` vs `あいだに` | 持续并行 vs 期间内一次发生 | `寝ているあいだ夢を見た。/ 寝ているあいだに電話が来た。` |
| `まで` vs `までに` | 持续到某点 vs 截止时点前完成 | `5時まで待つ。/ 5時までに送る。` |
| `たまま` vs `っぱなし` | 保持状态去做 vs 放任不管（负面） | `靴を履いたまま入った。/ 電気をつけっぱなしだ。` |
| `おかげで` vs `せいで` | 正面原因 vs 负面原因 | `先生のおかげで合格できた。/ 雨のせいで中止。` |
| `ものだから` vs `から/ので` | 带辩解语气 vs 普通理由 | `急いでいたものだから失礼した。/ 忙しいから行けない。` |
| `はずだ` vs `にちがいない` | 逻辑推断 vs 主观强确信 | `もう着いたはずだ。/ 彼が犯人にちがいない。` |
| `〜てくる` vs `〜ていく` | 变化朝现在 vs 变化朝未来 | `暑くなってきた。/ 暑くなっていくだろう。` |
