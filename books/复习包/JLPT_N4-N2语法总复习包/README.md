# JLPT N4-N2 语法总复习包

> 目标：按 `N4 / N3 / N2` 分层复习语法，覆盖考试高频与教材核心。  
> 说明：JLPT 官方不公开“唯一语法清单”，本复习包按主流教材、历年真题常见项做“尽可能全覆盖”的应试整理。  
> 结构：每个语法点统一提供 `含义 + 例句 + 易错点 + 使用场景`。

## 文件目录

- `N4语法总复习.md`
- `N3语法总复习.md`
- `N2语法总复习.md`

## 当前版本说明（细粒度补全版）

- 每个级别都包含两轮内容：
- 第一轮：核心高频语法（覆盖考试主干）。
- 第二轮：细粒度补充清单（补齐边界项、书面项、易混项）。
- 每条语法统一包含：`含义 / 例句 / 易错点 / 使用场景`。

## 使用建议（考前 7 天）

1. 第 1-2 天：通读各级别文法表格，只看“含义 + 例句”。
2. 第 3-4 天：重点攻克“易错点”，自己造句并对照改错。
3. 第 5 天：把 N3/N2 连接词、逆接、条件、推量做专项对比。
4. 第 6 天：做真题语法单项，回查本包中对应语法。
5. 第 7 天：只看“高频易混对照”，保持手感。

## 优先级建议

- 时间紧：优先 `N3/N2` 的 `逆接・让步・推量・限定・书面连接`。
- 时间足：按 N4→N3→N2 顺序完整过一遍。
