> 当前覆盖第1课到第12课（全书）。  
> 本文件为《MINNA NO NIHONGO INTERMEDIATE 1》整本汇总版。

## 第1课

### 本课核心目标

第1课的重点是把“请求、比喻、定义、让步”说得更像中级日语。  
核心变化是：同样是“请帮我做某事”，会根据对象和场合换更礼貌的表达。

### 会话与读写主题

- 話す・聞く：`お願いがあるんですが`
- 読む・書く：`畳`

### 文法详细总结

#### 1. `Vて + もらえませんか / いただけませんか / もらえないでしょうか / いただけないでしょうか`

- 用法：委婉请求对方做某事，四个形式礼貌程度递增。  
- 接续：`Vて形 + 目标表达`  
- 例句：  
  - 駅までの行き方を教えていただけないでしょうか。  
  - この資料を今日中に確認してもらえませんか。

#### 2. `Nのようだ / NのようなN / NのようにV・A`

- 用法：比喻、举例，表示“像……一样”。  
- 接续：`N + のようだ`、`N + のような + 名词`、`N + のように + 动词/形容词`  
- 例句：  
  - 彼はロボットのように正確に作業する。  
  - ここは春なのに、冬のように寒い。

#### 3. `V辞書形 + ことは / ことが / ことを`

- 用法：把动词名词化后作为句子成分（主题/主语/宾语）。  
- 接续：`V辞書形 + こと + 助词`  
- 例句：  
  - 毎日声に出して読むことは、発音の練習になる。  
  - 日本語で考えることが大切だ。

#### 4. `NをNと言う`

- 用法：定义、命名，“把A叫做B”。  
- 接续：`AをBと言う`  
- 例句：  
  - この料理を「お好み焼き」と言います。  
  - 日本で三月三日は「ひな祭り」と言います。

#### 5. `～という + 名词`

- 用法：说明名称、内容、属性，相当于“叫做……的”“所谓……的”。  
- 接续：`普通形/名词 + という + 名词`  
- 例句：  
  - 昨日、「和紙」という素材について勉強した。  
  - 彼は時間を守るという習慣がある。

#### 6. `いつ / どこ / 何 / だれ / どの / どんなに + ～ても`

- 用法：让步表达，“无论……都……”。  
- 接续：疑问词直接接 `～ても`（动词/形容词按句意变形）  
- 例句：  
  - どんなに忙しくても、朝ごはんは食べる。  
  - だれに聞いても、あの店はおいしいと言う。

### 本课易错点

1. 把 `～てもらえませんか` 当成命令句使用。  
2. `～のようだ` 和 `～みたいだ` 混用场景不分。  
3. `疑问词 + ても` 后面没写完整结论句。

### 本课复习清单

- 能按对象切换四档请求表达。  
- 能用 `のようだ / のような / のように` 各造一句。  
- 能用 `疑问词 + ても` 表达让步。

## 第2课

### 本课核心目标

第2课的主线是“说明概念 + 转述指令 + 口语比喻”。  
学完这课后，你可以更自然地解释“这是什么意思”、也能把别人说的话准确转给第三方。

### 会话与读写主题

- 話す・聞く：`何のことですか`
- 読む・書く：`外来語`

### 文法详细总结

#### 1. `～たら、～た`

- 用法：表示前项发生后发现了某个结果，常带“结果出乎当下发现”的语感。  
- 接续：`Vた形 + ら、～た`  
- 例句：  
  - 家に帰ったら、ドアの前に荷物が置いてあった。  
  - 窓を開けたら、外はもう雪だった。

#### 2. `～というのは～のことだ / ～ということだ`

- 用法：下定义、做解释，“所谓A就是B”。  
- 接续：`Aというのは + Bのことだ/ということだ`  
- 例句：  
  - 外来語というのは、外国から入ってきた言葉のことだ。  
  - 協調性というのは、相手と協力して動ける力ということだ。

#### 3. `～という + 名词`

- 用法：把后面的名词具体化，相当于“叫做……的……”。  
- 接续：`普通形/名词 + という + 名词`  
- 例句：  
  - 「リスキリング」という考え方が注目されている。  
  - 彼は「準備が八割」という信念を持っている。

#### 4. `普通形 + ように言う / 注意する / 伝える / 頼む`

- 用法：间接转述要求、提醒、嘱托。  
- 接续：`普通形（辞書形・ない形）+ ように + 传达类动词`  
- 例句：  
  - 先生は学生に、遅刻しないように注意した。  
  - 母に、帰りが遅くなるようなら連絡するように言われた。

#### 5. `～みたいだ / ～みたいな + N / ～みたいに + V・A`

- 用法：口语化比喻或不确定判断，“像……似的”。  
- 接续：`普通形/名词 + みたいだ`、`みたいな`、`みたいに`  
- 例句：  
  - 今日は春みたいに暖かい。  
  - 彼は何でも知っている先生みたいな人だ。

### 本课易错点

1. `～たら、～た` 和普通条件句混用。  
2. `ように言う` 前接错形。  
3. `～みたい` 在正式书面文频繁使用。

### 本课复习清单

- 能用 `～というのは～のことだ` 给3个词下定义。  
- 能把“老师的要求”用 `～ように言う` 转述出来。  
- 能区分 `～ようだ`（较正式）和 `～みたい`（较口语）。

## 第3课

### 本课核心目标

第3课围绕“许可、决定、愿望、样态判断”展开。  
这是从初级句型走向中级表达的关键一课，尤其是 `ことにする` / `ことになる` 的主客观区分。

### 会话与读写主题

- 話す・聞く：`遅れそうなんです`
- 読む・書く：`時間よ、止まれ！`

### 文法详细总结

#### 1. `～(さ)せてもらえませんか / ～(さ)せていただけませんか`

- 用法：请求对方允许“我去做某事”。  
- 接续：`使役形て + もらう/いただく`  
- 例句：  
  - 明日の会議で、最初に発表させていただけませんか。  
  - 今日は体調が悪いので、早退させてもらえませんか。

#### 2. `～ことにする`

- 用法：说话人主动做决定。  
- 接续：`V辞書形/ない形 + ことにする`  
- 例句：  
  - 健康のために、毎朝30分歩くことにした。  
  - 今月は外食しないことにする。

#### 3. `～ことにしている`

- 用法：持续性的个人规则、习惯。  
- 接续：`V辞書形/ない形 + ことにしている`  
- 例句：  
  - 寝る前にスマホを見ないことにしている。  
  - 週末は必ず家族に電話することにしている。

#### 4. `～ことになる`

- 用法：结果上被决定、变成某种状态（非个人单方决定）。  
- 接续：`V辞書形/ない形 + ことになる`  
- 例句：  
  - 来月から大阪支社へ転勤することになった。  
  - 会議は延期しないことになった。

#### 5. `～ことになっている`

- 用法：制度、规定、既定安排。  
- 接续：`V辞書形/ない形 + ことになっている`  
- 例句：  
  - この図書館では飲食してはいけないことになっている。  
  - 申請書は三日前までに提出することになっている。

#### 6. `～てほしい / ～ないでほしい`

- 用法：希望别人做/不要做某事。  
- 接续：`Vて + ほしい`、`Vないで + ほしい`  
- 例句：  
  - 明日は早めに来てほしい。  
  - 会議中は私語をしないでほしい。

#### 7. `～そうな + N / ～そうに + V`

- 用法：根据外观判断“看起来要……”。  
- 接续：`动词ます形去ます + そう`、`い形容词词干 + そう`  
- 例句：  
  - 雨が降りそうな空だ。  
  - 子どもたちが楽しそうに遊んでいる。

#### 8. `～なさそう`

- 用法：看起来不……（样态否定）。  
- 接续：`动词ます形去ます + なさそう`、`い形容词去い + なさそう`  
- 例句：  
  - 彼は今日は来なさそうだ。  
  - この道具は壊れなさそうだ。

#### 9. `～そうもない`

- 用法：强否定样态，“怎么看都不太可能……”。  
- 接续：`动词ます形去ます + そうもない`  
- 例句：  
  - 今夜中には終わりそうもない。  
  - 彼の説明だけでは納得できそうもない。

### 本课易错点

1. `ことにする` 与 `ことになる` 混用。  
2. `そう`（样态）和传闻 `そう` 混用。  
3. `～てほしい` 主语误用。

### 本课复习清单

- 能口头解释 `ことにする` 和 `ことになる` 的差别。  
- 能用 `～そうな / ～そうに / ～なさそう / ～そうもない` 各造一句。  
- 能在请假、请求许可场景中自然说出 `～(さ)せていただけませんか`。

## 第4课

### 本课核心目标

第4课强调“信息转述 + 口语缩约 + 书面语体 + 情感表达”。  
学完后你会更容易看懂真实会话里的缩略形式，也能更自然地表达“听说、好像、看起来”。

### 会话与读写主题

- 話す・聞く：`伝言、お願いできますか`
- 読む・書く：`電話嫌い`

### 文法详细总结

#### 1. `・・・ということだ`

- 用法：转述消息、总结判断，“据说……/也就是说……”。  
- 接续：`普通形 + ということだ`  
- 例句：  
  - 天気予報によると、台風は明日上陸するということだ。  
  - 彼は来月帰国するということです。

#### 2. `・・・の / ・・・の？`

- 用法：口语中省略后项，`の？` 常用于确认、询问理由。  
- 接续：`普通形 + の`（句末升调可作疑问）  
- 例句：  
  - どうしてそんなに急いでいるの？  
  - 今日は来ないの？

#### 3. `～ちゃう / ～とく / ～てる`

- 用法：口语缩约。  
- 接续：  
  - `～てしまう -> ～ちゃう`  
  - `～ておく -> ～とく`  
  - `～ている -> ～てる`  
- 例句：  
  - 財布を忘れちゃった。  
  - この資料、先に読んどいて。  
  - 今、駅で待ってる。

#### 4. `～(さ)せられる / ～される`（使役受身）

- 用法：被迫做某事，带有不情愿语感。  
- 接续：使役受身形（五段/一段动词各自变形）  
- 例句：  
  - 子どものころ、毎日ピアノを練習させられた。  
  - 会議で急に発表させられて困った。

#### 5. `～である`

- 用法：书面体判断句，比 `です` 更正式、客观。  
- 接续：`名词/な形容词语干 + である`  
- 例句：  
  - 日本語教育において、評価基準は明確であるべきだ。  
  - この結果は非常に重要である。

#### 6. 中止形 `～ます、～ます、・・・ / ～くて、～くて、・・・`

- 用法：并列动作、状态，不强行断句。  
- 接续：  
  - 动词 `ます形` 直接并列  
  - い形容词 `くて` 并列  
- 例句：  
  - 朝起きて、顔を洗い、朝ごはんを食べ、家を出る。  
  - この町は静かで、空気がきれいで、住みやすい。

#### 7. `～(た)がる`

- 用法：第三人称“表现出想……”。  
- 接续：`Vます形去ます + たがる`  
- 例句：  
  - 子どもは新しいゲームをやりたがっている。  
  - 彼はその話を聞きたがらない。

#### 8. `～(た)がっている`

- 用法：持续状态地显示愿望。  
- 接续：`Vます形去ます + たがっている`  
- 例句：  
  - 妹はずっと留学したがっている。  
  - 部長は結果を早く知りたがっています。

#### 9. `・・・こと / ・・・ということ`

- 用法：把句子内容抽象成“这件事/这个事实”。  
- 接续：`普通形 + こと / ということ`  
- 例句：  
  - 約束を守ることは、信頼につながる。  
  - 今日の会議が中止だということを全員に伝えてください。

### 本课易错点

1. `たい` 和 `たがる` 混用。  
2. 缩约形式直接用于正式邮件。  
3. `ということだ` 的语义不分。

### 本课复习清单

- 能把正式句改写成口语缩约句。  
- 能区分 `たい`、`たがる`、`たがっている`。  
- 能用 `である` 写出2句书面风格句子。

## 第5课

### 本课核心目标

第5课主打“文脉指示 + 推测语气 + 路线表达”。  
这课在真实会话里非常实用，尤其是问路、确认信息、给建议时的自然说法。

### 会话与读写主题

- 話す・聞く：`どう行ったらいいでしょうか`
- 読む・書く：`地図`

### 文法详细总结

#### 1. `あ～ / そ～`（会话中的文脉指示）

- 用法：在会话里指代已提到的信息。  
  - `あ`：说话人和听话人都知道。  
  - `そ`：一方知道、另一方未必知道。  
- 接续：`あの/あれ/あそこ`、`その/それ/そこ` 等。  
- 例句：  
  - あの人、昨日の発表ですごく緊張していたね。  
  - その映画って、どんな話ですか。

#### 2. `そ～`（文章中的文脉指示）

- 用法：在后句指代前句已出现的信息。  
- 接续：`その + 名词`、`そこで`、`それについて` 等。  
- 例句：  
  - 来月北海道へ行きます。そのときに雪祭りを見たいです。  
  - この問題は難しいです。それについて後で説明します。

#### 3. `・・・んじゃない？`

- 用法：口语推测、建议、轻度提醒。  
- 接续：`普通形 + んじゃない？`（名词/な形容词接 `な`）  
- 例句：  
  - 彼、今日来ないんじゃない？  
  - ちょっと休んだほうがいいんじゃない？

#### 4. `～たところに / ～たところで`

- 用法：表示“刚好在……时点”。  
- 接续：`Vた形 + ところに/で`  
- 例句：  
  - 出かけたところに、友達から電話がかかってきた。  
  - 帰ろうとしたところで、急な仕事が入った。

#### 5. `～(よ)うとする`

- 用法：正要做某动作、尝试去做。  
- 接续：`意向形 + とする`  
- 例句：  
  - 電車に乗ろうとしたとき、財布がないことに気づいた。  
  - 何度も説明しようとしたが、うまく言えなかった。

#### 6. `～(よ)うとする / しない`

- 用法：表示主观意图的有无（想做/不想做）。  
- 接续：`意向形 + とする / としない`  
- 例句：  
  - 彼は自分の失敗を認めようとしない。  
  - 子どもが野菜を食べようとしなくて困っている。

#### 7. `・・・のだろうか`

- 用法：带思考感的疑问与推量（“到底会……吗”）。  
- 接续：`普通形 + のだろうか`  
- 例句：  
  - この方法で本当に間に合うのだろうか。  
  - 彼はなぜあんなことを言ったのだろうか。

#### 8. `～との / での / からの / までの / への + 名词`

- 用法：把助词短语变成定语，连接名词。  
- 接续：`名词 + 助词 + の + 名词`  
- 例句：  
  - 駅からの道を地図に書いてください。  
  - 先生への質問をメモしておいた。

#### 9. `・・・だろう / ・・・だろうと思う`

- 用法：推量、判断，语气比 `と思う` 更明确。  
- 接续：`普通形 + だろう`  
- 例句：  
  - 明日は雨になるだろう。  
  - この案なら、みんな賛成するだろうと思う。

### 本课易错点

1. 会话中的 `あ`/`そ` 指示关系判断反了。  
2. `んじゃない？` 在正式场合使用过多。  
3. `～との～` 等结构里漏掉 `の`。

### 本课复习清单

- 能解释 `あ` 和 `そ` 在会话里的知识共享关系。  
- 能用 `～(よ)うとする` 说“正要做但被打断”。  
- 能用 `からの/への` 做路线和对象说明句。

## 第6课

### 本课核心目标

第6课重点在“口语表达真实化”：引用、话题标记、主观认知偏差、动作变化方向。  
这课会明显提升你听懂日常日语的能力。

### 会话与读写主题

- 話す・聞く：`行かせていただきたいんですが`
- 読む・書く：`メンタルトレーニング`

### 文法详细总结

#### 1. `・・・て / ・・・って・・・`（引用）

- 用法：口语引用，`って` 比 `と` 更随意。  
- 接续：`普通形 + って`（也可名词 + って）  
- 例句：  
  - 先生が、明日はテストだって言っていました。  
  - 田中さん、来週引っ越すって。

#### 2. `～って・・・`（主題）

- 用法：口语话题提示，相当于 `～は`。  
- 接续：`名词 + って`  
- 例句：  
  - 日本の生活って、最初は大変だよね。  
  - このアプリって、どう使うの？

#### 3. `～つもりはない`

- 用法：明确否定意志，“不打算……”。  
- 接续：`V辞書形 + つもりはない`  
- 例句：  
  - その件について、もう言い訳するつもりはない。  
  - 私は約束を破るつもりはありません。

#### 4. `～つもりだった`

- 用法：过去本打算……（但可能没实现）。  
- 接续：`V辞書形/ない形 + つもりだった`  
- 例句：  
  - 週末は勉強するつもりだったが、熱を出して寝ていた。  
  - 返信しないつもりだったけど、結局返してしまった。

#### 5. `～たつもり / ～ているつもり`

- 用法：主观上“自以为已经……”“自认一直在……”。  
- 接续：`Vた + つもり`、`Vている + つもり`  
- 例句：  
  - 説明したつもりだったが、相手に伝わっていなかった。  
  - 丁寧に話しているつもりでも、強く聞こえることがある。

#### 6. `～てばかりいる / ～ばかり～ている`

- 用法：总是做某事（常带负面或偏颇语感）。  
- 接续：`Vて + ばかりいる`、`ばかり + Vている`  
- 例句：  
  - 最近、甘い物を食べてばかりいる。  
  - 休日は寝てばかりいて、何もしなかった。

#### 7. `・・・とか・・・`

- 用法：列举例子（不完全列举）。  
- 接续：`普通形 + とか`、`名词 + とか`  
- 例句：  
  - 休日は映画を見るとか、散歩するとかしています。  
  - 会議では、コストとか納期とかを確認した。

#### 8. `～てくる`（事态出现）

- 用法：某种变化从无到有地“出现、显现”。  
- 接续：`Vて + くる`  
- 例句：  
  - 日本語を勉強していると、少しずつ聞き取れるようになってくる。  
  - 時間がたつと、問題点が見えてくる。

#### 9. `～てくる / ～ていく`（近づく / 離れる）

- 用法：以说话人时间轴为基准，描述变化走向。  
  - `てくる`：朝“现在”靠近。  
  - `ていく`：从“现在”往后延伸。  
- 接续：`Vて + くる / いく`  
- 例句：  
  - だんだん暑くなってきた。  
  - これからもっと高齢化が進んでいくだろう。

### 本课易错点

1. `って`（引用）和 `って`（话题）不分。  
2. `つもり` 句型把“客观事实”误写成“主观认知”。  
3. `てくる/ていく` 的时间基准点没立住。

### 本课复习清单

- 能各写2句 `って` 的引用和话题用法。  
- 能区分 `つもりはない`、`つもりだった`、`たつもり`。  
- 能用 `てくる/ていく` 描述天气、人口、技能变化。

## 第7课

### 本课核心目标

第7课集中训练“义务、容许、限制、情绪语气、感情使役”。  
这一课的口语色彩更强，尤其是 `かな`、`なんか`、`なんて` 的语感处理。

### 会话与读写主题

- 話す・聞く：`楽しみにしています・遠慮させてください`
- 読む・書く：`まんじゅう、怖い`

### 文法详细总结

#### 1. `～なくてはならない / いけない`、`～なくてもかまわない`

- 用法：  
  - `なくてはならない/いけない`：必须。  
  - `なくてもかまわない`：不做也可以。  
- 接续：`Vない形去ない + くては + ならない/いけない`  
- 例句：  
  - 明日は朝七時までに来なくてはならない。  
  - この書類は今日出さなくてもかまわない。

#### 2. `～なくちゃ / ～なきゃ（いけない）`

- 用法：口语化“必须……”。  
- 接续：`Vない形去ない + なくちゃ/なきゃ`  
- 例句：  
  - もう行かなきゃ。  
  - 早くレポートを出さなくちゃいけない。

#### 3. `・・・だけだ / [ただ]・・・だけでいい`

- 用法：限制范围，“只要……就行”。  
- 接续：`普通形 + だけだ`、`普通形 + だけでいい`  
- 例句：  
  - 必要なところだけ読めばいい。  
  - 明日は来るだけでいいです。

#### 4. `・・・かな`（终助词）

- 用法：自问、缓和断定、表达不确定。  
- 接续：句末直接加 `かな`  
- 例句：  
  - 明日、雨かな。  
  - これで伝わるかな。

#### 5. `～なんか・・・`

- 用法：轻视、谦逊、弱化某对象。  
- 接续：`名词/普通形 + なんか`  
- 例句：  
  - 私なんか、まだまだです。  
  - こんな失敗なんか気にしないで。

#### 6. `・・・なんて・・・`

- 用法：惊讶、评价、轻视，语气比 `なんか` 更主观。  
- 接续：`普通形/名词 + なんて`  
- 例句：  
  - 一人で富士山に登るなんて、すごいですね。  
  - うそをつくなんて、ひどいよ。

#### 7. `～(さ)せる`（感情使役）

- 用法：某事物“使（某人）产生某种感受”。  
- 接续：`使役形`，常见搭配 `安心させる/感動させる` 等。  
- 例句：  
  - そのニュースは家族を安心させた。  
  - 彼のスピーチは聴衆を感動させた。

#### 8. `～(さ)せられる / ～される`（感情的使役受身）

- 用法：被某事“弄得……”，表达受影响感。  
- 接续：`使役受身形`  
- 例句：  
  - 彼の一言には考えさせられた。  
  - その映画を見て、泣かされた。

#### 9. `・・・なら、・・・`

- 用法：基于前项条件给建议、判断、对应措施。  
- 接续：`普通形/名词 + なら`  
- 例句：  
  - 時間がないなら、要点だけ先に確認しよう。  
  - 日本で働くなら、敬語は避けて通れない。

### 本课易错点

1. `なくてはならない` 与 `なければならない` 书写混乱。  
2. `なんか/なんて` 用在正式文体导致语气不当。  
3. 感情使役里主语宾语方向写反。

### 本课复习清单

- 能按正式度排序：`なくてはならない -> なければならない -> なきゃ`。  
- 能写出2句 `なんか` 和2句 `なんて` 并解释语感。  
- 能区分 `～(さ)せる`（使别人）与 `～(さ)せられる`（我被影响）。

## 第8课

### 本课核心目标

第8课是“时间边界 + 状态保持 + 原因说明”的整合课。  
重点在区分 `あいだ / あいだに`、`まで / までに`，并能自然表达“保持某状态去做下一动作”。

### 会话与读写主题

- 話す・聞く：`迷子になっちゃったんです`
- 読む・書く：`科学者ってどう見える？`

### 文法详细总结

#### 1. `～あいだ`

- 用法：在X持续期间，Y也持续进行。  
- 接续：`Vている + あいだ`、`Nの + あいだ`  
- 例句：  
  - 電車に乗っているあいだ、ずっと本を読んでいた。  
  - 留学しているあいだ、日本語だけで生活した。

#### 2. `～あいだに`

- 用法：在X持续期间内，Y这一事件发生。  
- 接续：`Vている + あいだに`、`Nの + あいだに`  
- 例句：  
  - 料理しているあいだに、友達が来た。  
  - 日本にいるあいだに、京都へ行きたい。

#### 3. `～まで`

- 用法：表示动作/状态持续到某终点。  
- 接续：`N + まで`、`V辞書形 + まで`  
- 例句：  
  - 会議が終わるまで、ここで待っていてください。  
  - 昨日は夜中の二時まで勉強した。

#### 4. `～までに`

- 用法：表示截止时点，“在……之前完成”。  
- 接续：`N + までに`、`V辞書形 + までに`  
- 例句：  
  - 明日の朝までにレポートを送ってください。  
  - 30歳になるまでに一度は起業したい。

#### 5. `～た + 名词`（名词修饰）

- 用法：表示动作完成后的结果状态修饰名词。  
- 接续：`Vた形 + 名词`  
- 例句：  
  - 眼鏡をかけた人が受付にいます。  
  - 壊れた椅子は先に処分してください。

#### 6. `～によって・・・`

- 用法：按类别不同而变化，“因……而异”。  
- 接续：`N + によって`  
- 例句：  
  - 仕事の進め方は会社によって違う。  
  - 国によって食事のマナーはさまざまだ。

#### 7. `～たまま / ～のまま`

- 用法：保持某状态不变去做后项动作。  
- 接续：`Vた形 + まま`、`Nの + まま`  
- 例句：  
  - 窓を開けたまま寝てしまった。  
  - 昔のままの町並みが残っている。

#### 8. `・・・からだ`（原因・理由）

- 用法：明确陈述原因、判断依据，常用于较正式说明。  
- 接续：`普通形 + からだ`  
- 例句：  
  - 私が日本語を学ぶのは、日本で働きたいからだ。  
  - この案に反対するのは、費用が高すぎるからだ。

### 本课易错点

1. `あいだ` 和 `あいだに` 都翻成“期间”而误用。  
2. `まで` 和 `までに` 混用。  
3. `たまま` 的前项动词选错。

### 本课复习清单

- 能各写3句 `あいだ`/`あいだに` 对比句。  
- 能各写3句 `まで`/`までに` 对比句。  
- 能用 `からだ` 写出正式的“结论+理由”句子。

## 第9课

### 本课核心目标

第9课重点是“尊敬表达 + 比较表达 + 反事实表达”。  
这一课会把“礼貌地问对方状态”和“更精细的比较句”系统化。

### 会话与读写主题

- 話す・聞く：`どこが違うんですか`
- 読む・書く：`カラオケ`

### 文法详细总结

#### 1. `お + Vます形 + です`

- 用法：状态动词、往来发着动词的尊敬表达。  
- 接续：`お + Vます形 + です`（部分动词用特殊敬语）  
- 例句：  
  - 社長は今、応接室でお待ちです。  
  - 部長はもうお帰りですか。

#### 2. `～てもかまわない`

- 用法：表示许可，书面感比 `～てもいい` 略正式。  
- 接续：`Vて + もかまわない`、`いAくて + もかまわない`、`なA/Nで + もかまわない`  
- 例句：  
  - この席に座ってもかまいません。  
  - 提出は明日でもかまいません。

#### 3. `AはBほど～ない / AはBほどではない`

- 用法：比较否定，“A不如B那么……”。  
- 接续：`B + ほど + 否定`、`B + ほどではない`  
- 例句：  
  - この町は東京ほど人が多くない。  
  - 思っていたほど難しくはないです。

#### 4. `XほどYはない / いない`

- 用法：强调“没有比X更Y的”，主观最上级。  
- 接续：`N + ほど + 形容词/名词 + はない/いない`  
- 例句：  
  - 春の京都ほど美しい景色はない。  
  - 彼ほど日本語を勉強している学生はいない。

#### 5. `～ため[に]、～ / ～ためだ`（原因・理由）

- 用法：较书面的原因理由表达。  
- 接续：`普通形 + ため(に)`、`Nの + ため(に)`、`なAな + ため(に)`  
- 例句：  
  - 大雨のため、試合は中止になった。  
  - 渋滞したために、会議に遅れました。

#### 6. `～たら / ～ば、～た`（反事実）

- 用法：与事实相反的假设或遗憾。  
- 接续：`Vたら`、`Vば` 后接过去结果句  
- 例句：  
  - もっと早く出発していれば、間に合った。  
  - 地図を持っていたら、迷わなかったのに。

### 本课易错点

1. `お～です` 误用于自称动作。  
2. `ほど` 比较句里把肯定否定写反。  
3. `ため` 与 `から` 的语体差不区分。

### 本课复习清单

- 能把普通问句改成 `お～です` 的尊敬问法。  
- 能分别写出 `ほど～ない` 和 `ほど～はない` 对比句。  
- 能写3句反事实句（`～たら/～ば、～た`）。

## 第10课

### 本课核心目标

第10课主线是“高确信判断 + 结果表达 + 复合动词”。  
这一课之后，你可以更准确表达“按理说会……/不可能……/结果是……”。

### 会话与读写主题

- 話す・聞く：`そんなはずはありません`
- 読む・書く：`記憶型と注意型`

### 文法详细总结

#### 1. `～はずだ`

- 用法：基于逻辑、信息、经验作出高确信推断。  
- 接续：`普通形 + はずだ`、`なAな + はずだ`、`Nの + はずだ`  
- 例句：  
  - 田中さんは真面目だから、約束を守るはずだ。  
  - もう発送したので、明日には届くはずです。

#### 2. `～はずがない / ～はずはない`

- 用法：强否定推断，“不可能……”。  
- 接续：`普通形 + はずがない/はずはない`、`なAな`、`Nの` 同样接续  
- 例句：  
  - あの人がうそをつくはずがない。  
  - こんな大雨で、試合があるはずはない。

#### 3. `～はずだった`

- 用法：原本预期如此，但实际未实现。  
- 接续：`普通形 + はずだった`、`なAな`、`Nの` 同样接续  
- 例句：  
  - 今日の面談は10時のはずだった。  
  - 海外出張に行くはずだったが、延期になった。

#### 4. `～ことがある / ～こともある`

- 用法：表示“有时会……”。  
- 接续：`V辞書形/ない形 + ことがある/こともある`  
- 例句：  
  - 忙しいと、昼ご飯を食べないことがある。  
  - この路線は時々遅れることもあります。

#### 5. `～た結果 / ～の結果、～`

- 用法：强调经过后的客观结果。  
- 接续：`Vた + 結果`、`Nの + 結果`  
- 例句：  
  - 十分に話し合った結果、計画を見直すことにした。  
  - 検査の結果、異常はありませんでした。

#### 6. `～出す`（复合动词）

- 用法：动作、状态开始并突然出现。  
- 接续：`Vます形 + 出す`  
- 例句：  
  - 空が急に暗くなり、雨が降り出した。  
  - その話を聞いて、彼は急に笑い出した。

#### 7. `～始める / ～終わる / ～続ける`

- 用法：表示动作阶段（开始/结束/持续）。  
- 接续：`Vます形 + 始める/終わる/続ける`  
- 例句：  
  - 九時から会議が始まります。  
  - 彼は三時間ずっと話し続けた。

#### 8. `～忘れる / ～合う / ～換える`

- 用法：复合动词表达“忘做”“互相做”“更换做法/对象”。  
- 接续：`Vます形 + 忘れる/合う/換える`  
- 例句：  
  - 鍵をかけ忘れて出かけてしまった。  
  - 困ったときはチームで助け合う。

### 本课易错点

1. `はずだ` 和 `でしょう` 语气强度混用。  
2. `～た結果` 后项写成主观意图句。  
3. 复合动词前项变形错误（未用 `ます形`）。

### 本课复习清单

- 能区分 `はずだ / はずがない / はずだった`。  
- 能用 `～た結果` 写2句客观结果。  
- 能用4个复合动词造句：`降り出す/話し続ける/買い換える/言い忘れる`。

## 第11课

### 本课核心目标

第11课是“变化推移 + 比较 + らしい + ず形”的综合课。  
重点在区分 `らしい` 的两种功能，以及掌握书面连接 `～ず`。

### 会话与读写主题

- 話す・聞く：`お勧めのところ、ありませんか`
- 読む・書く：`白川郷の黄金伝説`

### 文法详细总结

#### 1. `～てくる / ～ていく`（变更）

- 用法：以现在为基准描述变化方向。  
  - `てくる`：变化累积到现在。  
  - `ていく`：从现在继续往后变化。  
- 接续：`Vて形 + くる/いく`  
- 例句：  
  - 最近、気温が下がってきた。  
  - これから物価は上がっていくと思う。

#### 2. `～たら[どう]？`

- 用法：提出建议，比 `～ほうがいい` 更柔和。  
- 接续：`Vた形 + ら(どう)`  
- 例句：  
  - 疲れているなら、少し休んだらどう？  
  - 分からなかったら、先生に聞いたらどうですか。

#### 3. `AよりBのほうが～`

- 用法：A、B比较，表示B更……。  
- 接续：`Aより + Bのほうが + 评价`  
- 例句：  
  - 電車よりバスのほうが安い。  
  - 見るより自分でやるほうが覚えやすい。

#### 4. `～らしい`（典型性质）

- 用法：具有某类事物典型特征。  
- 接续：`N + らしい`  
- 例句：  
  - 今日は春らしい天気ですね。  
  - 彼は学生らしい素直さがある。

#### 5. `～らしい`（传闻・推量）

- 用法：根据听闻信息或间接线索推测。  
- 接续：`普通形 + らしい`  
- 例句：  
  - 天気予報によると、明日は雪らしい。  
  - 田中さんはもう帰ったらしいです。

#### 6. `～として`

- 用法：表示立场、资格、观点评价。  
- 接续：`N + として`  
- 例句：  
  - 留学生として日本に来ました。  
  - 彼は研究者として有名です。

#### 7. `～ず[に]`（付带状況・手段）

- 用法：不做前项而做后项；书面语感。  
- 接续：`Vない形去ない + ず(に)`（`する -> せず`）  
- 例句：  
  - 朝ご飯を食べずに学校へ行った。  
  - メモを見ずに発表できました。

#### 8. `～ず、～`（并列・原因）

- 用法：书面并列或理由连接，后项多为结果、评价。  
- 接续：`Vない形去ない + ず、～`  
- 例句：  
  - 彼は謝らず、部屋を出ていった。  
  - 努力せず、成功だけを望むのは難しい。

#### 9. `～ている`（経験・経歴）

- 用法：表示经历、身份背景持续状态。  
- 接续：`Vている`  
- 例句：  
  - 彼は十年日本で働いている。  
  - 私は大学で経済学を学んでいます。

### 本课易错点

1. 两种 `らしい`（典型/传闻）语义不分。  
2. `～ず` 和 `～ないで` 语体混用。  
3. `てくる/ていく` 与第6课“移动方向”混淆。

### 本课复习清单

- 能分别用 `らしい` 写“性质句”和“传闻句”。  
- 能把 `～ないで` 改写成 `～ずに`。  
- 能用 `～として` 写3句“立场/身份”表达。

## 第12课

### 本课核心目标

第12课是全册收束课：理由表达深化、间接受身、状态持续与评价句型。  
语气上更贴近真实生活中的“解释、抱怨、道歉”。

### 会话与读写主题

- 話す・聞く：`ご迷惑をかけてすみませんでした`
- 読む・書く：`〖座談会〗日本で暮らす`

### 文法详细总结

#### 1. `～ものだから / ～もんだから`

- 用法：带解释、辩解语气的理由表达。`もんだから` 更口语。  
- 接续：`普通形 + もの/もんだから`、`なA/Nな + もの/もんだから`  
- 例句：  
  - 電車が遅れたものですから、遅刻しました。  
  - 今日は忙しいもんだから、また今度ね。

#### 2. `～(ら)れる`（間接受身・自動詞）

- 用法：因他人或外界动作而受到不利影响。  
- 接续：自动词受身形  
- 例句：  
  - 急に雨に降られて、服がぬれた。  
  - 友達に先に行かれて、道が分からなくなった。

#### 3. `～(ら)れる`（間接受身・他動詞）

- 用法：他人对别处对象的动作，间接给说话人带来困扰。  
- 接续：他动词受身形  
- 例句：  
  - 隣の部屋で夜中まで騒がれて、眠れなかった。  
  - 目の前でたばこを吸われて、気分が悪くなった。

#### 4. `～たり～たり`（交替・重复）

- 用法：A、B动作或状态反复、交替出现。  
- 接续：`Vたり`、`いAかったり`、`なA/Nだったり`  
- 例句：  
  - 最近、寒かったり暖かかったりしている。  
  - 忙しい日は食べたり食べなかったりだ。

#### 5. `～っぱなし`

- 用法：动作后状态一直放着不管，多带负面语感。  
- 接续：`Vます形去ます + っぱなし`  
- 例句：  
  - ドアを開けっぱなしにしないでください。  
  - テレビをつけっぱなしで寝てしまった。

#### 6. `～おかげで / ～おかげだ`

- 用法：积极结果的原因，“多亏……”。  
- 接续：`普通形 + おかげで/おかげだ`、`Nの + おかげで/おかげだ`  
- 例句：  
  - 先生のおかげで、試験に合格できた。  
  - 毎日練習したおかげで、発表がうまくいった。

#### 7. `～せいで / ～せいだ`

- 用法：消极结果的原因，“都怪……”。  
- 接续：`普通形 + せいで/せいだ`、`Nの + せいで/せいだ`  
- 例句：  
  - 渋滞のせいで、約束に遅れた。  
  - 寝不足のせいで、今日は集中できない。

### 本课易错点

1. `おかげ`（正向）和 `せい`（负向）混用。  
2. `ものだから` 用于纯客观说明句不自然。  
3. 間接受身里受害者视角没写出来。

### 本课复习清单

- 能区分并改写 `から/ので` 到 `ものだから`。  
- 能各写2句 `おかげ`、`せい` 的对照句。  
- 能用 `～っぱなし`、`間接受身` 描述日常“被影响”场景。

## 全书复习建议（第1课-第12课）

1. 按功能复习而不是只按课次复习：请求、比较、推量、受身、复合动词、因果。  
2. 每个文法至少做到“1条定义 + 2个自造句 + 1个易错点”。  
3. 对第9-12课优先做“语气对比”练习：`てもいい/てもかまわない`、`から/ので/ものだから`、`おかげ/せい`、`はず/らしい`。
