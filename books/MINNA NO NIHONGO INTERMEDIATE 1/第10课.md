# 第10课

### 本课核心目标

第10课主线是“高确信判断 + 结果表达 + 复合动词”。  
这一课之后，你可以更准确表达“按理说会……/不可能……/结果是……”。

### 会话与读写主题

- 話す・聞く：`そんなはずはありません`
- 読む・書く：`記憶型と注意型`

### 文法详细总结

#### 1. `～はずだ`

- 用法：基于逻辑、信息、经验作出高确信推断。  
- 接续：`普通形 + はずだ`、`なAな + はずだ`、`Nの + はずだ`  
- 例句：田中さんは真面目だから、約束を守るはずだ。
- 翻译：田中先生是认真的，所以他应该遵守诺言。

#### 2. `～はずがない / ～はずはない`

- 用法：强否定推断，“不可能……”。  
- 接续：`普通形 + はずがない/はずはない`、`なAな`、`Nの` 同样接续  
- 例句：あの人がうそをつくはずがない。
- 翻译：那个人不可能说谎。

#### 3. `～はずだった`

- 用法：原本预期如此，但实际未实现。  
- 接续：`普通形 + はずだった`、`なAな`、`Nの` 同样接续  
- 例句：今日の面談は10時のはずだった。
- 翻译：今天的面试本来是十点钟的。

#### 4. `～ことがある / ～こともある`

- 用法：表示“有时会……”。  
- 接续：`V辞書形/ない形 + ことがある/こともある`  
- 例句：忙しいと、昼ご飯を食べないことがある。
- 翻译：当我很忙的时候，我有时不吃午饭。

#### 5. `～た結果 / ～の結果、～`

- 用法：强调经过后的客观结果。  
- 接续：`Vた + 結果`、`Nの + 結果`  
- 例句：十分に話し合った結果、計画を見直すことにした。
- 翻译：经过充分讨论，我们决定修改我们的计划。

#### 6. `～出す`（复合动词）

- 用法：动作、状态开始并突然出现。  
- 接续：`Vます形 + 出す`  
- 例句：空が急に暗くなり、雨が降り出した。
- 翻译：天色突然暗下来，下起了雨。

#### 7. `～始める / ～終わる / ～続ける`

- 用法：表示动作阶段（开始/结束/持续）。  
- 接续：`Vます形 + 始める/終わる/続ける`  
- 例句：九時から会議が始まります。
- 翻译：会议将于九点开始。

#### 8. `～忘れる / ～合う / ～換える`

- 用法：复合动词表达“忘做”“互相做”“更换做法/对象”。  
- 接续：`Vます形 + 忘れる/合う/換える`  
- 例句：鍵をかけ忘れて出かけてしまった。
- 翻译：我忘记锁门就出去了。

### 本课易错点

1. `はずだ` 和 `でしょう` 语气强度混用。  
2. `～た結果` 后项写成主观意图句。  
3. 复合动词前项变形错误（未用 `ます形`）。

### 本课复习清单

- 能区分 `はずだ / はずがない / はずだった`。  
- 能用 `～た結果` 写2句客观结果。  
- 能用4个复合动词造句：`降り出す/話し続ける/買い換える/言い忘れる`。
