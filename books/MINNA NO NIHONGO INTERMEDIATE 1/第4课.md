# 第4课

### 本课核心目标

第4课强调“信息转述 + 口语缩约 + 书面语体 + 情感表达”。  
学完后你会更容易看懂真实会话里的缩略形式，也能更自然地表达“听说、好像、看起来”。

### 会话与读写主题

- 話す・聞く：`伝言、お願いできますか`
- 読む・書く：`電話嫌い`

### 文法详细总结

#### 1. `・・・ということだ`

- 用法：转述消息、总结判断，“据说……/也就是说……”。  
- 接续：`普通形 + ということだ`  
- 例句：天気予報によると、台風は明日上陸するということだ。
- 翻译：据天气预报，台风将于明天登陆。

#### 2. `・・・の / ・・・の？`

- 用法：口语中省略后项，`の？` 常用于确认、询问理由。  
- 接续：`普通形 + の`（句末升调可作疑问）  
- 例句：どうしてそんなに急いでいるの？
- 翻译：你为什么这么急？

#### 3. `～ちゃう / ～とく / ～てる`

- 用法：口语缩约。  
- 接续：  
  - `～てしまう -> ～ちゃう`  
  - `～ておく -> ～とく`  
  - `～ている -> ～てる`  
- 例句：財布を忘れちゃった。
- 翻译：我忘了带钱包。

#### 4. `～(さ)せられる / ～される`（使役受身）

- 用法：被迫做某事，带有不情愿语感。  
- 接续：使役受身形（五段/一段动词各自变形）  
- 例句：子どものころ、毎日ピアノを練習させられた。
- 翻译：当我还是个孩子的时候，我每天都被要求练习钢琴。

#### 5. `～である`

- 用法：书面体判断句，比 `です` 更正式、客观。  
- 接续：`名词/な形容词语干 + である`  
- 例句：日本語教育において、評価基準は明確であるべきだ。
- 翻译：在日语教育中，评价标准应该明确。

#### 6. 中止形 `～ます、～ます、・・・ / ～くて、～くて、・・・`

- 用法：并列动作、状态，不强行断句。  
- 接续：  
  - 动词 `ます形` 直接并列  
  - い形容词 `くて` 并列  
- 例句：朝起きて、顔を洗い、朝ごはんを食べ、家を出る。
- 翻译：早上起床，洗脸，吃早饭，出门。

#### 7. `～(た)がる`

- 用法：第三人称“表现出想……”。  
- 接续：`Vます形去ます + たがる`  
- 例句：子どもは新しいゲームをやりたがっている。
- 翻译：孩子们想玩新游戏。

#### 8. `～(た)がっている`

- 用法：持续状态地显示愿望。  
- 接续：`Vます形去ます + たがっている`  
- 例句：妹はずっと留学したがっている。
- 翻译：我姐姐一直想出国留学。

#### 9. `・・・こと / ・・・ということ`

- 用法：把句子内容抽象成“这件事/这个事实”。  
- 接续：`普通形 + こと / ということ`  
- 例句：約束を守ることは、信頼につながる。
- 翻译：信守承诺才能赢得信任。

### 本课易错点

1. `たい` 和 `たがる` 混用。  
2. 缩约形式直接用于正式邮件。  
3. `ということだ` 的语义不分。

### 本课复习清单

- 能把正式句改写成口语缩约句。  
- 能区分 `たい`、`たがる`、`たがっている`。  
- 能用 `である` 写出2句书面风格句子。
