# 第8课

### 本课核心目标

第8课是“时间边界 + 状态保持 + 原因说明”的整合课。  
重点在区分 `あいだ / あいだに`、`まで / までに`，并能自然表达“保持某状态去做下一动作”。

### 会话与读写主题

- 話す・聞く：`迷子になっちゃったんです`
- 読む・書く：`科学者ってどう見える？`

### 文法详细总结

#### 1. `～あいだ`

- 用法：在X持续期间，Y也持续进行。  
- 接续：`Vている + あいだ`、`Nの + あいだ`  
- 例句：電車に乗っているあいだ、ずっと本を読んでいた。
- 翻译：我在火车上一直在看书。

#### 2. `～あいだに`

- 用法：在X持续期间内，Y这一事件发生。  
- 接续：`Vている + あいだに`、`Nの + あいだに`  
- 例句：料理しているあいだに、友達が来た。
- 翻译：当我做饭的时候，一个朋友过来了。

#### 3. `～まで`

- 用法：表示动作/状态持续到某终点。  
- 接续：`N + まで`、`V辞書形 + まで`  
- 例句：会議が終わるまで、ここで待っていてください。
- 翻译：请在这里等候，直到会议结束。

#### 4. `～までに`

- 用法：表示截止时点，“在……之前完成”。  
- 接续：`N + までに`、`V辞書形 + までに`  
- 例句：明日の朝までにレポートを送ってください。
- 翻译：请在明天早上之前发送报告。

#### 5. `～た + 名词`（名词修饰）

- 用法：表示动作完成后的结果状态修饰名词。  
- 接续：`Vた形 + 名词`  
- 例句：眼鏡をかけた人が受付にいます。
- 翻译：接待处有一个戴眼镜的人。

#### 6. `～によって・・・`

- 用法：按类别不同而变化，“因……而异”。  
- 接续：`N + によって`  
- 例句：仕事の進め方は会社によって違う。
- 翻译：工作方式因公司而异。

#### 7. `～たまま / ～のまま`

- 用法：保持某状态不变去做后项动作。  
- 接续：`Vた形 + まま`、`Nの + まま`  
- 例句：窓を開けたまま寝てしまった。
- 翻译：我开着窗户睡着了。

#### 8. `・・・からだ`（原因・理由）

- 用法：明确陈述原因、判断依据，常用于较正式说明。  
- 接续：`普通形 + からだ`  
- 例句：私が日本語を学ぶのは、日本で働きたいからだ。
- 翻译：我学习日语是因为我想在日本工作。

### 本课易错点

1. `あいだ` 和 `あいだに` 都翻成“期间”而误用。  
2. `まで` 和 `までに` 混用。  
3. `たまま` 的前项动词选错。

### 本课复习清单

- 能各写3句 `あいだ`/`あいだに` 对比句。  
- 能各写3句 `まで`/`までに` 对比句。  
- 能用 `からだ` 写出正式的“结论+理由”句子。
