# 第7课

### 本课核心目标

第7课集中训练“义务、容许、限制、情绪语气、感情使役”。  
这一课的口语色彩更强，尤其是 `かな`、`なんか`、`なんて` 的语感处理。

### 会话与读写主题

- 話す・聞く：`楽しみにしています・遠慮させてください`
- 読む・書く：`まんじゅう、怖い`

### 文法详细总结

#### 1. `～なくてはならない / いけない`、`～なくてもかまわない`

- 用法：  
  - `なくてはならない/いけない`：必须。  
  - `なくてもかまわない`：不做也可以。  
- 接续：`Vない形去ない + くては + ならない/いけない`  
- 例句：明日は朝七時までに来なくてはならない。
- 翻译：我明天早上七点必须来。

#### 2. `～なくちゃ / ～なきゃ（いけない）`

- 用法：口语化“必须……”。  
- 接续：`Vない形去ない + なくちゃ/なきゃ`  
- 例句：もう行かなきゃ。
- 翻译：我现在得走了。

#### 3. `・・・だけだ / [ただ]・・・だけでいい`

- 用法：限制范围，“只要……就行”。  
- 接续：`普通形 + だけだ`、`普通形 + だけでいい`  
- 例句：必要なところだけ読めばいい。
- 翻译：只需阅读您需要的内容即可。

#### 4. `・・・かな`（终助词）

- 用法：自问、缓和断定、表达不确定。  
- 接续：句末直接加 `かな`  
- 例句：明日、雨かな。
- 翻译：我想知道明天会不会下雨。

#### 5. `～なんか・・・`

- 用法：轻视、谦逊、弱化某对象。  
- 接续：`名词/普通形 + なんか`  
- 例句：私なんか、まだまだです。
- 翻译：我还有很长的路要走。

#### 6. `・・・なんて・・・`

- 用法：惊讶、评价、轻视，语气比 `なんか` 更主观。  
- 接续：`普通形/名词 + なんて`  
- 例句：一人で富士山に登るなんて、すごいですね。
- 翻译：独自攀登富士山真是太棒了。

#### 7. `～(さ)せる`（感情使役）

- 用法：某事物“使（某人）产生某种感受”。  
- 接续：`使役形`，常见搭配 `安心させる/感動させる` 等。  
- 例句：そのニュースは家族を安心させた。
- 翻译：这个消息让家人松了口气。

#### 8. `～(さ)せられる / ～される`（感情的使役受身）

- 用法：被某事“弄得……”，表达受影响感。  
- 接续：`使役受身形`  
- 例句：彼の一言には考えさせられた。
- 翻译：他的话引起了我的思考。

#### 9. `・・・なら、・・・`

- 用法：基于前项条件给建议、判断、对应措施。  
- 接续：`普通形/名词 + なら`  
- 例句：時間がないなら、要点だけ先に確認しよう。
- 翻译：如果你没有时间，请先检查要点。

### 本课易错点

1. `なくてはならない` 与 `なければならない` 书写混乱。  
2. `なんか/なんて` 用在正式文体导致语气不当。  
3. 感情使役里主语宾语方向写反。

### 本课复习清单

- 能按正式度排序：`なくてはならない -> なければならない -> なきゃ`。  
- 能写出2句 `なんか` 和2句 `なんて` 并解释语感。  
- 能区分 `～(さ)せる`（使别人）与 `～(さ)せられる`（我被影响）。
