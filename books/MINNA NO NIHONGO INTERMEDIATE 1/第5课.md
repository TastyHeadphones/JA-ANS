# 第5课

### 本课核心目标

第5课主打“文脉指示 + 推测语气 + 路线表达”。  
这课在真实会话里非常实用，尤其是问路、确认信息、给建议时的自然说法。

### 会话与读写主题

- 話す・聞く：`どう行ったらいいでしょうか`
- 読む・書く：`地図`

### 文法详细总结

#### 1. `あ～ / そ～`（会话中的文脉指示）

- 用法：在会话里指代已提到的信息。  
  - `あ`：说话人和听话人都知道。  
  - `そ`：一方知道、另一方未必知道。  
- 接续：`あの/あれ/あそこ`、`その/それ/そこ` 等。  
- 例句：あの人、昨日の発表ですごく緊張していたね。
- 翻译：那个人对于昨天的宣布实在是太紧张了。

#### 2. `そ～`（文章中的文脉指示）

- 用法：在后句指代前句已出现的信息。  
- 接续：`その + 名词`、`そこで`、`それについて` 等。  
- 例句：来月北海道へ行きます。そのときに雪祭りを見たいです。
- 翻译：下个月要去北海道。我很想去看看那个时候的冰雪节。

#### 3. `・・・んじゃない？`

- 用法：口语推测、建议、轻度提醒。  
- 接续：`普通形 + んじゃない？`（名词/な形容词接 `な`）  
- 例句：彼、今日来ないんじゃない？
- 翻译：他今天不来吗？

#### 4. `～たところに / ～たところで`

- 用法：表示“刚好在……时点”。  
- 接续：`Vた形 + ところに/で`  
- 例句：出かけたところに、友達から電話がかかってきた。
- 翻译：正当我准备离开的时候，我接到了朋友的电话。

#### 5. `～(よ)うとする`

- 用法：正要做某动作、尝试去做。  
- 接续：`意向形 + とする`  
- 例句：電車に乗ろうとしたとき、財布がないことに気づいた。
- 翻译：当我准备上火车时，我发现我没有带钱包。

#### 6. `～(よ)うとする / しない`

- 用法：表示主观意图的有无（想做/不想做）。  
- 接续：`意向形 + とする / としない`  
- 例句：彼は自分の失敗を認めようとしない。
- 翻译：他拒绝承认自己的失败。

#### 7. `・・・のだろうか`

- 用法：带思考感的疑问与推量（“到底会……吗”）。  
- 接续：`普通形 + のだろうか`  
- 例句：この方法で本当に間に合うのだろうか。
- 翻译：这个方法真的够用吗？

#### 8. `～との / での / からの / までの / への + 名词`

- 用法：把助词短语变成定语，连接名词。  
- 接续：`名词 + 助词 + の + 名词`  
- 例句：駅からの道を地図に書いてください。
- 翻译：请在地图上画出从车站出发的路线。

#### 9. `・・・だろう / ・・・だろうと思う`

- 用法：推量、判断，语气比 `と思う` 更明确。  
- 接续：`普通形 + だろう`  
- 例句：明日は雨になるだろう。
- 翻译：明天会下雨。

### 本课易错点

1. 会话中的 `あ`/`そ` 指示关系判断反了。  
2. `んじゃない？` 在正式场合使用过多。  
3. `～との～` 等结构里漏掉 `の`。

### 本课复习清单

- 能解释 `あ` 和 `そ` 在会话里的知识共享关系。  
- 能用 `～(よ)うとする` 说“正要做但被打断”。  
- 能用 `からの/への` 做路线和对象说明句。
