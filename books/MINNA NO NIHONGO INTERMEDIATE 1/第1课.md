# 第1课

### 本课核心目标

第1课的重点是把“请求、比喻、定义、让步”说得更像中级日语。  
核心变化是：同样是“请帮我做某事”，会根据对象和场合换更礼貌的表达。

### 会话与读写主题

- 話す・聞く：`お願いがあるんですが`
- 読む・書く：`畳`

### 文法详细总结

#### 1. `Vて + もらえませんか / いただけませんか / もらえないでしょうか / いただけないでしょうか`

- 用法：委婉请求对方做某事，四个形式礼貌程度递增。  
- 接续：`Vて形 + 目标表达`  
- 例句：駅までの行き方を教えていただけないでしょうか。
- 翻译：你能告诉我怎么去车站吗？

#### 2. `Nのようだ / NのようなN / NのようにV・A`

- 用法：比喻、举例，表示“像……一样”。  
- 接续：`N + のようだ`、`N + のような + 名词`、`N + のように + 动词/形容词`  
- 例句：彼はロボットのように正確に作業する。
- 翻译：他的工作就像机器人一样精准。

#### 3. `V辞書形 + ことは / ことが / ことを`

- 用法：把动词名词化后作为句子成分（主题/主语/宾语）。  
- 接续：`V辞書形 + こと + 助词`  
- 例句：毎日声に出して読むことは、発音の練習になる。
- 翻译：每天大声朗读将帮助您练习发音。

#### 4. `NをNと言う`

- 用法：定义、命名，“把A叫做B”。  
- 接续：`AをBと言う`  
- 例句：この料理を「お好み焼き」と言います。
- 翻译：这道菜被称为“御好烧”。

#### 5. `～という + 名词`

- 用法：说明名称、内容、属性，相当于“叫做……的”“所谓……的”。  
- 接续：`普通形/名词 + という + 名词`  
- 例句：昨日、「和紙」という素材について勉強した。
- 翻译：昨天，我了解到了一种叫做“和纸”的材料。

#### 6. `いつ / どこ / 何 / だれ / どの / どんなに + ～ても`

- 用法：让步表达，“无论……都……”。  
- 接续：疑问词直接接 `～ても`（动词/形容词按句意变形）  
- 例句：どんなに忙しくても、朝ごはんは食べる。
- 翻译：不管多忙，我都会吃早餐。

### 本课易错点

1. 把 `～てもらえませんか` 当成命令句使用。  
2. `～のようだ` 和 `～みたいだ` 混用场景不分。  
3. `疑问词 + ても` 后面没写完整结论句。

### 本课复习清单

- 能按对象切换四档请求表达。  
- 能用 `のようだ / のような / のように` 各造一句。  
- 能用 `疑问词 + ても` 表达让步。
