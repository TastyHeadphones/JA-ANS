# 第3课

### 本课核心目标

第3课围绕“许可、决定、愿望、样态判断”展开。  
这是从初级句型走向中级表达的关键一课，尤其是 `ことにする` / `ことになる` 的主客观区分。

### 会话与读写主题

- 話す・聞く：`遅れそうなんです`
- 読む・書く：`時間よ、止まれ！`

### 文法详细总结

#### 1. `～(さ)せてもらえませんか / ～(さ)せていただけませんか`

- 用法：请求对方允许“我去做某事”。  
- 接续：`使役形て + もらう/いただく`  
- 例句：明日の会議で、最初に発表させていただけませんか。
- 翻译：我可以第一个出席明天的会议吗？

#### 2. `～ことにする`

- 用法：说话人主动做决定。  
- 接续：`V辞書形/ない形 + ことにする`  
- 例句：健康のために、毎朝30分歩くことにした。
- 翻译：为了健康，我决定每天早上步行30分钟。

#### 3. `～ことにしている`

- 用法：持续性的个人规则、习惯。  
- 接续：`V辞書形/ない形 + ことにしている`  
- 例句：寝る前にスマホを見ないことにしている。
- 翻译：我尽量在睡觉前不看智能手机。

#### 4. `～ことになる`

- 用法：结果上被决定、变成某种状态（非个人单方决定）。  
- 接续：`V辞書形/ない形 + ことになる`  
- 例句：来月から大阪支社へ転勤することになった。
- 翻译：下个月我将被调到大阪分公司。

#### 5. `～ことになっている`

- 用法：制度、规定、既定安排。  
- 接续：`V辞書形/ない形 + ことになっている`  
- 例句：この図書館では飲食してはいけないことになっている。
- 翻译：图书馆内禁止饮食。

#### 6. `～てほしい / ～ないでほしい`

- 用法：希望别人做/不要做某事。  
- 接续：`Vて + ほしい`、`Vないで + ほしい`  
- 例句：明日は早めに来てほしい。
- 翻译：我希望你明天早点来。

#### 7. `～そうな + N / ～そうに + V`

- 用法：根据外观判断“看起来要……”。  
- 接续：`动词ます形去ます + そう`、`い形容词词干 + そう`  
- 例句：雨が降りそうな空だ。
- 翻译：天空看起来要下雨了。

#### 8. `～なさそう`

- 用法：看起来不……（样态否定）。  
- 接续：`动词ます形去ます + なさそう`、`い形容词去い + なさそう`  
- 例句：彼は今日は来なさそうだ。
- 翻译：看来他今天不会来了。

#### 9. `～そうもない`

- 用法：强否定样态，“怎么看都不太可能……”。  
- 接续：`动词ます形去ます + そうもない`  
- 例句：今夜中には終わりそうもない。
- 翻译：看来今晚不会完结了。

### 本课易错点

1. `ことにする` 与 `ことになる` 混用。  
2. `そう`（样态）和传闻 `そう` 混用。  
3. `～てほしい` 主语误用。

### 本课复习清单

- 能口头解释 `ことにする` 和 `ことになる` 的差别。  
- 能用 `～そうな / ～そうに / ～なさそう / ～そうもない` 各造一句。  
- 能在请假、请求许可场景中自然说出 `～(さ)せていただけませんか`。
