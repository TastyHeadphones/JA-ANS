# 复习43-50（逐字人工精校）

## 语法要点速览

| 语法 | 用法 |
|---|---|
| 推测表达 | `そう`（样态/传闻）、`よう`（判断）、`はず`（有依据推断） |
| 敬语 | 尊敬语、自谦语、郑重表达的互换 |
| 使役与授受 | `使役形`、`〜ていただく`、`〜させる` |
| 助词综合 | `を/に/と/で/の/が` 的句法位置 |

## 3. 形式变换（1-14）

| 题号 | 题干（核心） | 答案 | 解析 |
|---|---|---|---|
| 3-1 | 地震が（ありました→　）そうです。 | あった | 简体过去用于传闻`そうだ`。 |
| 3-2 | （歩きました→　）すぎて、足が痛いです。 | 歩き | `Vます去ます+すぎる`。 |
| 3-3 | 白い服は（汚れます→　）やすいです。 | 汚れ | `Vます去ます+やすい`。 |
| 3-4 | 二人は（楽しいです→　）そうです／この間（結婚しました→　）ばかり。 | 楽しそうで、結婚した | 样态`そう`；`Vたばかり`。 |
| 3-5 | 字が小さくて（読みます→　）にくい。 | 読み | `Vます去ます+にくい`。 |
| 3-6 | 料理は（冷たいです→　）してし上がってください。 | 冷たく | い形容词连用形。 |
| 3-7 | 飲み物を買いに（行きます→　）来ます。 | 行って | `Vて来る`。 |
| 3-8 | 会社を（休みます→　）場合は電話してください。 | 休む | 条件句前接辞书形。 |
| 3-9 | 日本語の勉強を（始めます→　）ばかり。 | 始めた | `Vた+ばかり`。 |
| 3-10 | ちょうど今から（食事します→　）ところです。 | 食事する | `V辞書形+ところ`。 |
| 3-11 | 今日本に（いません→　）はずです。 | いない | `ない形+はず`。 |
| 3-12 | 事故が（ありました→　）ようです。 | あった | 普通形接`ようだ`。 |
| 3-13 | この仕事を（やります→　）いただけませんか。 | やらせて | `〜させていただく`。 |
| 3-14 | 学生に日曜日のことを（話します→　）。 | 話させる | 使役形。 |
