# 第45课（逐字人工精校）

## 语法要点速览

| 语法 | 用法 |
|---|---|
| 〜場合は | 条件场景说明 |
| 〜のに | 逆接（与预期相反） |
| 〜ので | 原因理由（客观、缓和） |

## 1. 「〜場合は」前项普通形

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 1-1 | （雨です…）場合は、野球の練習はありません。 | 雨の | 名词接「場合」用「の」。 |
| 1-2 | なかなか熱が（下がりません…）場合は、この薬を飲んでください。 | 下がらない | 动词否定普通形。 |
| 1-3 | 体の調子が（悪いです…）場合は、キャンプに参加しないでください。 | 悪い | い形容词普通形。 |
| 1-4 | 花火大会が（中止になりました…）場合は、来週行います。 | 中止になった | 动词过去普通形。 |
| 1-5 | 参加する人が（30人以上です…）場合は、安くなります。 | 30人以上の | 名词修饰「場合」。 |
| 1-6 | 予約を（キャンセルしたいです…）場合は、連絡してください。 | キャンセルしたい | い形容词型愿望普通形。 |
| 1-7 | 資料が（必要です…）場合は、自分でコピーしてください。 | 必要な | な形容词+な+場合。 |
| 1-8 | 保証書が（ありません…）場合は、修理代がかかります。 | ない | ない形普通体。 |

## 2. 词语选择填空（場合）

| 题号 | 题目（核心） | 答案 | 解析 |
|---|---|---|---|
| 2-1 | 電車にかばんを（　）場合は、どうしたらいいですか。 | 忘れた | 「忘れた場合」。 |
| 2-2 | 書き方を（　）場合は、新しい紙に書いてください。 | まちがえた | 「写错的情况」。 |
| 2-3 | 途中でコピーを（　）場合は、ここを押します。 | やめたい | 「想中止」场景。 |
| 2-4 | レポートの締め切りに（　）場合は、どうしたらいいですか。 | 間に合わない | 截止日期赶不上的情况。 |
| 2-5 | 今週、部長の都合が（　）場合は、会議は来週にしましょう。 | 悪い | 「都合が悪い」。 |
| 2-6 | 修理が（　）場合は、新しいのを買いましょう。 | 無理な | な形容词接续。 |
| 2-7 | 何かを（　）場合は、まずインターネットを使います。 | 調べる | 动词辞书形。 |
| 2-8 | （　）場合は、現金を持って行かないほうがいいです。 | 海外旅行の | 名词修饰「場合」。 |

## 3. 改写（用「〜のに」连接）

| 题号 | 题目（核心） | 标准答案 | 解析 |
|---|---|---|---|
| 3-1 | キャンプの準備をしていました。雨で急に中止になりました。 | キャンプの準備をしていたのに、雨で急に中止になりました。 | 与预期相反。 |
| 3-2 | もう12月です。暖かい日が続いています。 | もう12月なのに、暖かい日が続いています。 | 季节预期逆接。 |
| 3-3 | 父は歌が下手です。よくカラオケに行きます。 | 父は歌が下手なのに、よくカラオケに行きます。 | 性质与行为反差。 |
| 3-4 | 4月になりました。まだ桜が咲いていません。 | 4月になったのに、まだ桜が咲いていません。 | 时间与结果不一致。 |
| 3-5 | 雨が降っています。彼は釣りに行きました。 | 雨が降っているのに、彼は釣りに行きました。 | 条件不利仍行动。 |
| 3-6 | きのうは日曜日でした。会社へ行かなければなりませんでした。 | きのうは日曜日だったのに、会社へ行かなければなりませんでした。 | 休息日却要上班。 |
| 3-7 | このマンションは新しいです。よくエレベーターが故障します。 | このマンションは新しいのに、よくエレベーターが故障します。 | 新与故障形成反差。 |
| 3-8 | 楽しみにしていました。病気で旅行に行けませんでした。 | 楽しみにしていたのに、病気で旅行に行けませんでした。 | 心理预期落空。 |
| 3-9 | きょうはそんなに寒くないです。あの人は厚いコートを着ています。 | きょうはそんなに寒くないのに、あの人は厚いコートを着ています。 | 客观温度与穿着反差。 |

## 4. 「ので／のに」选择

| 题号 | 题目 | 答案 | 解析 |
|---|---|---|---|
| 4-1 | バスがなかなか来なかった（　）、遅くなってしまいました。 | ので | 原因→结果。 |
| 4-2 | 書類にはんこが必要だった（　）、押さないで出してしまいました。 | のに | 与应有做法相反。 |
| 4-3 | もう11時を過ぎた（　）、電話をかけないほうがいいです。 | ので | 客观理由。 |
| 4-4 | 財布をなくしてしまった（　）、友達にお金を借りました。 | ので | 自然因果。 |
| 4-5 | 1時間も待った（　）、友達が来なかった（　）、帰って来ました。 | のに、ので | 前半逆接，后半因果。 |
