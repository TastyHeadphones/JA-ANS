# 第34课（逐字人工精校）

## 1. 助词填空

| 题号 | 答案 | 解析 |
|---|---|---|
| 1-1 | に | 「〜に通っています」。 |
| 1-2 | を | 宾语标记。 |
| 1-3 | の／の | 连体修饰。 |
| 1-4 | か／の／を／に | 疑问+修饰+宾语+着落点。 |

## 2. 「〜とおりに」

| 题号 | 标准答案 | 解析 |
|---|---|---|
| 2-1 | 先生が言ったとおりに、書きました。 | 按对方所说去做。 |
| 2-2 | この図のとおりに、いすと机を並べてください。 | 按图示操作。 |
| 2-3 | 赤い線のとおりに、紙を折ってください。 | 按标线步骤。 |
| 2-4 | 佐藤さんが説明したとおりに、やってください。 | 按说明执行。 |
| 2-5 | ミラーさんに教えてもらったとおりに、ケーキを作りました。 | 按学到的方法重现。 |

## 3. 「〜たあとで」

| 题号 | 答案 | 解析 |
|---|---|---|
| 3-1 | 食事の | 名词化后接「あとの」。 |
| 3-2 | 出した | 「試験を出したあとで…」。 |
| 3-3 | 講義の | 名词修饰结构。 |
| 3-4 | した | 「練習をしたあとで…」。 |
| 3-5 | 帰った | 动作先后。 |
| 3-6 | 結婚式の | 名词化后接「あとで」。 |

## 4. て形填空

| 题号 | 答案 | 解析 |
|---|---|---|
| 4-1 | はって | 张贴动作连接。 |
| 4-2 | かぶって | 穿戴类动作连接。 |
| 4-3 | 入れて | 放入动作连接。 |
| 4-4 | 書いて | 书写动作连接。 |
| 4-5 | 着て、して | 两个连续动作。 |

## 5. 「〜ないで／〜て」

| 题号 | 答案 | 解析 |
|---|---|---|
| 5-1 | 入れて | 先放入再处理。 |
| 5-2 | 持たないで | “不要带着”。 |
| 5-3 | 消さないで | “不要关掉/擦掉”。 |
| 5-4 | 予約して | 先预约再行动。 |

## 6. 指示句（ないで）

| 题号 | 答案 | 解析 |
|---|---|---|
| 6-1 | 行かないで | 否定请求。 |
| 6-2 | 乗らないで | 交通手段否定。 |
| 6-3 | 買わないで | 避免多余购买。 |
| 6-4 | 帰らないで | 请求对方别先回去。 |
| 6-5 | 休まないで | 语气上是“别休息”。 |
| 6-6 | 決めないで | 先别下结论。 |

## 7. 连接形式选择

| 题号 | 答案 | 解析 |
|---|---|---|
| 7-1 | 座って | 动作接续。 |
| 7-2 | かけながら | 同时进行“戴着并…”。 |
| 7-3 | かけて | 顺接动作。 |
| 7-4 | 押しながら | 一边按一边转。 |
| 7-5 | 待たないで | 条件句中的否定请求。 |
| 7-6 | 急いで | 表方式。 |
| 7-7 | 入らないで | “没进浴缸就睡着”。 |
| 7-8 | 乗って、降りると | 路线说明固定搭配。 |
