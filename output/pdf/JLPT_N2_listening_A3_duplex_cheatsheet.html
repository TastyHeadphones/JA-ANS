JLPT N2 聴解ちょうかい - A3 正面

N2 听力重点总览：题型、信号词、陷阱、临场流程

官方框架：N2 听力约 50 分钟；题型为 5 类。
题数以当次试卷为准，听力时间也可能随录音长度略有差异。

1. 全场总原则

  • 先抓题目问什么：人、事、理由、时间、态度、下一步。
  • 不要“听到词就选”。JLPT 听力常把所有选项都提到，答案在最终判断。
  • 日语结论常在句尾；否定、转折、意志、婉拒都可能藏在最后。
  • 主语省略时先问：谁说的？谁要做？是本人、对方、第三者还是公司？
  • 每听到转折或更改，立刻刷新笔记，旧信息降级为干扰。

核心口诀：听“最后怎么定”，不是听“中间出现过什么”。

2. 临场 20 秒预读法

  • 有选项时先圈差异：动作、对象、时间、地点、数量、说话人。
  • 把疑问词写在题旁：为什么、哪一个、先做什么、谁的意见。
  • 选项太长时只记关键词，不翻译整句。
  • 无选项题先听首句和尾句，转折后优先。
  • 不回放训练：错了也继续，保持下一题节奏。

常见节奏：问题说明 - 预读 - 音声 - 设问 - 作答。具体以考场播放为准。

3. 答案雷达

反转
でもしかしただ実じつはやっぱり
真实理由
本当ほんとうは正直しょうじきに言いうとというのはだって
顺序
まず先さきにその前まえに〜てから後あとで
最终决定
結局けっきょく最終的さいしゅうてきにじゃあ〜にする〜ことになった

4. 排除雷达

已完成
もう〜た済すんだ終おわった
不用做
〜なくてもいい〜は要いらない〜しないで
别人做
私わたしがやるこちらで対応たいおうする担当たんとうが連絡れんらくする
计划更改
〜じゃなくて〜に変更へんこうやっぱり〜で

听到这些词后，选项里的旧动作、旧数字、旧负责人要重新判断。

1

問題もんだい1 課題理解かだいりかい

目标：解决具体任务，判断接下来最合适的动作。

核心问“这之后先做什么”“必须做什么”。看动作是否还没做、是否由题目中的人来做。
听法边听边删：已完成、别人负责、不用做、之后才做。抓 まず、先さきに、その前まえに。
陷阱常先说 A 后改 B；或 A 很重要但要 後あとで 才做。
何なにをしますかしなければなりません〜ておいて〜次第しだいとりあえず
2

問題もんだい2 ポイント理解りかい

目标：带着指定问题听，抓一个重点。

核心预读时圈问题点：理由、时间、心情、评价、最终选择。
听法理由题听 〜から、〜ので、というのは；时间数量题听更改和合计；心情题听评价词。
陷阱“表面理由”后可能有 本当ほんとうは 或 実じつは 推翻。
理由りゆう気持きもち評価ひょうか変更へんこう合わせあわせて
3

問題もんだい3 概要理解がいようりかい

目标：抓整体主旨、说话人观点和态度。

核心不要追所有细节。首句定主题，转折后定观点，尾句定结论。
听法听 大切たいせつなのは、必要ひつようがある、〜べきだ、つまり。
陷阱选项只覆盖例子或细节时通常不对；主旨题要选能统摄全文的说法。
何なにについて言いいたいこと考かんがえ要ようするに一方いっぽうで
4

問題もんだい4 即時応答そくじおうとう

目标：短句后选最自然回应，重点是话语功能。

核心先判断对方在请求、许可、邀请、确认、感谢、道歉、抱怨、后悔还是提醒。
听法抓句尾：〜てもらえますか、〜でしょうか、〜んですが、〜っけ、〜ばよかった。
陷阱回应语法通顺但意图不对会错；敬语场景要选礼貌层级一致的回答。
承知しょうちしました大丈夫だいじょうぶです残念ざんねんです助たすかります
5

問題もんだい5 統合理解とうごうりかい

目标：整合长信息，比较多条件，常要分别判断两个人。

核心信息播报时画 A/B/C/D 小表；讨论时给男、女或甲、乙分别记最终选择。
听法优先记约束：〜は外はずせない、预算、时间、距离、条件优先级。
陷阱两人可能选不同方案；私わたしは〜、あなたは？ 后要分开记。
比較ひかく条件じょうけん優先ゆうせんやっぱり〜にしよう

5. 典型错因 - 对照排查

听到就选把被提到的词当答案，没有等最终决定。训练：每题写“最后一句关键转折”。
主语错日语省略主语，尊敬语和自谦语没有听出谁做。训练：每轮对话标男、女、店员、老师、公司。
否定错没听完整句尾：〜わけではない、〜とは限かぎらない、〜なくてもいい。
数字错旧数字后有 やっぱり、変更へんこう、合わせあわせて、〜を除のぞいて。
语气错把 ちょっと…、検討けんとうします 当真同意；实际多是婉拒或保留。

6. 三步笔记模板

  • 任务题：动作 - 负责人 - 顺序 - 是否已完成。
  • 理由题：表面理由 - 转折 - 真实理由。
  • 综合题：方案 - 优点 - 缺点 - 谁最后选。
A: 条件/优缺点
B: 条件/优缺点
男/甲: 最后选择
女/乙: 最后选择

作答前 1 秒：这是不是问“先做”？是不是问“谁”？是不是问“理由”？选项是否只是出现过而非结论？

JLPT N2 聴解ちょうかい - A3 反面

高频表达库：缩约、敬语、即答、数字、潜台词

打印建议：A3 横向，双面，长边翻转。
日文汉字已用 HTML ruby 标注读音。

1. 口语缩约 - 秒还原

〜ちゃう / 〜じゃう〜てしまう：完了、遗憾、失误
〜とく〜ておく：预先做
〜てる / 〜でる〜ている：正在、状态
〜なきゃ〜なければならない：必须
〜なくちゃ〜なくてはならない：必须
〜ちゃだめ〜てはだめ：不可以
〜たって〜ても：即使
〜だって〜でも：也、即使
〜とこ〜ところ：正要、地方
〜っけ确认记忆：是……来着？
〜じゃん〜じゃない：不是……吗
〜かも〜かもしれない：也许

听力里很多答案不是词汇难，而是缩约没还原。

2. 敬语判断谁做

行いく / 来くる / いるいらっしゃる = 对方；伺うかがう / 参まいる = 我方
言いうおっしゃる = 对方；申もうす = 我方
見みるご覧らんになる = 对方；拝見はいけんする = 我方
するなさる = 对方；いたす = 我方
食たべる / 飲のむ召めし上あがる = 对方；いただく = 我方
知しっているご存ぞんじです = 对方；存ぞんじております = 我方

判断法：尊敬语多是对方、上级、客人动作；自谦语多是自己或己方动作。

3. 商务固定句

承知しょうちしました明白了，照办。
かしこまりました更郑重的“明白了”。
少々しょうしょうお待まちください请稍等。
お手数てすうですが麻烦您。
恐おそれ入いりますが不好意思，请问/请求前置。
〜していただけますか能否请您……
あいにく不巧，很遗憾地。
申もうし訳わけございません非常抱歉。

4. 数字时间陷阱

四時よじ不是 よんじ
七時しちじ容易听成 一時いちじ，靠语境校正
九時くじ不是 きゅうじ
一分いっぷん / 三分さんぷん注意促音和 ぷん。
六分ろっぷん / 八分はっぷん / 十分じゅっぷん常考变音。
一日ついたち / 二十日はつか日期特殊读法。
一人ひとり / 二人ふたり人数特殊读法。

数字后若听到“更改、合计、除外”，答案通常不是第一个数字。

5. 暧昧表达 = 潜台词

ちょっと…多半是不方便、婉拒。
考かんがえておきます先考虑；考试里常是保留或不积极。
検討けんとうします研究一下；不等于答应。
善処ぜんしょします会尽量处理；承诺度弱。
また今度こんど这次不去/不做。
都合つごうがつけば条件允许才行，不确定。
気持きもちはありがたいんですが感谢但大概率拒绝。
うーん、どうかな不太赞成。
まあ、いいんじゃない同意但热度低。

6. 即答功能 - 自然回应

请求承知しょうちしました / やっておきます / すみません、今いまちょっと…
许可どうぞ / かまいません / それはちょっと…
邀请いいですね / また今度こんどお願ねがいします
感谢いえいえ / とんでもないです / お役やくに立たてて何なによりです
道歉大丈夫だいじょうぶです / 気きにしないでください / こちらこそ
提醒/批评すみません、気きをつけます / 確認かくにんします
后悔仕方しかたないよ / 次つぎは気きをつけよう

7. 句尾语法速听

〜ざるを得えない不得不；不是想做。
〜ないこともない不是不能，弱肯定。
〜に限かぎらず不限于。
〜に沿そって按照、沿着。
〜ところだった差点发生，但没发生。
〜ばよかった后悔：早知道就……
〜わけではない并不是……
〜とは限かぎらない未必……
〜たところ刚刚做完。

8. 情绪评价词

正向
うれしいほっとした助たすかった満足まんぞく楽たのしみ
负向
がっかり残念ざんねん困こまった不安ふあん後悔こうかい
微妙
複雑ふくざつまあまあ微妙びみょうなんとも言いえない

观点题和心情题优先听评价词，不要只抓事实名词。

9. 连接词逻辑

追加

对比

总结

例示

10. 考前 15 分钟

  • 读一遍缩约形和即答功能，不背新词。
  • 听一段自然日语，让耳朵进入日语频道。
  • 快速复盘五类题：先做什么、指定点、主旨、自然回应、综合比较。
  • 提醒自己：不要因上一题失误影响下一题。
  • 最后检查铅笔、准考证、耳朵状态；考场只听一次，保持节奏。

最容易多拿分的顺序：即時応答そくじおうとう 功能判断 - 課題理解かだいりかい 排除法 - ポイント理解りかい 真实理由 - 統合理解とうごうりかい 小表格。

11. 练习时的错题记录表

错因分类最终决定 / 主语人物 / 否定句尾 / 数字更改 / 敬语角色 / 婉拒语气 / 主旨过窄。
复盘方式只重听答案前后 20 秒，写下导致答案的信号词，不整篇泛听。
跟读重点跟读包含 でも、やっぱり、実じつは、結局けっきょく 的句子。
目标从“听懂大概”变成“听出出题点”。

12. 最后确认清单

  • 我是否等到句尾？
  • 我是否确认了动作是谁做？
  • 我是否把“出现过”的选项和“答案”分开了？
  • 我是否注意到转折、变更、婉拒和合计？
  • 主旨题是否选了能概括全文的答案，而不是一个例子？
