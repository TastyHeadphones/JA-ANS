N2 听力正面：题型规则 + 完整原题式练习

A3 横向，双面长边翻转。无装饰框，正常分栏排版；日文汉字用 ruby 标假名。

0. N2听力先抓什么

  • 先判题型：下一步、指定点、主旨、即时回应、综合选择。
  • 所有选项都可能出现，答案不是最早出现的词，而是最终决定、真实理由、最后条件。
  • 下一步题优先听 まず / 先さきに / その前まえに / 〜ておいて。
  • 理由题优先听 本当ほんとうは / それより / 〜んですが 后半。
  • 数字题旧信息先写下，听到 変更へんこう / ではなく / やっぱり 后立刻更新。
删除もう〜た / 〜なくていい / 私わたしがやる / ではなく
保留必要ひつよう / 外はずせない / 先さきに / 決きめました
主旨不要选单个例子，选能概括转折后结论的选项。
复盘每错一题只写一个漏听信号词，不整套重刷。

A1. 課題理解かだいりかい｜会場準備かいじょうじゅんび・まず何なにをするか

設問せつもん：男おとこの学生がくせいはこのあとまず何なにをしますか。

音声原文おんせいげんぶん：女おんな：ポスターはもう貼はってくれたんだよね。ありがとう。
男おとこ：はい。他ほかにやることはありますか。
女おんな：受付うけつけの名簿めいぼを印刷いんさつしてほしいんだけど、その前まえに参加者さんかしゃの人数にんずうを確認かくにんして。資料しりょうのコピーは田中たなかさんがやってくれるから。
男おとこ：わかりました。

選択肢せんたくし：

1ポスターを貼はる
2人数にんずうを確認かくにんする
3名簿めいぼを印刷いんさつする
4資料しりょうをコピーする

正解せいかい：2

全選択肢解析ぜんせんたくしかいせき：

1もう貼はってくれた 表示已经完成，排除。
2その前まえに 后面的动作才是第一步。
3名簿打印是在确认人数之后。
4田中たなかさん 负责复印资料，不是男学生本人。

A2. ポイント理解りかい｜本当ほんとうの理由りゆう

設問せつもん：男おとこの人ひとはどうして講座こうざを変かえますか。

音声原文おんせいげんぶん：女おんな：夜よるの講座こうざ、続つづけるんですか。
男おとこ：仕事しごとが忙いそがしいから無理むりかなと思おもっていたんですが、本当ほんとうは内容ないようが少すこし簡単かんたんすぎるんです。先生せんせいに相談そうだんしたら、上級じょうきゅうクラスを勧すすめられました。

選択肢せんたくし：

1仕事しごとが忙いそがしいから
2授業料じゅぎょうりょうが高たかいから
3内容ないようが簡単かんたんすぎるから
4先生せんせいが変かわったから

正解せいかい：3

全選択肢解析ぜんせんたくしかいせき：

1忙いそがしい 是表面理由；本当ほんとうは 后面才是真理由。
2听力中没有提到学费。
3与真实理由完全一致。
4老师只是咨询对象，不是换课理由。

A3. 概要理解がいようりかい｜全体ぜんたいの主張しゅちょう

設問せつもん：話はなし手てが一番いちばん言いいたいことは何なんですか。

音声原文おんせいげんぶん：新あたらしい道具どうぐを使つかうと、作業さぎょうは速はやくなります。ただ、道具どうぐに任まかせるだけでは、問題もんだいが起おきたときに対応たいおうできません。便利べんりなものほど、使つかう人ひとが仕組しくみを理解りかいしておくことが大切たいせつです。

選択肢せんたくし：

1新あたらしい道具どうぐは使つかわないほうがいい
2道具どうぐを使つかえば問題もんだいはなくなる
3便利べんりな道具どうぐほど理解りかいして使つかうべきだ
4作業さぎょうを速はやくすることが最優先さいゆうせんだ

正解せいかい：3

全選択肢解析ぜんせんたくしかいせき：

1说话人没有否定工具本身。
2原文反而说可能出现问题。
3大切たいせつです 前后的结论一致。
4速度只是引入，主张是理解原理后使用。

A4. 課題理解かだいりかい｜修正作業しゅうせいさぎょうの順番じゅんばん

設問せつもん：女おんなの人ひとはこのあとまず何なにをしますか。

音声原文おんせいげんぶん：男おとこ：明日あしたの案内文あんないぶん、もう送おくった？
女おんな：まだです。地図ちずを入いれてから送おくろうと思おもって。
男おとこ：地図ちずは古ふるいから使つかわないで。先さきに集合時間しゅうごうじかんを直なおして、そのあと私わたしに見みせて。送信そうしんは私わたしが確認かくにんしてからでいいよ。

選択肢せんたくし：

1案内文あんないぶんを送おくる
2地図ちずを入いれる
3集合時間しゅうごうじかんを直なおす
4男おとこの人ひとに確認かくにんする

正解せいかい：3

全選択肢解析ぜんせんたくしかいせき：

1发送要等男方确认之后。
2地图是旧的，明确不要使用。
3先さきに 后面是第一动作。
4让男方看是修正之后的步骤。

A5. ポイント理解りかい｜苦情くじょうの理由りゆう

設問せつもん：客きゃくは何なにに一番いちばん困こまっていますか。

音声原文おんせいげんぶん：店員てんいん：先日せんじつの配送はいそうの件けんですね。遅おくれて申し訳もうしわけありません。
客きゃく：遅おくれたことも困こまりましたが、それより箱はこが破やぶれていて中なかの商品しょうひんが濡ぬれていたんです。交換こうかんできますか。
店員てんいん：すぐ確認かくにんいたします。

選択肢せんたくし：

1配送はいそうが遅おくれたこと
2箱はこが破やぶれて商品しょうひんが濡ぬれたこと
3商品しょうひんの値段ねだんが高たかかったこと
4店員てんいんの説明せつめいが長ながかったこと

正解せいかい：2

全選択肢解析ぜんせんたくしかいせき：

1遅おくれたことも 表示这只是次要信息。
2それより 后面是更重要的抱怨点。
3价格没有出现在听力中。
4说明时间长短没有出现在听力中。

A6. 概要理解がいようりかい｜働はたらき方かたの主張しゅちょう

設問せつもん：話はなし手ての考かんがえに最もっとも近ちかいものはどれですか。

音声原文おんせいげんぶん：在宅勤務ざいたくきんむは通勤時間つうきんじかんを減へらせる一方いっぽうで、同僚どうりょうとの雑談ざつだんが少すくなくなります。雑談ざつだんは無駄むだに見みえるかもしれませんが、問題もんだいを早はやく見みつける助たすけになることがあります。効率こうりつだけで働はたらき方かたを決きめないことが大切たいせつです。

選択肢せんたくし：

1在宅勤務ざいたくきんむはやめるべきだ
2雑談ざつだんは仕事しごとに不要ふようだ
3効率こうりつ以外いがいの面めんも考かんがえるべきだ
4通勤時間つうきんじかんを長ながくするべきだ

正解せいかい：3

全選択肢解析ぜんせんたくしかいせき：

1没有完全否定居家办公。
2原文说闲聊也有帮助。
3効率こうりつだけで 表示不能只看效率，和结论一致。
4不是建议延长通勤时间。

A7. ポイント理解りかい｜数字変更すうじへんこう

設問せつもん：会議かいぎは何時なんじからになりましたか。

音声原文おんせいげんぶん：男おとこ：会議かいぎは三時さんじからでしたよね。
女おんな：はい。でも部長ぶちょうの予定よていが変かわって、三時半さんじはんではなく四時よじからになりました。場所ばしょは同おなじです。

選択肢せんたくし：

1三時さんじ
2三時半さんじはん
3四時よじ
4場所ばしょは未定みてい

正解せいかい：3

全選択肢解析ぜんせんたくしかいせき：

1最初の確認で、最終情報ではない。
2三時半さんじはんではなく で否定。
3変更後の最終時刻。
4場所は同じなので未定ではない。

1B. 課題理解 - 追加速刷

听到什么就怎么删

受付うけつけはもう済すみました。次つぎに資料しりょうを配くばってください。答：資料しりょうを配くばる
解：もう済すみました 排除受付。
私わたしが予約よやくを取とりますから、田中たなかさんは人数にんずうを数かぞえてください。答：人数にんずうを数かぞえる
解：问田中さん时，予約是别人做。
雨あめなら外そとの看板かんばんは出ださなくていいです。入口いりぐちの紙かみだけ替かえてください。答：入口いりぐちの紙かみを替かえる
解：〜なくていい 是不用做。
部品ぶひんが届とどくまで修理しゅうりはできません。先さきに報告書ほうこくしょを書かきましょう。答：報告書ほうこくしょを書かく
解：〜までできません 表示修理暂时不可。
店長てんちょうへの確認かくにんは後あとでいいので、先さきに棚たなを整理せいりしてください。答：棚たなを整理せいりする
解：先さきに 直接指定第一步。
名札なふだは足たりています。不足ふそくしているのは椅子いすなので、倉庫そうこから出だしてください。答：椅子いすを出だす
解：选项里名札出现了，但不是问题。

2B. ポイント理解 - 追加速刷

问什么听什么

参加さんかしたかったんですが、熱ねつが出でてしまって。答：熱ねつが出でたから
解：问理由时，愿望不是理由，病因才是。
駅えきから近ちかいのはいいんですが、家賃やちんが予算よさんを超こえています。答：家賃やちんが高たかい
解：〜んですが 后常给真正问题。
去年きょねんより来場者らいじょうしゃは増ふえましたが、売上うりあげは少すこし下さがりました。答：売上うりあげは下さがった
解：问売上时，不要被来场者增加带偏。
最初さいしょはバスのつもりでしたが、荷物にもつが多おおいのでタクシーにします。答：タクシー
解：最初さいしょは 后面常反转。
説明せつめいは分わかりやすかったものの、実例じつれいが少すくないと感かんじました。答：実例じつれいが少すくない
解：ものの 前好后坏，评价重点在后。
今月こんげつは十件じゅっけん、そのうち新規しんきは三件さんけんです。答：三件さんけん
解：问新规数量时，十件是总数干扰。

3B. 概要理解 - 主旨排除

主旨题不要选例子

本ほんを読よむと、知識ちしきだけでなく想像力そうぞうりょくも育そだちます。答：读书的多方面价值
解：知识只是一个例子，主旨要更大。
町まちの店みせは商品しょうひんを売うるだけでなく、人ひとが集あつまる場所ばしょにもなっています。答：本地小店的社会作用
解：だけでなく 后扩展主题。
道具どうぐが便利べんりになるほど、使つかう側がわの判断はんだんが大切たいせつになります。答：便利工具也需要使用者判断
解：不是介绍道具种类。
失敗しっぱいを避さけるより、小ちいさく試ためして早はやく直なおすことが重要じゅうようです。答：先试再改的重要性
解：重要じゅうようです 收结论。

4B. 临考信号词补充

やっぱりAにします。答：最后选A。解：推翻前面的犹豫。
Aでもいいんですが、Bのほうが安心あんしんです。答：B。解：后半评价更强。

N2 听力反面：即时回应 + 综合理解练习

题目为原创仿真题：题干、听力原文、所有选项、正解、逐项解析。

0. 反面速记：即时回应和综合题

请求〜てもらえませんか - 接受就说 見みておきます / 承知しょうちしました
确认〜でしたっけ - 回答事实，不要换主语。
婉拒すみません + 理由 + 難むずかしいです 最自然。
综合先列 A/B/C 条件，再圈硬条件：时间、距离、预算、必须功能。
价格注意 学生がくせい / 会員かいいん / 引びき 是否真的适用。

常见坑：听到熟词就选、男女主语反了、过去信息当成现在、把未定条件当成决定。

B1. 即時応答そくじおうとう｜確認かくにんへの返事へんじ

設問せつもん：次つぎの発話はつわへの返事へんじとして最もっとも自然しぜんなものを選えらんでください。

音声原文おんせいげんぶん：今日きょうの会議かいぎ、部長ぶちょうもいらっしゃるんでしたっけ。

選択肢せんたくし：

1はい、いらっしゃる予定よていです。
2はい、私わたしが伺うかがいます。
3いいえ、会議かいぎはありません。

正解せいかい：1

全選択肢解析ぜんせんたくしかいせき：

1いらっしゃる 是对上级是否来的自然确认回答。
2伺うかがう 表示自己去，主语不对。
3问题不是问会议是否存在。

B2. 統合理解とうごうりかい｜旅行りょこうプラン

設問せつもん：二人ふたりはどのプランを選えらびますか。

音声原文おんせいげんぶん：Aプランは安やすいですが駅えきから遠とおいです。Bプランは駅えきに近ちかく、朝食ちょうしょくが付ついていますが、少すこし高たかいです。Cプランは静しずかですが、お風呂ふろがありません。
男おとこ：朝あさは早はやいから、駅えきに近ちかいのは外はずせないね。
女おんな：そうだね。朝食ちょうしょくもあるならBにしよう。

選択肢せんたくし：

1A
2B
3C
4決きめていない

正解せいかい：2

全選択肢解析ぜんせんたくしかいせき：

1便宜但离车站远，不符合必要条件。
2駅えきに近ちかい 和 朝食ちょうしょく 两个条件都满足。
3没有浴室，也不是对话中的决定因素。
4Bにしよう 已经明确决定。

B3. 統合理解とうごうりかい｜男女だんじょで別々べつべつに選えらぶ

設問せつもん：男おとこの人ひとと女おんなの人ひとはそれぞれどれを選えらびますか。

音声原文おんせいげんぶん：案内あんない：午前ごぜんコースは初心者向しょしんしゃむけ、午後ごごコースは経験者向けいけんしゃむけ、夜よるコースは録画ろくがで復習ふくしゅうできます。
男おとこ：初はじめてだから午前ごぜんがいいな。
女おんな：私わたしは仕事しごとの後あとしか時間じかんがないし、録画ろくがも見みたいから夜よるにする。

選択肢せんたくし：

1男おとこ：午前ごぜん / 女おんな：夜よる
2男おとこ：午後ごご / 女おんな：夜よる
3男おとこ：午前ごぜん / 女おんな：午後ごご
4男おとこ：夜よる / 女おんな：午前ごぜん

正解せいかい：1

全選択肢解析ぜんせんたくしかいせき：

1男生是初学者选上午；女生下班后有时间且想看录像，所以选晚上。
2男生没有选择下午。
3女生选晚上，不是下午。
4男女选择被反过来了。

B4. 即時応答そくじおうとう｜依頼いらいを受うける

設問せつもん：次つぎの発話はつわへの返事へんじとして最もっとも自然しぜんなものを選えらんでください。

音声原文おんせいげんぶん：この資料しりょう、明日あしたまでに確認かくにんしてもらえませんか。

選択肢せんたくし：

1はい、今日中きょうじゅうに見みておきます。
2いいえ、明日あした確認かくにんしてもらいます。
3では、資料しりょうを作つくってください。

正解せいかい：1

全選択肢解析ぜんせんたくしかいせき：

1请求对方确认资料，接受时要说自己会看。
2变成让别人确认，回应方向不对。
3资料已经存在，不是要求制作资料。

B5. 統合理解とうごうりかい｜講座こうざ選えらび

設問せつもん：二人ふたりはどの講座こうざを選えらびますか。

音声原文おんせいげんぶん：A講座こうざは土曜どようの午前ごぜんで安やすいですが、発表はっぴょう練習れんしゅうがありません。B講座こうざは平日へいじつの夜よるで、発表はっぴょう練習れんしゅうがあります。C講座こうざは録画ろくがだけです。
男おとこ：発表はっぴょうが苦手にがてだから練習れんしゅうしたいな。
女おんな：週末しゅうまつは仕事しごとだから、平日へいじつの夜よるなら行いけるよ。
男おとこ：じゃあBだね。

選択肢せんたくし：

1A
2B
3C
4決きめていない

正解せいかい：2

全選択肢解析ぜんせんたくしかいせき：

1安いが発表練習がなく、週末条件も合わない。
2発表練習あり、平日夜で両方の条件に合う。
3録画だけで、練習したい条件に合わない。
4Bだね で決定済み。

B6. 統合理解とうごうりかい｜条件除外じょうけんじょがい

設問せつもん：女おんなの人ひとはどの商品しょうひんを買かいますか。

音声原文おんせいげんぶん：案内あんない：赤あかは軽かるいが防水ぼうすいではありません。青あおは防水ぼうすいですが重おもいです。白しろは軽かるくて防水ぼうすいですが、値段ねだんが高たかいです。
女おんな：毎日まいにち持もつから軽かるさは大事だいじ。でも雨あめの日ひにも使つかうので防水ぼうすいも必要ひつようです。少すこし高たかくても白しろにします。

選択肢せんたくし：

1赤あか
2青あお
3白しろ
4買かわない

正解せいかい：3

全選択肢解析ぜんせんたくしかいせき：

1軽いが防水ではない。
2防水だが重い。
3軽さと防水の両条件を満たし、本人が白にすると言う。
4買わないとは言っていない。

B7. 即時応答そくじおうとう｜断ことわり方かた

設問せつもん：次つぎの発話はつわへの返事へんじとして最もっとも自然しぜんなものを選えらんでください。

音声原文おんせいげんぶん：今晩こんばん、残業ざんぎょうをお願ねがいできる？

選択肢せんたくし：

1すみません、今日きょうは予定よていがあって難むずかしいです。
2はい、昨日きのうは休やすみでした。
