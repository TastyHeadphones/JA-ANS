# 第〇部 聴解结构与配速

### 問題1 課題理解（5题）
- **问**："接下来**首先**要做什么？"
- **陷阱**：会提到多件事，但只有一件是"接下来马上做"。注意「まず」「とりあえず」「その前に」。

### 問題2 ポイント理解（6题）
- **特点**：题目和选项**先给你读**（有阅读时间）。
- **策略**：利用读题时间**圈出疑问点**（理由？时间？谁？），听时专攻该点。

### 問題3 概要理解（5题）
- **问**：主旨/说话人意图。**没有选项预读**。
- **策略**：抓**整体话题**，别纠结单词。听完再看选项。

### 問題4 即時応答（12题）★最多分
- **特点**：一句话→选最自然的回应。语速快、节奏密。
- **策略**：抓**句首的功能词**（〜てくれる？=请求／〜たら？=建议／〜ば良かった=后悔）。听懂**说话意图**比听懂每个词重要。常考**敬语应答、寒暄、委婉拒绝**。

### 問題5 統合理解（4题）
- **特点**：长对话/广播 → 综合判断。最后两题常是一段独白后两人讨论。
- **策略**：**边听边记笔记**（选项A/B/C/D各自特征）。最后两人对话各自选了什么，要分别记清。

# 第一部 解题攻略 — 真题套路版

> 给"觉得听力难"的你：N2 听力难，**不是因为词不会，而是因为没抓住"出题点"和"反转点"**。
> 本攻略结合历年真题套路，逐题型拆解：**问什么 → 陷阱在哪 → 怎么听 → 必背表达**。

---

### 0. 先搞懂：N2 听力为什么"听得懂却选错"

| 真相 | 说明 | 对策 |
|---|---|---|
| **答案常被"推翻"** | 先说A，再说「でも」「やっぱり」改成B | 听到转折词，前面信息作废 |
| **干扰项都被提到过** | 4个选项原文几乎都念过 | 别"听到就选"，要听**最终结论** |
| **关键在句尾** | 日语谓语在最后，否定/意志/转折藏句尾 | 一句话听完整，别抢答 |
| **省略 & 暧昧** | 日本人不说死，用「ちょっと…」拒绝 | 学会读"言外之意" |
| **语速 + 缩约** | 口语缩约（〜ちゃう/〜とく/〜なきゃ） | 提前熟悉缩约形（见第6节） |

🎯 **核心心法**：**听"最后怎么定的"，不是听"中间说了什么"。**

### 1. 問題1 課題理解（5题）— "接下来先做什么"

#### 问法（必为这一类）
- 「このあと、まず何をしますか。」
- 「男の人はこれから何をしなければなりませんか。」

#### 真题套路 & 陷阱
1. **会列出多件事**（A要做、B也要做、C别人来做），只有**1件是"自己接下来马上做"**。
2. **顺序陷阱**：「その前に〜」「先に〜」会打乱时间顺序。
3. **甩锅陷阱**：「それは私がやっておく」→ 这件事**不是问的人**做，排除。
4. **完成陷阱**：「もう〜は終わった/やった」→ 已完成，排除。

#### 怎么听（关键词雷达）
| 听到这些 → | 含义 |
|---|---|
| `まず` / `とりあえず` / `最初に` | ✅ 这就是"先做的" |
| `その前に` | ⚠️ 顺序提前，真正第一步在这 |
| `〜ておいて` / `〜といて` | 拜托对方做（可能不是自己做） |
| `あ、でも` / `じゃ、やっぱり` | 🔄 计划改变，重听后半 |
| `もう〜た` | ❌ 已完成，排除 |

#### 解题动作
- 题卷上常有图/选项 → **边听边划掉**被否定的，剩最后一个。

---

### 2. 問題2 ポイント理解（6题）— "抓指定的那个点"

#### 特点（最重要！）
- **题目和选项会先读给你听**（有约20秒预读时间）→ **务必利用！**

#### 真题套路
- 问 **理由**（なぜ／どうして）、**时间**、**人物心情**、**具体某项**。
- 答案往往**不在开头**，藏在对话**中后段**，且常用**因果/转折**引出。

#### 怎么听（两步法）
1. **预读时间**：在题卷上**圈出疑问词**——是问"理由"还是"时间"还是"谁"？
   - 例：题目「女の人はどうして引っ越すことにしましたか」→ 只听**搬家的理由**，别的略过。
2. **专注捕捉**：听到与问题相关的句子立刻锁定。

#### 理由题信号词
| 听到 → | 这后面就是理由 |
|---|---|
| `〜から` / `〜ので` / `〜ため` | 直接理由 |
| `というのは〜から` | 解释理由 |
| `実は〜` | 引出真实原因（常是答案） |
| `〜んです` | 强调说明原因 |

⚠️ **陷阱**：会说一个"表面理由"，再用「でも本当は〜」给出真实理由 → **真实理由才是答案**。

---

### 3. 問題3 概要理解（5题）— "整体在讲什么"

#### 特点
- **没有选项预读**（题卷空白）→ 完全靠耳朵抓主旨。
- 问「何について話していますか」「言いたいことは何ですか」「この人の考えは？」

#### 真题套路
- 一段独白/演讲/介绍 → 考**主题 or 说话人态度**。
- **不考细节**，纠结单个词反而误事。

#### 怎么听
1. **抓反复出现的关键词**（主题）。
2. **抓态度词**：是赞成/反对/担心/推荐？
   - `〜べきだ` `〜たほうがいい` `〜のではないか` `〜と思います` `大切なのは〜`
3. 听完**再看选项**，选"最概括"的，**排除以偏概全**（只覆盖一个细节的选项是错的）。

💡 **技巧**：开头第一句和结尾一句通常点题，**首尾必抓**。

### 4. 問題4 即時応答（12题）★最多分・最易丢分

#### 特点
- 一句话（A说）→ 选**最自然的回应**（3选1）。**超短、超快、节奏密**。
- **没有上下文**，纯靠听懂**那一句的功能/意图**。

#### 真题高频场景 & 应答套路

##### ① 请求 → 答应/拒绝
| A 说 | 意图 | 自然回应 |
|---|---|---|
| `〜てもらえる？` `〜てくれない？` | 请求 | `いいよ` `もちろん` `ごめん、今ちょっと…` |
| `手伝ってくれると助かるんだけど` | 委婉请求 | `あ、やっておきます` |

##### ② 提议/建议 → 接受/婉拒
| A 说 | 意图 | 回应 |
|---|---|---|
| `〜たらどう？` `〜ば？` | 建议 | `そうですね、そうします` `考えてみます` |
| `〜ない？`（一緒に） | 邀约 | `いいね、行こう` `あいにく予定が…` |

##### ③ 后悔/感叹/责备
| A 说 | 意图 | 回应 |
|---|---|---|
| `〜ばよかった` | 后悔 | `次は気をつけよう` `仕方ないよ` |
| `〜なんて、信じられない` | 惊讶/责备 | `本当に申し訳ない` `そうなんだ` |
| `〜ておけばよかったのに` | 责备(你当初该…) | `すみません、忘れてました` |

##### ④ 确认/反问
| A 说 | 意图 | 回应 |
|---|---|---|
| `〜んじゃない？` | 推测确认 | `たぶんね` `そうかも` |
| `〜っけ？` | 记不清在确认 | `〜だよ` `〜じゃなかった？` |
| `〜わけがない` | 强烈否定 | `だよね` `そうとも限らないよ` |

##### ⑤ 敬语应答（职场场景必考）
| A 说 | 场景 | 回应 |
|---|---|---|
| `お先に失礼します` | 下班先走 | `お疲れさまでした` |
| `いただいてもよろしいですか` | 谦让 | `どうぞ、召し上がってください` |
| `〜ていただけますか` | 礼貌请求 | `かしこまりました` `承知しました` |
| `恐れ入りますが…` | 客气开场 | `はい、何でしょうか` |

#### 解题核心
> **抓句首的功能词 + 句尾的语气**，判断对方"想干嘛"，选符合"对话逻辑"的回应。
> **干扰项套路**：①同音词陷阱（听到相似词的回应）②答非所问③时态/主语错位。

### 5. 問題5 統合理解（4题）— 长对话综合判断

#### 结构（两种）
- **前2题**：一段长对话/说明 → 综合理解后回答。
- **后2题**：先一段**信息播报（列举多个选项A/B/C/D）**，再**两人讨论各自选哪个** → 分别问"男的选什么""女的选什么"。

#### 真题套路 & 难点
- **信息量大**，4个方案的特征容易混。
- 两人**各有偏好**，常说「私は〜がいい、でもあなたは？」「じゃあ私はこっち」。

#### 怎么听（必须记笔记！）
1. **播报阶段**：在题卷空白处速记 4 个方案的**关键特征**：
   ```
   A: 便宜/远
   B: 贵/近/有早餐
   C: 安静/没浴室
   D: 新/网评好
   ```
2. **讨论阶段**：分别标记 **男 → ？ / 女 → ？**
3. 注意**反转**：「最初はAがいいと思ったけど、やっぱりBにする」→ 以**最后决定**为准。

💡 后2题是**两道独立题**，男女选的可能**不同**，分别听清。

### 6. 必练：口语缩约形（听不懂的元凶）

| 缩约 | 还原 | 意思 | 例 |
|---|---|---|---|
| `〜ちゃう/じゃう` | 〜てしまう | 做完/糟了 | `忘れちゃった` |
| `〜とく` | 〜ておく | 预先做 | `買っとく` |
| `〜てる` | 〜ている | 正在/状态 | `何してる？` |
| `〜なきゃ` | 〜なければ | 必须 | `行かなきゃ` |
| `〜なくちゃ` | 〜なくては | 必须 | `やらなくちゃ` |
| `〜たら？` | 提议 | 要不要…？ | `休んだら？` |
| `〜って` | 〜と/〜という | 引用/听说 | `来ないって` |
| `〜じゃん` | 〜じゃない | 不是…吗 | `いいじゃん` |
| `〜っけ` | 确认记忆 | …来着？ | `明日だっけ？` |
| `〜とこ` | 〜ところ | 正要/地方 | `今行くとこ` |
| `すいません` | すみません | 抱歉/劳驾 | — |
| `〜ちゃって` | 〜てしまって | 不小心…导致 | `遅れちゃって` |

### 7. 暧昧表达 = 听力的"潜台词"（高频考点）

| 说法 | 表面 | 真实意图 |
|---|---|---|
| `ちょっと…` | 有点… | **委婉拒绝**（不行/不方便） |
| `考えておきます` | 我考虑下 | 多半**婉拒** |
| `善処します` | 妥善处理 | 敷衍，不一定做 |
| `〜かな…と思って` | 想着…吧 | 试探性提议/犹豫 |
| `〜と言えなくもない` | 也不是不能说… | 委婉肯定（部分赞同） |
| `どうかな` | 怎么样呢 | 暗示**不太赞成** |
| `気持ちはわかるけど` | 心情能理解但… | **要拒绝/反对**了 |
| `まあ、いいんじゃない` | 还行吧 | 不太热情的同意 |

### 8. 实战节奏 & 应急策略

#### 临场纪律
1. **每题听前先看选项/图**（問題1・2有，抓紧10-20秒）。
2. **果断选、不回头**：纠结上一题会漏掉下一题开头（致命）。
3. **記号速记**：理由打"理"，时间打"⏰"，转折打"↻"，别写完整句。
4. **没听清也必选**：凭语感 + 排除法蒙一个，**绝不空**。

#### 听到这些立刻提高警觉（90%是答案附近）
```
でも / しかし / やっぱり / 実は / 本当は   → 反转，答案在后
というのは / だって / 〜から            → 理由来了
まず / その前に / 先に                  → 順序题答案
結局 / 最終的に / じゃあ〜にする          → 最终决定
```

#### 考前一晚 & 当天热耳
- **今晚**：把第6节缩约形、第4节即時応答场景**朗读2遍**（出声，练耳）。
- **考前30分**：戴耳机随便听一段日语播客/动画，让耳朵进入"日语状态"，避免开考前几题反应不过来。

## 第一部分：通用底层能力（所有题型都靠这个）

### 1. 缩约形（听不懂的头号元凶）— 必须秒还原

| 听到 | = | 意思 | 例 |
|---|---|---|---|
| 〜ちゃう / 〜じゃう | 〜てしまう | 完了/糟了 | `食べちゃった`＝吃完了 |
| 〜とく | 〜ておく | 预先做 | `やっとくね`＝先做好 |
| 〜てる / 〜でる | 〜ている | 正在 | `知ってる`＝知道 |
| 〜なきゃ | 〜なければ(ならない) | 必须 | `帰らなきゃ`＝得回去 |
| 〜なくちゃ | 〜なくては | 必须 | `急がなくちゃ` |
| 〜たって | 〜ても | 即使 | `言ったって無駄`＝说了也没用 |
| 〜だって | 〜でも/也 | 也/即使 | `子どもだって分かる` |
| 〜ったら | 〜たら/强调 | 如果/真是 | `お父さんったら` |
| 〜きゃ | 〜ければ | 的话 | `早くしなきゃ` |
| 〜ちゃ | 〜ては | 不可以(后接ダメ) | `見ちゃダメ` |
| 〜とこ | 〜ところ | 正要/地方 | `今出るとこ` |
| 〜っぱなし | 放着不管 | — | `つけっぱなし` |
| っす | です | 随便的です | `いいっす` |
| 〜じゃん | 〜じゃない | 不是…吗 | `できるじゃん` |
| 〜かも | 〜かもしれない | 也许 | `雨かも` |
| 〜とかって | 〜とか | 之类 | `田中さんとかって` |

### 2. 数字・时间・数量（听力必考陷阱）

#### 易混音（一字之差全错）
| 易混 | 区别 |
|---|---|
| 1時(いちじ) / 7時(しちじ) | しち 易听成 いち，看语境 |
| 4時(よじ) ≠ よんじ | 4点读 **よじ** |
| 9時(くじ) ≠ きゅうじ | 9点读 **くじ** |
| 1分(いっぷん)/3分(さんぷん)/4分(よんぷん)/6分(ろっぷん)/8分(はっぷん)/10分(じゅっぷん) | ぷん/ふん 变音 |
| 20日(はつか) ≠ にじゅうにち | 20号读 **はつか** |
| 一人(ひとり)/二人(ふたり) | 特殊读法 |
| 一日(ついたち) | 1号 |

#### 数字反转陷阱（答案常被改）
听到数字别急着记，常有：
- `3時…あ、ごめん、4時に変更で` → 答案 **4時**
- `5個ください…やっぱり3個でいいや` → **3個**
- `本社が15名、支社が…全部で20名` → 要**加总**算
- 信号词：`〜じゃなくて` `やっぱり` `〜に変更` `合わせて` `全部で` `〜を除いて`

### 3. 助数词（量词）速听
```
〜枚(まい)纸/薄物　〜本(ほん/ぼん/ぽん)细长　〜冊(さつ)书
〜台(だい)机器/车　〜匹(ひき/ぴき)小动物　〜頭(とう)大动物
〜個(こ)小物　〜杯(はい/ばい/ぱい)杯装　〜軒(けん)房子
〜名(めい)人(正式)　〜泊(はく/ぱく)住宿天数
```

## 第二部分：問題1 課題理解 —「接下来先做什么」

### 必听的"动作指令"表达

| 听到 → | 含义 | 判定 |
|---|---|---|
| まず / 最初に / とりあえず | 首先 | ✅ 第一步在这 |
| その前に / 先に | 在那之前/先 | ⚠️ 真正第一步被提前了 |
| 〜ておいて / 〜といて | 你（先）做好 | 拜托对方做（看是不是问的人） |
| 〜てから | 做完…再 | 顺序信号 |
| あとは / 後で / 〜たら | 之后/然后 | ❌ 不是"先做"的 |
| 〜はもういい / 済んだ / 終わった | 已完成 | ❌ 排除 |
| それは私が(やる) / こっちでやる | 那个我来做 | ❌ 不是问的人做 |
| 〜なくてもいい / 〜は要らない | 不用做 | ❌ 排除 |

### 典型反转
> 「資料をコピーして…あ、コピーはいいや、もう部長がやってくれた。それより先に会議室の予約お願い」
> → 答案是**预约会议室**（コピー被否定+已完成，会议室因"先に"成为第一步）

## 第三部分：問題2 ポイント理解 —「指定的那个点」

### 理由信号（问"なぜ/どうして"时锁定）

| 听到 → | 作用 |
|---|---|
| 〜から / 〜ので / 〜ため(に) | 直接理由 |
| 〜んです / 〜のだ | 强调说明（常带理由）|
| というのは〜から / だって | 解释理由 |
| 実は… | 引出**真实**理由（常是答案）|
| 〜せいで / 〜おかげで | 负面/正面原因 |

### ⚠️ 表面理由 vs 真实理由
> 「忙しいから行けない…っていうのは建前で、本当はあの人が苦手なんだ」
> → 答案是**和那个人合不来**（本当は后面才是真理由）

信号：`本当は` `実は` `〜というより` `正直(に言うと)` `ぶっちゃけ`

### 心情・评价词（问"心情/感想"时）
```
正面：うれしい／ほっとした／助かった／ありがたい／満足／楽しみ
负面：がっかり／残念／困った／面倒／不安／後悔／いらいら／ショック
微妙：複雑／まあまあ／微妙／なんとも言えない／思ったほどでは…
```

## 第五部分：問題4 即時応答 ★最多分・最易丢分（12题）

> 一句话→选最自然回应。**听"意图"，不是听"单词"**。下面按"对方想干嘛"全分类。

### A. 请求・依赖 → 答应 / 婉拒
| 对方说 | 意图 | 自然回应 |
|---|---|---|
| 〜てもらえる？／〜てくれない？ | 请求 | `いいよ／もちろん` ／ `ごめん、今ちょっと…` |
| 〜てもらえると助かるんだけど | 委婉请求 | `あ、やっておきます` |
| 〜ていただけますか | 礼貌请求 | `かしこまりました／承知しました` |
| 悪いんだけど、〜 | 不好意思但…(请求) | `全然いいよ、何？` |
| 手が空いてたら〜 | 有空的话… | `今大丈夫ですよ` |

### B. 提议・劝诱 → 接受 / 拒绝
| 对方说 | 意图 | 回应 |
|---|---|---|
| 〜たら？／〜ばいいじゃない | 建议 | `そうだね、そうする／考えてみる` |
| 〜ない？（一緒に） | 邀约 | `いいね、行こう` ／ `あいにく予定が…` |
| 〜ことない？（そんな〜） | 反劝/没必要 | `そうかな…／確かに` |
| 〜たらどうですか | 提议(礼貌) | `そうしてみます` |

### C. 许可・确认 → 给/不给
| 对方说 | 意图 | 回应 |
|---|---|---|
| 〜てもいい？ | 求许可 | `どうぞ／うん、いいよ` ／ `あー、それはちょっと…` |
| 〜てもかまいませんか | 求许可(礼貌) | `ええ、かまいませんよ` |
| 〜んじゃない？ | 推测确认 | `たぶんね／そうかも` |
| 〜だっけ？／〜よね？ | 确认记忆 | `〜だよ／〜じゃなかった？` |

### D. 后悔・遗憾・责备
| 对方说 | 意图 | 回应 |
|---|---|---|
| 〜ばよかった | 后悔 | `次は気をつけよう／仕方ないよ` |
| 〜ておけばよかったのに | 责备(你当初该…) | `すみません、忘れてました` |
| 〜なんて(信じられない) | 惊讶/责备 | `本当にごめん／そうなんだ` |
| 〜ちゃって(困った) | 麻烦了 | `大丈夫？／どうしよう` |
| 〜とは思わなかった | 没想到 | `ね、びっくりだよね` |

### E. 感谢・道歉 → 应答（必背固定句）
| 对方说 | 回应 |
|---|---|
| 助かりました／ありがとうございます | `いえいえ／とんでもないです／お役に立てて何よりです` |
| 申し訳ありません／すみませんでした | `気にしないでください／大丈夫ですよ／こちらこそ` |
| お世話になりました | `こちらこそ／とんでもないです` |
| お疲れさまでした | `お疲れさまでした` |
| ご迷惑をおかけしました | `いえ、お気になさらず` |

### F. 寒暄・职场固定应答 ★高频
| 对方说 | 场景 | 回应 |
|---|---|---|
| お先に失礼します | 先下班 | `お疲れさまでした` |
| いただきます／召し上がってください | 吃饭 | `どうぞ／いただきます` |
| よろしくお願いします | 拜托 | `こちらこそ／承知しました` |
| 行ってきます | 出门 | `行ってらっしゃい` |
| ただいま | 回来 | `おかえりなさい` |
| 恐れ入りますが… | 客气开场 | `はい、何でしょう` |
| お邪魔します | 进门 | `どうぞ、お上がりください` |

### G. 暧昧拒绝（听出"言外之意"）★必考
| 对方说（表面） | 真实意图 |
|---|---|
| ちょっと…(ね) | **拒绝/不方便** |
| 考えておきます／検討します | 多半**婉拒** |
| また今度／そのうち | 委婉推掉 |
| 気持ちはありがたいんだけど… | **要拒绝**了 |
| う〜ん、どうかな… | 不太赞成 |
| 都合がつけば… | 不一定来 |

→ 对这些的自然回应：`そうですか、残念です／わかりました、また誘いますね`

### H. 即時応答 3 大干扰套路（避坑）
1. **同音/近音陷阱**：句中有「橋(はし)」，错误选项接「箸(はし)」的回应。
2. **答非所问**：回应在语法上通顺，但**逻辑对不上意图**。
3. **主语/时态错位**：对方说的是"我"，错误选项当成"你"来回应。

## 第六部分：問題5 統合理解 — 长对话综合

### 结构 & 笔记法
- **前2题**：一段长说明/对话 → 综合判断。
- **后2题**：先**信息播报（A/B/C/D 四方案）**，再**两人讨论各自选哪个**。

#### 必做：边听边速记 4 方案
```
A: 安／遠
B: 高／近／朝食付
C: 静か／風呂なし
D: 新しい／口コミ◎
→ 男: ?   女: ?
```

### 抓"最终决定"的信号
| 听到 → | 含义 |
|---|---|
| じゃあ〜にする／〜にしよう | 决定了 |
| やっぱり〜がいい | 改主意(以这个为准) |
| 最初は〜と思ったけど | ⚠️后面会反转 |
| 私は〜、あなたは？ | 两人**各选各的**，分开记 |
| 〜は外せないな | 这个条件是关键 |

> ⚠️ 后2题男女**可能选不同**方案，分别听清各自最后说的。

## 第八部分：敬语听力（职场场景必出）

### 尊敬语 ↔ 自谦语 听辨（听出"谁做的"）
| 普通 | 尊敬(对方做) | 自谦(我做) |
|---|---|---|
| 行く/来る/いる | いらっしゃる／お越しになる | 伺う／参る／おる |
| 言う | おっしゃる | 申す／申し上げる |
| 見る | ご覧になる | 拝見する |
| する | なさる | いたす |
| 食べる/飲む | 召し上がる | いただく |
| 聞く | お聞きになる | 伺う／拝聴する |
| 知っている | ご存じだ | 存じておる／存じ上げる |
| くれる | くださる | — |
| もらう | — | いただく／頂戴する |

→ **听到尊敬语=对方/上级动作；听到自谦语=自己/己方动作**，用这个判断"谁去/谁做"。

### 商务固定说法
```
承知しました／かしこまりました ＝ 明白了(遵命)
少々お待ちください ＝ 请稍等
お手数ですが ＝ 麻烦您
恐れ入りますが ＝ 不好意思(请求前)
〜していただけますか ＝ 能否请您…
〜させていただきます ＝ 请允许我…
あいにく〜 ＝ 不巧…
申し訳ございませんが ＝ 非常抱歉但…
```

| **别人的理由≠他的理由** | 2017-2-2（俳優・ストーリー＝女の理由）／2023-2-3（「おっしゃる方も」）／2017-2-5（「っていう人もいる」） |
| **过去的≠现在的** | 2018-2-3（和菓子＝きっかけ、今は仕事のため）／2022-2-1（メモ＝前のアドバイス、問「新しく」） |
| **先夸后转** | 2016-2-2（味✓接客✓→でも内装）／2018-2-5（集客✓会場✓→資料配ればよかった） |
| **わけじゃない否定** | 2023-2-4「そういうわけじゃない」／2024-2-5「不満があったわけじゃなく」 |
| **比较级后置**：第二个才是答案 | 2023-2-6「非常に難しかった→**さらに困難だったのは**」 |
| **「一番/特に/まず」限定** | 2017-2-5、2023-2-1、2022-2-6、2024-2-6——候选全被提及，听「やっぱり/特に」 |
| **条件全过最后一条不过** | 2021-2-4（免許✓週末✓→場所が遠い❌） |

### 3. 問題3 题型分布（7套统计）

| 类型 | 比例 | 抓法 |
|---|---|---|
| 専門家・職人の語り | 约6成 | `〜が大切/重要`／转折`けど・しかし`后／`〜のではないでしょうか` |
| 主张・呼吁（議会・校長・講演） | 约2成 | `〜べきだ`／`ご協力いただければ`／反对意见先让步再反驳 |
| 现象解说（天気雨・タンポポ・カラス） | 约2成 | 罗列的细节都服务于一个上位词（仕組み/役割/運ばれ方/意味） |

**选项陷阱**：以偏概全（只覆盖一个例子）和「きっかけ/理由/方法/魅力」偷换。2018-3-2（ペット服）整段→「きっかけ」；2022-3-4（出張料理）尾句「やりがい」→「魅力」。

### 4. 問題4 考点统计（7套78题）— N2文法直考化是大趋势

| 文法/惯用考点 | 出现（年-题） |
|---|---|
| `〜っぱなし` | 2022-6（エアコン）——加上2010/2012/2013，**5次出场** |
| `〜ところだった`（差点） | 2024-5 → 回应「間に合ってよかった」 |
| `〜ないこともない`（二重否定） | 2024-10 → ＝可以/部分肯定 |
| `〜ざるを得ない` | 2017-6、2024-8 → 回应「仕方ない」 |
| `に限る/に限らず/を問わず` | 2022-7、2024-3、2022-10 |
| `〜はともかく` | 2018-10、2023-1 → 接「另一面如何」 |
| `〜わりに/にしては` | 2017-12、2022-3 |
| `〜からには/以上は` | 2021-5、2018-5 → 回应「頑張ります/期待してる」 |
| `〜次第/てからでないと` | 2021-7、2016-3 |
| `に決まってる/できっこない/ありえない` | 2024-1、2022-11、2021-6 |
| 惯用：`甲斐がある/頭が下がる/文句のつけようがない/見直す/ついてない` | 2018-6、2017-11、2016-9、2018-1、2018-4 |
| `〜つつ/つつある/に反して/ずに済む` | 2023-9/4/10/8（2023年問題4几乎全是文法题） |

### 5. 問題5 结构（7套14大题全部吻合）

- **永远 4 个选项**：学校4／旅館4／チケット4／グループ4／施設4／コース4／映画4／会場4。速记每项**1-2个特征词**。
- **最终决定句**：`じゃ、そうしよう`／`それにします`——决定前必有 2-3 轮排除。
- **排除理由模板**：費用かかる／時間かかる／好みが分からない／負担が大きい／（2022衣装）セリフ変更できない → 共性：**成本与风险**。
- **Q1/Q2 双问男女各选各的、几乎必不同**（2016留学/2017チケット/2018ボランティア/2021イベント/2022コース/2023映画/2024会場——7年全是）。
- **「人気だから」必被否定**：2016-5-3 女想选最有人气的学校→被「日本人が多くて伸びない」劝退。**从众选项＝错误选项**。
- **条件句常由领导最后给出**：2017「目的に合って予算内」／2021「お金と手間をかけずに」——**最后30秒的条件句是筛选钥匙**。

### 7. 跨年规律（最值得记的 5 条）

1. **JLPT 会原样复用旧题**：2021-07 問題2「サッカー・体力」＝2013-07 原题；問題4「レストラン行ったことある？」＝2012-07 原题。**做旧真题不会白做**。
2. **設問里的限定词决定一切**：「今日の授業で」「今年度から」「新しく」「一番」「まず」——约 1/3 的题 4 个选项全在原文出现过，唯一区分就是设问限定词。**预读时圈出限定词**。
3. **「他人的理由」是問題2第一大干扰源**（每套至少1题）：`〜という人もいる`／`〜とおっしゃる方も`／对话里另一人的理由。设问问谁，就只记谁的话。
4. **婉拒/排除链稳定**：`〜たいんですが＋ただ/実は＋ちょっと…＋すみません`（2021バイト辞退・2016部長辞退）→ 含义永远是"拒绝"。
5. **答案在"剩下的那个"**：問題1 和問題5 的本质相同——不是听出正解，而是**听掉三个错解**。训练时养成边听边划掉的习惯。

## 問題1 課題理解（5题）—「このあと、まず何をしますか」

### 1-1 ★陷阱：その前に（顺序提前）

**場面**：会社で、女の人と男の人（新人）が話しています。男の人はこのあとまず何をしますか。

> F: 田中くん、午後の会議の準備、お願いね。資料の印刷と、お弁当の注文と、机の並べ替え。
> M: はい。じゃ、まず印刷からやります。
> F: あ、待って。**その前に**お弁当ね。注文、11時までだから。
> M: あ、そうでした。わかりました。

**質問**：男の人はこのあとまず何をしますか。
1. 資料を印刷する　2. お弁当を注文する　3. 机を並べ替える　4. 会議に出る

<details><summary>正解・解析</summary>

**正解：2**
男の人は「まず印刷」と言ったが、女の人が「**その前に**お弁当」と訂正 → 真正的第一步被提前。
🏷 陷阱：`その前に／先に` 顺序提前 → 《表現集》第二部分
</details>

### 1-2 ★陷阱：已完成排除＋甩锅排除

**場面**：大学の事務室で、留学生と職員が話しています。留学生はこのあとまず何をしますか。

> F: 奨学金の申し込みですね。申込書と、写真と、先生の推薦状が必要ですが。
> M: 申込書は**もう書きました**。写真も持ってきました。
> F: じゃ、あとは推薦状ですね。
> M: それが、先生が今週出張で…。
> F: でしたら、推薦状は**こちらから先生にメールしておきます**。代わりに、この同意書に今日中にサインをもらってきてください。寮の管理人さんのでいいので。
> M: わかりました。

**質問**：留学生はこのあとまず何をしますか。
1. 申込書を書く　2. 写真を撮る　3. 先生に推薦状を頼む　4. 管理人にサインをもらう

<details><summary>正解・解析</summary>

**正解：4**
申込書・写真＝**已完成**排除；推薦状＝职员说「こちらから〜しておきます」**别人来做**排除 → 剩下的就是答案。
🏷 陷阱：`もう〜た`（完了）＋`こちらでやる`（甩锅）→ 《表現集》第二部分
</details>

### 1-3 ★陷阱：あ、でも／やっぱり（计划反转）

**場面**：アルバイトの店で、店長と女の人が話しています。女の人はこのあとまず何をしますか。

> M: おはよう。開店まであと30分ね。床の掃除は？
> F: さっき終わりました。
> M: じゃ、レジのお金の確認をお願い。**あ、でも**、今日は雨だから、**先に**入口に傘立てを出しといて。
> F: はい、わかりました。

**質問**：女の人はこのあとまず何をしますか。
1. 床を掃除する　2. レジのお金を確認する　3. 傘立てを出す　4. 看板を出す

<details><summary>正解・解析</summary>

**正解：3**
床掃除＝已完成；レジ確認被「**あ、でも…先に**」推翻 → 听到转折词，前面的指令作废。
🏷 陷阱：`あ、でも／先に` 反转 → 《表現集》第二部分・《攻略》核心心法
</details>

### 1-4 ★陷阱：数字变更（数量加算）

**場面**：会社で、男の人と女の人が話しています。女の人は資料を何部コピーしますか。

> M: 明日の説明会の資料、コピーお願いできる？参加者は30人ね。
> F: 30部ですね。
> M: あ、**やっぱり**、課長と部長の分も**入れて**。
> F: じゃ、32部ですね。
> M: うん、お願い。

**質問**：女の人は資料を何部コピーしますか。
1. 28部　2. 30部　3. 32部　4. 34部

<details><summary>正解・解析</summary>

**正解：3**
30 部之后被「やっぱり〜も入れて」追加 2 人份 → 30+2=32。听到数字**别急着定**，等到最后确认。
🏷 陷阱：数字反转/加总（`やっぱり``〜も入れて``全部で`）→ 《表現集》第一部分 2
</details>

### 1-5 ★陷阱：〜なくてもいい（不用做）排除

**場面**：家で、母親と息子が話しています。息子はこのあとまず何をしますか。

> F: ちょっと出かけてくるから、お願いがあるの。洗濯物を取り込んで、それから宅配便が来たら受け取って。
> M: ゴミ出しは？
> F: ああ、ゴミは**今日はもう出さなくていい**わ、明日の朝で。それより、雨が降りそうだから洗濯物**を先に**ね。
> M: はーい。

**質問**：息子はこのあとまず何をしますか。
1. ゴミを出す　2. 洗濯物を取り込む　3. 宅配便を受け取る　4. 買い物に行く

<details><summary>正解・解析</summary>

**正解：2**
ゴミ出し＝「出さなくていい」**不用做**排除；宅配便＝条件性任务（来たら）非第一步；「先に」点名洗濯物。
🏷 陷阱：`〜なくてもいい` 排除＋`先に` 锁定 → 《表現集》第二部分
</details>

## 問題2 ポイント理解（6题）— 先预读题目，只听问的那个点

### 2-1 ★陷阱：表面理由 vs 真实理由

**場面**：会社で、女の人と男の人が話しています。男の人はどうして引っ越すことにしましたか。

> F: 引っ越すんだって？今の部屋、駅から近くて便利なのに。
> M: うん、家賃もちょっと高いしね。
> F: ああ、それで。
> M: いや、家賃は**まあ何とかなるんだけど**、**実は**隣の工事が始まってさ、朝から音がすごくて。在宅勤務の日は仕事にならないんだ。
> F: それはつらいね。

**質問**：男の人はどうして引っ越すことにしましたか。
1. 家賃が高いから　2. 駅から遠いから　3. 工事の音がうるさいから　4. 会社が遠いから

<details><summary>正解・解析</summary>

**正解：3**
家賃は表面理由，且被「まあ何とかなる」弱化；「**実は**」之后才是真实理由。
🏷 陷阱：`実は／本当は` 真理由 → 《表現集》第三部分
</details>

### 2-2 ★陷阱：理由的否定连发

**場面**：店で、店員と女の人が話しています。女の人はどうしてかばんを買いませんでしたか。

> M: こちら、今年の新作です。いかがですか。
> F: デザインはすごく好き。色もいいし。
> M: 軽いですし、ポケットも多いですよ。
> F: うーん、でも**思ったより小さくて**、私のパソコンが入らないんですよね。残念だけど、今日はやめておきます。

**質問**：女の人はどうしてかばんを買いませんでしたか。
1. デザインが好きじゃないから　2. 色が好きじゃないから　3. 値段が高いから　4. パソコンが入らないから

<details><summary>正解・解析</summary>

**正解：4**
デザイン・色都被**夸了**（干扰项全被提到过），「でも」之后才是不买的理由。
🏷 陷阱：干扰项全念过＋`でも` 后是重点 → 《攻略》第0节
</details>

### 2-3 ★陷阱：心情题（微妙系）

**場面**：試合のあとで、女の人が男の人と話しています。男の人は今どんな気持ちですか。

> F: 残念だったね、決勝で負けちゃって。
> M: うん、悔しくないと言ったらうそになるけど…。でも、練習してきたことは全部出せたから、**すっきりした気持ち**なんだ。
> F: そっか。いい顔してるもんね。

**質問**：男の人は今どんな気持ちですか。
1. 悔しくてたまらない　2. やることはやったので満足している　3. 練習が足りなかったと後悔している　4. 次の試合が不安だ

<details><summary>正解・解析</summary>

**正解：2**
「悔しくないと言ったらうそになる**けど**」→ 转折后「全部出せた・すっきり」才是核心心情。
🏷 陷阱：心情题抓转折后＋微妙系词汇 → 《表現集》第三部分「心情・评价词」
</details>

### 2-4 ★陷阱：时间变更

**場面**：電話で、女の人と男の人が話しています。二人は何時に待ち合わせますか。

> F: 明日、映画の前にお茶しない？映画は2時からだから、1時に駅でどう？
> M: ごめん、午前中ちょっと用事があって、1時は**ぎりぎりかも**。
> F: じゃあ、1時半は？
> M: うん、**それなら**大丈夫。
> F: オッケー、じゃ1時半に駅ね。

**質問**：二人は何時に待ち合わせますか。
1. 1時　2. 1時半　3. 2時　4. 2時半

<details><summary>正解・解析</summary>

**正解：2**
最初の提案（1時）被婉拒，2時は映画開始時間（干扰数字）。最终确认「じゃ1時半に駅ね」。
🏷 陷阱：数字变更＋干扰数字 → 《表現集》第一部分 2
</details>

### 2-5 ★陷阱：谁做的（敬语判断主语）

**場面**：会社で、女の人と男の人が話しています。資料を作ったのは誰ですか。

> F: この資料、よくできてますね。佐藤さんが作ったんですか。
> M: いえ、私は印刷しただけで、**作成は部長がなさいました**。グラフのところだけ、私がお手伝いしましたが。
> F: そうですか。さすが部長ですね。

**質問**：資料を作ったのは誰ですか。
1. 女の人　2. 佐藤さん（男の人）　3. 部長　4. 課長

<details><summary>正解・解析</summary>

**正解：3**
「作成は部長が**なさいました**」→ 尊敬语＝**对方/上级**的动作。男の人做的只是印刷+帮忙画图。
🏷 陷阱：尊敬语/自谦语判断"谁做的" → 《表現集》第八部分
</details>

### 2-6 ★陷阱：为谁做（主语/对象错位）

**場面**：学校で、男の人と女の人が話しています。女の人はどうして病院へ行きますか。

> M: あれ、午後の授業、休むの？具合悪い？
> F: ううん、私は元気。**母が**明日手術することになって、今日その説明を家族も一緒に聞かなきゃいけないの。
> M: そうか、それは心配だね。気をつけて。

**質問**：女の人はどうして病院へ行きますか。
1. 自分の具合が悪いから　2. 母の手術の説明を聞くから　3. 母を見舞うから　4. 検査を受けるから

<details><summary>正解・解析</summary>

**正解：2**
「私は元気」先排除本人生病；去医院是为了**听母亲手术的说明**（不是探病、不是检查）。
🏷 陷阱：主语错位＋选项偷换动作 → 《表現集》第五部分 H-3
</details>

## 問題3 概要理解（5题）— 没有选项预读，抓主旨/态度

### 3-1 講演（主张型）

**場面**：講演会で、医者が話しています。

> M: 皆さん、睡眠時間を削って頑張ることが努力だと思っていませんか。実は、寝る時間を決めて毎日同じリズムで生活している人のほうが、仕事の効率も記憶力も高いという調査結果があります。何時間寝るかも大事ですが、**それより大切なのは、毎日同じ時刻に寝て、同じ時刻に起きること**です。まず今夜から、寝る時刻を決めてみてください。

**質問**：医者が一番言いたいことは何ですか。
1. 睡眠時間を削って努力すべきだ　2. 長く寝れば寝るほどいい　3. 毎日同じ時刻に寝起きすることが大切だ　4. 調査の結果は信用できない

<details><summary>正解・解析</summary>

**正解：3**
「**それより大切なのは〜**」＝主张信号，首句的常识被否定、尾句呼吁行动 → 首尾呼应锁定主旨。
🏷 套路：`大切なのは／〜てみてください` 主张信号 → 《表現集》第四部分
</details>

### 3-2 留守番電話（用件型）

**場面**：留守番電話のメッセージを聞いています。

> F: もしもし、鈴木です。あさっての飲み会のことなんだけど、お店、予約してた「さくら」が貸し切りになっちゃって。それで、**同じ時間で、駅の反対側の「うみ」っていう店に変えました**。時間は変わらず7時ね。場所、地図を後でメールで送るから見ておいて。じゃ、よろしく。

**質問**：このメッセージの用件は何ですか。
1. 飲み会の中止の連絡　2. 飲み会の店の変更の連絡　3. 飲み会の時間の変更の連絡　4. 店の予約のお願い

<details><summary>正解・解析</summary>

**正解：2**
時間は「変わらず」明确不变（排除3）；只有**店**变了。用件题听「それで〜ました」的结论句。
🏷 套路：留守電题＝抓"打电话的目的" → 《攻略》問題3
</details>

### 3-3 先生の話（建议型）

**場面**：日本語学校で、先生が話しています。

> M: 漢字が覚えられないという相談をよく受けます。何十回も書いて覚えようとする人が多いんですが、書くだけではすぐ忘れてしまいます。**それより、覚えた漢字をメールや日記ですぐ使ってみてください**。使った漢字は忘れません。書く練習が無駄だとは言いませんが、「書いて終わり」にしないことです。

**質問**：先生は漢字の勉強について、どう考えていますか。
1. 何十回も書いて覚えるべきだ　2. 覚えた漢字を実際に使うことが大切だ　3. 書く練習は全く無駄だ　4. 日記は日本語で書かないほうがいい

<details><summary>正解・解析</summary>

**正解：2**
「それより〜てみてください」＝建议核心；注意「無駄だとは言いません」→ 选项3「全く無駄」**以偏概全**，排除。
🏷 套路：部分否定 `〜とは言わないが` ≠ 全盘否定 → 《攻略》問題3「排除以偏概全」
</details>

### 3-4 ニュース（话题型）

**場面**：テレビでアナウンサーが話しています。

> F: 今年は夏の気温が高かった上に、台風が続いたため、レタスやキャベツなど野菜の値段が去年の2倍近くまで上がっています。スーパーでは、形の悪い野菜を安く売ったり、カット野菜の量を増やしたりする工夫が広がっています。野菜の高い状態は、**来月まで続く見込み**だということです。

**質問**：アナウンサーは何について話していますか。
1. 台風の進路　2. 野菜の値段の上昇　3. スーパーの新しいサービス　4. 今年の夏の暑さ

<details><summary>正解・解析</summary>

**正解：2**
天气、超市的应对都是**为主题服务的细节**；反复出现的关键词是「野菜の値段」。
🏷 套路：抓反复出现的关键词＝主题 → 《攻略》問題3
</details>

### 3-5 朝礼（态度型）

**場面**：会社の朝礼で、社長が話しています。

> M: 先日、お客様からお礼の手紙をいただきました。商品がよかったからではありません。電話に出た社員の**声が明るくて、気持ちがよかった**からだそうです。私たちはつい、商品の質ばかり考えてしまいますが、お客様が最初に触れるのは、皆さんの声であり、あいさつです。**技術より先に、まず気持ちのいい対応を**。今日も一日よろしくお願いします。

**質問**：社長が社員に一番伝えたいことは何ですか。
1. 商品の質を上げてほしい　2. お客様に手紙を書いてほしい　3. 気持ちのいい対応を心がけてほしい　4. 電話の数を減らしてほしい

<details><summary>正解・解析</summary>

**正解：3**
「商品がよかったからでは**ありません**」先否定干扰项1；「技術より先に、まず〜を」＝主张句式。
🏷 套路：`〜より先に、まず〜` 优先级主张 → 《表現集》第四部分
</details>

## 問題4 即時応答（12题）— 听"意图"，1秒判断

> 每题：A 说一句 → 从 3 个回应中选最自然的。

### 4-1（请求）
> M: 悪いんだけど、この荷物、上の棚に上げてもらえる？

1. うん、**いいよ。よいしょっと**。
2. 棚は二階にありますよ。
3. 荷物が多いですね。

<details><summary>正解：1</summary>「悪いんだけど＋〜てもらえる？」＝请求 → 答应/婉拒。2、3 答非所问。🏷《表現集》第五部分 A</details>

### 4-2（求许可）
> F: すみません、この資料、ちょっと借りてもいいですか。

1. ええ、**どうぞ**。
2. 図書館で借りました。
3. 貸してください。

<details><summary>正解：1</summary>「〜てもいいですか」＝求许可 → 给/不给。3 把立场反了（主语错位）。🏷《表現集》第五部分 C・H-3</details>

### 4-3（邀约→婉拒）
> M: 今度の土曜、みんなでバーベキューするんだけど、来ない？

1. 行ってらっしゃい。
2. **あいにく、その日はちょっと予定があって…**。
3. 昨日はごちそうさま。

<details><summary>正解：2</summary>「〜ない？」＝邀约 → 接受/婉拒。「あいにく＋ちょっと…」是标准婉拒。🏷《表現集》第五部分 B・G</details>

### 4-4（后悔→共感）
> F: ああ、傘、持ってくればよかった。

1. **ね、まさか降るとは思わなかったよね**。
2. 傘は売り場の二階です。
3. 持ってきてくれてありがとう。

<details><summary>正解：1</summary>「〜ばよかった」＝后悔 → 共感/安慰。3 与事实相反（没带）。🏷《表現集》第五部分 D</details>

### 4-5（寒暄：先下班）
> M: お先に失礼します。

1. お邪魔します。
2. **お疲れさまでした**。
3. いただきます。

<details><summary>正解：2</summary>职场固定应答。🏷《表現集》第五部分 F（必背）</details>

### 4-6（责备→道歉）
> F: 出かける前に言っておいてくれればよかったのに。

1. **ごめん、うっかりしてて**。
2. どういたしまして。
3. もう出かけましたよ。

<details><summary>正解：1</summary>「〜ておけばよかったのに」＝责备"你当初该…" → 道歉。🏷《表現集》第五部分 D</details>

### 4-7（敬语：问上级在不在）
> M: 部長はもうお帰りになりましたか。

1. はい、**先ほど帰られました**。
2. はい、私が帰りました。
3. 部長を返してください。

<details><summary>正解：1</summary>「お帰りになる」尊敬语＝**部长**的动作；2 把主语换成自己。🏷《表現集》第八部分</details>

### 4-8（感谢→应答）
> F: 先日は手伝っていただいて、本当に助かりました。

1. **いえいえ、お役に立てて何よりです**。
2. はい、助けてください。
3. お大事に。

<details><summary>正解：1</summary>感谢的固定应答三件套：いえいえ／とんでもないです／お役に立てて何よりです。🏷《表現集》第五部分 E</details>

### 4-9（确认记忆→纠正）
> M: 会議って、3時からだっけ？

1. **いや、3時半からだよ**。
2. 会議室は3階だよ。
3. もう3時間もやってるよ。

<details><summary>正解：1</summary>「〜だっけ？」＝确认 → 肯定/纠正。2、3 是**同音近音干扰**（3階/3時間）。🏷《表現集》第五部分 C・H-1</details>

### 4-10（同音陷阱：箸 vs 橋）
> F: ねえ、そこのお箸、取ってくれる？

1. **はい、どうぞ**。
2. あの橋は長いですね。
3. 走って渡りましょう。

<details><summary>正解：1</summary>食卓场景の「はし」＝**箸**；2、3 故意接「橋」。🏷《表現集》第五部分 H-1</details>

### 4-11（建议→接受）
> M: 顔色悪いよ。今日はもう帰ったら？

1. **そうだね、そうさせてもらうよ**。
2. 顔は洗いましたよ。
3. 帰りに買ってきて。

<details><summary>正解：1</summary>「〜たら？」＝建议 → 接受/婉拒。🏷《表現集》第五部分 B</details>

### 4-12（困扰→帮助）
> F: どうしよう、財布、家に忘れてきちゃって。

1. **えっ、じゃ、貸そうか？**
2. いい財布ですね。
3. 家はどこですか。

<details><summary>正解：1</summary>「〜ちゃって＋どうしよう」＝困扰 → 主动提供帮助（〜ようか？）。🏷《表現集》第五部分 D・第七部分「困扰」</details>

## 問題5 統合理解（4题）— 长对话，记要点

### 5-1 家族会議（条件整合）

**場面**：家で、家族3人が父の誕生日プレゼントについて話しています。

> F1(母): お父さんの誕生日、何にする？去年はセーターだったわね。
> M(息子): 最近ジョギング始めたから、運動靴は？
> F2(娘): いいけど、靴はサイズが難しいよ。本人がいないと。
> F1: そうね。じゃ、時計は？運動用の、歩数も測れるやつ。
> M: あ、それ**この間自分で買ってた**よ。
> F2: えー。じゃあ、走るとき用のリュックはどう？軽くて、水も入れられるの。
> F1: それなら**サイズの心配もない**し、まだ持ってないわね。
> M: うん、**それにしよう**。

**質問**：3人は父に何をあげますか。
1. セーター　2. 運動靴　3. 時計　4. リュック

<details><summary>正解・解析</summary>

**正解：4**
靴＝サイズ問題で消し；時計＝**已拥有**消し；「それにしよう」＝最终决定信号。
🏷 套路：逐项排除＋`〜にしよう` 决定信号 → 《表現集》第六部分
</details>

### 5-2 社員旅行（反转决定）

**場面**：会社で、男の人と女の人が社員旅行の行き先を話しています。

> M: 旅行の行き先、温泉と京都と、海のホテルの3つまで絞ったんだけど。
> F: **最初は温泉がいいと思ってた**んだけど、若い社員に聞いたら、温泉は去年も行ったからって。
> M: ああ、確かに。じゃ、京都か海か。
> F: 京都は見るところが多いけど、今の季節、観光客がすごいよ。バスも乗れないって。
> M: それは疲れに行くようなもんだね。**となると…**。
> F: うん、**消去法だけど**、みんなでゆっくりできるし。
> M: 決まりだね。

**質問**：二人はどこに行くことにしましたか。
1. 温泉　2. 京都　3. 海のホテル　4. まだ決めていない

<details><summary>正解・解析</summary>

**正解：3**
「最初は〜と思ってた」＝必反转；温泉（去年去过）、京都（人多）先后被排除 →「となると」＋「決まりだね」锁定剩下的海。**答案没被直接念出来**，靠排除。
🏷 套路：`最初は〜けど` 反转＋消去法 → 《表現集》第六部分
</details>

### 5-3 講座選び（信息播报＋两人各选各的）★後2問は分けて記録

**場面**：日本語学校で、先生が4つの特別講座について説明しています。そのあと、男の学生と女の学生が話しています。

> 先生(F): 来月から特別講座を4つ開きます。**Aコース**は会話中心で、毎週金曜の夜です。**Bコース**は漢字の集中講座、土曜の朝ですが、初級の方向けです。**Cコース**は聴解対策、試験前の1か月だけの短期集中で、平日の昼間に行います。**Dコース**は作文で、オンラインでいつでも受けられますが、課題が毎週出ます。
>
> M: どれにする？俺、平日の昼間はバイトがあるから無理だな。
> F: オンラインなら時間が自由でいいんじゃない？
> M: でも毎週課題が出るのはきついよ。**金曜の夜なら行けるから、話す練習にするよ**。試験も会話から逃げられないし。
> F: 私は**漢字…と思ったけど**、初級向けでしょ？私にはちょっと簡単すぎるかも。
> M: じゃ、試験前の短期のは？昼間、授業ないって言ってたよね。
> F: うん、**聴解が一番苦手だから、それにする**。短期集中なら頑張れそう。

**質問1**：男の学生はどの講座を受けますか。
1. Aコース　2. Bコース　3. Cコース　4. Dコース

**質問2**：女の学生はどの講座を受けますか。
1. Aコース　2. Bコース　3. Cコース　4. Dコース

<details><summary>正解・解析</summary>

**質問1 正解：1（A 会話）**　平日昼（C）＝バイトで無理；オンライン（D）＝課題きつい；「金曜の夜なら行ける」＝A。
**質問2 正解：3（C 聴解）**　漢字（B）＝「〜と思ったけど」反转（太简单）；「聴解が一番苦手だから、それにする」＝C。

🏷 套路：**边听边速记 4 方案**（A会話金夜／B漢字土朝初級／C聴解昼短期／D作文オンライン課題）→ 男女**各选各的，分开记**。两人**不选同一个**是本题型常态。→ 《表現集》第六部分
</details>
