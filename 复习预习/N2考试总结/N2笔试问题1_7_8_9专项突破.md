# 一、問題（もんだい）1 漢字（かんじ）読（よ）み

## 1. 出题规律

問題（もんだい）1 常考 5 类：

1. **音读词**：二字熟语，如 記憶（きおく）、治療（ちりょう）、批評（ひひょう）、拡充（かくじゅう）。
2. **训读动词**：納（おさ）める、傾（かたむ）く、拾（ひろ）う、務（つと）める。
3. **训读形容词**：著（いちじる）しい、険（けわ）しい、偉（えら）い、怖（こわ）い。
4. **长音/促音/浊音陷阱**：密閉（みっぺい）、圧倒的（あっとうてき）、垂直（すいちょく）、破片（はへん）。
5. **近形近义混淆**：模範（もはん）/模倣（もほう）、声援（せいえん）/支援（しえん）、運賃（うんちん）/運転（うんてん）。

## 2. 判断流程

1. 先判断词性：名词熟语多为音读，动词/形容词多为训读。
2. 看是否有固定搭配：学費（がくひ）を納（おさ）める、役割（やくわり）を果（は）たす。
3. 排除不存在的读音：N2 干扰项经常是「看起来像」但日语没有的读音。
4. 最后检查长音、促音、浊音：きょう/きょ、しょう/しょ、っ、濁音（だくおん）。

# 二、問題（もんだい）7 文法形式（ぶんぽうけいしき）

## 1. 先看接续，再看意思

| 接续 | 常见句型 |
|---|---|
| 動詞辞書形（どうしじしょけい） | べきだ、ことはない、わけにはいかない、恐（おそ）れがある |
| 動詞た形（どうしたけい） | 末（すえ）に、あげく、ばかり、ところ |
| 動詞ます形去ます | かねる、かねない、抜（ぬ）く、がち、ながら |
| 動詞ない形（どうしないけい） | ざるを得（え）ない、ないことには、ない限（かぎ）り |
| 名詞（めいし） | にとって、に対（たい）して、を問（と）わず、に限（かぎ）って |
| 普通形（ふつうけい） | ものの、とはいえ、ことから、わけだ、はずだ |

## 2. 高频逻辑分类

### A. 逆接・让步

- ものの：虽然做了，但结果不如预期。例：予約（よやく）したものの、行（い）けなくなった。
- とはいえ：虽说如此，但补充限制。例：春（はる）とはいえ、朝（あさ）は寒（さむ）い。
- にもかかわらず：尽管。书面。例：雨（あめ）にもかかわらず、人（ひと）が集（あつ）まった。
- ながらも/つつも：一边承认一边转折。例：危険（きけん）だと思（おも）いつつ、続（つづ）けた。

### B. 原因・结果

- ばかりに：就因为…导致坏结果。例：一言（ひとこと）言（い）ったばかりに、誤解（ごかい）された。
- 末（すえ）に：经过长时间努力/考虑后。例：相談（そうだん）した末（すえ）に、進学（しんがく）を決（き）めた。
- あげく：折腾到最后，多负面。例：迷（まよ）ったあげく、買（か）わなかった。
- ことから：根据某事实得出原因/判断。例：客（きゃく）が増（ふ）えたことから、営業時間（えいぎょうじかん）を延（の）ばした。

### C. 限定・强调

- すら：连…都。例：名前（なまえ）すら覚（おぼ）えていない。
- さえ：只要/甚至。例：水（みず）さえあれば生（い）きられる。
- こそ：正是。例：今（いま）こそ努力（どりょく）すべきだ。
- のみならず：不仅。例：学生（がくせい）のみならず、教師（きょうし）も参加（さんか）した。

### D. 时间・变化

- にともなって/につれて：随着。例：人口（じんこう）が増（ふ）えるにつれて、問題（もんだい）も増（ふ）えた。
- うちに：趁/在…过程中。例：若（わか）いうちに旅行（りょこう）したい。
- たびに：每当。例：この歌（うた）を聞（き）くたびに友人（ゆうじん）を思（おも）い出（だ）す。
- きり：自从…之后就没再。例：一度（いちど）会（あ）ったきりだ。
- ぶり：时隔。例：五年（ごねん）ぶりに帰国（きこく）した。

### E. 判断・评价

- べきだ：应该。例：約束（やくそく）は守（まも）るべきだ。
- わけにはいかない：受道义/社会原因不能。例：熱（ねつ）があるが、会議（かいぎ）を休（やす）むわけにはいかない。
- かねる：难以。例：判断（はんだん）しかねる。
- っこない：绝不可能，口语。例：一人（ひとり）ではできっこない。
- ことはない：没必要。例：心配（しんぱい）することはない。

### F. 敬语

- いらっしゃる：行（い）く/来（く）る/いる 的尊敬语。客人动作。
- 伺（うかが）う：行（い）く/聞（き）く/訪（たず）ねる 的自谦语。自己动作。
- 申（もう）し上（あ）げる：言（い）う 的自谦语。
- お/ご + ます形 + する：自谦。例：ご説明（せつめい）します。
- お/ご + ます形 + になる：尊敬。例：先生（せんせい）がお帰（かえ）りになる。

## 3. 易混句型速辨

| 易混 | 区别 |
|---|---|
| にしては / わりに | にしては：以某身份/标准来说出乎意料；わりに：按程度比例来说不相称 |
| ものの / のに | ものの更书面；のに主观不满更强 |
| 末（すえ）に / あげく | 末（すえ）に可正可负；あげく多为坏结果 |
| おかげで / せいで / せいか | おかげで积极；せいで消极；せいか不确定原因 |
| きり / ぶり | きり强调之后没再发生；ぶり强调隔了多久再次发生 |
| ことはない / ないことはない | ことはない＝没必要；ないことはない＝不是不能/多少可以 |
| わけにはいかない / ほかない | 前者是道义上不能；后者是没有其他选择 |

# 三、問題（もんだい）8 並（なら）べ替（か）え

## 1. 解题五步

1. 找句尾：句尾决定前面需要什么形式。
2. 找固定搭配：如 `〜にとって`、`〜ばかりか`、`〜たびに`、`〜うえで`。
3. 找修饰对象：名词前一定有修饰语；「こと」「もの」「人（ひと）」前面常接句子。
4. 找助词组合：`Nにとっては`、`Nを通（つう）して`、`Nに対（たい）して`、`Vたうえで`。
5. 拼完整句后再数星号位置。不要看到 ★ 就直接选。

| 时间块 | Vたびに / Vる前（まえ）に / Vた後（あと）で |

# 四、問題（もんだい）9 文章（ぶんしょう）文法（ぶんぽう）

## 1. 問題9 的本质

問題（もんだい）9 不是单纯文法题，而是「段落逻辑题」。每个空都要判断：

1. 指示词指向谁：これ/それ/そう/このような。
2. 前后关系：转折、递进、总结、例示、原因、结果。
3. 主语是否一致：谁在做动作，作者还是一般人。
4. 时态和体：ている、ていた、ていく、てきた。
5. 文体是否统一：文章体中常用 という、ことから、わけだ、ものだ。

## 2. 常见空格类型

| 类型 | 常见选项 | 判断方法 |
|---|---|---|
| 指示词 | これ/それ/そう/このような | 往前找一句或整段内容 |
| 连接词 | しかし/ただ/つまり/そこで/一方（いっぽう） | 看前后是反向、补充、总结还是行动 |
| 原因 | ため/ことから/によって/ばかりに | 后句是否是结论或坏结果 |
| 变化 | てきた/ていく/ようになった | 看过去到现在，还是现在到未来 |
| 评价 | はずだ/わけだ/ものだ/ことだ | 看作者是推断、解释还是一般规律 |
| 让步 | とはいえ/ものの/にもかかわらず | 前后内容是否相反 |

## 3. 問題9 解题流程

1. 先读标题和第一段第一句，确认主题。
2. 每空先看前一句和后一句，不够再看整段。
3. 空格前有「しかし」「だが」后面多为反向；空格后有「つまり」则前面通常是解释材料。
4. 出现「この」「その」「そう」时，必须圈出指代内容。
5. 最后一空通常考总结、作者感想或文章主旨。
