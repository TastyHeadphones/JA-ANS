# 第〇部 文字语汇·文法 攻略与配速

## 二、文字・語彙（問題1～6）

### 問題1 漢字読み（5题）
- **考点**：音读混淆、难读音读、音训混同。
- **高频陷阱**：把握（はあく）、懸念（けねん）、堪能（たんのう）、措置（そち）、遂行（すいこう）、秩序（ちつじょ）。
- **策略**：先排除「明显多/少一个长音、浊音」的干扰项。注意 促音・长音・浊音 是出题点。

### 問題2 漢字表記（5题）
- **考点**：给假名选正确汉字。
- **陷阱**：同音异字（保証/保障/補償、関心/感心、対象/対照/対称）。
- **策略**：靠**偏旁意义**判断。如「障」含"阻碍"→保障；「償」含"赔偿"→補償。

### 問題3 語形成（5题）
- **考点**：接头/接尾词、复合词。
- **必背接尾**：`〜化`（少子化）、`〜性`（可能性）、`〜的`（社会的）、`〜風`（和風）、`〜気味`（風邪気味）、`〜だらけ`（泥だらけ）、`〜づらい/〜にくい`（言いづらい）、`〜っぽい`（子どもっぽい）。
- **必背接头**：`不〜`（不可能）、`無〜`（無関心）、`非〜`（非常識）、`未〜`（未解決）、`再〜`（再確認）、`猛〜`（猛反対）、`好〜`（好景気）。

### 問題4 文脈規定（7题，分值高）
- **考点**：根据句意选最贴切的词（动词/副词/外来语）。
- **高频副词**：`いきなり`（突然）、`めったに〜ない`（很少）、`さすが`（不愧）、`せっかく`（特意）、`わざわざ`（特地）、`せいぜい`（顶多）、`あいにく`（不巧）、`案外`（出乎意料）。
- **策略**：注意**搭配（コロケーション）**，如「噂が流れる」「迷惑をかける」「責任を果たす」。

### 問題5 言い換え類義（5题）
- **考点**：选与划线词意思最接近的词。
- **例**：`あいまい` ≈ はっきりしない／`くたくた` ≈ とても疲れた／`めいわく` ≈ 困る。
- **策略**：把划线词换成自己理解的意思，再找同义选项。

### 問題6 用法（5题，最难）
- **考点**：给一个词，选它**用法正确**的句子。
- **陷阱**：四个选项里词义相近但搭配/语境错误。
- **策略**：确认①词性②搭配助词③褒贬色彩④习惯搭配。逐句代入判断哪句"自然"。

## 三、文法（問題7～9）

### 問題7 文法形式判断（12题，最多分）
- **考点**：选最合适的文法/句型/助词。
- **必考分类**（详见《N2语法总复习》）：
  - **逆接**：ものの／とはいえ／にもかかわらず／つつも
  - **原因结果**：ばかりに／あげく／末に／ことから
  - **限定强调**：にすぎない／にほかならない／ばかりか／のみならず
  - **不得已**：ざるを得ない／ないわけにはいかない／かねる／かねない
  - **书面连接**：において／にあたって／をめぐって／に基づいて
  - **判断评价**：べきだ／ものだ／というものだ／とは限らない
- **策略**：先看**接续形式**（动词形/名词）缩小范围，再判断**语义逻辑**。

### 問題8 文の組み立て（5题）★难点
- **考点**：4个★碎片排序，选**★位置**的词。
- **解题5步**：
  1. 找**固定搭配**（如「〜をめぐって」「〜に応じて」必须成对的部分）。
  2. 定**句尾接续**（结尾词决定前面接什么形）。
  3. 用**意义逻辑**把碎片连成通顺句。
  4. 数到**第3个空（★常在此）**填词。
  5. **通读复查**，确认整句自然。
- **提醒**：答案是**★所在位置的词**，不是整句顺序，别填错位置！

### 2. 問題1/2（読み・表記）出题规律

- **音读陷阱三件套**：促音有无（みっぺい）、长短音（ちりょう/きおく）、浊音（そうがく/はへん）。干扰项几乎全是「不存在的读音」。
- **训读考"形近动词组"**：絡まる/挟まる/詰まる、崩れる/乱れる/荒れる、凍る/湿る/濁る——按组背。
- **形容词系**：〜やか（鮮やか/穏やか/爽やか/和やか）、〜しい（著しい/険しい/頼もしい/輝かしい）是固定题库。

### 3. 問題4/5/6（語彙）高频区

- **拟声拟态词每年必出 1-2 个**：のんびり/ひそひそ/でたらめ/じたばた/きっぱり/ぞろぞろ/じろじろ/こそこそ/はきはき/着々/ほっと。
- **片假名词每年 1 个**：リーダー/アピール/契機系のきっかけ/アレンジ/クリア/タイミング/インパクト/テンポ/マナー。
- **言い換え答案偏好「简单词」**：难词→基础词（勘定→会計、概要→大体の内容、油断→気をつけていない）。
- **用法题（問題6）错误选项的本质**：把词放进**搭配对象错误**的句子（鑑賞→只能接艺术；解約→只能接合同）。判断标准是**搭配**，不是意思。

### 4. 問題7/8（文法）与听力問題4 直接重合

7 套中以下句型在**笔试文法和听力即時応答都考过**：
`にしては／ものなら／たとえ〜ても／おそれがある／かねる／っこない／わけにはいかない／ように（してほしい）／〜ばよかった／欠かせない／きり／ぶり／たびに／せいで・せいか／を問わず／に限って／すら／がち`

→ **一份句型清单，两个大题同时受益**。背法：每个句型配「一句话场景＋一个自然回应」。

### 5. 敬语题固定套路（問題7 每年 1〜2 题）

- 判断**动作是谁的**：对方/客人→尊敬（いらっしゃる/おいでになる/ご覧になる/お〜です）；自己→自谦（伺う/申し上げる/拝見する/お〜する）。
- 高频陷阱：**「伺う」用在客人身上**（2022・2024 连续两年同一个坑）。

### 6. 並べ替え（問題8）三步法（7套35题通用）

1. 找**句型搭档**（と思うと/か〜ないかのうちに/ながらも/たびに/うえで——四个选项里必有一对是固定搭配）。
2. 用**接续规则**锁位置（簡体＋名詞、ます形＋かねる、た形＋きり）。
3. 最后用句意验证顺序。★考点 90% 是 N2 句型本体，并非阅读能力。

# 第三部 叠词（ABAB）一覧

> 畳語（じょうご）= 同じ語を重ねて作った語。ABAB 型・AA型 をまとめる。

### 一、副詞型畳語

| 様々 | さまざま | ナ形・副詞 | 各种各样 | 様々な問題が起きた。様々な人に会った。 |

### 四、混同しやすいペアまとめ

| ペア | 違い |
|---|---|
| **段々** vs **徐々に** | 「段々」口语常用；「徐々に」书面・正式 |
| **次々と** vs **続々と** | 「次々と」= 一个接一个（有顺序）；「続々と」= 接连不断（成群出现） |
| **様々な** vs **色々な** | 「様々な」书面语强，强调多样性；「色々な」日常通用 |
| **どきどき** vs **わくわく** | 「どきどき」= 紧张/担心（负面也可）；「わくわく」= 兴奋期待（正面） |
| **ぐずぐず** vs **もたもた** | 「ぐずぐず」= 磨蹭+不情愿/发牢骚；「もたもた」= 单纯行动迟缓 |
| **ふわふわ** vs **ふかふか** | 「ふわふわ」= 轻飘飘/蓬松（面包・感觉）；「ふかふか」= 柔软有弹性（床・布）|
| **つるつる** vs **ぬるぬる** | 「つるつる」= 干燥光滑；「ぬるぬる」= 湿润黏滑 |
