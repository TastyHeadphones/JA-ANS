# 第〇部 文字语汇·文法 攻略与配速

## 二、文字（もじ）・語彙（ごい）（問題（もんだい）1～6）

### 問題（もんだい）1 漢字読（かんじよ）み（5题）
- **考点**：音读混淆、难读音读、音训混同。
- **高频陷阱**：把握（はあく）、懸念（けねん）、堪能（たんのう）、措置（そち）、遂行（すいこう）、秩序（ちつじょ）。
- **策略**：先排除「明显多/少一个长音、浊音」的干扰项。注意 促音・长音・浊音 是出题点。

### 問題（もんだい）2 漢字表記（かんじひょうき）（5题）
- **考点**：给假名选正确汉字。
- **陷阱**：同音异字（保証（ほしょう）/保障（ほしょう）/補償（ほしょう）、関心（かんしん）/感心（かんしん）、対象（たいしょう）/対照（たいしょう）/対称（たいしょう））。
- **策略**：靠**偏旁意义**判断。如「障」含"阻碍"→保障（ほしょう）；「償」含"赔偿"→補償（ほしょう）。

### 問題（もんだい）3 語形成（ごけいせい）（5题）
- **考点**：接头/接尾词、复合词。
- **必背接尾**：`〜化`（少子化（しょうしか））、`〜性`（可能性（かのうせい））、`〜的`（社会的（しゃかいてき））、`〜風`（和風（わふう））、`〜気味`（風邪気味（かぜぎみ））、`〜だらけ`（泥だらけ（どろだらけ））、`〜づらい/〜にくい`（言いづらい（いいづらい））、`〜っぽい`（子どもっぽい（こどもっぽい））。
- **必背接头**：`不〜`（不可能（ふかのう））、`無〜`（無関心（むかんしん））、`非〜`（非常識（ひじょうしき））、`未〜`（未解決（みかいけつ））、`再〜`（再確認（さいかくにん））、`猛〜`（猛反対（もうはんたい））、`好〜`（好景気（こうけいき））。

### 問題（もんだい）4 文脈規定（ぶんみゃくきてい）（7题，分值高）
- **考点**：根据句意选最贴切的词（动词/副词/外来语）。
- **高频副词**：`いきなり`（突然）、`めったに〜ない`（很少）、`さすが`（不愧）、`せっかく`（特意）、`わざわざ`（特地）、`せいぜい`（顶多）、`あいにく`（不巧）、`案外（あんがい）`（出乎意料）。
- **策略**：注意**搭配（コロケーション）**，如「噂（うわさ）が流（なが）れる」「迷惑（めいわく）をかける」「責任（せきにん）を果（は）たす」。

### 問題（もんだい）5 言（い）い換（か）え類義（るいぎ）（5题）
- **考点**：选与划线词意思最接近的词。
- **例**：`あいまい` ≈ はっきりしない／`くたくた` ≈ とても疲（つか）れた／`めいわく` ≈ 困（こま）る。
- **策略**：把划线词换成自己理解的意思，再找同义选项。

### 問題（もんだい）6 用法（ようほう）（5题，最难）
- **考点**：给一个词，选它**用法正确**的句子。
- **陷阱**：四个选项里词义相近但搭配/语境错误。
- **策略**：确认①词性②搭配助词③褒贬色彩④习惯搭配。逐句代入判断哪句"自然"。

## 三、文法（ぶんぽう）（問題（もんだい）7～9）

### 問題（もんだい）7 文法形式判断（ぶんぽうけいしきはんだん）（12题，最多分）
- **考点**：选最合适的文法/句型/助词。
- **必考分类**（详见《N2语法总复习》）：
  - **逆接**：ものの／とはいえ／にもかかわらず／つつも
  - **原因结果**：ばかりに／あげく／末（すえ）に／ことから
  - **限定强调**：にすぎない／にほかならない／ばかりか／のみならず
  - **不得已**：ざるを得（え）ない／ないわけにはいかない／かねる／かねない
  - **书面连接**：において／にあたって／をめぐって／に基（もと）づいて
  - **判断评价**：べきだ／ものだ／というものだ／とは限（かぎ）らない
- **策略**：先看**接续形式**（动词形/名词）缩小范围，再判断**语义逻辑**。

### 問題（もんだい）8 文（ぶん）の組（く）み立（た）て（5题）★难点
- **考点**：4个★碎片排序，选**★位置**的词。
- **解题5步**：
  1. 找**固定搭配**（如「〜をめぐって」「〜に応（おう）じて」必须成对的部分）。
  2. 定**句尾接续**（结尾词决定前面接什么形）。
  3. 用**意义逻辑**把碎片连成通顺句。
  4. 数到**第3个空（★常在此）**填词。
  5. **通读复查**，确认整句自然。
- **提醒**：答案是**★所在位置的词**，不是整句顺序，别填错位置！

### 2. 問題（もんだい）1/2（読（よ）み・表記（ひょうき））出题规律

- **音读陷阱三件套**：促音有无（みっぺい）、长短音（ちりょう/きおく）、浊音（そうがく/はへん）。干扰项几乎全是「不存在的读音」。
- **训读考"形近动词组"**：絡（から）まる/挟（はさ）まる/詰（つ）まる、崩（くず）れる/乱（みだ）れる/荒（あ）れる、凍（こお）る/湿（しめ）る/濁（にご）る——按组背。
- **形容词系**：〜やか（鮮（あざ）やか/穏（おだ）やか/爽（さわ）やか/和（なご）やか）、〜しい（著（いちじる）しい/険（けわ）しい/頼（たの）もしい/輝（かがや）かしい）是固定题库。

### 3. 問題（もんだい）4/5/6（語彙（ごい））高频区

- **拟声拟态词每年必出 1-2 个**：のんびり/ひそひそ/でたらめ/じたばた/きっぱり/ぞろぞろ/じろじろ/こそこそ/はきはき/着々（ちゃくちゃく）/ほっと。
- **片假名词每年 1 个**：リーダー/アピール/契機系（けいきけい）のきっかけ/アレンジ/クリア/タイミング/インパクト/テンポ/マナー。
- **言（い）い換（か）え答案偏好「简单词」**：难词→基础词（勘定（かんじょう）→会計（かいけい）、概要（がいよう）→大体（だいたい）の内容（ないよう）、油断（ゆだん）→気（き）をつけていない）。
- **用法题（問題（もんだい）6）错误选项的本质**：把词放进**搭配对象错误**的句子（鑑賞（かんしょう）→只能接艺术；解約（かいやく）→只能接合同）。判断标准是**搭配**，不是意思。

### 4. 問題（もんだい）7/8（文法（ぶんぽう））与听力問題（もんだい）4 直接重合

7 套中以下句型在**笔试文法和听力即時応答都考过**：
`にしては／ものなら／たとえ〜ても／おそれがある／かねる／っこない／わけにはいかない／ように（してほしい）／〜ばよかった／欠（か）かせない／きり／ぶり／たびに／せいで・せいか／を問（と）わず／に限（かぎ）って／すら／がち`

→ **一份句型清单，两个大题同时受益**。背法：每个句型配「一句话场景＋一个自然回应」。

### 5. 敬语题固定套路（問題（もんだい）7 每年 1〜2 题）

- 判断**动作是谁的**：对方/客人→尊敬（いらっしゃる/おいでになる/ご覧（らん）になる/お〜です）；自己→自谦（伺（うかが）う/申（もう）し上（あ）げる/拝見（はいけん）する/お〜する）。
- 高频陷阱：**「伺（うかが）う」用在客人身上**（2022・2024 连续两年同一个坑）。

### 6. 並（なら）べ替（か）え（問題（もんだい）8）三步法（7套35题通用）

1. 找**句型搭档**（と思（おも）うと/か〜ないかのうちに/ながらも/たびに/うえで——四个选项里必有一对是固定搭配）。
2. 用**接续规则**锁位置（簡体＋名詞（めいし）、ます形（けい）＋かねる、た形（けい）＋きり）。
3. 最后用句意验证顺序。★考点 90% 是 N2 句型本体，并非阅读能力。
