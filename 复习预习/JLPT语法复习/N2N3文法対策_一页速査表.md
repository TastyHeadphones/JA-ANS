# JLPT N2/N3 文法対策 一页速査表

> **N3 核心**：句间逻辑 + 时态区分 + 授受敬语  
> **N2 核心**：语气强弱 + 书面程度 + 逻辑关系的精准选择

---

## 一、推量・推断（N3→N2 升级对比）

| 文法 | 含义 | 例句 | 书面度 | 易错点 |
|---|---|---|---|---|
| `〜そうだ`（样态） | 看起来要… | `雨が降りそうです。` | ☆ | ≠传闻そうだ |
| `〜そうだ`（传闻） | 听说… | `田中さんは来月結婚するそうです。` | ☆ | ≠样态そうだ |
| `〜ようだ` | 好像…（有依据） | `電気がついている。中にだれかいるようだ。` | ☆☆ | 比みたい书面 |
| `〜みたいだ` | 好像…（口语） | `このケーキ、おいしいみたい。` | ☆ | 口语版ようだ |
| `〜らしい` | 听说/具有…特点 | `彼は日本語が上手らしいです。` | ☆☆ | 兼含传闻+典型性 |
| `〜かもしれない` | 可能… | `あしたは雪かもしれません。` | ☆ | 概率最低 |
| `〜はずだ` | 理应… | `9時発だから、もう着いたはずだ。` | ☆☆ | 基于前提逻辑 |
| `〜はずがない` | 不可能… | `彼が約束を忘れるはずがない。` | ☆☆ | 强否定推断 |
| `〜にちがいない` | 一定是… | `あの人が犯人にちがいない。` | ☆☆☆ | 主观确信强 |
| `〜に決まっている` | 肯定是… | `そんな話、うそに決まっている。` | ☆ | 口语反驳语气 |
| `〜わけだ` | 难怪/原来如此 | `毎日練習しているから、上手なわけだ。` | ☆☆☆ | ≠单纯"所以" |
| `〜わけではない` | 并非都… | `お金があれば幸せというわけではない。` | ☆☆ | 部分否定 |
| `〜わけがない` | 不可能… | `一日で全部終わるわけがない。` | ☆☆ | 比はずがない更主观 |

## 二、逆接・让步・对比

| 文法 | 含义 | 例句 | 书面度 | 易错点 |
|---|---|---|---|---|
| `〜のに` | 明明…却… | `勉強したのに、点が上がらない。` | ☆ | ≠用途のに |
| `〜ても` | 即使…也… | `高くても買います。` | ☆ | 通用让步 |
| `〜くせに` | 明明…却…（责备） | `子どものくせに生意気だ。` | ☆ | 勿对上级用 |
| `〜ながら`（逆接） | 虽然…却… | `知っていながら、教えなかった。` | ☆☆ | ≠并行ながら |
| `〜ものの` | 虽然…但是… | `就職したものの、仕事が合わない。` | ☆☆☆ | 书面逆接 |
| `〜つつ（も）` | 虽然…但… | `危ないと思いつつ、やってしまった。` | ☆☆☆ | 书面反省 |
| `〜とはいえ` | 虽说… | `春とはいえ、朝はまだ寒い。` | ☆☆☆ | 后项有保留 |
| `〜にもかかわらず` | 尽管…仍… | `雨にもかかわらず、多くの人が来た。` | ☆☆☆ | 正式逆接 |
| `〜にしては` | 以…来说却… | `初めてにしては上手です。` | ☆☆ | "和预期不符" |
| `〜わりに` | 虽…却… | `この店は高いわりにおいしくない。` | ☆ | 口语化对比 |
| `〜反面` | 另一方面… | `便利な反面、危険も多い。` | ☆☆☆ | 双面评价 |
| `〜一方（で）` | 一方面…另一方面… | `都会は便利な一方で、物価が高い。` | ☆☆☆ | 对比分析 |
| `〜どころか` | 别说…连…都… | `寝るどころか、休む時間もない。` | ☆☆ | 程度更甚 |
| `〜どころではない` | 不是…的时候 | `今は旅行どころではない。` | ☆☆ | ≠どころか |

## 三、原因・结果

| 文法 | 含义 | 例句 | 极性 | 书面度 |
|---|---|---|---|---|
| `〜ので` | 因为…（客观礼貌） | `頭が痛いので、休みます。` | 中 | ☆☆ |
| `〜から` | 因为…（主观） | `疲れたから、もう寝ます。` | 中 | ☆ |
| `〜ため（に）` | 因为…/为了… | `台風のため、電車が止まった。` | 中 | ☆☆☆ |
| `〜おかげで` | 多亏… | `先生のおかげで合格できた。` | ✅正面 | ☆☆ |
| `〜せいで` | 都怪… | `寝不足のせいで頭が痛い。` | ❌负面 | ☆☆ |
| `〜せいか` | 也许因为… | `風邪のせいか、だるい。` | ❌偏负 | ☆☆ |
| `〜ばかりに` | 正因为…才…（坏结果） | `油断したばかりに失敗した。` | ❌负面 | ☆☆☆ |
| `〜あげく（に）` | 折腾到最后 | `けんかしたあげく、別れた。` | ❌负面 | ☆☆☆ |
| `〜末（に）` | 最终… | `悩んだ末に、留学を決めた。` | 中 | ☆☆☆ |
| `〜結果` | 结果… | `検討した結果、計画を変更した。` | 中 | ☆☆☆ |
| `〜ことから` | 由此可知/因为 | `声が大きいことから、彼だとわかった。` | 中 | ☆☆☆ |
| `〜恐れがある` | 恐怕会… | `台風で洪水の恐れがある。` | ❌负面 | ☆☆☆ |
| `〜かねない` | 恐怕会…（坏事） | `このままでは事故が起こりかねない。` | ❌负面 | ☆☆☆ |

## 四、条件・假定

| 文法 | 含义 | 例句 | 用法要点 |
|---|---|---|---|
| `Vば` | 如果… | `安ければ買います。` | 书面逻辑条件 |
| `Vたら` | 如果…就… | `時間があったら、連絡してください。` | 口语最通用 |
| `Vと` | 一…就…（规律） | `春になると花が咲きます。` | 不接命令/请求 |
| `Nなら` | 如果是… | `駅前なら便利ですよ。` | 接前文话题 |
| `〜限り` | 只要…就… | `努力する限り、可能性はある。` | 辞书形 |
| `〜ない限り` | 除非…否则不… | `連絡がない限り、出発しない。` | 双重否定常考 |
| `たとえ〜ても` | 即使…也… | `たとえ失敗しても、挑戦する。` | 强让步 |
| `〜にしろ/にしても` | 不管…都/即使… | `行くにしろ行かないにしろ、早く決めて。` | 常成对 `AにしろBにしろ` |
| `〜さえ〜ば` | 只要…就… | `時間さえあれば、行ける。` | 最低条件 |
| `〜ならでは` | 只有…才有 | `これは地元ならではの料理だ。` | 后接名词高频 |
| `〜としたら` | 如果说…的话 | `彼が犯人だとしたら、説明がつく。` | 假设推理 |
| `〜以上（は）` | 既然…就… | `約束した以上、守るべきだ。` | 后项责任义务 |
| `〜からには` | 既然… | `行くからには、全力でやる。` | 决心表达 |
| `〜上は` | 既然…（更书面） | `引き受けた上は最後までやる。` | 正式宣言 |

## 五、时间轴

| 文法 | 含义 | 例句 | 对比要点 |
|---|---|---|---|
| `Vるところだ` | 正要… | `これから出かけるところです。` | 三个ところ要区分时态 |
| `Vているところだ` | 正在… | `今、会議をしているところです。` | 瞬间截面 |
| `Vたところだ` | 刚刚… | `さっき昼ご飯を食べたところです。` | ≠一直持续 |
| `Vたばかりだ` | 刚…（主观） | `日本に来たばかりです。` | 时间可不"刚刚" |
| `Vうとする` | 正要…/试图… | `家を出ようとしたとき、電話が来た。` | 后项常被打断 |
| `〜うちに` | 趁着… | `若いうちに挑戦したほうがいい。` | 变化前窗口 |
| `〜ないうちに` | 趁还没… | `暗くならないうちに帰りましょう。` | 与前に语感不同 |
| `〜間に` | 在…期间（某时点） | `日本にいる間に、富士山に登りたい。` | 一次性动作 |
| `〜間` | 在…整个期间 | `母が料理している間、子どもは寝ていた。` | 持续背景 |
| `〜たびに` | 每当…就… | `この曲を聞くたびに、故郷を思い出す。` | 周期性反复 |
| `〜ついでに` | 顺便… | `買い物のついでに銀行へ行く。` | 前项是主要动作 |
| `〜次第` | 一…就…（书面） | `準備ができ次第、連絡します。` | 后项多未来动作 |
| `〜にあたって` | 值此…之际 | `契約にあたって、条件を確認した。` | 庄重书面 |
| `〜に際して` | 临…之际 | `入国に際して、書類が必要です。` | ≈にあたって |
| `〜に先立って` | 在…之前（正式） | `試験に先立って説明会を行う。` | 仪式/流程 |
| `〜て以来` | 自从…以来 | `日本に来て以来、毎日勉強している。` | 后项多持续变化 |
| `〜をきっかけに` | 以…为契机 | `留学をきっかけに価値観が変わった。` | 变化起点 |

## 六、目的・变化・习惯

| 文法 | 含义 | 例句 | 对比要点 |
|---|---|---|---|
| `Vるために` | 为了…（目的强） | `合格するために毎日勉強する。` | 意志动词前项 |
| `Vるように` | 为了能…（状态目标） | `忘れないように、メモします。` | 能力/状态类 |
| `Vるようになる` | 变得会… | `漢字が読めるようになりました。` | 结果渐变 |
| `Vないようになる` | 变得不再… | `夜更かししないようになりました。` | 注意否定位置 |
| `Vるようにする` | 努力做到… | `毎日歩くようにしています。` | 主观持续努力 |
| `Vないようにする` | 尽量不… | `砂糖を取りすぎないようにしています。` | 语气比禁止柔 |
| `〜ことになる` | 决定为…（外部） | `来月転勤することになりました。` | ≠ことにする |
| `〜ことにする` | 决定…（自己） | `明日から運動することにします。` | 自我意志 |
| `〜ことにしている` | 一直坚持… | `朝6時に起きることにしています。` | 长期自律规则 |
| `〜ように言う/頼む` | 让某人务必… | `母に早く帰るように言われました。` | 间接命令结构 |

## 七、授受・受身・使役（N3核心）

| 文法 | 含义 | 例句 | 易错点 |
|---|---|---|---|
| `AはBにNをあげる` | A 给 B | `私は妹に本をあげた。` | 以给出方为视角 |
| `AはBにNをもらう` | A 从 B 收到 | `先生に辞書をもらった。` | 以收到方为视角 |
| `Aは私にNをくれる` | （他人）给我方 | `友達が私に傘をくれた。` | 主语≠我 |
| `Vてあげる` | 为别人做 | `友達に日本語を教えてあげた。` | 对上级慎用 |
| `Vてくれる` | 别人为我做 | `母が弁当を作ってくれた。` | 主语是施惠者 |
| `Vてもらう` | 请别人为我做 | `駅員に道を教えてもらった。` | 主语是受益者 |
| `受身形（〜られる）` | 被… | `私は先生にほめられた。` | 助词 `に` 常考 |
| `使役形（〜させる）` | 让/使… | `母は子どもに野菜を食べさせる。` | 一类动词变形易错 |
| `使役受身（〜させられる）` | 被迫… | `上司に残業させられた。` | 含不情愿语感 |
| `お/ご〜ください` | 请您…（尊敬） | `こちらにお名前をお書きください。` | ≠てください |
| `お/ご〜になる` | （您）…（尊敬） | `社長はもうお帰りになった。` | 只用于对方 |
| `お/ご〜する` | 我来…（谦让） | `私がご案内します。` | 只用于自己 |
| `させていただく` | 谨让我… | `本日は説明させていただきます。` | 过度用显啰嗦 |

## 九、义务・不得已・能力限制

| 文法 | 含义 | 例句 | 语体 |
|---|---|---|---|
| `〜なければならない` | 必须… | `明日までに提出しなければならない。` | 标准 |
| `〜ないといけない` | 必须…（口语） | `明日は早く起きないといけません。` | 口语 |
| `〜ざるを得ない` | 不得不… | `急用で帰らざるを得ない。` | 书面 |
| `〜ないわけにはいかない` | 不能不… | `親だから心配しないわけにはいかない。` | 书面 |
| `〜ずにはいられない` | 忍不住… | `その話を聞いて笑わずにはいられなかった。` | 书面 |
| `〜ないではいられない` | 忍不住… | `彼のことを考えないではいられない。` | 书面 |
| `〜かねる` | 难以…（礼貌拒绝） | `その件はお答えしかねます。` | 商务 |
| `〜ようがない` | 无法… | `情報がなくて、判断しようがない。` | 无奈 |
| `〜ようにも〜ない` | 想…也无法… | `忙しくて、休もうにも休めない。` | 无奈 |
| `〜得る/得ない` | 可能/不可能 | `それは十分あり得る話だ。` | 书面 |
| `〜まい` | 不会…/不打算… | `彼は約束を破るまい。` | 古典书面 |
| `〜ずにはすまない` | 不…不行 | `本人に謝らずにはすまない。` | 书面责任 |
| `〜ないではすまない` | 不…难以解决 | `説明しないではすまない状況だ。` | 书面责任 |

## 十、限定・强调

| 文法 | 含义 | 例句 | 对比要点 |
|---|---|---|---|
| `〜しか〜ない` | 只有… | `財布に千円しかない。` | 后项必须否定 |
| `〜だけ` | 仅… | `今日は見るだけです。` | 比しか中性 |
| `〜ばかり` | 尽是…/刚… | `最近、甘い物ばかり食べている。` | 含负面语感 |
| `〜さえ` | 连…都 | `名前さえ書けばいいです。` | 最低要素 |
| `〜すら` | 连…都（书面） | `名前すら知らない。` | 比さえ更书面 |
| `〜まで` | 甚至… | `先生まで反対した。` | 意外范围强调 |
| `〜こそ` | 正是… | `今こそ行動すべきだ。` | 强调焦点 |
| `〜にすぎない` | 只不过… | `それは噂にすぎない。` | 书面降级 |
| `〜にほかならない` | 正是… | `これは努力の結果にほかならない。` | 书面断定 |
| `〜ばかりか` | 不但…反而… | `遅刻したばかりか、謝りもしなかった。` | 后项程度更高 |
| `〜ばかりでなく` | 不仅…而且… | `英語ばかりでなく中国語も話せる。` | 正式连接 |
| `〜のみならず` | 不仅…还… | `国内のみならず海外でも人気だ。` | 书面 |
| `〜に限らず` | 不限于… | `年齢に限らず応募できます。` | 范围扩展 |
| `〜を問わず` | 不论… | `経験を問わず採用します。` | 常接二元名词 |
| `〜にかかわらず` | 不管…都… | `天気にかかわらず開催します。` | ≠にもかかわらず（逆接） |
| `〜に限って` | 偏偏…（抱怨） | `急ぐ日に限って電車が遅れる。` | 含抱怨语气 |
| `〜に限る` | 最好是… | `暑い日は冷たい麺に限る。` | 主观推荐 |

## 十一、评价・规范（N2书面）

| 文法 | 含义 | 例句 | 场景 |
|---|---|---|---|
| `〜ものだ` | 本来就…/感慨 | `人は年を取るものだ。` | 常识/感叹/回忆 |
| `〜ものではない` | 不该… | `人を見た目で判断するものではない。` | 道德规劝 |
| `〜というものだ` | 这才叫… | `最後まであきらめないのが努力というものだ。` | 评价书面 |
| `〜べきだ` | 应该… | `約束は守るべきだ。` | 论述立场 |
| `〜べきではない` | 不应… | `そんな言い方をするべきではない。` | 提醒批评 |
| `〜ことだ` | 最好…（建议） | `早く寝ることだ。` | 指导性表达 |
| `〜ことはない` | 没必要… | `心配することはない。` | 安慰 |
| `〜とは限らない` | 未必… | `日本人がみんな寿司好きとは限らない。` | 客观论述 |
| `〜というものではない` | 不能简单说… | `高ければいいというものではない。` | 反绝对化 |
| `〜だけあって` | 不愧是… | `プロだけあって、説明がわかりやすい。` | 肯定评价 |
| `〜だけに` | 正因为…更… | `人気店だけに、いつも混んでいる。` | 原因强化 |
| `〜からして` | 从…来看就… | `この発音からして、彼は上級者だ。` | 局部→整体判断 |
| `〜かのようだ` | 就像…一样 | `彼は何も知らないかのような顔をしている。` | 比喻推测 |
| `〜までもない` | 用不着… | `これは説明するまでもない。` | "明显无需" |
| `〜かと思いきや` | 本以为…结果却… | `静かかと思いきや、急に騒ぎ出した。` | 戏剧性反转 |
| `〜ないことはない` | 也不是不… | `食べないことはないが、好きではない。` | 委婉部分肯定 |

## 十二、N3高频易混 vs N2高频易混

### N3 关键区别
| 易混项 | 区别核心 | 对照例句 |
|---|---|---|
| `ようだ` vs `らしい` | 眼前证据 vs 传闻/典型性 | `空が暗い、雨が降るようだ。/ 田中さんは来るらしい。` |
| `ことになる` vs `ことにする` | 外部决定 vs 自我决定 | `転勤することになった。/ 毎日走ることにした。` |
| `ために` vs `ように` | 目的行为 vs 能力/状态目标 | `合格するために勉強する。/ 忘れないようにメモする。` |
| `ば` vs `たら` | 书面逻辑 vs 口语通用 | `安ければ買う。/ 安かったら買う。` |
| `ので` vs `から` | 客观礼貌 vs 主观直接 | `体調が悪いので休みます。/ 疲れたから寝る。` |
| `受身` vs `てもらう` | 被动受害 vs 受益 | `自転車を壊された。/ 自転車を直してもらった。` |
| `ばかり` vs `だけ` | 偏多/不满 vs 中性限定 | `甘い物ばかり食べる。/ 水だけ飲む。` |
| `のに（逆接）` vs `のに（用途）` | 明明…却 vs 用来… | `薬を飲んだのに治らない。/ 切るのに便利だ。` |

### N2 关键区别
| 易混项 | 区别核心 | 对照例句 |
|---|---|---|
| `わけではない` vs `わけがない` | 部分否定 vs "不可能" | `便利なわけではない。/ できるわけがない。` |
| `にしては` vs `わりに` | 基准不符 vs 对比落差 | `初心者にしては上手。/ 値段のわりに高品質。` |
| `ものの` vs `とはいえ` | 叙述逆接 vs 评论逆接 | `就職したものの…/ 春とはいえ…` |
| `ばかりに` vs `せいで` | 懊悔感 vs 一般负面原因 | `油断したばかりに失敗した。/ 雨のせいで中止。` |
| `ざるを得ない` vs `ないわけにはいかない` | 书面强硬 vs 义务柔和 | `中止せざるを得ない。/ 行かないわけにはいかない。` |
| `ずにはいられない` vs `ないではいられない` | 书面凝练 vs 口语略强 | `泣かずにはいられない。/ 泣かないではいられない。` |
| `に基づいて` vs `をもとに` | 正式客观（法律） vs 数据分析 | `法律に基づいて。/ データをもとに分析。` |
| `につれて` vs `に従って` | 自然渐变 vs 渐变+规则遵守 | `春になるにつれて…/ 指示に従って…` |
| `にかかわらず` vs `にもかかわらず` | 不管/无论 vs 尽管…却 | `天気にかかわらず開催。/ 雨にもかかわらず来た。` |

## 速记口诀

```
推断强弱：かもしれない < でしょう < らしい < はずだ < にちがいない
负面原因：せいで（一般）< ばかりに（懊悔/后悔）
逆接书面：が < のに < ものの < とはいえ < にもかかわらず
书面程度：口语 ☆ → 一般 ☆☆ → 正式书面 ☆☆☆
授受方向：あげる（给出）/ もらう（收到）/ くれる（给我）
```
