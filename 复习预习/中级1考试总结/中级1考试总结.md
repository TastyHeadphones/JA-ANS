# 第一部分　N3 语法搭配与例句

> 体例：`文型`｜释义 — **接续** ＋ 例句（注假名）。按功能分组，考点高频在“接续细节”和“近义辨析”。
> 「普通形」= 简体形；な形・名词在 `のに／ので／はずだ／ようだ` 等前要不要 `な/の/だ`，是最常见陷阱。

## 1. 推量・样态・传闻

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `Vそうだ`（样态） | 看起来要…/快… | ます形去ます＋そうだ：`雨（あめ）が降（ふ）りそうです。`／い形去い＋そう：`おいしそう`／`いい→よさそう、ない→なさそう` |
| `〜そうだ`（传闻） | 听说… | 普通形＋そうだ：`田中（たなか）さんは結婚（けっこん）するそうです。`（不接ます形） |
| `〜ようだ` | 好像…（依所感/证据） | 動詞普通形・い形＋ようだ／な形＋な／名詞＋の：`誰（だれ）かいるようだ。``まるで夢（ゆめ）のようだ。` |
| `〜みたいだ` | 好像…（口语） | 普通形（な形・名词去だ）＋みたい：`風邪（かぜ）をひいたみたいだ。` |
| `〜らしい` | 听说／有典型特点 | 普通形（な形・名词去だ）＋らしい：`彼（かれ）は上手（じょうず）らしい。` |
| `〜かもしれない` | 可能… | 普通形＋かもしれない：`明日（あした）は雪（ゆき）かもしれない。` |
| `〜にちがいない` | 一定… | 普通形＋にちがいない：`合格（ごうかく）したにちがいない。` |
| `〜はずだ／はずがない` | 理应…／不可能… | 動・い形＋はずだ；な形＋な；名詞＋の：`今日（きょう）は休（やす）みのはずだ。`／`言（い）うはずがない。` |
| `〜わけだ／わけではない／わけがない` | 难怪／并非都／不可能 | 普通形接：`上手（じょうず）なわけだ。`／`みんな好（す）きなわけではない。`／`行（い）けるわけがない。` |

## 2. 时间・阶段・过程

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜ところだ` | 正要/正在/刚（三态） | 辞書形＋ところ（正要）`出（で）かけるところ`／ている＋ところ（正在）／た形＋ところ（刚）`食（た）べたところ` |
| `Vたばかりだ` | 刚…（主观） | た形＋ばかり：`日本（にほん）に来（き）たばかりです。` |
| `〜最中だ` | 正在…的关键过程 | ている＋最中／名詞＋の＋最中：`会議（かいぎ）の最中（さいちゅう）に地震（じしん）が起（お）きた。` |
| `Vるうちに／Vないうちに` | 趁…/趁还没… | 辞書形・ている＋うちに `熱（あつ）いうちに食（た）べて`／ない形＋うちに `暗（くら）くならないうちに帰（かえ）ろう` |
| `〜間に／〜間` | 在…期间 | 辞書形・ている＋間に（某时点）`いる間（あいだ）に登（のぼ）りたい`／ている＋間（整段持续） |
| `〜て以来` | 自从…以来 | て形/名詞＋以来：`来（き）て以来（いらい）、毎日（まいにち）勉強（べんきょう）している。` |
| `Vようとする` | 正要/试图… | 意向形＋とする：`家（いえ）を出（で）ようとしたとき、電話（でんわ）が来（き）た。` |

## 3. 条件・原因・逆接

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜ば` | 如果…（偏书面逻辑） | ば形／い形→ければ／な形・名詞＋なら：`急（いそ）げば間（ま）に合（あ）う。` |
| `〜たら` | 如果…（口语最通用） | た形＋ら：`時間（じかん）があったら連絡（れんらく）して。` |
| `Vと` | 一…就…（规律/必然） | 辞書形・ない形＋と：`春（はる）になると花（はな）が咲（さ）く。` |
| `〜ても` | 即使…也… | て形＋も／くても／でも：`高（たか）くても買（か）う。` |
| `〜ので／から` | 因为（客观礼貌/主观直接） | 普通形＋ので（な形＋な）`痛（いた）いので休（やす）む`／＋から `疲（つか）れたから寝（ね）る` |
| `〜ため（に）` | 因为…（书面） | 動・い形＋ため／な形＋な／名詞＋の：`台風（たいふう）のため電車（でんしゃ）が止（と）まった。` |
| `〜おかげで／せいで／せいか` | 多亏/都怪/或许因为 | 普通形接：`おかげで合格（ごうかく）`／`雨（あめ）のせいで中止（ちゅうし）`／`寝不足（ねぶそく）のせいか頭（あたま）が痛（いた）い` |
| `〜のに`（逆接） | 尽管…却… | 普通形＋のに（な形＋な・名詞＋な）：`薬（くすり）を飲（の）んだのにまだ熱（ねつ）がある。` |
| `〜くせに` | 明明…却…（责备） | 普通形／な形＋な／名詞＋の＋くせに：`知（し）っているくせに教（おし）えてくれない。` |

## 4. 目的・变化・习惯・决定

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `Vように` | 为了能…（状态/能力目标） | 辞書形・可能形・ない形＋ように：`忘（わす）れないようにメモする。` |
| `Vために／Nのために` | 为了…（意志可控目的） | 辞書形＋ために／名詞＋の＋ために：`働（はたら）くために勉強（べんきょう）する。` |
| `Vようになる` | 变得会…/养成 | 辞書形・可能形＋ようになる：`漢字（かんじ）が読（よ）めるようになった。` |
| `Vようにする` | 努力做到… | 辞書形・ない形＋ようにする：`毎日（まいにち）歩（ある）くようにしている。` |
| `〜ことになる／ことにする／ことにしている` | 外部决定/自我决定/坚持 | `転勤（てんきん）することになった`／`運動（うんどう）することにする`／`六時（ろくじ）に起（お）きることにしている` |
| `Vように言う／頼む` | 让某人务必… | 辞書形・ない形＋ように＋言う/頼む：`早（はや）く帰（かえ）るように言（い）われた。` |

## 5. 限定・比较・程度・并列

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜しか〜ない` | 只有…（后必否定） | 名詞・辞書形＋しか〜ない：`千円（せんえん）しかありません。` |
| `〜だけ` | 仅…（中性限定） | 名詞＋だけ／普通形＋だけ：`見（み）るだけです。` |
| `〜ばかり` | 尽是…/老…/刚… | 名詞＋ばかり（尽是）／て形＋ばかりいる（老）／た形＋ばかり（刚） |
| `〜ほど／〜ほど〜ない` | 到…程度／不如… | `死（し）ぬほど忙（いそが）しい。`／`去年（きょねん）ほど寒（さむ）くない。` |
| `〜くらい／ぐらい` | 大约/达到…程度 | 名詞・数量・普通形＋くらい：`十分（じゅっぷん）ぐらいかかる。` |
| `〜さえ／〜でも` | 连…都／即使·举例 | 名詞＋さえ `名前（なまえ）さえ書（か）ければいい`／名詞＋でも `子（こ）どもでもわかる` |
| `〜とか／〜たり〜たり／〜し／〜や〜など` | 列举·并列 | `映画（えいが）とか買（か）い物（もの）とか`／`掃除（そうじ）したり料理（りょうり）したり`／`安（やす）いしおいしいし` |

## 6. 授受・被动・使役・敬语（高频）

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `Vてあげる／くれる／もらう` | 为别人/别人为我/请别人 | `教（おし）えてあげた`／`作（つく）ってくれた`／`教（おし）えてもらった` |
| 受身形 `〜（ら）れる` | 被…（动作主用に） | `先生（せんせい）にほめられた。`／`ケーキを食（た）べられた。` |
| 使役形 `〜（さ）せる` | 让/使… | `野菜（やさい）を食（た）べさせる。`／`留学（りゅうがく）させる。` |
| 使役受身 `〜（さ）せられる` | 被迫…（含不情愿） | `上司（じょうし）に残業（ざんぎょう）させられた。` |
| `お／ご〜になる` | 尊敬（对方动作） | お＋ます形＋になる／ご＋漢語＋になる：`お帰（かえ）りになる。` |
| `お／ご〜する` | 自谦（自己动作） | お＋ます形＋する：`お荷物（にもつ）をお持（も）ちします。` |
| `お／ご〜ください` | 请您… | `こちらにお書（か）きください。` |
| `〜させていただく` | 谨让我… | 使役て形＋いただく：`説明（せつめい）させていただきます。` |

## 7. 书面连接・论述表达

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜について／に関して` | 关于… | 名詞＋について（に関して更正式）：`教育（きょういく）について調（しら）べる。` |
| `〜に対して` | 对于…/与…相对 | 名詞＋に対して：`質問（しつもん）に対（たい）して答（こた）える。` |
| `〜にとって` | 对…而言 | 名詞＋にとって：`私（わたし）にとって家族（かぞく）が大切（たいせつ）。` |
| `〜として` | 作为… | 名詞＋として：`教師（きょうし）として働（はたら）く。` |
| `〜によると／によって` | 根据/由于·因…而异 | `天気予報（てんきよほう）によると雨（あめ）`／`国（くに）によって習慣（しゅうかん）が違（ちが）う` |
| `〜結果` | …的结果 | た形＋結果／名詞＋の＋結果：`考（かんが）えた結果（けっか）、留学（りゅうがく）を決（き）めた。` |
| `〜という／ということだ` | 所谓/听说·即 | `「もったいない」という言葉（ことば）`／`値上（ねあ）げするということだ` |

## 8. 状态・尝试・倾向

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `Vてある／Vておく` | 结果状态/预先做 | `メモが貼（は）ってある`／`資料（しりょう）を印刷（いんさつ）しておく` |
| `Vてみる` | 试试… | て形＋みる：`方法（ほうほう）を試（ため）してみる。` |
| `Vることがある／ことはない` | 有时会…/不必… | `食（た）べないことがある`／`急（いそ）ぐことはない` |
| `Vてもかまわない` | …也没关系 | `先（さき）に帰（かえ）ってもかまわない。` |
| `Vないといけない／ないわけにはいかない` | 必须/不能不 | `起（お）きないといけない`／`行（い）かないわけにはいかない` |
| `Nっぽい／N向け／N向き` | 倾向/面向/适合 | `子（こ）どもっぽい`／`初心者（しょしんしゃ）向（む）け`／`私（わたし）向（む）き` |

## 9. 高频易混辨析（必背）

| 易混项 | 区别 | 对照例句 |
|---|---|---|
| `ようだ` vs `らしい` | 前者据眼前证据；后者含传闻/典型性 | `雨（あめ）が降（ふ）るようだ。`／`田中（たなか）さんは来（く）るらしい。` |
| `ことになる` vs `ことにする` | 外部决定 vs 自我决定 | `転勤（てんきん）することになった。`／`走（はし）ることにした。` |
| `ために` vs `ように` | 意志可控目的 vs 能力/状态目标 | `合格（ごうかく）するために。`／`忘（わす）れないように。` |
| `ば` vs `たら` | 书面逻辑 vs 口语通用 | `安（やす）ければ買（か）う。`／`安（やす）かったら買（か）う。` |
| `ので` vs `から` | 客观礼貌 vs 主观直接 | `悪（わる）いので休（やす）む。`／`疲（つか）れたから寝（ね）る。` |
| `ばかり` vs `だけ` | 偏多/不满 vs 中性限定 | `甘（あま）い物（もの）ばかり。`／`水（みず）だけ。` |
| 受身 vs `てもらう` | 常带被害 vs 受益 | `壊（こわ）された。`／`直（なお）してもらった。` |

# 第二部分　中級から学ぶ日本語　第1〜4課

> 每课结构：主题 → 重点词汇（注假名）→ 核心文型（接续＋例句）→ 课文大意。
> 文型是考试主战场，务必能“看接续、辨语义、会造句”。

## 第1課　まなぶ ―「なぞなぞ」

**主题**：外语学习与文化差异 —— 学外语不只是学单词语法，更是拓宽看世界的方式。

**重点词汇**

| 词汇 | 释义 | 词汇 | 释义 |
|---|---|---|---|
| 始（はじ）める | 开始 | 心配（しんぱい）する | 担心 |
| それでも | 尽管如此 | 親（おや） | 父母/家长 |
| 口（くち）を出（だ）す | 插嘴/多管闲事 | ある日（ひ） | 某一天 |
| 〜だって／〜って | 听说（口语） | なぞなぞ | 谜语 |
| 手（て）を止（と）める | 停下手中的事 | 場面（ばめん） | 场面/情景 |
| 口々（くちぐち）に | 你一言我一语地 | そこで | 于是 |
| 相手（あいて） | 对方 | 文化（ぶんか） | 文化 |
| 学（まな）ぶ | 学习/领会 | 広（ひろ）げる | 拓宽 |

**核心文型**

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜ようになる／〜なくなる` | 能力·习惯·状态的变化 | 辞書形＋ようになる `漢字（かんじ）が読（よ）めるようになった`／ない形→なくなる `夜更（よふ）かししなくなった` |
| `〜がる／〜がっている` | 描述第三者表面的情绪/愿望 | ます形词干・い形词干＋がる：`子（こ）どもがゲームをほしがっている。`（不用于自己） |
| `Vては／Vると＋いけない` | 担心某动作引出不良结果 | `言（い）いすぎてはいけない`／`風邪（かぜ）をひくといけないから早（はや）く寝（ね）た` |
| `Vないで／Vずにいる` | 持续处于未做某事的状态 | ない形＋ないでいる／ずにいる：`忙（いそが）しくて話（はな）せずにいる。`（ずに更书面） |

**课文大意**：母亲担心孩子学英语会变讨厌、又怕插嘴惹烦，便默默观察。某天孩子讲起英语课上“外国孩子睡前洗脸”的文化差异场景，母亲意识到孩子是在通过英语学习更广阔的世界，于是开始期待孩子下次的分享。

## 第2課　みつける ―「花（はな）の名前（なまえ）」

**主题**：重新发现身边事物 —— 习惯坐车后不再走路，偶然步行却发现了平时忽略的风景。

**重点词汇**

| 词汇 | 释义 | 词汇 | 释义 |
|---|---|---|---|
| 交通機関（こうつうきかん） | 交通工具 | 便利（べんり）な | 方便 |
| 利用（りよう）する | 利用/使用 | この間（あいだ） | 前几天 |
| 見当（けんとう）をつける | 估摸方向 | 気（き）が進（すす）まない | 不太情愿 |
| 宣伝（せんでん）する | 宣传 | 景色（けしき） | 风景 |
| 咲（さ）く | （花）开 | 周（まわ）り | 周围 |
| 会釈（えしゃく）する | 微微点头致意 | 通（かよ）う | 往来/通勤通学 |
| 方向（ほうこう） | 方向 | 進（すす）む | 前进/进步 |

**核心文型**

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜もせずに〜てしまった` | 连X都没做（却）做了Y（疏忽/违常） | ます形词干＋もせずに：`かたづけもせずに出（で）かけてしまった。` |
| `〜ことにする／した` | 主动做出决定 | 辞書形・ない形＋ことにする：`明日（あした）から早（はや）く起（お）きることにします。` |
| `〜ことにしている／していた` | 已养成的个人习惯/坚守的规则 | `甘（あま）い物（もの）は食（た）べないことにしている。`（区别于ようにしている“努力做”） |
| `〜ながら` | 两动作同时进行（主语相同，主动作在后） | ます形去ます＋ながら：`音楽（おんがく）を聞（き）きながら宿題（しゅくだい）をする。` |
| `〜ようと思いながら、〜できない` | 想做却一直没做到（遗憾） | 意向形＋と思いながら＋なかなか〜ない：`書（か）こうと思（おも）いながら、ずっと書（か）けないでいる。` |

**课文大意**：来日后因交通便利渐渐不走路。某天地铁停运被迫步行，途中发现大公园里盛开的花、认出学过的花名、看到远山与熟悉的店铺、与路人会釈。到站虽略累，却感到“走路真好”，决定以后有空就走路。

## 第3課　たべる ―「ごちそう」

**主题**：饮食与家庭关系 —— 全家围坐一桌共餐的风景正在消失。

**重点词汇**

| 词汇 | 释义 | 词汇 | 释义 |
|---|---|---|---|
| 準備（じゅんび）する | 准备 | 温（あたた）める | 加热 |
| 変（か）わる／変化（へんか）する | 变化 | 姿（すがた） | 样子/身影 |
| 囲（かこ）む | 围坐 | 昔（むかし） | 从前 |
| 声（こえ）を上（あ）げる | 发出（笑）声 | あふれる | 充满/溢满 |
| 栄養（えいよう） | 营养 | バランス | 均衡 |
| 風景（ふうけい） | 景象 | ごちそう | 盛宴/丰盛饭菜 |

**核心文型**

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜うちに` | 趁某状态持续做某事；或不知不觉发生变化 | 辞書形・ている・い形・な形＋な・N＋の＋うちに：`朝（あさ）のうちに宿題（しゅくだい）を終（お）わらせる。``読（よ）んでいるうちに眠（ねむ）ってしまった。` |
| `〜うちは` | 在某状态持续期间，某条件成立（暗含之后会变） | `朝（あさ）のうちはすずしいが、昼（ひる）は暑（あつ）くなる。` |

**课文大意**：小学生独自对着电视吃饭、初中生姐姐热好饭匆匆吃完去学英语、父亲深夜才归、母亲最后一个吃 —— 全家围坐的风景渐渐消失。超市食物充足又方便，但“妈妈花时间做饭、一家人围坐”如今几乎只能说是“怀念的风景”了。

## 第4課　たとえる ―「猫（ねこ）に小判（こばん）」

**主题**：谚语与比喻 —— 用动物（尤其猫）打比方，简洁有趣地传情达意。

**重点词汇**

| 词汇 | 释义 | 词汇 | 释义 |
|---|---|---|---|
| ことわざ | 谚语 | 紹介（しょうかい）する | 介绍 |
| 目（め）が回（まわ）る | 忙得晕头转向 | 手（て）を借（か）りる | 借助帮助 |
| それほど（〜ない） | 没那么… | 役（やく）に立（た）つ | 有用 |
| 額（ひたい） | 额头（借指面积小） | 狭（せま）い | 狭窄 |
| たとえる | 比喻/打比方 | 小判（こばん） | 江户椭圆金币 |
| 価値（かち） | 价值 | 手（て）に入（い）れる | 弄到手 |
| 続（つづ）ける | 持续 | 口（くち）に出（だ）す | 说出口 |

**核心文型**

| 文型 | 释义 | 接续 ＋ 例句 |
|---|---|---|
| `〜ほど〜はない／いない` | 没有比…更…（最高级·否定式） | N＋ほど＋N＋はない／〜ほど＋い形＋ものはない：`チンさんほどよく勉強（べんきょう）する学生（がくせい）はいない。` |
| `〜ほど`（程度） | 达到…的程度（常夸张） | 辞書形・た形・形容词＋ほど：`おなかが痛（いた）くなるほど笑（わら）った。` |
| `〜ないでほしい` | 希望对方不要做某事（委婉） | ない形＋ないでほしい：`今日（きょう）は帰（かえ）らないでほしい。` |
| `〜とは思えない` | 无法相信/不觉得…（主观强否定） | 普通形・N＋とは思えない：`練習（れんしゅう）しなければ話（はな）せるようになるとは思（おも）えない。`（常配いくら〜ても/たとえ〜ても） |

**课文大意**：日语里有许多以动物为主题的谚语，尤以“猫”的最有趣：「猫の手も借りたい」（忙到想借猫的手）、「猫の額ほどの庭」（小得像猫额头的院子）、「猫に小判」（再贵重的东西对不懂的人也没用）。除猫外，狗牛马等与人长期共生的动物也常入谚语，因为比喻能简洁有趣地传意、世代流传。
