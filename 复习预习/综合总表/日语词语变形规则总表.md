# 日语词语变形规则总表（现代日语全覆盖版）

> 用途：考试速查 + 系统复习。  
> 范围：现代标准日语（初级到中高级考试常用活用）。  
> 说明：这里的“所有”按教学和考试体系中的主要变形规则整理，不含古典日语活用。每条规则都配例句。

---

## 0. 先看总分类

| 词类 | 是否活用 | 主要变形 |
|---|---|---|
| 动词(どうし) | 活用 | `ます形`、`て形`、`た形`、`ない形`、`可能形`、`受身形`、`使役形`、`使役受身形`、`命令形`、`意向形`、`条件形` 等 |
| い形容词(けいようし) | 活用 | `く`、`くて`、`かった`、`くない`、`ければ`、`そう`、`すぎる` 等 |
| な形容词(けいようどうし) | 轻活用 | `だ/です`、`ではない`、`だった`、`なら`、`に`、`そう`、`すぎる` 等 |
| 名词 + 判断词 | 轻活用 | `だ/です`、`ではない`、`だった`、`なら`、`で`、`に` 等 |

---

## 1. 动词基础：三大类与五段行变换

## 1.1 三大类

| 类别 | 识别方式 | 例词 |
|---|---|---|
| 五段动词 | 词尾在 `う段`，活用时词尾按 `あ/い/う/え/お` 变换 | `書(か)く` `読(よ)む` `話(はな)す` `待(ま)つ` |
| 一段动词 | 多为 `〜える/〜いる` 且去 `る` 直接接活用 | `食(た)べる` `見(み)る` `起(お)きる` |
| 不规则 | `する`、`来(く)る` 及其复合 | `勉強(べんきょう)する` `持(も)って来(く)る` |

## 1.2 五段行变换示意

| 词尾 | ない形(あ段) | ます形(い段) | 辞书形(う段) | 条件形(え段) | 意向形(お段) |
|---|---|---|---|---|---|
| `く` | `か` | `き` | `く` | `け` | `こ` |
| `ぐ` | `が` | `ぎ` | `ぐ` | `げ` | `ご` |
| `す` | `さ` | `し` | `す` | `せ` | `そ` |
| `つ` | `た` | `ち` | `つ` | `て` | `と` |
| `ぬ` | `な` | `に` | `ぬ` | `ね` | `の` |
| `ぶ` | `ば` | `び` | `ぶ` | `べ` | `ぼ` |
| `む` | `ま` | `み` | `む` | `め` | `も` |
| `る` | `ら` | `り` | `る` | `れ` | `ろ` |
| `う` | `わ` | `い` | `う` | `え` | `お` |

例句：`毎日(まいにち)日本語(にほんご)を書(か)きます。`

## 2. 动词基本活用（敬体/普通体）

## 2.1 `ます形` 系列

| 规则 | 公式 | 例句 |
|---|---|---|
| 非过去肯定 | `ます形 + ます` | `わたしは毎朝(まいあさ)コーヒーを飲(の)みます。` |
| 非过去否定 | `ます形 + ません` | `きょうは肉(にく)を食(た)べません。` |
| 过去肯定 | `ます形 + ました` | `昨日(きのう)、図書館(としょかん)で勉強(べんきょう)しました。` |
| 过去否定 | `ます形 + ませんでした` | `先週(せんしゅう)は運動(うんどう)しませんでした。` |

## 2.2 普通体（辞书/ない/た）

| 规则 | 公式 | 例句 |
|---|---|---|
| 非过去肯定 | `辞書形` | `毎晩(まいばん)本(ほん)を読(よ)む。` |
| 非过去否定 | `ない形` | `あしたは学校(がっこう)へ行(い)かない。` |
| 过去肯定 | `た形` | `きのう映画(えいが)を見(み)た。` |
| 过去否定 | `ない形 + なかった` | `昨夜(さくや)はあまり寝(ね)なかった。` |

---

## 3. 动词连接形：`て形 / た形`

## 3.1 五段动词音变规则

| 词尾 | て形 | た形 | 例词 | 例句 |
|---|---|---|---|---|
| `う/つ/る` | `って` | `った` | `買(か)う` `待(ま)つ` `帰(かえ)る` | `駅(えき)で友達(ともだち)を待(ま)っている。` |
| `む/ぶ/ぬ` | `んで` | `んだ` | `読(よ)む` `遊(あそ)ぶ` `死(し)ぬ` | `本(ほん)を読(よ)んで寝(ね)た。` |
| `く` | `いて` | `いた` | `書(か)く` | `名前(なまえ)を書(か)いてください。` |
| `ぐ` | `いで` | `いだ` | `泳(およ)ぐ` | `川(かわ)で泳(およ)いだことがある。` |
| `す` | `して` | `した` | `話(はな)す` | `先生(せんせい)と話(はな)して安心(あんしん)した。` |

例外：`行(い)く` 的 `て形/た形` 是 `行って / 行った`。  
例句：`毎日(まいにち)学校(がっこう)へ行(い)っている。`

## 3.2 一段与不规则

| 类别 | て形 | た形 | 例句 |
|---|---|---|---|
| 一段 | `〜て` | `〜た` | `野菜(やさい)を食(た)べて健康(けんこう)になった。` |
| する | `して` | `した` | `宿題(しゅくだい)をしてから寝(ね)ます。` |
| 来る | `来(き)て` | `来(き)た` | `友達(ともだち)が家(いえ)に来(き)た。` |

## 4. `ない形`（否定词干）

| 类别 | 规则 | 例句 |
|---|---|---|
| 五段 | `う段 -> あ段 + ない`（`う -> わ`） | `煙草(たばこ)は吸(す)わない。` |
| 一段 | `る -> ない` | `朝(あさ)ごはんを食(た)べない日(ひ)もある。` |
| する | `しない` | `きょうは残業(ざんぎょう)しない。` |
| 来る | `来(こ)ない` | `あした彼(かれ)は来(こ)ない。` |

---

## 5. 可能形（能做）

| 类别 | 规则 | 例句 |
|---|---|---|
| 五段 | `う段 -> え段 + る` | `わたしは日本語(にほんご)が話(はな)せる。` |
| 一段 | `る -> られる`（口语也有 `れる`） | `この字(じ)は小(ちい)さくて見(み)られない。` |
| する | `できる` | `一人(ひとり)で全部(ぜんぶ)はできない。` |
| 来る | `来(こ)られる` | `日曜日(にちようび)なら来(こ)られる。` |

易错点：可能形常搭配助词 `が`。  
例句：`辛(から)い物(もの)は食(た)べられます。`

---

## 6. 受身形（被动）

| 类别 | 规则 | 例句 |
|---|---|---|
| 五段 | `う段 -> あ段 + れる` | `弟(おとうと)にケーキを食(た)べられた。` |
| 一段 | `る -> られる` | `先生(せんせい)に名前(なまえ)を呼(よ)ばれた。` |
| する | `される` | `この問題(もんだい)はよく研究(けんきゅう)されている。` |
| 来る | `来(こ)られる` | `急(きゅう)に来(こ)られて困(こま)った。` |

受害被动例句：`雨(あめ)に降(ふ)られて、服(ふく)がぬれた。`

| 来る | `来(こ)させる` | `親(おや)は子どもを早(はや)く家(いえ)に来(こ)させた。` |

## 9. 命令、禁止、意向

## 9.1 命令形

| 类别 | 规则 | 例句 |
|---|---|---|
| 五段 | `う段 -> え段` | `早(はや)く行(い)け。` |
| 一段 | `る -> ろ / よ` | `もっと食(た)べろ。` |
| する | `しろ / せよ` | `静(しず)かにしろ。` |
| 来る | `来(こ)い` | `ここへ来(こ)い。` |

## 9.2 禁止形

规则：`辞書形 + な`  
例句：`ここで写真(しゃしん)を撮(と)るな。`

## 9.3 意向形（“要不…吧”）

| 类别 | 规则 | 例句 |
|---|---|---|
| 五段 | `う段 -> お段 + う` | `週末(しゅうまつ)は海(うみ)へ行(い)こう。` |
| 一段 | `る -> よう` | `早(はや)く寝(ね)よう。` |
| する | `しよう` | `そろそろ勉強(べんきょう)しよう。` |
| 来る | `来(こ)よう` | `あした早(はや)く来(こ)よう。` |

## 10. 条件与假定：`ば / たら / と / なら`

## 10.1 `ば` 条件

| 类别 | 规则 | 例句 |
|---|---|---|
| 五段 | `う段 -> え段 + ば` | `時間(じかん)があれば、映画を見ます。` |
| 一段 | `る -> れば` | `安(やす)ければ買(か)います。` |
| する | `すれば` | `説明(せつめい)すればわかります。` |
| 来る | `来(く)れば` | `彼(かれ)が来(く)れば始(はじ)めます。` |
| ない形 | `なければ` | `急(いそ)がなければ大丈夫です。` |

## 10.2 `たら` 条件

规则：`た形 + ら`  
例句：`駅(えき)に着(つ)いたら電話(でんわ)してください。`

## 10.3 `と` 条件

规则：`辞書形 + と`（自然结果、规则）  
例句：`春(はる)になると桜(さくら)が咲(さ)く。`

## 10.4 `なら` 提示条件

规则：`名词/普通体 + なら`  
例句：`京都(きょうと)へ行(い)くなら、秋(あき)がおすすめです。`

## 11. 推量、传闻、样态相关变形

| 规则 | 公式 | 例句 |
|---|---|---|
| 推量（普通） | `辞書形 + だろう` | `あしたは雨(あめ)が降(ふ)るだろう。` |
| 推量（礼貌） | `辞書形 + でしょう` | `試験(しけん)は難(むずか)しいでしょう。` |
| 传闻 | `普通形 + そうだ` | `天気予報(てんきよほう)によると、台風(たいふう)が来(く)るそうです。` |
| 样态（看起来）动词 | `ます形去ます + そうだ` | `雨が降(ふ)りそうです。` |
| 样态（看起来）い形 | `い去掉 + そうだ` | `このかばんは重(おも)そうです。` |
| 样态（看起来）な形 | `词干 + そうだ` | `元気(げんき)そうですね。` |

易错例：`いい -> よさそう`。  
例句：`この店(みせ)の料理(りょうり)はよさそうです。`

## 12. 动词派生助动词

## 12.1 `〜たい`（想做）

规则：`ます形 + たい`（按い形容词活用）  
例句：`来年(らいねん)日本(にほん)へ留学(りゅうがく)したい。`

## 12.2 `〜たがる`（第三人称想做）

规则：`ます形 + たがる`  
例句：`子(こ)どもは外(そと)で遊(あそ)びたがっている。`

## 12.3 `〜やすい / 〜にくい`

规则：`ます形 + やすい/にくい`  
例句：`このペンは書(か)きやすい。`  
例句：`この漢字(かんじ)は覚(おぼ)えにくい。`

## 12.4 `〜すぎる`

规则：`ます形 + すぎる`（动词）  
例句：`きのう飲(の)みすぎました。`

## 12.5 `〜始める / 〜続ける / 〜終わる`

规则：`ます形 + 補助動詞`  
例句：`雨(あめ)が降(ふ)り始(はじ)めた。`  
例句：`毎日(まいにち)勉強(べんきょう)し続(つづ)ける。`  
例句：`レポートを書(か)き終(お)わった。`

## 13. `て` 系复合变形

| 规则 | 公式 | 例句 |
|---|---|---|
| 进行/状态 | `ている` | `今(いま)ご飯(はん)を食(た)べている。` |
| 准备先做 | `ておく` | `会議(かいぎ)の前(まえ)に資料(しりょう)を読(よ)んでおく。` |
| 完成/遗憾 | `てしまう` | `宿題(しゅくだい)を忘(わす)れてしまった。` |
| 许可 | `てもいい` | `ここに座(すわ)ってもいいですか。` |
| 禁止 | `てはいけない` | `ここで泳(およ)いではいけません。` |
| 必须 | `なければならない` | `あした早(はや)く起(お)きなければならない。` |
| 请求 | `てください` | `もう一度(いちど)言(い)ってください。` |
| 尝试 | `てみる` | `この薬(くすり)を飲(の)んでみます。` |
| 授受（给我） | `てもらう` | `友達(ともだち)に手伝(てつだ)ってもらった。` |
| 授受（给别人） | `てあげる` | `母(はは)に料理(りょうり)を作(つく)ってあげた。` |
| 授受（别人给我） | `てくれる` | `先生が漢字を教(おし)えてくれた。` |
| 去做后回来 | `てくる` | `ちょっと水(みず)を買(か)ってきます。` |
| 做后离开 | `ていく` | `荷物(にもつ)を持(も)っていきます。` |

## 14. 受益与敬语方向变形（动词换形）

| 原形 | 谦让 | 尊他 | 例句 |
|---|---|---|---|
| `言(い)う` | `申(もう)す/申し上(あ)げる` | `おっしゃる` | `部長(ぶちょう)がそうおっしゃいました。` |
| `行(い)く/来(く)る` | `参(まい)る` | `いらっしゃる` | `先生(せんせい)はもういらっしゃいました。` |
| `食(た)べる/飲(の)む` | `いただく` | `召(め)し上(あ)がる` | `どうぞ召し上がってください。` |
| `見(み)る` | `拝見(はいけん)する` | `ご覧(らん)になる` | `資料(しりょう)を拝見しました。` |
| `する` | `いたす` | `なさる` | `わたしがいたします。` |
| `くれる` | - | `くださる` | `先生が本をくださいました。` |

补充规则：`お/ご + Vます语干 + になる`（尊他）  
例句：`少々(しょうしょう)お待(ま)ちください。`  

补充规则：`お/ご + Vます语干 + する`（谦让）  
例句：`荷物(にもつ)をお持(も)ちします。`

## 15. い形容词活用总表

## 15.1 基本变形

| 规则 | 公式 | 例句 |
|---|---|---|
| 非过去肯定 | `高(たか)い` | `この本(ほん)は高(たか)い。` |
| 非过去否定 | `〜くない` | `この本は高くない。` |
| 过去肯定 | `〜かった` | `昨日(きのう)は寒(さむ)かった。` |
| 过去否定 | `〜くなかった` | `去年(きょねん)は暑(あつ)くなかった。` |
| 连用 | `〜く` | `字(じ)を大(おお)きく書(か)いてください。` |
| 并列/原因 | `〜くて` | `この店は安(やす)くておいしい。` |
| 条件 | `〜ければ` | `安ければ買います。` |
| 推测样态 | `〜そう`（去`い`） | `この問題は難(むずか)しそうだ。` |
| 过度 | `〜すぎる` | `このかばんは重(おも)すぎる。` |

## 15.2 特殊词 `いい`

| 规则 | 变形 | 例句 |
|---|---|---|
| 现在肯定 | `いい` | `この店はいい。` |
| 否定 | `よくない` | `体調(たいちょう)がよくない。` |
| 过去 | `よかった` | `きのうは天気(てんき)がよかった。` |
| 过去否定 | `よくなかった` | `昨日の成績(せいせき)はよくなかった。` |
| 样态 | `よさそう` | `この靴(くつ)はよさそうです。` |

## 18. 高频复合规则（考试特别爱考）

| 规则 | 结构 | 例句 |
|---|---|---|
| 目的（能力目标） | `V可能形 + ように` | `日本語が話せるように毎日練習している。` |
| 目的（意志目标） | `V辞書形 + ために` | `大学(だいがく)に入(はい)るために勉強する。` |
| 避免结果 | `Vない + ように` | `遅(おく)れないように早(はや)く出る。` |
| 用途 | `V辞書形 + のに使う` | `はさみは紙(かみ)を切(き)るのに使う。` |
| 难易 | `Vます + にくい/やすい` | `この靴(くつ)は歩(ある)きにくい。` |
| 变化结果 | `N/A/V可能 + ようになる` | `ひらがなが読めるようになった。` |
| 习惯变化 | `V辞書/ない + ようにしている` | `毎日運動するようにしている。` |
| 被动进行 | `受身形 + ている` | `この寺(てら)は多(おお)くの人(ひと)に愛(あい)されている。` |
| 使役被动 | `使役受身` | `子どものころ毎日ピアノを練習させられた。` |

## 20. 常见易错总清单（带例句）

| 易错点 | 错误 | 正确 | 例句 |
|---|---|---|---|
| `う` 结尾的五段否定 | `買あない` | `買わない` | `高(たか)いから買わない。` |
| 可能与被动混用 | `食べれる`（考试写法争议） | `食べられる` | `辛い物は食べられない。` |
| `いい` 的活用 | `いいくない` | `よくない` | `この方法(ほうほう)はよくない。` |
| 样态 `そう` 与传闻 `そう` 混用 | `高いそう`（想表达“看起来高”） | `高そう` | `このかばんは高そうです。` |
| `しか` 后接肯定 | `一人しか来ました` | `一人しか来ませんでした` | `会議に一人しか来ませんでした。` |
| `ために` 与 `ように` 混用 | `話せるために` | `話せるように` | `話せるように練習する。` |
