## 1. 文法(ぶんぽう)总图

| 句型 | 核心意思 | 最常见考法 |
|---|---|---|
| `疑問詞(ぎもんし) + 普通形(ふつうけい) + か` | “……什么 / 为什么 / 多少” | 把直接问句改成嵌入句 |
| `普通形 + かどうか` | “是否……” | 判断题、填空题 |
| `Vてみる` | “试着……” | 试穿、试吃、试问、试做 |

## 2. `疑問詞(ぎもんし) + 普通形(ふつうけい) + か`

### 用法

把原来的问句，嵌进更大的句子里，变成“我想知道……”“请告诉我……”“我记不得……”。

### 结构

- `何(なに) / どこ / だれ / いつ / どうして / いくら / 何個(なんこ) ... + 普通形 + か`

### 典型例句

- `ケーキが何個(なんこ)あるか、教(おし)えてください。`
- `ワットさんがどうして来(こ)なかったか、わかりますか。`
- `このカメラがいくらだったか、覚(おぼ)えていません。`

### 常接的后项

- `知(し)っています / 知りません`
- `わかります / わかりません`
- `覚えています / 覚えていません`
- `教えてください`
- `聞(き)いてみます`

### 你要记住的点

- 句子里面已经有疑问词时，用 `か`，不用 `かどうか`。
- 这是“间接疑问”，不是句末直接发问，所以语气比原问句更嵌入、更完整。

### 易错点

- `何時(なんじ)かどうか` 这种说法不对。
- 有疑问词，就直接用：`何時か`、`どこか`、`だれが来(く)るか`。

## 3. `普通形(ふつうけい) + かどうか`

### 用法

表示“是否……”“会不会……”“对不对……”，也就是 yes/no 判断。

### 结构

- `V普通形 + かどうか`
- `い形容詞(けいようし) + かどうか`
- `な形容詞 / 名詞(めいし) + かどうか`

### 形式上的注意

- 这一课里，`名詞(めいし)` 和 `な形容詞(けいようし)` 前面一般直接接 `かどうか`，不特别写 `だ`。
- 例如：`必要(ひつよう)かどうか`、`静(しず)かかどうか`

### 典型例句

- `会議(かいぎ)に出席(しゅっせき)するかどうか、まだわかりません。`
- `この答(こた)えが正(ただ)しいかどうか、もう一度(いちど)考(かんが)えてください。`
- `この荷物(にもつ)が必要(ひつよう)かどうか、確認(かくにん)してください。`

### 和 `〜か` 的区别

| 句型 | 用在什么时候 |
|---|---|
| `〜か` | 句中已经有“什么、谁、哪里、为什么、多少”等疑问词 |
| `〜かどうか` | 只是问“是不是 / 有没有 / 对不对” |

### 一眼判断法

- 能翻成“什么 / 谁 / 哪里 / 为什么”时，用 `〜か`
- 只能翻成“是否”时，用 `〜かどうか`

## 4. `Vてみる`

### 用法

表示“试着做做看，看看结果”。重点是“先做一下，再看情况”。

### 结构

- `Vて + みる`

### 典型例句

- `この靴(くつ)をはいてみます。`
- `この料理(りょうり)を食(た)べてみます。`
- `一度(いちど)聞いてみます。`
- `その店(みせ)に入(はい)ってみました。`

### 常见场景

- `着(き)てみる`：试穿上衣
- `はいてみる`：试穿鞋、裤子
- `かぶってみる`：试戴帽子
- `食べてみる`：试吃
- `飲(の)んでみる`：试喝
- `聞いてみる`：试着问问
- `読(よ)んでみる`：试着看看

### 活用也会考

| 形式 | 例子 |
|---|---|
| 现在 | `食べてみます` |
| 过去 | `食べてみました` |
| 请求 | `食べてみてください` |
| 意志 | `聞いてみます` |

### 易错点

- `〜てみる` 不是单纯“看见”，而是“试着做”。
- `聞いてみます` 不是“我听见”，而是“我去问问看”。

## 5. 第40课重点题型

这些题型很像考试里的标准出法。

### 题型1：把直接问句改成间接疑问

- `ケーキが何個あるか、教えてください。`
- `シュミットさんはどんな料理(りょうり)が好(す)きか、知りたいです。`
- `ワットさんはどうして来なかったか、わかりますか。`

### 题型2：区分 `〜か` 和 `〜かどうか`

- `この手紙(てがみ)が25グラム以下(いか)かどうか、量(はか)ってください。`
- `この答えが正しいかどうか、もう一度考えてください。`
- `何(なに)をあげるか、まだ決(き)めていません。`

### 题型3：`〜てみる`

- `サイズが合(あ)うかどうか、かぶってみます。`
- `足(あし)が痛(いた)くないかどうか、はいて歩(ある)いてみます。`
- `来(く)るかどうか、聞いてみてください。`

## 6. 第40课考前速记

- 有疑问词，用 `〜か`
- 没疑问词、表示“是否”，用 `〜かどうか`
- `Vてみる` = 试着做一下，看看结果
- 高频动词：`教える / 知る / わかる / 覚える / 聞く / 量る`

## 1. 授受(じゅじゅ)关系总图

| 方向 | 基本动词 | 敬语 / 谦语 |
|---|---|---|
| 我给别人 | `あげる` | 给晚辈也可说 `やる` |
| 别人给我 / 我方 | `くれる` | `くださる` |
| 我从别人那里得到 | `もらう` | `いただく` |

### 一句话记忆

- `あげる`：我给出去
- `くれる`：别人给我这边
- `もらう`：我收到

## 2. `あげる / くれる / もらう`

### `あげる`

主语把东西给别人。

- `わたしは友達(ともだち)に本(ほん)をあげました。`

### `くれる`

别人把东西给我，或给“我方(わたし側)”。

- `友達がわたしに本をくれました。`
- `娘(むすめ)が人形(にんぎょう)をくれました。`

这里的“我方”也可以是：

- `わたし`
- `わたしたち`
- `家族(かぞく)`、`うちの人`

### `もらう`

主语从别人那里得到东西。

- `わたしは友達に本をもらいました。`
- `祖父(そふ)に手袋(てぶくろ)をもらいました。`

### 视角差别

这两个句子写的是同一件事，只是角度不同：

- `田中(たなか)さんがわたしに辞書(じしょ)をくれました。`
- `わたしは田中さんに辞書をもらいました。`

## 3. `くださる / いただく`

### `くださる`

是 `くれる` 的敬语。主语通常是老师、上司、长辈、客户等。

- `先生(せんせい)が辞書をくださいました。`
- `部長(ぶちょう)が花(はな)を持(も)って来(き)てくださいました。`

### `いただく`

是 `もらう` 的谦让语，表示“我恭敬地接受”。

- `先生に辞書をいただきました。`
- `課長(かちょう)におみやげをいただきました。`

### 记忆重点

- 上位者给我：`くださる`
- 我从上位者那里得到：`いただく`

## 4. `〜てあげる / 〜てくれる / 〜てもらう`

这组比单纯“给东西”更常考，因为它表示“帮谁做事”。

| 句型 | 意思 | 例句 |
|---|---|---|
| `〜てあげる` | 我为别人做 | `わたしは妹(いもうと)に宿題(しゅくだい)を教えてあげました。` |
| `〜てくれる` | 别人为我做 | `友達が宿題を教えてくれました。` |
| `〜てもらう` | 我得到别人帮忙 | `わたしは友達に宿題を教えてもらいました。` |

### 视角区别

同一件事：

- `田中さんがかばんを持(も)ってくれました。`
- `わたしは田中さんにかばんを持ってもらいました。`

前一句主语是“帮我的人”；后一句主语是“受帮助的人”。

## 5. `〜てやる`

`やる` 和 `〜てやる` 多用于：

- 晚辈
- 小孩
- 动物

### 例句

- `わたしは犬(いぬ)にえさをやります。`
- `姉(あね)は弟(おとうと)に宿題を見(み)てやりました。`

### 注意

对老师、上司、陌生人不要用 `やる`，会显得不礼貌。

## 6. `〜てくださいませんか`

### 用法

比 `〜てくれませんか` 更礼貌，表示委托、请求。

### 例句

- `ちょっと見てくださいませんか。`
- `ここにはんこを押(お)してくださいませんか。`
- `渡辺(わたなべ)さんに伝(つた)えてくださいませんか。`

### 考试里要会的感觉

- 对老师、同事、店员、前辈说都比较自然
- 比直接命令式柔和很多

## 7. 第41课重点题型

### 题型1：判断授受方向

- `わたしの誕生日(たんじょうび)に、娘が人形をくれました。`
- `祖父にきれいな手袋をもらいました。`
- `先生に日本語(にほんご)の辞書をいただきました。`

### 题型2：判断“谁帮谁做事”

- `友達がタイ料理を作(つく)ってくれました。`
- `引(ひ)っ越(こ)しのとき、友達に手伝(てつだ)ってもらいました。`
- `わたしは妹に絵本(えほん)を読(よ)んでやりました。`

### 题型3：礼貌请求

- `ミラーさん、ここにはんこを押してくださいませんか。`
- `エアコンの調子(ちょうし)が悪(わる)いので、ちょっと見てくださいませんか。`
- `会議(かいぎ)は3時(さんじ)からだと、渡辺さんに伝えてくださいませんか。`

## 8. 第41课最容易错的点

### 1. `くれる` 不能写成“我给别人”

- `わたしは友達に本をくれました` ×
- `わたしは友達に本をあげました` ○

### 2. `もらう` 的重点在“收到的人”

- `わたしは先生に本をいただきました。`

这里主语是“我”，不是老师。

### 3. `〜てくれる` 和 `〜てもらう` 是同一事件的不同视角

- `友達が教えてくれました。`
- `わたしは友達に教えてもらいました。`

### 4. 对长辈别用 `やる`

- 对小孩、弟弟、猫(ねこ)、狗等才自然。

## 9. 第41课考前速记

- 我给别人：`あげる`
- 别人给我：`くれる`
- 我收到：`もらう`
- 长辈给我：`くださる`
- 我从长辈那里领受：`いただく`
- 别人为我做：`〜てくれる`
- 我得到别人帮忙：`〜てもらう`
- 礼貌请求：`〜てくださいませんか`

- `普通形 + かどうか`
- `Vてみる`

### 立刻会写的句子

- `何時(なんじ)から始(はじ)まるか、知っていますか。`
- `行(い)くかどうか、まだ決めていません。`
- `一度(いちど)食べてみます。`

## 第41课一句话背完

- `あげる`：给出去
- `くれる`：给我这边
- `もらう`：收到
- `くださる / いただく`：敬语版
- `〜てくれる / 〜てもらう`：帮我做事
- `〜てくださいませんか`：礼貌请求

### 立刻会写的句子

- `友達がノートをくれました。`
- `先生に辞書をいただきました。`
- `友達に宿題を手伝ってもらいました。`
- `ちょっと教えてくださいませんか。`

---

## 最后提醒

- 第40课最关键的是：先分清“疑问词”还是“是否”。
- 第41课最关键的是：先看“东西 / 动作”是朝谁移动，再决定用哪一个授受动词。
- 如果考试里让你改句子，先判断“视角”再写：是站在给的人一边，还是站在收的人一边。
